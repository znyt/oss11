　　古剑山踏入了距离山巅十丈的距离之内，他踏出一步之后，便不再前进，而是停顿了下来。
<br/>　　到了十丈之内，每一步都危险万分，他不得不谨慎，需要将状态提升到巅峰。
<br/>　　嘭！嘭！嘭！
<br/>　　沉默的声响，不时的响起，拓跋流明和苏莫等人，几乎是每踏出一步，都是身形震颤。
<br/>　　剑山弥漫的锋锐之气，和他们的身体碰撞，让得虚空不断的崩裂，闷响连连。
<br/>　　苏莫的身躯，混沌之光伴随着血脉之光，照耀八方，锋锐的剑山之气，让他体外的混沌之力，不断的被撕裂。
<br/>　　若不是血脉之力防御极强，仅凭混沌之力，根本不足以抵挡。
<br/>　　“好奇的防御！”
<br/>　　无论是拓跋流明，还是韩生和云沐，都不禁暗暗惊讶。
<br/>　　他们自然看出苏莫几位不凡，防御力超强，混沌之力和血脉之力，均是极度强大。
<br/>　　“你是叫秦陨？”韩生转头，看向苏莫，沉声问道。
<br/>　　因为，他好像记得，他终极神王殿绫裳剑宫，好像新进一位弟子，貌似就叫秦陨。
<br/>　　不过，他是毁灭剑宫弟子，对此并不是很清楚。
<br/>　　“是！”苏莫淡淡的点了点头。
<br/>　　“你是绫裳剑宫弟子？”韩生又问道。
<br/>　　“是！”苏莫再次的点了点头，他并未隐瞒，也无需隐瞒。
<br/>　　毕竟，他杀死多名终极神王殿弟子，触犯殿规之事，并未公开，全部被紫霄压了下去。
<br/>　　“那就好！”韩生微微颌首，既然这个秦陨也是终极神王殿的弟子，待会倒是可以协助他。
<br/>　　其他人闻言，均是面色微沉，没想到这个秦陨，居然是终极神王殿的弟子。
<br/>　　他们知晓，终极神王殿，是不会允许别人登上剑山的之巅的，那么，这个秦陨也是潜在的敌人。
<br/>　　千幻邪君眸中精光微闪，原来这个小子，是终极神王殿的弟子，那么就有些棘手了。
<br/>　　毕竟，他若是斩杀一位终极神王殿的妖孽，而且是比万古天骄还要妖孽的天才，很容易无法承受后果。
<br/>　　再者，此地还有四品低级修为的顶级妖孽韩生，他若是出手，很容易被韩生攻击。
<br/>　　不过，略一沉吟，千幻邪君便不再顾虑，终极神王殿的弟子又怎样？
<br/>　　终极神王殿，不会为了一个死人，而大动干戈。
<br/>　　毕竟，他乃是无双界域无双神殿之人，无双神殿的实力，丝毫不弱于终极神王殿。
<br/>　　韩生停在了距离山巅十丈之处，不再前进，倒不是他害怕了，而且，他要先等待别人的情况。
<br/>　　若是其他人纷纷失败，他自然不需要出手了，自己最后再尝试攀登剑山之巅。
<br/>　　拓跋流明同样停在了距离山巅十丈之处，他没有急着行动，而是面现沉吟之色。
<br/>　　少倾，他转头看向韩生和苏莫，冷笑一声，道：“韩生还有秦陨，你们是自己下去，还是要我将你们打下去？”
<br/>　　他可不允许韩生和秦陨在此虎视眈眈，毕竟，他若是进入了十丈之内，风险急速暴增，到时候对方若是出手，会让他非常的危险。
<br/>　　“韩生、秦陨，你们自行下去吧！不要让本小姐给予你们天罚！”云沐也开口了，他距离山巅还有十三丈。
<br/>　　之所以之前没有让韩生和秦陨离开，是因为，大家都在下方，沉受的压力不大，一旦冲突，不会很容易的解决。
<br/>　　而此时此刻，他们均是接近了山巅，每个人都承受了巨大的压力。
<br/>　　而目前，以她们的实力，只需轻轻一击，就能让韩生和秦陨，彻底失去战斗力。
<br/>　　“你们……！”
<br/>　　韩生闻言，顿时面色微变，他没想到这个时候，拓跋流明等人就开始针对他了。
<br/>　　不过，他倒是没有惧怕，大不了先行后退一些，等会再攀登。
<br/>　　“秦陨，让他们先行！”韩生向苏莫吩咐一声，便开始缓缓的后退。
<br/>　　然而，苏莫却是并未听从韩生之言，就在对方说完之后，他又踏出了一步，距离山巅，只有十五丈了。
<br/>　　“嗯？”拓跋流明见此，顿时双眉一挑，眸中闪过一丝煞气。
<br/>　　“秦陨，你在自不量力吗？”云沐见此，面带寒霜，虽然她挺佩服眼前的青年，但是，在这种关键的时候，必须先排除风险。
<br/>　　千幻邪君见此，嘴角勾起一丝冷笑，朗声道：“各位，此人是终极神王殿的弟子，肯定不允许我等登上山巅，我们先将他打下去。”
<br/>　　千幻邪君心中暗笑，等会他们一起出手，将这个秦陨击杀了，也没人会怪在他的头上。
<br/>　　毕竟，有拓跋流明和云沐两人，随他一起出手呢。
<br/>　　“秦陨，快退回去！”
<br/>　　韩生见此，面色微变，虽然他和这个秦陨不熟，但是，同为终极神王殿的弟子，他必须保证对方的安全。
<br/>　　苏莫闻言，停下了脚步，环视众人，目光落在了韩生的身上，道：“退避并不是办法，唯有坚定道心，不断的前进，方有登上山巅的可能！”
<br/>　　“这……！”韩生闻言一怔，他自然明白苏莫的意思，也知道该勇往直前，但是，他们两人的实力，大大的弱于拓跋流明三人。
<br/>　　“诸位，我们还等什么，将他们打下去，我们才能没有后顾之忧！”千幻邪君朗喝，显得有些迫不及待。
<br/>　　云沐瞥了千幻邪君一眼，她知道对方的真实身份，因为她不仅非常熟悉对方。
<br/>　　虽然对方的样貌和气息，甚至是源力等等，都有所改变，但是，她还是认了出来。
<br/>　　而且，所有进入万道剑冢之人，除了千幻邪君有这个能力来到此处之外，已经别无他人了。
<br/>　　而且，修为方面，千幻邪君也正好相符。
<br/>　　千幻邪君言罢，手中剑光一闪，犀利的剑芒不断的暴涨。
<br/>　　而拓跋流明和云沐，均是微微点头，不过，两人并未向千幻邪君这般郑重，并未取出长剑，只是手掌之上源力涌动，随时准备出手。
<br/>　　毕竟，他们没有像千幻邪君那般，领教到苏莫的实力，所以，并未放在心上。
<br/>　　他们在意的是韩生，虽然韩生的实力，在他们眼里也不算厉害，但是在剑山之上，他们也不得不小心。
<br/>　　“千幻邪君！”
<br/>　　苏莫瞥了千幻邪君一眼，眸中冷光一闪，对方的宝剑他认识，所以，现在他也得知了对方的身份。
<br/>　　“出手！”
<br/>　　千幻邪君朗喝一声，没有耽搁时间，率先出手了，一剑狠狠的斩向苏莫。
