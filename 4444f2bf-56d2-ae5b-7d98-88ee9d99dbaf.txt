　　流星郡炎武卫据点。
<br/>　　萧逸一行人，刚刚完成最后一个任务，回到据点。
<br/>　　却得到了萧逸升任队长之职取消的事。
<br/>　　其他炎武卫以及队长，感到不解。
<br/>　　周平等人，略微有些喜意。
<br/>　　但更多的，是愤怒。
<br/>　　“怎么这样？副统领明明说好要给队长你升职的。”周平说道。
<br/>　　胡三以及其余人，纷纷面露怒色。
<br/>　　“副统领怎能出尔反尔，我去讨个说法。”
<br/>　　众人虽然希望继续跟随萧逸。
<br/>　　但他们更愿意看到萧逸高迁升职。
<br/>　　“别急。”萧逸摆摆手，道，“我去问问吧，你们在此等待。”
<br/>　　说罢，萧逸走向最中央的房间。
<br/>　　房内，副统领似是早知萧逸会到来，道，“来了。”
<br/>　　“嗯。”萧逸点点头。
<br/>　　萧逸还未说话。
<br/>　　林重率先道，“是来问为何取消升职之事吧？”
<br/>　　“嗯。”萧逸再次点点头。
<br/>　　林重沉声道，“有人要打压你，不让你当队长。”
<br/>　　“谁？”萧逸皱了皱眉头。
<br/>　　林重回答道，“王都五位统领之一，万统领，万剑一，亦是你们宗门的长老。”
<br/>　　说着，林重将周副统领到来之事，悉数告诉萧逸。
<br/>　　萧逸闻言，面露疑惑之色。
<br/>　　“副统领将事情如实告知我，就不怕…”萧逸问着。
<br/>　　林重摆摆手，打断道，“怕什么。”
<br/>　　“我不仅不怕，还要继续给你升职队长。”
<br/>　　“哦？”萧逸更加疑惑了。
<br/>　　林重沉声道，“炎武卫，是国主的炎武卫。”
<br/>　　“什么时候轮到他一个剑宗长老来弄这些龌蹉事了。”
<br/>　　“再说了，我与你接触了大半个月。”
<br/>　　“发现你这人，狂是狂了些，但重情重义。”
<br/>　　“可不像王都流言里的目中无人、嚣张跋扈。”
<br/>　　“反倒是万剑一，偷偷派个副统领过来。”
<br/>　　“让我打压你。”
<br/>　　“这根本就是小人行径。”
<br/>　　“老夫活了大半辈子，一看就知道这万剑一是个小人。”
<br/>　　“老夫若还帮他，岂不是也成了是非不分的小人？”
<br/>　　“副统领言重了。”萧逸拱了拱手。
<br/>　　林重摆摆手，道，“你的队长之职，我仍旧会给你提升。”
<br/>　　“待会儿会记录在案，并将你的炎武令升为队长令牌。”
<br/>　　“但，我始终只是副统领。”
<br/>　　“而万剑一，可是正统领。”
<br/>　　“我就是提升了，他还是能够一道命令给你取消掉。”
<br/>　　林重说着，讪笑两声，道，“所以，我只能对外宣布，将你升职之事取消了。”
<br/>　　“现在再偷偷帮你办了。”
<br/>　　萧逸皱眉道，“队长职务，会记录在案。”
<br/>　　“就算副统领替我偷偷办了，上面的资料中，还是能查询到。”
<br/>　　“有何用？”
<br/>　　“有。”林重点了点头，“炎武卫规定。”
<br/>　　“队员无论升职还是降职，都必须要队员亲自到场。”
<br/>　　“拿队员的贴身令牌，做职位记录和改动。”
<br/>　　“我现在给你升了职位。”
<br/>　　“万剑一若想给你降职，只能找到你。”
<br/>　　“找不到你，你不接令，他奈你不何。”
<br/>　　萧逸疑惑道，“副统领什么意思？”
<br/>　　“很简单。”林重道，“炎武卫的职责，乃是巡守各郡。”
<br/>　　“你大可离开流星郡，到其它郡接任务。”
<br/>　　“以你的本事，只要不是一直待在一个郡。”
<br/>　　“万剑一手下的副统领，没办法轻易找到你
<br/>　　“待得你功绩足够提升副统领后。”
<br/>　　“再回来找我。”
<br/>　　“副统领之职，可就不是万剑一说撤就撤的了。”
<br/>　　“那需要五位统领中，超过一半以上赞成，才能撤职。”
<br/>　　萧逸摇摇头，沉声道，“若万统领铁了心要对付我。”
<br/>　　“我就是全国到处跑也没用。”
<br/>　　“炎武卫，有准确的信息情报。”
<br/>　　“他麾下的副统领齐齐出动，要找我并不难。”
<br/>　　“他敢？”林重瞪了瞪眼睛。
<br/>　　“炎武卫，个个都任务繁忙。”
<br/>　　“若他敢为了打压你，特地派遣副统领全炎武王国到处找你。”
<br/>　　“净弄这些龌蹉事的话。”
<br/>　　“看我不去王都总部告他个玩忽职守之罪。”
<br/>　　“嗯？”萧逸皱了皱眉头。
<br/>　　他发现，林重在说这句话时，似乎底气十足。
<br/>　　“副统领，是林家本家的人？”萧逸问道。
<br/>　　“不错。”林重点了点头。
<br/>　　林重，确实是林家本家的人。
<br/>　　论辈分，他属林家老一辈。
<br/>　　与林家上代家主，炎武卫大统领，乃是堂兄弟。
<br/>　　……
<br/>　　林重，替萧逸的令牌做好升职记录。
<br/>　　而后，萧逸告别了林重，离开了据点。
<br/>　　之后，带上了周平等人，离开了流星郡。
<br/>　　虽说如今萧逸已是正队长。
<br/>　　但队员们，还是要带上的。
<br/>　　所谓队长能够单独行动。
<br/>　　指的是在一个郡内。
<br/>　　像如今，如果要去黑雨城执行任务的话。
<br/>　　萧逸完全可以独自行动，无需带任何人。
<br/>　　可若穿梭于郡与郡之间，就必须带上队员们了。
<br/>　　否则，一个队长撇下队员。
<br/>　　自己一人全炎武王国四处跑，这算什么队长？
<br/>　　若想全炎武王国内，都独自行走的话。
<br/>　　必须达到副统领职位。
<br/>　　当然了，萧逸现在已经很满意了。
<br/>　　每到一个郡，自己都能单独行动，方便了许多。
<br/>　　只是离开这个郡，前往其它郡时，要带上众人离开罢了。
<br/>　　另外一点，炎武卫的许多规定，有些奇葩。
<br/>　　像升职降职，必须队员在场，且要让队员交出随身令牌。
<br/>　　方能做出改动。
<br/>　　而并非单单传达一条命令。
<br/>　　若队员死活不肯交令牌，谁也奈何不得。
<br/>　　除非将这位队员打趴，强行夺过令牌。
<br/>　　这其实，也是‘实力为尊’这条尊重的一个表现。
<br/>　　职位较高者，若打不过职位较低者。
<br/>　　有什么资格给别人降职？
<br/>　　炎武卫，每一份任务，都极其危险。
<br/>　　队员们水里来，火里去。
<br/>　　炎武卫，自是给予每个队员足够的尊重。
<br/>　　……
<br/>　　流星郡边缘，某座大城。
<br/>　　萧逸一行人，即将离开。
<br/>　　周平沉声道，“队长，我们真的要离开流星郡吗？”
<br/>　　众人本都是流星郡的武者。
<br/>　　虽说炎武卫巡守各郡。
<br/>　　但很多时候，炎武卫忙自己郡的任务都忙不过来，一般不会去其它郡的据点。
<br/>　　除非有特别必要。
<br/>　　“怎么？你们不想离开？”萧逸问道。
<br/>　　“倒不是。”众人笑笑。
<br/>　　“我们早就想到其它郡见识见识。”
<br/>　　“特别是能跟着队长你，我们哪里都去。”
<br/>　　“呵呵。”萧逸笑笑，道，“那便走吧。”
<br/>　　“从今日起，炎武卫，萧逸分队，巡守各郡！“
