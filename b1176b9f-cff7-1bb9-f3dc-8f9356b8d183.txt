　　这里已经彻底远离大部队，除了空中一路倒追着而来的大群飞行丧尸，四周竟然一只丧尸都没有，空空荡荡极其诡异。
<br/>　　江凌若还在坚持着整个领域，但在一路退后的过程中，第五小队的成员已经适应过来把握好了节奏，每当江凌若将飞行丧尸放进领域内的时候，他们都会第一时间挥出武器，从各个角度将眼前的飞行丧尸重伤，甚至有些能够一击毙命。
<br/>　　“准备，加大进度。”江凌若有条不紊的操控战局，强大的意志力和能量，让整个领域壁长时间都牢牢挡住了这群丧尸。
<br/>　　太不可思议！
<br/>　　所有人在江凌若话音落下的一刻都猛然提紧心神，握紧武器紧盯前方。
<br/>　　哗——
<br/>　　领域壁忽然张开一个巨大缝口，好几只飞行丧尸不要命的往内钻，相互间你争我抢，一张张满嘴利齿的大口‘咔咔’的咬合着，想要将其内的人类全部咬碎。
<br/>　　刷！
<br/>　　江凌若的巨大镰刀第一个发出攻势，依旧是手起刀落的犀利一削，上来就将最前方的一头飞行丧尸斩成两半。
<br/>　　她身后的好人则是配合着上前，相互合作的与其他飞行丧尸缠斗起来，进攻防守骚扰都配合完美。
<br/>　　很快这些冲进来的飞行丧尸就全部变成了地上的尸堆，个头巨大的体型摆在那里极有震撼力。
<br/>　　江凌若没有浪费时间，不断的将领域壁一开一合，源源不断的将飞行丧尸卡着数量放入，就这样在第五小队的全体人员配合下，地上的丧尸尸体越来越多，堆成了两簇小山。
<br/>　　在时间里江凌若的领域越来越弱，范围开始不住的缩小，到最后只能堪堪维持。
<br/>　　“注意，五秒后各自散开。”江凌若在最后一刻下令，哪怕她的能量已经近乎无法再继续维持，也依然声音冷静无比。
<br/>　　第五小队的其他人早已料到会有这一刻，他们没有慌张无措，而是立即各自成对站在了即将奔走的不同方向，散开之后就不再是团队作战，而是要形成三五人的小队形式游走战斗。
<br/>　　剩下的飞行丧尸还有很多，他们只能依靠这片地点的丛林树木来迂回作战！
<br/>　　“散！”江凌若一声令下，同时‘哗’的一声，整个领域壁顿时消失。
<br/>　　刷刷刷！
<br/>　　第五小队的五十人快速分散而开，与飞行丧尸一冲而上的大·片覆盖时间重合，只见黑压压的一大·片扑面而来，近乎一瞬间就将那片地带铺满。
<br/>　　四周则是各自奔跑中的第五小队成员，时不时的闪过刀光剑影，战斗看上去尤其激烈。
<br/>　　江凌若的速度也极快，一下子跳到了附近一棵树的枝头，同时手中巨大镰刀不断在半空中挥舞，削落了无数的黑色羽毛或利爪，这些都是追赶而来的飞行丧尸身上所带。
<br/>　　而她每过一棵树枝，那整棵树都会一下子被撞倒，成群的飞行丧尸蜂拥而至在丛林中横冲直撞，将大·片大·片的树木都撞断在地。
<br/>　　整片地带一片狼藉不下于大型战役现场，飞行丧尸的体型过大，人类的体型太小，怎么看都像是被驱赶的一方。
<br/>　　江凌若没有再继续下令，独自一人不断在各个枝头上跳跃，手起刀落又是好几头飞行丧尸瞬间死亡。
<br/>　　这些飞行丧尸等阶都比她低，哪怕体型过大只要她发挥没有失常，每一刀的挥出都带着极强杀伤力，只是丧尸群的数量太多，尤其在高处往下看，这整片地带的空间都被占满，无论是地上的一群还是半空中正扑着翅膀的。
<br/>　　江凌若眼神发冷，没有任何犹豫的从高处枝头一跳而下，同时手中银色镰刀流光大放。
<br/>　　轰！轰！
<br/>　　几声爆炸般的破空声响起，每一次的银色镰刀挥舞，都带着黑血淋漓的飞行丧尸脑袋滚落在地，以及‘嘭’的一声大体型的冲撞声。
<br/>　　江凌若十阶新人类的杀伤力尤其可怕，除了体型上无法避免的劣势之外，几乎没有任何一头飞行丧尸可以近她的身。
<br/>　　她想逃完全可以，但想杀光这些数量恐怖的飞行丧尸却不可能，甚至继续持续战斗轻伤重伤甚至生命危险都有可能，更别说在场的其他第五小队成员，他们大多数都还是七阶新人类。
<br/>　　好在小队模式的训练狼牙战团相当成熟，第五小队的全体成员无论是否出自于三战队，都是精英中的精英，在这种游击战中配合绝佳。
<br/>　　开启分散战斗模式之后的极长时间内，都还未有伤亡出现，唯有地上不断增多的飞行丧尸尸体，一下子铺了满地高高堆起。
<br/>　　速战速决是最好的办法，再拖延下去只有他们等死的份，这是人人皆知的想法。
<br/>　　只是还不等他们想出一个速战速决的法子，一只一直在远处观望的飞行丧尸忽然飞上高空，带有一双睿智的眸子里凶光炸闪，而后毫不犹豫的朝着远处飞离。

<br/>　　江凌若自然清楚这一点，眼下她还有机会选择带着人撤离战场，前往圆圈附近与其他的小队汇合共同解决这一麻烦。
<br/>　　但在话出口的一瞬间，她却改了主意，道：“继续七点钟方向撤离。”
<br/>　　话落，她便第一个在前方开道，一边灵活的奔走在道路上，一边将一头头等阶不如她的飞行丧尸斩杀出一条黑血之路。
<br/>　　其他人也渐渐集中跟在江凌若身后，不断与周围的飞行丧尸大战厮杀，没有了灵活游走的优势，队伍中也开始有人受伤，甚至还有严重的直接服用了珍贵的丧尸病毒解毒剂！
