　　等到叶尘再次醒来的时候，发现自己已经身在云州狂仙门之中，
<br/>　　“小尘，你总算是醒了！”
<br/>　　床边，苏岚紧紧握着叶尘的手，满脸激动之色，眼眶通红发肿，显然之前哭过。
<br/>　　而除了苏岚之外，苏蔓、叶天歌、楚飞烟、唐清雅、雪姬等人，也全都在屋子里。
<br/>　　“曦月！”
<br/>　　叶尘大吼了一声，直接从床上猛然坐起，不过很快，全身上下传来阵阵剧烈的疼痛感，脑袋也一阵眩晕。
<br/>　　“叶尘，你放心好了，若曦妹妹只是受了一点轻伤，如今就在隔壁的房间休养，你很快就能见到她啦！”
<br/>　　楚飞烟还以为叶尘说的萧若曦，连忙开口安慰道，同时美眸深处，透着一丝丝羡慕之色。
<br/>　　叶尘在重伤之余，还在时时刻刻惦记的女人，不光楚飞烟羡慕不已，连同苏蔓、唐清雅和雪姬，也都不由得升起一丝嫉妒之念。
<br/>　　然而，她们又哪里知道，叶尘口中的曦月，根本不是萧若曦！
<br/>　　只可惜，此事只有他、萧若曦和曦月三人知道，并不足与外人道。
<br/>　　“没事就好！”
<br/>　　叶尘心念一动，神念在吞天鼎中扫过，发现曦月的残魂犹在，顿时松了一口长气，
<br/>　　“对了，华山那边的军情如何？”
<br/>　　听叶尘问起这个，叶天歌开口道：
<br/>　　“小尘，那日在华山，神、仙、魔三界的高手，几乎全部被你斩杀，妖界高手虽然给军方制造了一些麻烦，不过在得知其他三界全军覆没的消息，便吓得向北极方向逃窜，因而我华夏的损失并不大……”
<br/>　　听到叶天歌的这番话，叶尘微微点了点头，
<br/>　　自从将叶天歌复活之后，叶尘虽然并没有认他这个父亲，不过却也默许了他和苏岚在一起的事实，两人关系已经有所缓和。
<br/>　　叶尘右手一招，从乾坤镯中取出一堆丹药，在众人惊愕的目光中，一口气全部吞了下去，继而运转吞天神功，一转眼的功夫，体内的伤势已经好了许多，然后从容起身下床，在众人身上扫视了一遍之后，语出惊人道：
<br/>　　“我有一件要紧的事情，可能要离开地球一段时间！”
<br/>　　“离开……地球？”
<br/>　　听到这话，所有人全都懵了，甚至都怀疑，叶尘是不是因为受伤，脑袋出了问题？
<br/>　　“对！离开地球！我知道，你们或许无法理解，不过一时半会，我也向你们解释不清楚，但是我有不得不离开的理由！”
<br/>　　众人见叶尘的样子，不像是在开玩笑，顿时脸色全都大变。
<br/>　　“小尘，虽然岚姨不知道你要做什么，可是你身上还有伤呢！不如先把伤养好了再走也离开也不迟啊！”
<br/>　　叶尘叹了一口气，
<br/>　　“岚姨，时间紧迫，我一刻也不能耽搁！”
<br/>　　“你们不必再劝我了，我马上就会离开，我走之后，你们对外，只需说我正在闭关即可！”
<br/>　　“相信经过华山一战，四界应该不敢再轻举妄动。”
<br/>　　众人见叶尘态度坚决，似乎已经下定了决心，知道再劝也没有用，顿时相顾愕然。
<br/>　　……
<br/>　　又休养了一天一夜，动用吞天神功，吞噬了上千粒灵丹，叶尘身上的伤势，也总算修复了几成，虽然离完全恢复还有一定的距离，不过至少已经活动自如。
<br/>　　第二日，
<br/>　　叶尘并没有再继续修复，而是让楚飞烟发动苏叶集团的力量，搜集了数以千计的极品美玉。
<br/>　　叶尘先将这些极品美玉，全部打造成了储存玉简，然后将他脑海之中的功法秘籍，以及炼丹之术，铸器之法，阵法之道，乃至四品以下的常用丹方，甚至还有一些修炼心得，全都一股脑的刻录在了这些玉简之上。
<br/>　　最终，叶尘将这些玉简，全部放入狂仙门的藏经楼之中，并且又在周围设下层层禁制，只有楚飞烟、叶天歌、苏蔓母子等寥寥数人可以进入其中。
<br/>　　非但如此，叶尘还将自己这一年来，搜集的各种灵器法宝，也几乎全都留了下来，以供自己的亲朋和门下弟子使用。
<br/>　　除此之外，他的三名大学室友，叶尘也命人把他们召到了云州，再经过他们的允许之后，拜入狂仙门门下，交由如今狂仙门修为最高的祝明飞亲自教导。
<br/>　　处理完云州的后事之后，叶尘还是有些不放心，又分别杀入仙、妖、魔一趟，肆无忌惮的除掉了一些潜在的威胁，令三界为之胆寒的同时，也让整个星球，都清楚的知道一件事：他叶狂仙，还活着！
<br/>　　而等着一系列的善后工作做完之后，时间也过去了十多日。
<br/>　　眼看着吞天鼎之中的曦月残魂，已经开始越来越虚弱，叶尘也知道，不能再耽搁了！
<br/>　　这一日，
<br/>　　叶尘将苏岚等人全部召集在一处，
<br/>　　“岚姨、蔓姐、飞烟、清雅、磊子、雪姬、祝家兄弟……”
<br/>　　叶尘的目光，在众人身上一一扫视了一遍，唯独叶天歌，他始终喊不出那个字，索性直接跳过，
<br/>　　“我这一走，快则一两年，慢则十年八年，待我将那件要紧事处理好，定会尽快回来看你们！”
<br/>　　众人听到这话，顿时个个一脸不舍，苏蔓母女和唐清雅，更是泪眼汪汪，唯独祝明飞笑道：
<br/>　　“叶兄，你真的不考虑带上我一起？”
<br/>　　原来，祝明飞曾经提出要与他一同前往通天路，不过叶尘思量之后，最终还是拒绝了，
<br/>　　他虽然找到了通天路所在，可是那条路的后面是什么，谁也不知道！
<br/>　　他是为了救曦月，所以才不得不拼上一把，没必要再带上其他人一起冒险。
<br/>　　更何况，祝明飞乃是元婴境高手，由他镇守在狂仙门，他才能走的安心一些。
<br/>　　“祝兄，先容我去探探路，倘若十年之内，我不能回来，说明里面凶险万分，你需慎重考虑！”
<br/>　　祝明飞自然明白叶尘这话的意思，脸上顿时露出一抹严肃之色，冲着叶尘一拱手，道：
<br/>　　“好！我就在此等你十年！倘若……”
<br/>　　祝明飞说到这里，忽然觉得下面的话，此时说出来有些不吉利，当即话锋一转，道：
<br/>　　“无论如何，这通天路，我是一定要走上一走，有劳叶兄代为探路了！”
<br/>　　叶尘又在众人身上一一扫视了一遍，忽然哈哈大笑了数声，跟着身体微微一晃，便消失不见。
