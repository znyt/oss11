　　这一斧霸道绝伦，一斧劈下斧芒如天河倾泄，威势无与伦比。
<br/>　　这一斧的威力的之强，可以轻易斩杀一般的五品之人。
<br/>　　不过，这一斧威力再强，在苏莫面前，依旧是小巫见大巫。
<br/>　　只见苏莫手臂一抖，终极神剑逆刺而上，璀璨的混沌剑芒直冲九霄，瞬间便击中的斧芒。
<br/>　　轰！
<br/>　　一声爆响，斧芒炸裂，化为万千气流迸射方圆万里。
<br/>　　“杀！”
<br/>　　暴喝声惊天动地，拓跋坤头顶虚影升腾，元神都释放了出来。
<br/>　　拓跋坤拥有三生元神，分别是一把金色的战斧，一座黄色的小山，以及一块灰色的大石。
<br/>　　三种元神，虽然都不是特别强大的超级元神，但是其元神之力波动，均是极为不凡。
<br/>　　一瞬间，三种元神全部进入了他手中的战斧之中，战斧爆发出璀璨的夺目的光芒，恐怖的气息浩荡诸天。
<br/>　　轰隆隆！
<br/>　　战斧重重的一挥，巨大的斧芒再次斩击而出，这一击强大到了极致，威力暴增。
<br/>　　因为，这不仅是修为的力量，还是集合了三大元神最巅峰的力量，最强一击。
<br/>　　当然，这种做法极为危险，稍有不慎，可能元神湮灭，当场陨落。
<br/>　　拓跋坤这是背水一战，他知道苏莫的强大，不舍命一战很难胜利，再者，他有底气，就算这一击无法击败苏莫，他的三大元神也能逃离出来。
<br/>　　轰隆隆！
<br/>　　巨大的斧芒，仿佛分开了整个世界，以无与伦比的速度，裹挟惊天的恐怖力量，重重的斩向苏莫的头顶。
<br/>　　这一击的威力，让无数的为之心惊，就连龙腾和姜枫然等人，都不禁心中咯噔一下，担心苏莫被击败。
<br/>　　“找死！”
<br/>　　苏莫见此，眸中闪过了犀利的精光，对方这一击的确是强大，但是，用这种方法对付他，这分明就是在找死。
<br/>　　“死吧！~”苏莫再次出剑了，混沌源力滚滚，终极神剑爆发出万丈混沌之光，一剑逆斩而上。
<br/>　　霎时之间，一道巨大剑轮，如同朝阳升空，直上苍穹。
<br/>　　混沌剑轮！
<br/>　　这一剑威力更加的恐怖，因为，苏莫不仅动用了吞噬元神、两种万化神剑元神、不败战气元神，甚至连血脉之力都动用了。
<br/>　　巨大的混沌剑轮，不仅恐怖无匹，吞噬之力更是前所未有的强大。
<br/>　　以往，苏莫施展混沌剑轮，其上只有吞噬法则的力量，很少催动吞噬元神。
<br/>　　但现如今，他早已不用在隐藏，吞噬元神可以肆无忌惮的运用了。
<br/>　　混沌剑轮恐怖的威势，让得无数观战之人瞪大了眼眸，眸中露出不可思议之色。
<br/>　　这一剑的强大，让所有人心神震动，心中掀起惊涛骇浪。
<br/>　　轰！
<br/>　　一声惊天的爆响，震动诸天星河，巨大的混沌剑轮瞬间便和斧芒相击在了一起。
<br/>　　轰隆隆！
<br/>　　爆响震耳欲聋，强大的斧芒瞬间崩溃，根本无法阻挡住混沌剑轮。
<br/>　　更加惊人的是，崩溃的斧芒，以及拓跋坤的三大元神，如同泥牛入海一般，全部都让混沌剑轮吞噬一空。
<br/>　　斩灭了斧芒的混沌剑轮，威势不减反增，直接向上方的拓跋坤袭杀而去。
<br/>　　“完了！”
<br/>　　拓跋坤面无人色，在他的三大元神被吞噬的那一刻，他的生机便在飞速的消散。
<br/>　　毫无意外，拓跋坤的肉身，在恐怖的剑轮之下灰飞烟灭。
<br/>　　巨大混沌剑轮，冲上了无尽的虚空，直到小世界的尽头，才最终爆炸开来，让得这方小世界都微微震动了起来。
<br/>　　众人震撼不已，拓跋坤这么强大，依旧被强势抹杀，这苏莫的天赋战力，果然冠绝神域。
<br/>　　见识了这一招的强大，现场所有的天才，都能肯定，在整个神域，没有任何人能在同阶之中，挡住苏莫一招。
<br/>　　“拓跋坤……！”地藏神王面无人色，声嘶力竭的狂吼，但是，这一切都已经晚了，他出手都来不及了。
<br/>　　“畜生！”
<br/>　　地藏神王眸中杀机冲天，大手一伸，一掌向苏莫拍去。
<br/>　　浑厚的掌劲，形成了一个高大数十丈的石碑虚影，向苏莫袭杀而去。
<br/>　　这一招快如疾电，并且强大到了无法形容的地步，石碑虚影一出，这方小世界根本承受不足其巨大的力量，开始飞速崩塌。
<br/>　　苏莫在这一击之下，显得极为脆弱，如同一叶扁舟，即将倾覆。
<br/>　　“大胆！”
<br/>　　就在此刻，一声暴喝，震动环宇，只见龙族大长老出手了，大手一探，一只巨大的金色龙爪虚影，跨越长空，一把将石碑虚影抓在了手中。
<br/>　　轰隆隆！
<br/>　　龙爪狠狠的一捏，石碑便被捏的粉碎，沉闷的声响从龙爪之内传出，震动八荒，即便逸散而出的力量不多，也让整个小世界都在颤抖。
<br/>　　多亏了大长老出手的早，不然这方小世界都会彻底的毁灭。
<br/>　　“地藏，你敢出手攻击我龙族之人，是想要找死吗？”大长老向地藏神王怒目而视，暴喝震天。
<br/>　　既然苏莫是龙腾要保护的人，他自然也会保护苏莫。
<br/>　　当然，苏莫这种天才，只要是真心报效龙族，即便没有龙腾，他也会出手。
<br/>　　“我……！”地藏神王无言以对，他向苏莫出手，的确是有些冲动了，若是一击能将苏莫灭杀，倒还罢了，现在没有得手，难免无法解释。
<br/>　　他最大的希望，当然是将苏莫待会地王宫，若是带不回地王宫，退而求其次，将苏莫击杀也无不可。
<br/>　　“肆意出手对付我龙族之人，老夫可以将你镇杀当场！”大长老再次开口道，眸中杀机闪烁。
<br/>　　“龙昌大长老，我只是一时激动，还望恕罪！”地藏神王阴沉着脸，向大长老抱拳。
<br/>　　在龙族面前，他不得不低头，不然，对方真的有将他镇杀的底气的和实力。
<br/>　　“地藏神王，我龙族也不追近你的责任了，带着你地王宫的弟子，离开龙都城吧！”龙腾面色冷漠，朗声说道。
<br/>　　“这……！”地藏神王闻言，顿时面色微变，今日离开，以后就不好再对付苏莫了。
<br/>　　龙腾之言，让得天尘宇等天王宫之人眼眸一亮，地王宫之人走了，那么，就没有人和他们抢夺苏莫了。
<br/>　　虽然目前看来，他们也是机会渺茫，但是，只要留下来就能找到机会。
<br/>　　吞虚龙帝面色淡淡，并未开口表示什么，适才，他本也能出手挡下地藏神王的攻击，不过，他不会出手。
<br/>　　苏莫脸色冷漠，他倒是没有说什么，此事自有大长老和龙腾为他做主。
<br/>　　“还不快滚！”大长老不怒自威，再次暴喝一声。
<br/>　　“告辞！”面对大长老的怒喝，地藏神王的脸色黑如锅底，招呼一声麾下弟子，便灰头土脸的离去。
<br/>　　他不得不离开，不然真的会死在龙族。
<br/>　　闻人天都和战乱两人，纷纷扫了一眼苏莫，便跟随着地藏神王离开了。
<br/>　　在一位龙族长老的‘护送’之下，地藏神王和麾下弟子，飞出了小世界，离开了龙都城。
<br/>　　地王宫的人并不多，死了两位弟子，现如今除了闻人天都和战乱之外，还剩下一位修为较高的弟子。
<br/>　　一行四人，离开龙都城之后，便火速向轮回神朝的方向而去。
<br/>　　“长老，难道我们就这样离开了？”闻人天都跟着地藏神王的身后，出声问道。
<br/>　　“离开也不是坏事！”
<br/>　　地藏神王铁青着脸，继续道：“今日想对付苏莫，基本上不可能了，回去再想办法。”
<br/>　　在他看来，不管是天王宫还是终极神王殿，今日都拿不下苏莫，还要日后再想办法。
<br/>　　“苏莫投靠了龙族，有龙族保护，日后也很难有办法！”闻人天都眸光微闪，声音低沉的说道。
<br/>　　“投靠龙族有怎样？除非他一辈子不离开龙都城！”地藏神王眸中杀机深寒，这苏莫杀了他地王宫这么多人，就算不死为力拓跋青云，也一定要杀死。
<br/>　　闻人天都和战乱两人闻言，对视一眼，眸中均是闪过隐晦的光芒，看来苏莫日后也是不安全啊。
<br/>　　明枪易躲暗箭难防，随时随地，都有可能遭受到杀机。
