　　房内，萧逸为昏迷中的依依上了些金创药，这才彻底放下心来。
<br/>　　这些药，是从议事厅回来前，问三长老拿的。
<br/>　　萧逸一改之前的无赖之色，变得满脸冷酷，“得快些提升修为了，要不然以萧若寒父子俩的阴险，往后肯定会有更多的阴谋陆续而来。”
<br/>　　这个世界，以武为尊，只要拥有足够的实力，一切阴谋都会在拳头之下破碎。
<br/>　　对于萧若寒请杀手暗杀自己一事，他始终没有告诉三长老。
<br/>　　实在是，这事说出来也不会有人信。
<br/>　　而且，萧若寒阴险就阴险在恰恰是今天晚上让杨伏出手。
<br/>　　以萧逸的聪明，以及当年在地球常年混迹于黑道，见惯那些龌龊之事，他几乎已经猜出了萧若寒父子的阴谋。
<br/>　　若是自己在那时被杨伏成功暗杀，那么就算事情传出去，萧若寒父子也可以立刻摆脱嫌疑。
<br/>　　毕竟，在这一整天的会议中，他们都在商量废除自己少家主之位的事，摆明了是在针对自己。
<br/>　　若是这个时候自己被人暗杀，那么他们的嫌疑最大。
<br/>　　那么，到时候，所有长老和执事都会认为，萧若寒父子绝对不会这么蠢，绝对不会做这样的蠢事。
<br/>　　明摆会有最大嫌疑，还会去做这种事吗，显然不会。
<br/>　　萧若寒就是捉准了正常人的想法，才会特地让杨伏今天晚上暗杀自己。
<br/>　　这也是萧逸自己在会议厅不说出今晚被暗杀的事，反正说出来也没用。
<br/>　　另外一点，萧若寒父子逼迫自己在‘少家主之位’和‘修炼名额’上做选择也是一项阴谋。
<br/>　　正常来看，自己的少家主之位名存实亡，根本没有任何权利可言，倒不如留一个‘修炼名额’，让自己进入紫云洞修炼，提升修为更好。
<br/>　　但其实，一旦自己放弃少家主之位。那么，没了这身‘老虎皮’，萧若寒父子俩绝对能在半个月内，找出一百种莫须有的办法，将自己的‘修炼名额’也一并除去。
<br/>　　这阴谋，环环相扣，防不胜防。
<br/>　　当然了，萧若寒父子打地好算计。
<br/>　　可萧逸一眼就看出了他们的想法，所以今晚才死不退让，更是耍无赖，咬定了族规。
<br/>　　“这两个阴险的家伙，一个又一个地阴谋往老子头上盖。好，很好，都给我等着，等我将修为提上去，这些账，老子一笔一笔和你们算。”萧逸眼中露出了冷酷的杀意。
<br/>　　每当萧逸露出这种眼神时，都代表着他已经对‘目标’烙下了必杀的印记。
<br/>　　这时，萧逸从怀中了拿出了杨伏的乾坤袋。
<br/>　　里面有一大堆乱七八糟的东西，毒药、夜行衣、武器、食物等等。
<br/>　　除此之外，还有数百两银子，二十颗淬体丹，一颗后天丹。
<br/>　　“身家果然丰厚。”萧逸满意地点了点头。
<br/>　　后天丹，乃是二品丹药，只有二品以上的炼药师才能炼制出来，极为珍贵。
<br/>　　与淬体丹一样，这是一种能让武者增加修炼速度的丹药。
<br/>　　不过淬体丹是凡境武者使用；后天丹则是后天境武者使用；两者间蕴含的药力天差地别。
<br/>　　没有多想，萧逸立刻抓紧时间修炼，吞服淬体丹。
<br/>　　在紫色武魂冰鸾剑的吸收下，淬体丹精纯的药力顷刻间游走于四肢百骸，变为真气，一部分淬炼肉体，一部分并丹田吸收。
<br/>　　凡境中，除了炼体外，还要淬炼丹田。
<br/>　　当武者达到凡境九重时，若想突破到后天境，就必须凝结气丹。
<br/>　　所谓的气丹，就是用真气淬炼丹田，使丹田越来越凝实坚固，最后衍变为气丹。
<br/>　　气丹凝，后天成。
<br/>　　一分钟后，十颗淬体丹已经被吸收完毕。境界从凡境二重突破到凡境三重。
<br/>　　凡境一重时，突破到二重，只需要三颗淬体丹。
<br/>　　如今突破到三重，却需要十颗了。
<br/>　　“果然，武道修炼会越来越难，突破所需的真气也越来越庞大。”萧逸喃喃自语道。
<br/>　　当剩下的十颗淬体丹完全被吸收，体内的真气浓郁了许多，但仍旧没有突破。
<br/>　　别的普通武者，想要吸收完二十颗淬体丹，起码需要十天十夜。若算上中途要休息，二十天都不一定能吸收完。
<br/>　　而且，凡境武者的肉体承受力有限，武魂也无法不间断地长时间操控着吸收，这个时间还要拉长到数月。
<br/>　　而萧逸有了冰鸾剑武魂的辅助，轻松无视了这些限制。
<br/>　　“接下来，该试试后天丹的效果了。”萧逸拿出了最后一颗丹药。
<br/>　　后天丹，是后天境武者才能服用的丹药。
<br/>　　丹药入腹，足足五分钟，药力才完全被吸收完毕。一颗后天丹的药力，竟比十颗淬体丹加起来的药力还要强悍。
<br/>　　境界突破到凡境四重。
<br/>　　“凭借我的形意五绝，凡境五重时，应该就能打败凡境六重的武者。”萧逸暗暗思索着。
<br/>　　既然半个月后就要参加家族的比武，更是和萧若寒有了约定，他当然要考虑这件事。
<br/>　　“不过，武道修炼，等级越高，要突破就越难。如果单纯吸收天地灵气，恐怕要突破到凡境五重起码需要大半个月时间。”
<br/>　　半个月突破到凡境五重，这还是萧逸拥有紫色武魂的情况下。若是橙阶、黄阶武魂，一年都不一定能突破。
<br/>　　要是能有更多的丹药就好了。
<br/>　　凭借冰鸾剑武魂的强大，自己的修为绝对会飞速上涨。
<br/>　　“丹药，丹药。若是我成为炼药师，自己炼药，便能有源源不断地丹药了。”萧逸眼前一亮。
<br/>　　自己动手，丰衣足食。
<br/>　　不过，想要成为炼药师，要求极为苛刻。一百个武者中，都不一定有一个能有资质成为炼药师。
<br/>　　“不管怎么样，总归要试试。”萧逸有了想法后，就不会轻易放弃，而是决心尝试。
<br/>　　.....
<br/>　　翌日，天蒙蒙亮，萧逸便从睡梦中醒来，便直奔家族炼药堂而去。
<br/>　　炼药堂，是整个萧家最重要的地方之一。
<br/>　　里面放置了许多珍贵的药材，还有许多丹药。
<br/>　　每个月，家族子弟都会来这里领取自己的丹药。
<br/>　　来到炼药堂后，一阵浓郁的药香扑面而来。
<br/>　　萧逸看到，正堂的边缘，一位中年人正在查阅着书籍。
<br/>　　而四周，则有数十个药童忙碌地整理、搬运着药材。
<br/>　　最中央，两个一品炼药师辛勤地炼着丹药。
<br/>　　萧家作为紫云城三大家族之一，族人众多，所需要的淬体丹也极多。当然，萧家也会出售丹药，以此作为家族的收入来源之一。
<br/>　　整个萧家，也不过只有五位炼药师，三品一位，一品四位。
<br/>　　“嗯？”正堂边缘的中年人抬头看了萧逸一眼，认出了萧逸，皱眉道，“萧逸，今天还未到丹药发放的日子，你来这里做什么？”
<br/>　　中年人是萧家的一位执事，同时也是一位一品炼药师，掌管炼药堂，名为萧百炼。
<br/>　　萧百炼暗暗疑惑，今天不单止不是丹药发放的日子，而且萧逸的丹药不是一向都由三长老代为领取的吗。
<br/>　　“百炼执事，我是想来拿炼药录。”萧逸说道。
<br/>　　炼药录，是一门基础的炼药师手册。虽然只讲述了基础的炼药知识，但极为珍贵。起码，紫云城中只有三大家族拥有。
<br/>　　萧家鼓励所有族人尝试成为炼药师，所有子弟可随意拿来观看。
<br/>　　“炼药录？”萧百炼愣了愣，问道，“你拿那东西干嘛？你别告诉我你想成为炼药师？”
<br/>　　萧百炼话说出口时，连自己都觉得好笑。
<br/>　　“不行吗？”萧逸反问道。
<br/>　　“当然行。”萧百炼点点头，从抽屉中取出一本书籍，书籍封面写着‘炼药录’三个大字，递给了萧逸。
<br/>　　“我想拿回去慢慢看，可以吗？”萧逸知道这本炼药录的珍贵，犹豫地问道。
<br/>　　“没问题。”萧百炼耸耸肩，道，“反正整个萧家，所有人都看过这本书，但几乎没有一个人达到成为炼药师的要求。放在我这里也是白放，你拿回去就是。”
<br/>　　“不过切记，不能损毁书籍，看完好要完好归还。记好了，如果炼药录有什么三长两短，就算有三长老护着你，我也会将你屁股打到开花。”
<br/>　　“知道了。”萧逸撇撇嘴。
<br/>　　......
<br/>　　萧逸离开后不久，一名头发花白的老者缓缓踱步入了炼药堂。
<br/>　　老者出现的瞬间，所有药童和炼药师，包括萧百炼在内，所有人都站了起来，恭敬地行了一礼，“大长老”。
<br/>　　不错，老者正是萧家唯一的三品炼药师，萧离火。
<br/>　　整个紫云城最强大的炼药师，也是萧家威望最高之人。
<br/>　　萧离火摆摆手，道，“无需多礼，你们忙去吧。”
<br/>　　说罢，萧离火来到萧百炼面前。
<br/>　　“百炼，后天丹练成了吗？”萧离火问道。
<br/>　　萧百炼满脸通红地摇摇头，道，“没有。”
<br/>　　若是能练出后天丹，则代表该炼药师的品阶达到二品。
<br/>　　“哎。”萧离火叹了口气，道，“萧家的二品炼药师断层太久了，你是最有望成为二品的一个，连你都迟迟炼制不出后天丹。”
<br/>　　“大长老教训得极是。”萧百炼愧疚地低下头。
<br/>　　“我不是在教训你。”萧离火淡淡地摇摇头，道，“可惜了，如果家主当年没有离开就好了。”
<br/>　　“当年他已经是二品炼药师，如果现在还在萧家，肯定已经成为三品炼药师了吧，我也就不用为萧家的日后烦心了。”萧离火满脸唏嘘。
<br/>　　萧离火年纪很大了，但家族里的后天丹全都由他一人炼制。如果过些年，他撒手归西，萧家可就麻烦了。
<br/>　　“罢了，尽人事听天命就是。”萧离火满脸憔悴疲惫，道，“百炼，今天炼药堂中没发生什么事吧。”
<br/>　　“没有。”萧百炼回答道，“一切正常，没有出现毁炉或者炼制失败的情况。”
<br/>　　“那就好。”萧离火点点头。
<br/>　　“哦，对了。”萧百炼忽然说道，“刚才萧逸来过，取走了炼药录。”
<br/>　　“萧逸？还取走了炼药录？”萧离火有些惊讶。
<br/>(本章完)
