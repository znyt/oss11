婷婷对于他叔叔好像有些不客气，但是毕竟是他人的家事，也就只管看着，不说话。<br/>　　<br/>　　    “你啊！咦！这位是谁，婷婷你不介绍一下么？”说着，转头看向沈婷婷，倒是让沈婷婷有些翻白眼。至于说沈博文，自从进来后就一直对沈婷婷微笑着。<br/>　　<br/>　　    “叔，这是我一个朋友，叫陈默。”她知道自己的叔叔为什么看自己，内心一阵黯然，本来有些兴头，也随之暗淡下去。<br/>　　<br/>　　    “陈默，这是我小叔，沈博文！”<br/>　　<br/>　　    “您好，沈先生！”陈默上前伸手和沈博文握了握手，笑着问候到。<br/>　　<br/>　　    不过陈默的这个动作和称呼，也让陈博文知道他和沈婷婷没有什么特殊关系，就是普通朋友关系，要不然他不会这么称呼自己的。<br/>　　<br/>　　    沈博文脑海中想着，手里倒是也没有停下来，伸手和陈默相互握了握手，也是一团笑容。他今天收到自己侄女的信息，说是介绍一个供货商，所以也就过来看看。<br/>　　<br/>　　    这侄女可是自己大哥的心头肉，而且在家里也很是受宠，自己的老爹也是非常疼爱的，所以怎么也要露个面，给侄女装装脸面。<br/>　　<br/>　　    不过进来后看到是个年轻人，就心里一沉，思索着两人是什么关系，没有想到陈默他自己打招呼暴露出两个人的关系，也让沈博文一阵放松，没有关系就好，不然就麻烦了，婷婷早就有被自家大哥安排好了，要不是婷婷一直有些不愿意，可能两人早就结婚了。要是现在出了意外的话，还不被自家大哥埋怨死啊。<br/>　　<br/>　　    “好，好！来坐下说！”沈博文对陈默说道，并且自己也坐了下来，他身后跟着的几个人也相继坐了下来。<br/>　　<br/>　　    沈博文又给两人介绍了一下自己身后的两个人，一个是采购部的经理朱建乐，一个是他的助手邢克明。<br/>　　<br/>　　    除了沈婷婷没有反应外，其他几个人又是一圈的问好。但是这里也没有人计较，一来沈婷婷的来历背景大，所以没有啥好埋怨的，二来沈博文也没有说什么，他可是亲叔叔来着。三来陈默也不计较什么，他可是知道招惹女汉子的后果，还是不惹的好。<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
