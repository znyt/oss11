　　彩云仙子自最初吹奏仙乐助阵，并没有真正动过手，不然局面早就不一样了。对此，清长风也不敢说什么，冲她笑笑，“有劳姑姑。”
<br/>　　彩云仙子纯粹是来帮忙的。
<br/>　　轩辕山，在人间一向充满了神秘，在人间大多数法师的想象中，轩辕山是一个门派，这虽然没错，但他们不知道的是，轩辕山是一个十分宽松的门派，更像是一个联盟，七大长老、还有一些异兽、当年飞升的人类，等等，各自都占据了一座山头，有的是自己清修，有的则开宗立派，接引弟子。
<br/>　　就像道门诸多门派一样，虽然都信奉三清，但每个门派都有自己的一套体制，互相之间并不算亲密，只有在外敌入侵（这种事轩辕山从来没发生过），或者为了轩辕山共同的利益，大家才会一起行动。各山头自己的行动，想找别人帮忙，那就看关系了。
<br/>　　清长风和假叶少阳找了这彩云峰设下埋伏，就是因为两人关系不错，别的门派还真不一定愿意多事，就连讪讪禅师，也是因为之前叶少阳一行人闯过累次，双方结下因果，今天才特意来助阵的。
<br/>　　所有门派之中，唯一的例外，就是法术公会。如果清长风等人的门派都属于分公司的话，那么法术公会就是轩辕山的直营单位。
<br/>　　它不是星月奴自己的，星月奴不过是总部董事会任命的Ceo，业务方面，别的公司都要配合一些，清长风跟星月奴是师兄弟，虽然各怀心事，看对方不太顺眼，但场面上还是得互相帮忙的。
<br/>　　彩云仙子走后，将她的仙乐大阵也撤掉了，只剩下大洞真经的碑文，还在困在叶少阳二人。
<br/>　　“道风，你斩却两尸，我度过七重雷劫，今日一战，正好分个高下！”
<br/>　　道风看着他，淡淡说道：“你若真想一分高下，便将捡到的法器还我，不然，让你那些埋伏在四周的弟子，一起上杀了我便是。”
<br/>　　清长风一滞，袖子里的双手本能地握紧，一只手是打神鞭，一直手是番天印。
<br/>　　这两样东西，都是刚才道风的化身所用的，死了之后被他捡到的。
<br/>　　两样都是九段光的神器，就算是他清长风，也十分垂涎，看到就捡了，但被道风这么一说，脸上有些挂不住。
<br/>　　自己……毕竟是轩辕山的一门之主，那么多弟子都埋伏在四周，暗中窥视着，这么明目张胆地抢夺人家法器也就算了，关键是自己嘴上还一直说跟他公平决斗……
<br/>　　更不要说，在彩云峰周围，还藏着不少几个宗主，都是来看热闹的，并没有现身，自己这么做的确让人笑话。
<br/>　　清长风眼珠一转，有了主意，反正道风今天必死，先把东西给他，姿态做足，等杀了他，自己再拿这两件法器，便是名正言顺了。
<br/>　　当下将两件法器丢给他。
<br/>　　“来吧！”
<br/>　　清长风双手结印，人影一闪而逝，消失无踪，与此同时，天空逐渐变暗，形成了一朵巨大的乌云，一时间狂风乍起，电闪雷鸣。
<br/>　　咔嚓！
<br/>　　一道闪电在黑暗中惊现，却不是一般的形状，而是五根又细又长的手指，如同鬼爪。
<br/>　　道风骤然暴起，一鞭子抽在鬼爪上，立刻电光四射，纷飞落下。
<br/>　　伴随着轰鸣的雷声，鬼爪不断闪现，从各个角度不断进攻，道风也使出了浑身手段，见招拆招……因为两人实力相当，战斗看上去虽然紧迫，但似乎不够刺激。
<br/>　　然而，若是将他们两人换成一般法师，不管是道风的打神鞭，还是清长风的幻术鬼爪，可能一下都撑不住就挂了。
<br/>　　“这个道风，似乎也没有那么强。”峡谷对面、稍微远一点的那座山，几个人站在山顶，一个个都眼睛不眨地望着两人的战斗。
<br/>　　轩辕山好久没有这样的热闹可以看了，轩辕山几大宗门的老大和强者来了许多，又不想直接帮忙，于是各自观战，后来感觉一个人看热闹太无聊，于是不由自主地聚在一起，一边观战一边讨论起来。
<br/>　　方才开口评论的，就是一个穿黄衫的中年人。
<br/>　　“你没看出吗，他并没有出全力。”身后一个秃头的胖子说道，但他没穿僧衣，就这么看上去，也不知道是和尚还是秃子。
<br/>　　黄衫道士道：“你这说得，难道北斗就用了全力了？”
<br/>　　回头看了一眼，他身后站着一个身穿白衣、头发胡子全白的老者，说道：“仙翁，这道风杀了可惜，不如你引渡一番，化其戾气，收归门下调教一番，岂不是好。”
<br/>　　仙翁微微一笑，“星月奴恒之入骨，老朽可不敢触这个眉头，再者，老朽还真没资格教化于他。”
<br/>　　这时人群中钻出来一个头陀模样的矮胖子，嚷嚷道：“列位，列位，休要说这些无用之事，倘若清长风落败，我等帮是不帮？”
<br/>　　大伙各自沉吟起来，黄衫道士说道：“列位稍候，我去长老会打探一番。他们八成还不知道此事。”说完化作一道青烟，朝身后的群山之中飞去。
<br/>　　咔嚓！咔擦！
<br/>　　战场之中，随着鬼爪不断落下，道风总算支撑不住，抵挡的姿势也越发狼狈起来，一时间险象环生。清长风当然不会放过机会，不过他生怕有诈，在试探了几次，确定道风真是扛不住了之后，这才展开了疯狂的攻击。
<br/>　　“噗嗤！”
<br/>　　一只从，从后面穿过了道风的胸膛，五指并拢，结印朝他腹部扣去，道风一个纵身出去，躲过致命攻击，人摇摇欲坠。
<br/>　　清长风身形显露，带着胜利的微笑，拢着双手，站在原地望着他。
<br/>　　道风摇摇欲坠，用打神鞭撑着地，抬头看向清长风。
<br/>　　“道风，许久不见，你实力并没有长进，而我却度过了七重雷劫，如今的你，已不是我对手。”说完摇了摇头，感觉有些无趣，作法召唤飞旋在身边的金字，朝道风绞杀过去。
<br/>　　道风双手握着打神鞭，将体内参与的邪气释放出来，苦苦地抵抗着。
