        &lt;h3&gt;第三卷第一百三十八章  凌辰的决定&lt;/h3&gt;
<br/>
<br/>        凌辰回头见刘诚之没有跟着自己，觉得奇怪，便又返回去，看到下面的情况：
<br/>
<br/>        董亚子拽着刘诚之的两只耳朵，俩个人在亲、亲。
<br/>
<br/>        凌辰以为他看错了，眨了眨眼睛，确定没有看错。
<br/>
<br/>        他尴尬的“嗨”：“我好像是不应该来。眼睛被辣了。”
<br/>
<br/>        董亚子迅速的松开了刘诚之，刘诚之一脸懵圈。
<br/>
<br/>        董亚子满脸通红：“凌辰哥哥，不是你想的那样，不是的。”
<br/>
<br/>        凌辰看着刘诚之那傻乎乎的样子问：“那是什么样的？”
<br/>
<br/>        刘诚之：“她、她、她强、迫我.......”
<br/>
<br/>        凌辰实在是忍不住，万年冷山上裂开了一道缝：
<br/>
<br/>        “亚子妹妹，你厉害啊!”
<br/>
<br/>        “凌辰哥哥，不是你想的哪样。”董亚子着急的辩解。
<br/>
<br/>        “不，你应该喊诚之哥哥。哈哈哈。”凌辰笑着离开了。
<br/>
<br/>        这下好了，董亚子不黏着他了，董亚子对他多少有点意思，是个人都能看得出来，但是，他对董亚子没有感觉。
<br/>
<br/>        这么多年，宁雅已经占满了他的心。
<br/>
<br/>        可是宁雅又伤透了他的心，一边和他甜言蜜语，一边找男朋友。
<br/>
<br/>        他在宁雅心中到底是什么？
<br/>
<br/>        宁雅每次出去相亲都会告诉他，说她看不上对方，可是她的父亲就是想要给她找一个门当户对的爱人。他每次听着，都在想，自己一定要强大，宁雅，等着我。
<br/>
<br/>        想到宁雅，他就烦躁，狠狠的踢了一下脚下。
<br/>
<br/>        刘诚之和董亚子走在凌辰的身后。
<br/>
<br/>        董亚子瞪着刘诚之。
<br/>
<br/>        刘诚之很委屈的轻声说：“我的first  kiss都让你夺走了。”
<br/>
<br/>        董亚子：“哼，是我的first  kiss被你夺走了。”
<br/>
<br/>        刘诚之：“那就扯平了，你不要瞪我了，不见。”
<br/>
<br/>        刘诚之快速赶上凌辰，小心脏通通的跳，真是，还想着和凌总一样谈一场深情而甜甜的恋爱呢，first  kiss就这样没了。
<br/>
<br/>        “凌总，我们什么时候回去？”刘诚之想着要离董亚子再远一点。
<br/>
<br/>        “不是，你是着急回去呢？还是舍不得离开？”凌辰很好笑。
<br/>
<br/>        “什么叫舍不得离开啊？再不离开我怕我不保。”刘诚之悄悄的指了指后面的董亚子。
<br/>
<br/>        “刘诚之，你说什么呢？你们谁都不能离开!”董亚子在后面生气的喊，心里窝火着呢。
<br/>
<br/>        刘诚之一定是故意的。
<br/>
<br/>        “为什么？”凌辰转过身皱着眉头问。
<br/>
<br/>        凌辰皱着眉头的样子冷冰冰的，一股子寒意。
<br/>
<br/>        “他非，礼了我，要对我负责！”董亚子一着急脱口而出。
<br/>
<br/>        凌辰：“刘诚之，听到了没有，你要对董总负责！”
<br/>
<br/>        刘诚之：“为什么要我负责？是她拽着我的，我凭什么负责！”
<br/>
<br/>        董亚子看着刘诚之，她这是被嫌弃了？在学校多少人围着她转，回来后，家里的门都被踏破了，要不是心里有凌辰，她早都嫁人了，你一个小小的助理竟然还嫌弃我？
<br/>
<br/>        看着刘诚之那嫌弃的样子还有凌辰那揶揄的眼神，她心里是又气又羞，“刘诚之，你等着！”转身就走了。
<br/>
<br/>        “凌总，现在的女人真是，太奇怪了。”刘诚之心有余辜的说，不过，那个董亚子的嘴唇，软软的。
<br/>
<br/>        “刘诚之，没发现啊，你这翻脸的速度很快啊。”凌辰看着刘诚之那意犹未尽的样子。
<br/>
<br/>        “你别冤枉我了，我就是回去提醒她，让她不要打你的注意了，我告诉她小雅姐姐家世好，人又漂亮又能干，她就和老虎一样逼着我，然后拽着我的耳朵，然后就是你看到的样子了。”刘诚之解释着。
<br/>
<br/>        “你小雅姐姐家世好？又漂亮又能干？关我什么事？关你什么事？”凌辰冷冷的说。
<br/>
<br/>        “不是，凌总，你和小雅姐姐怎么了？最近好像一直都不见你和小雅姐姐联系。”刘诚之问凌辰，虽然凌辰一直是冷冰冰的，但是他们在一起还不错，凌辰就像一个大哥哥一样。
<br/>
<br/>        “不关你的事情。”凌辰有些生气。
<br/>
<br/>        “怎么就不关我的事情了？这关系到我以后工作的问题，找个温柔娴淑的老板娘，我们以后跟着你一起幸福，找个母老虎，天，画面感太强，不忍心想。”刘诚之顶着三九寒天的冷气说。
<br/>
<br/>        “想宁雅给你当老板娘，你可以滚了，去她男朋友那里去吧，不要在我这里了。”凌辰这一段时间的隐忍已经爆发了。
<br/>
<br/>        “啊，她有男朋友了啊？不是她有男朋友你们还聊的热火朝天的？”刘诚之觉得不可思议了。
<br/>
<br/>        “原来你这一段时间不联系她，是她有男朋友了啊？”刘诚之工作上很细心，但是有时间又很大条，这简直就是在凌辰的伤口上抹盐，抹盐不说，还加了一把朝天椒：“我说，你这么多年努力，连个女人都追不回来，天天搞暖昧，追求和暖昧还是有区别的。”
<br/>
<br/>        “你说什么？”凌辰简直就是暴怒了，要不是旁边人来人往，他都想动手揍刘诚之了。
<br/>
<br/>        “嗨嗨，注意形象，别那么大声，你要是厉害你就去将宁雅重新追回来啊，喊我能把老婆喊回来？你要是对着我喊，能把老婆喊回来，我就姓你的姓！”刘诚之看着凌辰那个样子也来气了。
<br/>
<br/>        还是不是男人了？
<br/>
<br/>        “人家都已经在一起了，我还追什么追？”凌辰被刘诚之一喊，竟然就泄气了。
<br/>
<br/>        “你看见在一起了？不是这几天我们都在一起，你也没有看见宁雅啊？她亲口告诉你了？”刘诚之有些不解，这才多久的时间没有见啊？
<br/>
<br/>        “没有，等于告诉我了一样。”凌辰想起那个电话，对他来说就是一场恶梦。
<br/>
<br/>        “没有看见的事情就不要下定论，不过啊，凌总，世上森林一大片，你也不应该为了一棵树放弃一片森林，天涯何处无芳草，何必单恋一枝花。”刘诚之摇摇头。
<br/>
<br/>        “原来你是这样的刘诚之。”凌辰看着刘诚之，像是才认识刘诚之。
<br/>
<br/>        “唉，我这是安慰你受伤的小心脏，怕你难过嘛！”刘诚之又不怕死的来了一句。
<br/>
<br/>        “不难过了，我想通了，像你说的，我和她一直暖昧不清，爱着她，却不敢告诉她，怕他们家里不愿意，男人，只有自己强大了，才会拥有一切，包括爱情！”凌辰说：“刘诚之，两年内我不谈感情好好做事情！”
<br/>
<br/>        “嘿，有志气，不愧是我看上的老总，看好你了，说不定两年之后小雅姐就来向你提亲了，加油！哈哈哈！”
<br/>
<br/>        “刘诚之，你是越来越大胆了啊！”凌辰看着刘诚之。
<br/>
<br/>        “谢谢凌总借给我的胆子！”刘诚之不知道凌辰和宁雅到底发生了什么，但是凌辰从医院出来整个人像是从冰窖里来出来的一样，搞得他也提心吊胆的。
<br/>
<br/>        傲娇的凌辰很想探听一下宁雅的消息，拿起电话想给宁雅打电话，可是想到那天接电话的声音，心里堵得慌，又将电话放进了口袋。
<br/>
<br/>        终于想到陈子昂，给陈子昂打个电话问问情况吧。
<br/>
<br/>        可是电话响了好久，陈子昂都没有接电话，凌辰叹了口气，挂了电话。
