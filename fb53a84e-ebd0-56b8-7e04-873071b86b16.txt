两头狼扑了过来，带出一股强劲的风向。
<br/>
<br/>        陆寒霆伸手一推，直接将怀里的夏夕绾给推开了，他沉声吼道，“快跑！夏夕绾，快跑，不要回头！”
<br/>
<br/>        夏夕绾被推开了几步，在千钧一发之际，陆寒霆还是推开了她。
<br/>
<br/>        说好了一起死的，他怎么这会儿就舍不得了？
<br/>
<br/>        两头狼扑在了他的身上，他开始跟狼殊死搏斗。
<br/>
<br/>        夏夕绾没有跑，她看着跟两头狼打在一起的陆寒霆，陆寒霆身手矫健，动作快准狠，一时之间竟然让两头狼无法下口。
<br/>
<br/>        但是人的体力是有限的，更何况这里有两头狼，这时一头狼看出了破绽，一口咬住了陆寒霆的右臂。
<br/>
<br/>        夏夕绾当即看陆寒霆出了一头的冷汗，一张俊脸已经煞白了，夏爸说过那一次他为了救她，直接用劳斯莱斯幻影豪车撞上了前面的车，近乎疯狂，他的右臂就在那一次废了。
<br/>
<br/>        现在右臂被咬住，他一时难以挣脱，直接落在下风了。
<br/>
<br/>        这时陆寒霆抬头，猩红的狭眸向她看了过来，“夏夕绾，你耳朵聋了，机会已经给你了，快跑！”
<br/>
<br/>        这时另一头狼已经看准了时机，它张开血盆大口直接向陆寒霆扑去，想要将陆寒霆撕成碎片。
<br/>
<br/>        夏夕绾瞳仁一缩，惊呼了一声，“不要！”
<br/>
<br/>        夏夕绾扑了过去，直接扑到了陆寒霆的身上。
<br/>
<br/>        陆寒霆没想到她会扑过来，现在只觉得视线里一黑，她纤柔的身体已经扑到了他的怀里。
<br/>
<br/>        他几乎是瞬间就搂着她的身体转了一个圈，将她护到了他的身下，但是狼的速度更快，锋利的爪子抓向了夏夕绾心口上方的位置，直接拉出了三道血痕。
<br/>
<br/>        夏夕绾的鲜血溅到了狼的身上。
<br/>
<br/>        这时凶猛攻击的两头狼突然顿住了，它们停止了攻势，两双绿油油的双眼落在夏夕绾的心口上，现在衣服都被抓烂了，露出了夏夕绾心口上方那朵红色到近乎妖冶的帝女花。
<br/>
<br/>        两头狼慢慢的收回了眼里森然的绿光，变得乖顺。
<br/>
<br/>        见两头狼停止了，陆寒霆这才大口大口的喘息，他精硕的胸膛上下起伏，目光狠戾的盯着身下的夏夕绾，“夏夕绾，你是不是不要命了，机会已经给你了，为什么不跑？”
<br/>
<br/>        夏夕绾疼的小脸煞白，腻白的美人尖额头都是细碎的冷汗，说话都开始哆嗦。
<br/>
<br/>        这时陆寒霆将她用力的扯抱进了怀里，薄唇落在她的秀发上，狠狠的里外都亲了一遍，他用尽全身的力气在克制自己，嗓音暗哑，“夏夕绾，你没有机会了，刚才不跑，以后你就跑不了了！”
<br/>
<br/>        他的话更像是宣誓。
<br/>
<br/>        夏夕绾现在很疼，只能这样给他抱着。
<br/>
<br/>        陆寒霆抱了一会儿就松开了她，他垂眸看着她心口的伤势，被狼抓出来的那三道血痕触目惊心的。
<br/>
<br/>        陆寒霆脱下身上的大衣披在了她的香肩上，然后将她给打横抱起，“我们现在就出去。”
<br/>
<br/>        “不行。”夏夕绾拒绝道，“我还要留在这里。”
<br/>
<br/>        今天进入这片禁地毫无收获，她不可能就这样出去的！
