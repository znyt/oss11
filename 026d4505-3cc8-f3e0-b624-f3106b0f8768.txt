　　有了奢比尸的加入，玄天从一人大战三祖巫变成了一人大战四祖巫，风云变色，战场之中，瞬息势变。
<br/>　　玄天一人大战四大祖巫，依旧显得不慌不乱，挥手之间虚空破碎，各种招式中毫无花哨，朴实无华，有的只是最简单、最直接、最有效的一拳、一指、一掌、一腿等，但就是这样最简单的招式之中，却又比其它大招还要厉害。
<br/>　　其实，对于玄天来说，招式已经只能算是一种手段了，到了他这种境界，对道则、规则的运用，要比许多招式厉害更多。其实，招式也只是规则运用的一部分而已。
<br/>　　在玄天面前，一切大招都是虚妄的，他挥手之间，尽是道则攻击，这也是他能够一人战四大祖巫的原因。
<br/>　　祖巫们注重肉身，一力破万法，相反，对于身体之内的法则运用就要弱了许多，这就给他玄天巨大的优势。他甚至还可以一边大战，一边学习巫族身上的法则，之后再以彼之道还施彼身。
<br/>　　剩下两位祖巫中，后土祖巫不赞成屠杀人族，其实她心里对人族是充满愧疚之情的，因为是他们巫族最先背弃盟约的。但，因为玄天斩杀祝融祖巫之事，此刻她又不得不出手对付人族，她的心中充满复杂之情。
<br/>　　至于玄冥祖巫，倒是心中没有太多想法，只是单纯想为祝融报仇，为巫族除去后患，仅此而已。
<br/>　　面对两大祖巫的屠杀，剩下的人族又该如何才能抵挡？
<br/>　　难道人族的命运真的要如此终结了吗？
<br/>　　就在两大祖巫要动手屠杀人族，收集他们的魂魄怨力炼制戮妖幡之际，高空之上，再次飞来四道流光，长虹贯日，拦截在两大祖巫身前，停了下来。光芒散去，现出四位青年，两男两女。
<br/>　　男的脸色坚毅，一个一身青衫，一个一身白衣，女的清纯亮丽，一个一身墨色衣裙，一个一身白色衣裙。他们一看到两位祖巫，便拦了下来。
<br/>　　“你们是谁？”
<br/>　　玄冥两人有些警惕的看着出现在自己面前的四人，她一眼就看出眼前四人都是准圣高手，现在他们突然出现在人族战场之中，肯定是带有特别的目的。若是他们是人族的仇敌还好，但若是他们要来帮助人族，恐怕……
<br/>　　“青玄！”
<br/>　　青衫男子看着眼前大战中的巫妖两族，看着堆积如山的尸骸，血流成河，看向巫族的目光中充满不忿，皱了皱眉头，慢慢道。
<br/>　　“玉玲珑！”墨色衣裙女子面色凝重，淡淡道。
<br/>　　“白衣！”
<br/>　　“风晓月！”
<br/>　　白衫男子与白色衣裙少女接着说道。
<br/>　　玄冥与后土皱了皱眉头，有些尴尬，因为她们都没有听说过这四人的名字。不过，洪荒如此大，能者异士辈出，什么地方出现几个隐士之修，她们不认识，也是有可能的。不过，这些都不重要，重要的是现在要弄清楚这四人的目的。玄冥看了一眼四人，道：
<br/>　　“不知四位为何到此，而且还阻我去路？”
<br/>　　“久仰巫族十二祖巫大名，今日一见，果然非凡。我等想要与两位讨教讨教。”青玄淡淡出声道。
<br/>　　玄冥祖巫目光一下子就冷了下来，感情还真是还阻止她巫族杀人啊，真不知道此四人与人族有何关系，竟值得他们得罪巫族而去帮助人族？不再多想，她冷声道：
<br/>　　“我不管你们是何人，敢阻拦我巫族杀人，都要死！”
<br/>　　青玄四人不再说话，而且取出各自的灵宝，戒备起来。青玄手中握着一柄青色长枪，名为青龙抢，是一件上品先天灵宝，威力巨大。白衣手中握着一柄银色灵剑，剑身银亮，剑光绚烂，剑柄玉身，是一柄无双好剑。
<br/>　　玉玲珑手中一翻，取出一台黑色长琴，琴身上刻着四大凶兽图纹，七根琴弦长短不一，也是一件先天灵宝。风晓月取出一条破白色绫布，上面绣着利刃神兵，凤凰朱雀等神兽，一看就知道这不是普通绫条，而是一件厉害的灵宝。
<br/>　　“哼！战！”
<br/>　　玄冥祖巫冷哼一声，不再废话，对方目的都已经很明显了，自己再废话那么多干什么。一个字，就是——干！
<br/>　　祖巫肉身强横，玄冥更是祖巫中的佼佼者，一身实力堪比准圣中期巅峰了，想要对付她，一人自然是不够，所以，需要青玄与风晓月两人联手，才能对抗玄冥。
<br/>　　而后土祖巫更是了得，作为最后一个出世的祖巫，她的资质趋势十二祖巫中最好的，一身肉身实力堪比准圣后期，脚踏大地，更是能够源源不断地借助大地之力来作战，端是厉害。玉玲珑与白衣两人都有些招架不住。不过，后土心梗并没有太强杀意，所以他们还能勉强应付。
<br/>　　玉玲珑的天魔琴是音攻性灵宝，能射放出魔音，影响人的思维，灵魂越弱，影响力越强。再加上她修炼的就是音系功法，能够震慑人的元神灵魂，杀人于无形。而且祖巫之中，肉身强大，没有元神，灵魂也不经常修炼，所以，玉玲珑的音攻倒是正好克制祖巫。
<br/>　　但也只是有些影响而已，因为玉玲珑的实力，还不足以杀灭祖巫的灵魂。但，就算如此，也能够为其他三人大战祖巫，赢得一些优势了。
<br/>　　一时之间，东海之滨，首阳山人族祖地，几处战场，完全爆发，而且还愈演愈烈，战斗在不断升级，范围不断扩大，他们并不是在一个地方大战。战场是在不断移动着的，谁也不知道，下一刻，他们会去到哪里。
<br/>　　此刻，关注着这场战争的各路大神们，神念横空，笼罩大地，又是议论纷纷，一阵惊叹，人族不得了啊。
<br/>　　“武祖一人大战四大祖巫，真是强悍，洪荒之中，准圣之下，谁人能敌？”
<br/>　　“人族其他强者也是非常厉害，如燧人氏三祖，竟然有三件功德圣器，好厉害！”
<br/>　　“咦！那是……”有人发现了青玄等四个新冒出来的人。
<br/>　　“那四人是谁？为何会在此刻出现，帮助人族？”
<br/>　　“难道他们也是人族？”有人这样猜测道。
<br/>　　“不是，若真是人族，那还得了？”
<br/>　　“咦，那不是玉清圣人最后一次听道所带去的四人吗？当时还坐在玉清圣人身后呢？”有听过道祖讲道的大神认出了四人的身份。
<br/>　　“那不就是玉清圣人的徒弟么？”
<br/>　　“什么？！”
<br/>　　众人大惊，得到这样一个结论，元始的四个徒弟都出手帮助人族，那岂不就是代表着人族的背后有玉清圣人在撑腰吗？三清为一体，有了玉清，太清、上清还会远吗？
<br/>　　人族的背后是三位圣人！
<br/>　　得到如此一个结论，众人心惊不已，一想到这里，众人不能淡定了。人族何德何能，能让圣人如此对待？
<br/>　　这一刻，众人沉默了，看向人族，越看越觉得心惊胆颤。
