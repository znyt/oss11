　　“这……！”
<br/>　　姜神夜面带迟疑之色，他还是不太想加入万界山，对于这种未知的势力，他心中有很多的不确定。
<br/>　　“加入万界山，你能得到的好处远超在三神宗！”真贤道圣微笑着说道，但他的眼眸深处，却是留露出了一缕寒芒。
<br/>　　不愿意加入万界山，那就只能埋骨与此了，这是万界山的规矩。
<br/>　　历来不愿意加入万界山的天才，只有死路一条，因为万界山是隐藏在黑暗中的势力，不想让任何人知晓。
<br/>　　虽然真贤道圣很看重姜神夜，但他不得不如此做。
<br/>　　“姜兄，加入吧，你我以后就是同门！”苏莫向姜神夜使了个眼色，他注意到了真贤道圣眸中的寒芒。
<br/>　　姜神夜也不傻，闻言深深的吸了口气，随即重重的点了点头。
<br/>　　“很好，本道已经数十年没有为万界山选拔到弟子了，这一次能一次性选拔到两位，值得庆贺！”真贤道圣满脸微笑。
<br/>　　他负责管理古灵星河和其附近几个星河的弟子选拔，这数十年来因为没有选拔到弟子，难免被万界山其他同门嘲弄。
<br/>　　虽然这几十年来，他也发现了少量天赋达到标准的天才，但只是达到标准而已，天赋还是不够妖孽，达不到他自己的要求。
<br/>　　这一次的苏莫和姜神夜，绝对算是妖孽中的妖孽，尤其是苏莫，连他都看不透，不知道天赋具体达到了什么地步。
<br/>　　“这是我们万界山的界主雕像，现在你们行入门叩拜之礼吧！”真贤道圣走到纯金雕像身边，对苏莫两人说道。
<br/>　　苏莫和姜神夜闻言，走到雕像身前，恭敬的向纯金雕像行了叩拜大礼。
<br/>　　“好，从今日起，你们二人就是万界山的记名弟子！现在跟本道来吧！”真贤道圣朗声说道，言罢，便转身向一座古朴的宫殿飞了过去。
<br/>　　苏莫和姜神夜见此，立刻跟了上去。
<br/>　　少倾之后，两人跟着真贤道圣，打开了宫殿外的阵法，来到了宫殿中。
<br/>　　宫殿很大，其内摆放着一排排的檀木长桌，摆满了大半个宫殿。
<br/>　　而在这些檀木长桌之上，摆放着大量的玉牌，每一个玉牌都有巴掌大小，总数不下上万个，而且其中偶尔还有碎裂的玉牌。
<br/>　　这些玉牌之上都有名字，显然都对应不同的人。
<br/>　　真贤道圣手掌一番，取出两个同样的玉牌，玉牌上正是苏莫和姜神夜的名字。
<br/>　　“在玉牌上滴上一滴你们的精血！”真贤道将玉牌递给了苏莫和姜神夜。
<br/>　　“精血？”
<br/>　　苏莫闻言疑惑，不解的问道：“前辈，这些是何物？”
<br/>　　“这叫灵魂玉牌，滴了精血之后，你们日后若是死了，玉牌也会碎裂，同时，通过此灵魂玉牌，不仅可以感知到你们的位置，还能对你们发出召唤！”真贤道圣说道。
<br/>　　苏莫闻言微微皱眉，可以通过这玉佩感知到他的位置，那他以后会不会受制于万界山了？
<br/>　　不过，既然加入了万界山，他也无法拒绝，现在只能照做。
<br/>　　逼出一滴精血滴在玉牌上，苏莫又将玉牌交给了真贤道圣，姜神夜也同样如此做了。
<br/>　　随后，真贤道圣将两人的玉牌，摆在了长桌上。
<br/>　　“一切已经完成，你们现在可以得到在神灵池洗礼的机会！”真贤道圣说道。
<br/>　　“神灵池？是什么地方？”姜神夜好奇的问道。
<br/>　　“到了你就知道了！”真贤道圣微微一笑，随即大步离开了宫殿之中。
<br/>　　苏莫和姜神夜急忙跟了上去，既然是入门奖励，他们自然不会拒绝。
<br/>　　“两位，你们既然已经加入了万界山，也不必叫本道前辈了，大家都是同门师兄弟，唤本道一声师兄便可！”真贤道圣一边大步而行，一边微笑着说道。
<br/>　　“是，真贤师兄！”苏莫两人点了点头，同时两人心中也觉得有些怪异，喊一名武圣境后期的强者为师兄，这让他们觉得有些不可思议。
<br/>　　三人绕着万界山的山腰走了半圈，很快便来到了后山。
<br/>　　穿过一片奇花异草密布的丛林，三人来到了一个被众多奇花包围的水潭之前。
<br/>　　“这是……真圣之气？”苏莫惊讶的望着面前的水潭，这水潭不大，方圆不足二十丈，潭水呈淡黄之色，真是真圣之气。
<br/>　　只不过，这真圣之气极为浓郁，已经化为了液体。
<br/>　　“这的确是真圣之气，但我们称之为神灵之气，而这潭水正是神灵之气浓缩的精华，被称为神灵液，普天之下只有万界山才拥有！”真贤圣师说道。
<br/>　　“真贤师兄，我们的身体已经融炼了不少真圣之气，已经无法再融合吸收了！”姜神夜无奈说道，他之前吸收融炼了四股真圣之气，身体已经接近饱和了。
<br/>　　“真圣之气你们无法吸收融炼，但这神灵液不同！”真贤圣师摇了摇头道。
<br/>　　“哦！”苏莫两人了然，但也没太当回事。
<br/>　　真贤道圣见两人一脸兴趣缺缺的模样，嘴角微微一勾，略带戏虐的说道：“只要经过神灵池的洗礼，将来必成武圣。”
<br/>　　“什么？”两人闻言一震，顿时张大了嘴巴。
<br/>　　必成武圣？
<br/>　　这也太夸张了吧！
<br/>　　据说想要成为武圣，已经不单是全靠天赋了，还要有机遇、运气、契机等等因素。
<br/>　　就算是融合了真圣之气，也只是增加成为武圣的几率而已。
<br/>　　没有人敢肯定，自己一定能成为武圣。
<br/>　　就比如姜神夜，被人称为有虚神之资，虽然不一定能成为虚神，但他自信将来成为武圣，应该没有什么问题，但即便如此，他也不敢保证必成武圣。
<br/>　　毕竟修炼到了后期，一切都不确定。
<br/>　　而这小小的神灵池，经过一次洗礼，就能必成武圣？怎么可能？
<br/>　　见苏莫和姜神夜一脸不可置信的表情，真贤道圣又补充道：“当然了，若是提前陨落，或者是修炼出了岔子，那就另当别论了！”
<br/>　　“真贤师兄，此言当真？”苏莫眼眸闪亮，惊讶的问道，若真是如此，那还得了！
<br/>　　他若是多弄些神灵液出去，给父亲等苍穹神宫的人洗礼一番，岂不是将来个个都能成为武圣？
<br/>　　“这还能有假？就算是一头猪扔进去，过些年都能修炼到武帝！”真贤道圣笑眯眯的说道。
