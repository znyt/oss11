<br/>        叶翎出身很高，是将门之后，从小就在大院里长大，只可惜她父母殉职在了一场汽车爆炸里，连她的亲哥哥都不知所踪，于是她被商贾簪缨世家的顾家给收养了。

<br/>        夏妍妍很毒，三言两语就挑中了叶翎的痛处。

<br/>        这一件事被当时的顾家一手压了下去，外面并没有风声出去，但世交的亲人还有在场的这几个亲密朋友都是知情的。
<br/>
<br/>        夏夕绾当即担忧的看了叶翎一眼。
<br/>

<br/>
<br/>        霍璇垂在身侧的两只手一下子拽紧了拳，“叶翎，你是不是忘了事后夜瑾哥哥狠狠的给过你一巴掌？”

<br/>        这时夏妍妍突然出声道，“夜瑾，你来了？”
