　　粮草被焚毁对于唐军来说可以算得上致命的打击，就算是临时筹措，想要弄到可供二十万大军吃上十天的粮草也不是那么容易的。
<br/>　　而且唐军在这里驻扎也有一段时间了，周边那些有钱有粮的富户早就已经跑的连影子都看不到，余下的那些穷苦老百姓家中根本就没有什么吃的，就算是把活人都算上，也不够唐军二十万人吃一顿的，所以筹粮的事情根本想都不要想。
<br/>　　另外还有一点就是那些粮草是当着无数唐军士兵的面烧的，就算是候君集想‘瞒’也根本瞒不住，那些士兵只要有一顿没有吃到饱饭，立刻就会有哗变的可能。
<br/>　　至于说等后方将粮食运到，不好意思，后方也是要筹措粮草的，运到前线怎么也要十天时间，所以就算是把消息送回天竺道，负责后勤的官员也不可能马上变出粮食给前方送过来。
<br/>　　这样的情况下退兵成了唐军唯一的选择，而且为了不让拜占庭人追上，唐军撤退的速度还必须要快。
<br/>　　伊力克完全没有想过明尼城主会成功，在他看来这一次的偷袭最大的作用就是用他来测试唐军的实力。可让他万万没想到的是，明尼城主竟然成功了，不但成功的烧掉了唐军的粮草，而且还带回来三千来人。
<br/>　　如果不是过于担心唐军的实力，这一切功劳其实都应该是伊力克的，可是因为他顾虑太多，故而与战功失之交臂，每每想起，就让他痛心疾首。
<br/>　　不行，这样子决对不行，战功绝对不能让明尼城主全部得去，自己怎么也要从中分一杯羹，考虑了半天之后，伊力克终于下定了决心，命人去后面催促主力部队的同时，带着他的两万部下气势汹汹的向着正在组织后撤的唐军扑了上去。
<br/>　　候君集等人站在明尼城的城墙顶上，眼睁睁看着下面两万拜占庭军队从城下经过，老脸上不由露出一抹满含深意的笑容，扭头对身边的裴行俭说道：“干的不错，回去准备吧，估计要不了多久，士兵们的步枪就可以饱饮敌人的鲜血了。”
<br/>　　“大将军，敌人的主力还要两天才会赶到，我们现在就还击是不是早了些？”裴行俭有些疑惑的问道。
<br/>　　“万事有度，过犹不及！”候君集的目光盯着城下路过的拜占庭军队，像是在看着一群死人：“你只知示敌以弱，敌人会追上来，可是你没有想过，敌人也有脑子，也会观察我们，如果我们连一场仗都不打，只怕很快就会引起敌人的警觉。”
<br/>　　“是，大将军！”裴行俭露出一抹恍然的神情，抱拳躬身：“谢大将军指点！”
<br/>　　候君集转过头，深深看了一眼裴行俭：“罢了，我们这些老家伙总不能把一身的本事全都带进棺材里，所以能学多少就看你们的了。”
<br/>　　“是！”
<br/>　　……
<br/>　　龙义，第九十二师师长，按照级别来说，等同于过去十六卫将军之职。
<br/>　　只是相比于以前的十六卫将军，他现在的权力要大上许多，部下一万五千人吃喝拉撒全都要管，如果不是上过两年的军校，估计现在他早就已经玩不转了。
<br/>　　要各道，以前的十六卫，有长史，有书记官，还有行军司马，总之闲人很多。
<br/>　　可是现在除了一个师部，几个作战参谋，下面就是一些旅长、团长，以前那些应该由长史、书记官管的东西都被归划到了师部统一管理，这就造成了师部参谋必须一专多能，一个人要干好几个人的事，就连他这个师长有些时候也不得不参与进去。
<br/>　　不过这一次让他头疼的却不是文书方面，而是作战方面的事情。
<br/>　　候君集这个当初的顶头上司昨天传来命令让他们假装粮草被毁，还要假装撤退。而今天一早又下令他们第九十二师负责阻拦敌人，务必把身后两万敌人的先锋拦下来。
<br/>　　不过拦下来归拦下来，却不能一时激动把这两万人全给吃了，必须让他们有一种随时可以取得胜利，但却需要付出一定代价的感觉。
<br/>　　也就是说，九十二师明明有能力一个反击把身后的两万拜占庭人彻底吃掉，但却必须假模假式的跟他们打，不能把敌人消灭不说，还要让自己一方合理的减员。
<br/>　　这对于龙义来说简直太难了，因为九十二师的武器装备根本就不适合打这种拖拖拉拉的战役，近百门贞观炮完全有能力一个覆盖射击将敌人的进攻打垮，然后步兵再一个冲锋，火箭弹与铁菠萝加上步枪的组合就能把敌人留下一半，接下来便是追击就可以了。
<br/>　　但现实却是他根本无法动用贞观炮，甚至连火箭弹都不能用多少，步枪更是不能玩儿什么齐射。
<br/>　　这仗要怎么打，到底要怎么打，思来想去龙义几乎愁白了头发。
<br/>　　最后，还是他的副师长给了他一个建议：“师长，要不咱们干脆让士兵把枪口抬高些算了，这样一来听着动静不小，但是造成的伤害却十分有限。”
<br/>　　“对，对对！”龙义正愁的一个劲拍脑袋，被副师长这么一提醒茅塞顿开，大笑着说道：“咱们就这么办，只用五分之一的力量来进攻，其余人都把子弹打到天上去，这样一来就可以完成老将军交待的任务了！”
<br/>　　有了这样的决定，很快作战任务就被布置了下去，哪个排负责阻敌，哪些排负责放空枪，哪个排负责一边打一边装死，总之用了半天时间才计划明白。
<br/>　　而这个时候，伊力克的部队也已经缀上了九十二师的尾巴，双方只隔着十余里，眼看着能追上他们。
<br/>　　拜占庭人在这个时候几乎都疯了，作为拥有大唐先进装备的‘精锐’，他们没理由看着明尼城的三流部队在他们眼前耀武扬威打胜仗。
<br/>　　追上唐军，将唐军断后的部队彻底吃掉，这才是他们需要的，只要能做到这一点，伊力克就算圆满完成任务，接下来就等着享受胜利果实便好。
