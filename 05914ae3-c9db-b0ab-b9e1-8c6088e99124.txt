　　赤霞洞天、紫薇圣地、九霄圣地、太史世家、云顶圣门、苍云圣宗、星象圣门、日月圣宗、皇祖圣朝、广云天宫、苍海圣殿、紫光圣门、无极圣山。
<br/>　　这便是神武学府背后的十三个大势力，这十三个大势力，分布于古周星以及周围几个星辰之上。
<br/>　　这十三个大势力，每一个在古灵星河之内，都是普通人族势力中顶尖的存在。
<br/>　　十几个大势力联合起来，比之巫族主脉，都不会相差太多。
<br/>　　当然，古灵星河无比浩瀚，各种势力错综复杂，不说一些其他异族，就单是普通之人族之中，像这十三个势力一般的大势力，都有很多。
<br/>　　而神武学府之内，十三个派系也是各有特点，比如紫光圣门派系，这个派系比较吸引一些修炼光属性武学的天才。
<br/>　　再比如皇祖圣朝派系，因为皇祖圣朝是一个国家，这个派系喜欢拉拢一些能征善战之辈，为圣朝开疆扩土。
<br/>　　聂乾说了盏茶的功夫，将他知道的关于十三个派系的情况，全部都告诉了苏莫。
<br/>　　苏莫听完之后，陷入了沉默之中，一时间也不知道那个派系更适合他。
<br/>　　“苏兄，虽然加入派系会陷入各种争斗之中，但其实好处无穷，我和韩盈天赋不高，以前想加入派系，都没有派系愿意要我们！”聂乾脸色落寞的说道。
<br/>　　像他和韩盈这种弟子，只能等到在学府中待满三十年之后，才能有资格加入十三个势力之一。
<br/>　　那个时候，他们修为有所成就，这十三个大势力才愿意接收他们。
<br/>　　“哦，有什么好处？”苏莫好奇的问道。
<br/>　　“苏兄，你有所不知，因为加入了派系，就代表了将来加入那个派系背后的势力，所以能得到那个势力重点培养！”聂乾说道。
<br/>　　“是啊！苏兄，每个派系之中，还经常举行各种试炼、探险等等，好处无穷，修炼的速度比我们这些普通弟子快多了！“韩盈赞同的说道。
<br/>　　“还有这种事？”苏莫闻言愕然，这些他可是从来没有听说过，若真是如此的话，加入派系反而是有百利而无一害了。
<br/>　　“当然，不然为什么那么多人提前加入派系！”聂乾说道。
<br/>　　“苏兄，你看我们所在的南苑，看似弟子不多，只有万人左右，其实南苑有不下五万弟子，只不过那些人多是各大派系的人，平常很少在学府中！”韩盈道。
<br/>　　“原来如此！”苏莫心中了然。
<br/>　　“其实，严格意义上来讲，普通的弟子，只能算是神武学府的外围弟子，加入了派系才算是核心弟子！”韩盈沉声道。
<br/>　　苏莫默认的点了点头，对于两人的话深有感触，好像的确是这样。
<br/>　　神武学府就是为十几个势力招揽人才的地方，加入了他们的派系，他们才会大力的培养。
<br/>　　若是不加入派系，就只能自己独自修炼，将来三十年期满，才能加入其中一个势力。
<br/>　　这种差别是巨大的，因为提前加入派系，受到重点培养，将来进入所在的势力之后，无论是修为实力还是身份地步，都要比后期加入要高的多。
<br/>　　看来自己以前不加入派系，倒是有些孟浪了。
<br/>　　虽然加入派系可能会卷进各大派系的争斗之中，但不加入派系，也不是很安稳。
<br/>　　“嗯，那我就加入一个派系，你们觉得哪个合适？”苏莫点了点头，又征求两人的意见。
<br/>　　“这个苏兄你自己定夺吧！反正都相差不大！”聂乾摇头道。
<br/>　　“哪个势力最富有，机遇最大？“苏莫问道。
<br/>　　“最富有？”
<br/>　　聂乾闻言沉吟了片刻，道：“要说最富有、最有机遇的势力，莫过于皇祖圣朝了，这个国家经常征战，向外扩张，掠夺无尽财富，而且对于拥有战功之人，奖励非常的大！”
<br/>　　“嗯！”苏莫暗暗点了点头，这个皇祖圣朝是十几个势力之中唯一的国家，征战扩张是再说难免的。
<br/>　　征战？
<br/>　　呵呵！他的战魂就是不怕人多，呼吸间灭你百万大军！
<br/>　　苏莫心中暗暗决定，若是这个皇祖圣朝派系的人来拉拢他，他就直接加入。
<br/>　　聂乾、韩盈与苏莫聊了片刻之后，便纷纷返回了自己的洞府中。
<br/>　　两人伤势未愈，均是回去疗伤了。
<br/>　　苏莫独自坐在洞府中，静静的沉思，神武学府也并不是久留之地，不知道什么时候，巫族的人可能就会找上们来。
<br/>　　他要尽快的提升实力，加入派系倒也不错，到时候就算巫族之人找来，他也有了靠山，也比较好应对。
<br/>　　随即，苏莫开始在洞府中整理战利品。
<br/>　　天冥星一役，他得到了大量的战利品，都还没有来得及整理。
<br/>　　足足耗费了一个时辰，苏莫才将身上的战利品整理完毕，财富非常的惊人。
<br/>　　下品灵晶足有百余万，极品灵石更是说不清了，万亿都不止，更低级的上品灵石同样是天文数字。
<br/>　　各种兵器宝物、功法武技等，亦是数之不尽。
<br/>　　当然，大多都是低级的宝物和武学秘籍，帝级以上的连百分之一都没有。
<br/>　　即便如此，也已经非常的可观了。
<br/>　　毕竟，他还分给了聂乾和韩盈，以及苍穹神宫一小部分。
<br/>　　片刻之后，整理完战利品，苏莫便抛却心中杂念，开始了修炼。
<br/>　　……
<br/>　　内府，距离火元峰较远的地方，一座比火元峰小很多的山峰上。
<br/>　　山上建筑不多，只有寥寥几座宫殿。
<br/>　　其中一座宫殿中，大殿中端坐着百余名年轻人，尤天韩亦是在其中。
<br/>　　这些人个个身上气息浩大，弥漫着尊者的气息，尤天韩的修为在这些人中，也只是垫底的存在。
<br/>　　而在中央主座之上，端坐着一名身穿银色长袍的青年，这名青年很高，足有八尺出头。
<br/>　　青年脸颊欣长，眼眸开合间给人一种锐利如鹰之感。
<br/>　　此人年级轻轻，便已经颇有一种上位者的感觉。
<br/>　　“诸位，太史轩被一个叫苏莫的人杀了，此事你们都知晓了吧？”银袍青年目光环视众人，面无表情的说道。
<br/>　　“圣子，该如何做，你吩咐吧！”一名青年说道。
<br/>　　主座上的银袍青年，正是太史世家派系的圣子，神武学府十八位圣子之一，太史天意！
<br/>　　“敢杀我太史世家的嫡系子弟，简直是自寻死路，我的要求只有一个字，杀！”太史天意微眯着眼眸，口中言语冰冷至极。
<br/>　　此事，苏莫必须要死，不然的话，太史世家派系，将成为其它派系的笑柄。
<br/>　　毕竟，太史轩可不是普通的太史世家派系的人，而是太史世家嫡系子弟。
<br/>　　苏莫敢杀太史轩，那是在打太史世家派系的脸，在打他太史天意的脸！
