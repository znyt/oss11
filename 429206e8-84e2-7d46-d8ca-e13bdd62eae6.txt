　　“咳咳，这两位是学院的特招生，他们天赋不错，但错过了第一轮比试选拔，我名下正好有两个名额，就给了他们。”
<br/>　　琥乾干咳一声，瞥了一眼新生前五，随后目光落在慕清霄与小医仙身上。
<br/>　　“特招生？”
<br/>　　闻言，两位长老诧异的目光落在两人身上，随后看向琥乾，笑骂道：“原来如此，你这老家伙也怕丢脸！”
<br/>　　另一名老者微微点头，道：“不过，在这样的年龄，就快要追上外院导师，潜力与往年的天才也不差，确实值得培养。”
<br/>　　站在一旁，听闻他们的对话，慕清霄隐隐的听出一些端倪。
<br/>　　看来，无论是在何处，每一届新生实力与表现，都关乎着学校领导的面子，迦南学院亦是如此。
<br/>　　而这一届的新生，琥乾并不是很满意，正巧碰到慕清霄与小医仙，于是就让两人跳过比试选拔，直接进入新生行列中。
<br/>　　慕清霄展现出的八星斗师，小医仙是七星斗师，放在新生行列中，绝对是拔尖的存在，哪怕在内院，也不至于垫底。
<br/>　　“哇塞，真是漂亮的女孩子，看来内院美女排名要变化一番了。”
<br/>　　“是啊，我也是第一次如此貌美的女孩子，感觉内院美女，跟她比起来，差好多啊！”
<br/>　　在小医仙出现后，令得老者身后，将近二十名学员，眼中都闪过一丝惊艳，纷纷吹起口哨，嘴皮子上挑逗起来，整就一副流氓的模样。
<br/>　　“都给我闭嘴，谁在多说一句屁话，扣除半个月的塔修时间。”
<br/>　　苏长老厉喝一声，身后嬉皮笑脸的年轻学员，立马闭口不言，显然，所谓的塔修时间，对他们来说，都是极为宝贵的东西。
<br/>　　慕清霄眼底倒是掠过一丝精芒，塔修，指的恐怕就是他的目标，天焚炼气塔吧？
<br/>　　训斥过后，苏长老微微一笑，道：“你们都是新生，我希望你们能尽量适应这里，那样你们会获得极大的好处。”
<br/>　　“看我身后这群混蛋，往年他们虽通过了外院选拔赛，却也仅仅是排名末尾的存在，可如今只过去一年的时间，我敢保证，他们的实力，放在你们队伍里，绝对能碾压你们。”
<br/>　　说话的同时，苏长老不经意的瞥了一眼慕清霄和小医仙，暗道：“当然，除了这小子和这小姑娘以外。”
<br/>　　“行了，废话也不多说，虽然你们已经通过了选拔赛，获得了进入内院的资格，但还没有结束。”
<br/>　　苏长老微微一笑，手掌一扬，一把漆黑色晶片，便是出现在了手中，随手一掷，晶片射出，犹如长了眼睛一般，悬浮在全体新生的面前。
<br/>　　将晶片拿在手中，慕清霄顿时感觉到一股奇异的能量自其中渗出，在晶片上有一个透明的镜面，其上印着一个清晰的数字。
<br/>　　“五!”
<br/>　　在苏长老将晶片发放出去的时候，身后将近二十名老生满是火热，丝毫不掩饰眼中的垂涎之色。
<br/>　　“有些东西解释起来挺复杂的，你们现在只需要知道一件事情，那就是，你们手中的晶片极其宝贵。”
<br/>　　说着，苏长老指了指自己身后，将近二十名老生，道：“等你们正式进入内院就能明白，你们看看我身后这帮牲口的眼神就知道了。”
<br/>　　看到老生们的眼神，新生觉得有些不对劲，心中满是警惕，迅速将晶片收了起来。
<br/>　　“接下来，你们需要进入森林，然后，成功抵达森林尽头的内院。”
<br/>　　同时，苏长老又指了指身后的老生，语气平淡，道：“森林中没有魔兽，但你们要小心他们，按照内院的规定，在森林中，不限制出手，也就是说，他们可以攻击你们。”
<br/>　　“记住，你们手中晶片上的数字，对他们有着莫大的诱惑力，所以，他们会除了下杀手外，用尽一切手段，获得你们晶片上的‘火能’。”
<br/>　　见新生露出困惑之色，苏长老耐着性子，道：“火能，你们以后会理解它的宝贵，现在，你们只需要躲避老生的追击，或则打败他们，只要不被他们抓住，顺利抵达内院，那么就能获得最后的奖励，越早达到，奖励就越丰富。”
<br/>　　苏长老目光在全体新生身上扫过，指向背后的森林。
<br/>　　“对于森林抢夺赛，我们内院，称为‘火能猎捕赛’，现在我宣布，这一届‘火能猎捕赛’，正式开始，新生们，开始逃亡吧。”
<br/>　　......
<br/>　　......
<br/>　　茂密的森林中，光线自树叶缝隙间洒落而下，星星点点的光斑照射在布满枯叶的地面上，将整个林间，衬托的略显梦幻。

<br/>　　伸手揉了揉她的小脑袋，慕清霄道：“已经无碍，何况只是对付一些学生，刚才长老说的火能，我们也要收集一点，对于你有不少益处。”

<br/>　　话语声刚落，林间就响起一道戏虐声：“运气真不错，又遇到一队，咦？这不是刚才，容貌极为漂亮的女孩子吗？”

<br/>　　慕清霄脸上浮现一个古怪的笑意，道：“森林挺大的，你们可真不走运。”
<br/>　　闻言，五名老生皆是一愣，不明所以。
<br/>　　回过神来，站在中间的男子脸色一沉，道：“小子，你挺狂的啊，身边还跟着个美女，难怪前面两支队伍中的新生，对你骂骂咧咧。”
