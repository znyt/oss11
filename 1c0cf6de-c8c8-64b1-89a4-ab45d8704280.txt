　　段江伟和洛冥心头迫切，不等直升机找到地方停稳就跳下了机舱，周围几乎没有空地，都被躺着的人占满。
<br/>　　当两人走下来之后更近距离的观看，才确认这些躺着的几万人，正是狼牙战团的成员无疑。
<br/>　　只是所有人都躺在地上，周围场景诡异的可怕，一片死寂中唯有直升机螺旋桨的声音在减小。
<br/>　　段江伟不断环顾四周，想要找到一两个他认识的人，可是人实在太多满地都是，根本无处下手。
<br/>　　“这是死了？还是？”洛冥忍不住出声。
<br/>　　是不是死了？还是昏迷？为何会这么诡异的躺在这里一动不动？
<br/>　　这是两人心中·共同的想法，可就在洛冥的声音响起后不久，远处的直升机终于找到一小块平地降落，与之同时在半空中划出呼呼风声的螺旋桨声音，也就此消失。
<br/>　　没有了螺旋桨的‘呼呼’声，这片区域就真的万籁空寂，死一般的寂静让洛冥和段江伟心头发懵。
<br/>　　只是这时候，忽的段江伟耳朵一动，猛然转身看向身后。
<br/>　　这一动作把洛冥吓了一跳：“怎么了？”
<br/>　　“有声音！”段江伟瞪大双目，表情有些呆傻的开口：“好像是呼噜声。”
<br/>　　洛冥愣住，第一反应就是自己听错了，第二反应则是段江伟听错了。
<br/>　　怎么可能会有呼噜声？
<br/>　　难不成有人在这里睡觉？
<br/>　　还睡的这么死打起了呼噜？
<br/>　　可就在洛冥根本不信的情绪刚刚升起之时，四周就渐渐响起了一阵阵呼噜声，接二连三又此起彼伏。
<br/>　　之前在高空他们不可能听得到这呼噜声，下来后又有直升机的螺旋桨声音在吵，便将这些呼噜声都盖了过去。
<br/>　　当什么声音都没有，这些熟睡而发出的自然呼噜声，便一下子明显起来！
<br/>　　站在一片躺尸中的洛冥和段江伟两人当场石化，不可置信的情绪被无限放大，对视中的两人更是从对方的眼中看到了同样的神情。
<br/>　　有没有搞错？
<br/>　　不会是他们想的那样吧？
<br/>　　“找！找他们长官找出来！”洛冥铁青着脸下令。
<br/>　　段江伟连忙将自己八阶的五感放到最大，在上万名狼牙战团成员中寻找人，周围躺着的这些普通士兵已经已经没必要叫醒询问，至少也得找个少将出来问话。
<br/>　　睡觉？
<br/>　　南署基地大战结束后，狼牙战团的人竟然在这里集体睡觉？
<br/>　　我靠！
<br/>　　两人从一开始的被战役浩大的场景所震撼，到找不到活人时的无尽担忧，此时发现不对劲后，便心情糟糕又暴躁到了极点。
<br/>　　狼牙战团果然就和他们的长官一样不着调！
<br/>　　这种创世纪大战结束后全体躺地上睡觉的事，也就他们才做得出来！
<br/>　　段江伟和洛冥花了很长时间，才从极难找到的地面缝隙中走到前方，实在是太拥挤了，这些狼牙战团的成员睡成一片几乎都挨在一起，根本让他们难以下脚。
<br/>　　好不容易穿过这几万人的队伍之后，两人都有种难以言喻的心累感。
<br/>　　“这些人我认得，是侦查队的。”段江伟指着旁边睡的跟死猪一样的几百人开口。
<br/>　　洛冥点点头：“呼噜声都大到震天了，越往前走呼噜声越大，看来前面才是核心成员，但这些老兵都什么情况，怎么睡觉呼噜声这么大？”
<br/>　　段江伟想了想，解释道：“人在疲惫的时候往往会打呼噜。”
<br/>　　洛冥沉默不语，心中也明白这场大战让狼牙战团的成员精疲力竭。
<br/>　　两人继续往前走，中途也看到了睡成了一个大字型的蒋天庆。
<br/>　　“这是狼牙中将，侦查队的队长蒋天庆。”段江伟认出了蒋天庆，朝着洛冥问道：“要不要叫醒？”
<br/>　　洛冥摆摆手：“让这孩子再睡会儿，据我所知狼牙的每场大战，蒋天庆最少也是三天三夜不睡觉？”
<br/>　　“是。”段江伟点头：“侦查队负责的任务往往需要二十四小时时刻紧盯，蒋天庆也总是透支工作，南署基地的这场大尸潮战役，侦查队恐怕全员都没睡过觉。”
<br/>　　洛冥心中的情绪微微激荡，走到这里看过了这么多狼牙成员的睡相，从他们疲惫的表情和都来不及清洗整理的衣着，不难想象这场大战他们有多全力以赴，以至于连大战结束后汇报的精力都没有，就这么就地而睡。
<br/>　　“继续往前走，把楚涵那小子找出来，他皮厚肉糙经得起折腾。”洛冥刻意黑着脸道。
<br/>　　段江伟微不可查的嘴角勾了勾，跨过侦查队继续往前。
<br/>　　紧接着他们就看到了风格明显的射击队，每人手中都握着一把枪，身上各处能放东西的地方都是弹夹，射击队与侦查队一样，躺在那里死睡对身边有人毫无所觉。
<br/>　　大战结束，南署基地已无一只丧尸，在知道没有危险的时候，他们也刚躺下还没多久，正处于深度睡眠阶段，一时半会儿还真难叫得醒。
<br/>　　“那是陈少爷中将。”段江伟指着前方不远处躺尸的大胖子，嘴角抽·搐着开口：“这家伙呼噜声最大，竟然还流口水。”
<br/>　　洛冥环顾了一圈四周，指着这些射击队成员，眨了眨眼睛不解问道：“侦查队睡成那样可以理解，陈少爷能力逆天责任也大，我也能理解，可射击队其他人怎么也睡成这幅样子？看他们的衣着最干净，应该没有靠近过丧尸群。”
<br/>　　段江伟想了想，俯下·身轻轻拉了拉身旁一名射击队成员手中的枪，却发现根本拔不出来。
<br/>　　见到这一幕的洛冥瞳孔一缩：“睡觉都不放手，抓这么紧。”
<br/>　　段江伟笑了笑，笑容里有着某种敬佩情绪：“据我所知这场大战中，楚涵让狼牙工程队改造了南署基地，新造了几百个射击高台，就是周围的这些柱子。”
<br/>　　听着这番话，洛冥立即明白了过来：“你是说这七天大战时期，射击队的成员都守着自己的射击地点，不眠不休时刻准备着开枪？”
<br/>　　“应该是，具体的大战过程还需要事后才知道。”段江伟点头，道：“但这些射击队成员每个人都视枪如命，哪怕睡的没知觉都不会松开紧握枪支的双手，可见……”
<br/>　　接下来的话段江伟没有细说，但洛冥怎么可能不明白，他表情渐渐变得认真严肃：“继续往前。”
<br/>　　“是。”
