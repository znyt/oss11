　　“对，我是怕”涂小安朝着台下人群开口。
<br/>　　很多人闻言便一愣，脸上得意起来：“你们看，最强新生承认自己怕了，他怂了”
<br/>　　涂小安嘴角微勾，一双妖治的眼眸朝着人群横扫过去，每一个接触他眼神的人，心脏猛然被一揪，一股莫名死亡的恐惧感侵袭了每一个人的心。
<br/>　　好可怕的眼神，不带一丝感情，那一霎那，他们感觉自己被天地间最可怕的毒蛇盯上。
<br/>　　心有余悸的时候，只见台上的那个清瘦身影，腾空悬浮了起来，立身天地之间般的含词轻吐：“我怕自己打死他”
<br/>　　呃...所有人猛然瞪大双眼，一脸懵逼，这个人？？？。
<br/>　　飞起来了？他既然能如大神学长们一样的凌空而立，这本身就代表了超强的实力。
<br/>　　都知道，未入三阶，任何生物都不具备飞行之能，但是人类修武者，有功法辅助，很多天才，虽然还达到三阶，但也能借助不少手段做到这一点。
<br/>　　但古学院，真正能做到凌空而立的，恐怕双手双脚也数得过来，都是一等一的天才高手。
<br/>　　他悬浮起来，说的话，让人下意识的不由相信了。
<br/>　　凌岩航看到涂小安凌空悬浮，表情难看起来，听他说的话，更是差点没一口老血吐出来，这话将自己置于何地，藐视，赤果果的藐视自己。
<br/>　　在怎么说，他也是天才班的学员，是新生的学长。
<br/>　　人争一口气，佛争一炷香，凌岩航脸上扭曲了起来：“休要口出狂言，有本事就重新比过，我到要看看你怎么打死我”
<br/>　　“好，那我现在就给你机会，希望你不要后悔”
<br/>　　涂小安悬浮凌空，足足高了凌岩航一个人的身高，俯视而下。
<br/>　　他跟凌岩航没有交际，本也不打算伤这个人，让他认输就算了，可惜呀，对方不识抬举。
<br/>　　他意念一动，贴在凌岩航喉眼的飞刀骤然回到了他的手上，把玩起来，道：“凌岩航，你说我的飞刀是暗器，那么这次我不用飞刀”
<br/>　　“不过，这次，我就不会对你手下留情了”
<br/>　　涂小安高高在上，微微抬着下巴，眉峰间仿佛郁结着化不开的冰霜，眼神似是冷淡无波又似是含着高高在上的鄙夷。
<br/>　　人要是没用自知之明，就是找死，刚才他都暴露了掌控雷电的能力了，就算没有飞刀，他拿什么胜自己。
<br/>　　这种人，是利益熏心了，光想着不想被淘汰，却不知道自己几斤几两，难道他还将自己当成往年的一些新生？
<br/>　　贴在喉咙处的飞刀没了，凌岩航整个人轻松起来，也随之大喜，机会果真被自己争取来了。
<br/>　　“小学弟，刚才是我大意，这一次，你就算在用飞刀，我也不会让你得逞，为了公平，等下你败了，我也同样给你一次机会”凌岩航不想被人落下话柄，故意说了这样一句话。
<br/>　　给我机会？
<br/>　　涂小安看白痴一样的看着他，他脑子秀逗了？
<br/>　　既然掌控雷电的能力都暴露了，那么涂小安也就不客气了，意念一动，周身电光浮现，有银蛇在他身边窜动，噼里啪啦的声音不绝于耳。
<br/>　　“你...”凌岩航看到这一幕，浑身都鸡皮疙瘩了起来。
<br/>　　忘记了，这小子能掌控雷电，是个元素属性能力者，这种人是最可怕的。
<br/>　　“你是要能扛住我一道雷，就算我输怎么样”
<br/>　　涂小安冷淡无波的看着紧张起来的凌岩航。
<br/>　　“你不要太狂，我抗你一道雷算什么，尽管来吧”凌岩航对自己的身体还是有自信的。
<br/>　　刚才几个大神学长凌空，被他的雷劈中，虽然落了下来，但一点事情都没有。
<br/>　　这说明他释放的雷电，威力并不算强。
<br/>　　几个念头，凌岩航硬气足了起来，想吓唬自己？
<br/>　　元素属性能力是可怕，但是也要看境界的。
<br/>　　一阶的修武者要是释放雷电，二阶的高手站着给你劈，也不会有多少伤害。
<br/>　　“我就站着这里不动，吃你一道雷，但要是我扛住了，你是不是...”
<br/>　　涂小安道：“我立刻认输”
<br/>　　“好，一言为定”凌岩航一口笃定下来，还生怕涂小安会后悔。
<br/>　　这个最强新生有多少实力，凌岩航心里还真的没底，最少他刚才的飞刀绝技，就能让自己束手无策。
<br/>　　现在这种打赌，显然是对自己最有利的。
<br/>　　抗你一道雷，我堂堂二阶六段的修武者，还抗不住你一道雷吗，笑话。
<br/>　　“大家快看，最强新生好可怕呀，全身都是电光流转”
<br/>　　台下的一个个人都看待了，涂小安现在悬空而立，周身有电光流转，银蛇窜动，整个人看上去好似天神下方，雷公附体。
<br/>　　“蛇精，不可取他性命”
<br/>　　忽然，涂小安的脑海中出现了副院长苍老的声音。
<br/>　　“老头，是他非逼我出手，我这次要是下手轻了，都感觉对不住他”涂小安冷冷的声音传过去。
<br/>　　副院长道：“你们两个的实力不在一个档次上，你若全力出手，他必死无疑，你别忘记了我们的约定，不可在古学院杀人，否则我就将你赶出去”
<br/>　　“老头，你在威胁我吗，你以为本蛇王很稀罕你的古学院，比武台上，生死由命，刀剑无眼，就算死了，也只能怪自己实力不济，你们古学院比武难道没死过人？”
<br/>　　这个世界是弱肉强食的，尤其是修武界，哪怕是古学院，每一年都有不少学员因为比斗而丢了性命，这是非常正常的。
<br/>　　只不过，涂小安是条蛇精，副院长怎么能让学员死在一条蛇精的手中。
<br/>　　但他也知道，这件事情怪不得蛇精了，是凌岩航咎由自取，他都拦不住。
<br/>　　“你全当给老夫一个面前，只要留他性命就可，别的随你”副院长苍老的声音传过去，算是退了一步。
<br/>　　“二十颗灵石，我留这个人一命”
<br/>　　涂小安的脾气早不是最开始那么好了，以前他不愿意杀人类，但自从吞吃了一个人类修武者之后。
<br/>　　早就不算什么了，该救的人，他救过，该杀的人，他杀过。
<br/>　　这种非要撞上来送死的，他都不好意思拒绝。
<br/>　　听到二十颗灵石，副院长顿时吹鼻子瞪眼：“你这是敲诈”
<br/>　　“那随便你，最多我杀了这个人，就离开古学院，你们也拦不住我，要是惹怒了本蛇王，信不信我让你这个古学院变成一座蛇窟”
<br/>　　此话一出，沉默数秒，副院长道：“十颗，我给你十颗灵石”
<br/>　　这老头看来真的是有什么事情有求于自己，这都退让?
<br/>　　涂小安其实就是想试一下他的底线，毕竟明知道自己是条蛇精，还任意他在古学院，这点，本身就说不过去。
<br/>　　“一口价，十二颗”
<br/>　　“好，老夫答应你”
<br/>　　涂小安闻言就笑了，还真的被自己敲诈成功了。
<br/>　　凌岩航这种货色，涂小安根本没放在眼中，杀与不杀，就看心情。
<br/>　　为了十二颗灵石，那就给他来一个半身不遂优惠套餐吧。
