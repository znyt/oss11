　　微微犹豫了一下，刘哔后退了几步，默默拉开了自身和棺材的距离。
<br/>　　他可是知道，这个世界上真的会有鬼魂存在，不管事件再怎么变动，大体的世界观应该还是一致的。
<br/>　　对于这种不能用常理度之的东西，他心中还是抱有一些敬畏感。
<br/>　　“松绳，起钉。”站在棺材旁边，九叔抬头看了看天色。
<br/>　　紧接着，九叔转过身来，对着众人说道：“诸位，今天是任公威勇重见天日，凡年龄三十六，二十二，三十五，还有四十八，属鸡属牛者，一律转身回避。”
<br/>　　听到九叔的吩咐，刘哔默默转过了身体，倒不是他符合条件，而是他吃不准自己的年龄到底有多大。
<br/>　　其实主要原因，他是不知道九叔说的是实岁还是虚岁，如果按照实岁来说，他年龄够了，但是按照虚岁来说，他又不符合了。
<br/>　　故此，为了避免意外发生，他还是转过了身去，虽然他也不知道这到底有什么讲究。
<br/>　　“回避完毕，大家整理衣冠，开馆。”稍等了片刻，等听到九叔的话语之后，刘哔方才转过身来，继续观察者场中的情况。
<br/>　　不知是不是巧合，一阵尖锐的叫声响起，刘哔皱了皱眉头，扭头看去，从一旁的树林中飞出了一片乌鸦。
<br/>　　这可不是一个好的兆头，九叔脸上也带上了一层忧虑，和众人一起来到了棺前。
<br/>　　施工队缓缓打开了棺材盖，一个身着古代官服的老人露了出来，一股肉眼可见的黑气，从棺材中飘了出来。
<br/>　　随后，一阵哭嚎声响起，任发和任婷婷跪在了地上，对棺中老人祭拜。
<br/>　　刘哔站在一旁，眉头深深皱起，他见过死人，也见过死去的活人，但是像现在这么诡异的情况，他还是第一次见到。
<br/>　　棺木中，老人的尸首平躺着，脸上满是皱褶，虽然是一张在正常不过的脸庞，却没有寻常老人的那种慈祥，反倒给人一种凶戾无比的感觉。
<br/>　　奇怪的是，死去了这么久的时间，这具尸体居然没有丝毫腐烂，而九叔和刘哔，都是发现了这一点，脸上的表情，颇为不解。
<br/>　　“九叔，这墓穴还能用吗？”跪拜完毕，任发从地上爬起，满脸心疼的向九叔问道。
<br/>　　毕竟这也是难得一见的好穴，如果就这么放弃掉了，也着实可惜，他原本的想法，是将自家老爷子再重新给安回去。
<br/>　　“蜻蜓点水，一点再点，肯定不会点在同一个地位上，这个穴，没用了。”轻叹一声，九叔摇了摇头。
<br/>　　看着已经被打开的棺材，任发满脸惊慌，有些着急的开口：“那怎么办呢？”
<br/>　　“我提议，就地火化。”看了一眼任发，九叔给出了自己的建议。
<br/>　　目前为止，火化确实是最为稳妥快捷的方式，但出乎意料的是，任发却果断拒绝了。
<br/>　　“火化？不行，生父生前最怕的就是火，我不能这样做。”低了低头，任发一脸的为难。
<br/>　　撇了撇嘴，刘哔脸上带起一抹冷笑，他当然知道任发不可能将尸体火化，因为那个尸体身上，可是藏着不知道什么东西。
<br/>　　只是信封上的内容太过于含糊，他也不知道具体是什么，只知道，是某些不得了的大人物，所需要的一种东西罢了。
<br/>　　“任老爷，不火化会有麻烦的。”九叔有些无奈，他不知道里面的事情，倒也真的相信了任发的借口。
<br/>　　但任发当然不会同意火化，对着九叔连连摇头：“怎么都行，就是不能火化，你想想其他办法吧。”
<br/>　　看着固执的任发，九叔也只能放弃了一开始的打算，道：“好吧，那就暂且寄存在我们义庄，到明天我会帮老太爷另外找一个墓穴，让他早些安息。”
<br/>　　这次，任发倒是也没有反对，目前看来，这是唯一的解决办法，也和他原本计划中的差不许多。
<br/>　　“好，先盖上棺盖，抬到义庄。”见事情已经定下，任发身边那个带眼睛的年轻人，挥了挥手，对施工队指挥道。
<br/>　　随后，告别一声，任发和任婷婷两人都坐上轿子离开了，旁边施工队，也扛起棺材，下了山去。
<br/>　　九叔见众人走的差不多了，转过身来，向三人吩咐道：“你们就在墓穴点个梅花香阵，烧成什么样回来告诉我。”
<br/>　　三人点了点头，应了下来，九叔看了一眼，还是有些不太放心的嘱咐道：“每个坟头都要上香啊。”
<br/>　　说罢，不等三人回应，就走下了山去，他还要去找一下任发，有些事情要询问他一下。
<br/>　　目送着九叔下了山，三人来到墓穴旁边，文才点燃了一把香，三根一组，在地上摆了一个梅花的形状。
<br/>　　刘哔在一旁饶有兴致的查看着，待梅花成型，方才开口，向文才询问道：“师兄，这个梅花香阵，到底有什么作用啊？”
<br/>　　听到刘哔的问题，文才头都没抬，手中握起一把土，散在了梅花阵周边，待一切完成后，方才站起身来，拍了拍手，向刘哔解释了起来。
<br/>　　“其实我也不太理解，这是师父教我的，说可以检测到坟中是不是有鬼魂或者僵尸。”停顿了提下，文才将手上的香放置在一旁。
<br/>　　“如果坟内有鬼呢，香会燃烧的特别快，因为这代表有鬼魂在吃香，如果梅花心和梅花瓣燃烧的速度不同，说明有外鬼分香，需要驱赶，如果香烧的特别慢，甚至灭掉，估计坟里的尸体就要变成僵尸了。”一边指着地上的阵法，文才一边细心的向刘哔解释道。
<br/>　　此时，一阵‘噼啪’的声音响起，文才话语一顿，地上所有的香都在同一时间燃的飞快，片刻之后，悉数灭掉。
<br/>　　“额，这个怎么解释？”刘哔有些疑惑，不知道这代表着什么。
<br/>　　“我也不知道，第一次碰见这种情况，师兄可能知道怎么回事。”挠了挠头，文才脸色有些难看，细心的拔起一束香，拿在了手里。
<br/>　　说罢，两人向一旁正在上香的秋生走去，但刚站起身来，文才和秋生两人就撞了一个满怀。
<br/>　　“快，看看，怎么会烧成这样？”文才拿着三炷香，不解的问道。
<br/>　　接过香查看了一眼，秋生拉起两人，一脸惊慌的说道：“走，快去告诉师父！”
<br/>　　说罢，一马当先的向山下跑去，刘哔不解的看了一眼坟头位置，顺从的被秋生给拉下了山。
