　　进入了界障裂缝之中，苏莫只感觉天旋地转，周身尽是混乱的气流。
<br/>　　他努力稳住身形，固守本心，急速的在这片昏暗的空间之中穿梭。
<br/>　　界障，并不是普通的阵法气罩，不仅隔绝了宇宙和神域，更是蕴含通天彻底的玄妙。
<br/>　　没有方向，精神力无法离体感知，苏莫只能不断的飞行。
<br/>　　突破到虚神境三变，战魂化为元神，精神灵识，已经化为了神识，又被称为是神念。
<br/>　　但是，即便是神念，在这片昏暗的空间之中，也无法探测出任何东西。
<br/>　　在苏莫想来，界障，不仅隔绝神域和宇宙，更担负着一种能量转化的能力。
<br/>　　毕竟，神域，是靠九十九万九千九百个阵基，吸收宇宙星辰的力量，那么，吸收的这些灵气，肯定要进行转化。
<br/>　　不然，就算神域灵气再浓郁，也只能算是上好的修炼之地而已，与神并不搭边。
<br/>　　苏莫不断的飞行，不知道过了几个时辰之后，他前方的气流剧烈的沸腾了起来。
<br/>　　恐怖的气流将他包裹，飞速的转动，仿佛星河倒卷，一片混乱。
<br/>　　苏莫根本稳不住身形，即便他修为高达虚神境三变，也被转的晕头转向，脑中一片浑浑噩噩。
<br/>　　嘭！
<br/>　　不知道过了多久，苏莫只感觉自己的身体，接触到了大地，静止了下来。
<br/>　　“这是哪里？”
<br/>　　苏莫甩了甩脑袋，待脑中渐渐清明，他缓缓的站了起来。
<br/>　　抬头看去，这不是荒郊野外，而是在一座古朴的大殿之中。
<br/>　　大殿之中，狂风呼啸，气流滚滚，遮蔽了视线，仿佛是空间风暴，在不断的席卷。
<br/>　　“呵呵，几千年了，终于有人来了！”
<br/>　　就在此刻，一声轻笑声突然响起，苏莫脑中还没彻底清醒过来，一根黑色的长鞭，如同灵蛇一般，从前方飞射而来，立刻缠住了他的身体。
<br/>　　苏莫心中一惊，下一刻，长鞭一拉，直接将他拽了出去。
<br/>　　嗖！
<br/>　　苏莫被拽到了大殿之外，眼前豁然开朗，轻呼一口气，都感觉神情气爽。
<br/>　　只见此刻，他身处在一片连绵的建筑群中，周围有一座座阁楼耸立。
<br/>　　而此时，他的身前，伫立着一道年轻的身影。
<br/>　　此人，身穿黄色锦袍，身形微胖，脸庞圆润，一双小眼精光四溢。
<br/>　　苏莫在打量着微胖黄衣青年，对方也在打量着他。
<br/>　　“虚神境三变，你是怎么来的？”微胖青年眸中露出惊讶之色，诧异的问道。
<br/>　　众所周知，只有超越了虚神之境，才有可能从下界来到神域。
<br/>　　而眼前的青年，不过才虚神境三变，这怎么可能？
<br/>　　微胖青年心中惊疑不定，他管理这个飞升殿几千年了，目前只是接引到第一个飞升者，没想到居然只有虚神境三变的修为。
<br/>　　“我……我是在别人的帮助之下，才来到此地！”苏莫略一迟疑，撒了个小谎。
<br/>　　眼前之人，貌似是人族，但是修为深不可测，无法窥探，苏莫不得不小心。
<br/>　　“别人的帮助？”微胖青年闻言，有些不太相信，不过他不了解下界，也没有深究。
<br/>　　随即，他再次打量了一番苏莫，问道：“你的身上，有混沌之力的气息，你修炼了混沌族的功法？”
<br/>　　“在下修炼了一些浅薄的混沌之力，与混沌族好像并无关系！”苏莫向对方抱拳，微笑着说道，他可不会承认自己是混沌族，毕竟，神域的格局他并不了解。
<br/>　　“原来如此！”
<br/>　　微胖青年点了点头，随即朗声道：“你既然不是混沌族之人，那可愿意加入我大夏神王殿？”
<br/>　　“大夏神王殿？”苏莫闻言一怔，面带迟疑之色。
<br/>　　他刚来到此地，什么情况都不了解，哪里能随意加入某个势力！
<br/>　　“你没有选择，你既然是人族，又从我大夏界域的飞升殿来到神域，必须加入我大夏神王殿！”微胖青年面色一板，威严的说道。
<br/>　　飞升殿，在每一个界域和神朝都有，下界之人来到神域，会受到飞升殿的牵引，至于从哪里的飞升殿出来，并没有定律。
<br/>　　不过，在神域，有不成文的规定，无论哪个势力，若是接引到其它种族之人，要将飞升者送到所在的种族。
<br/>　　比如，苏莫若说自己是混沌族，便会被送到混沌神朝。
<br/>　　再比如，苏莫若是魔族，会被送到亘古魔国。
<br/>　　而现在，苏莫是人族，又说与混沌族无关，微胖青年自然要将苏莫留下来。
<br/>　　毕竟，众所周知，只要是能从下界来到神域之人，均是下界的顶级妖孽。
<br/>　　大夏神王也曾经吩咐过，若是有人族飞升者，要尽量将其留下，所以微胖青年，自然要邀请苏莫加入。
<br/>　　“阁下，我刚到神域，什么都不了解，你总该告诉我，大夏神王殿是什么势力，才好让我加入吧？”苏莫面色郑重的说道。
<br/>　　“说的也是！”
<br/>　　微胖青年略一沉吟，点了点头，继续道：“你跟我来吧？”
<br/>　　言罢，微胖青年转身向外走去，苏莫见此，无奈的跟了上去。
<br/>　　一边行走，苏莫也在一边查看这方神域的天地。
<br/>　　此地，天穹之上有九个太阳，每一个都光芒刺目，关辉无限。
<br/>　　此地的空间非常的凝固，其坚实的程度，几乎是天荒星域的万倍都不止。
<br/>　　就算是以苏莫的修为，行走之间，都能感觉到空间的阻力。
<br/>　　空气之中，飘荡着淡淡的源气，只不过，这源气较为驳杂。
<br/>　　“如此坚固的空间，怕是飞行都很困难！”苏莫面色凝重，心中暗道。
<br/>　　不过，他对此并无意外。
<br/>　　毕竟，这是神域，若是空间不坚固，一个普通的神灵，一巴掌都能将神域给毁了。
<br/>　　可以说，这种坚固的空间，是在限制武者的破坏力，亦或者说，是在限制神域所有神灵的破坏力。
<br/>　　苏莫神识一动，离体而出，亦是受到了巨大的阻力。
<br/>　　他的神识，只能探出千丈之远。
<br/>　　这让苏莫有些无语，不过，这些都无所谓了，毕竟，这空间会影响所有人，不只是他一个。
<br/>　　微胖青年带着苏莫，很快便走进了旁边的一座阁楼之中。
<br/>　　这阁楼之内，古朴而又苍凉，只有几个蒲团和木椅。
<br/>　　“坐吧！”
<br/>　　微胖青年率先坐下，挥了挥手说道。
<br/>　　苏莫点了点头，便在旁边的木椅上坐了下来。
<br/>　　“你叫什么名字？”微胖青年凝视这苏莫，问道。
<br/>　　“上官昊。”苏莫道，他并未说出真名，毕竟，来到了神域，无时无刻都有被帝一魂找到的风险。
<br/>　　微胖青年点了点头，略一沉吟，沉声说道。“我们大夏神王殿，在天池神朝，算是比较强大的势力，强者无数，是你理想的栖身之地。”
<br/>　　“天池神朝？”苏莫闻言，顿时心中一动，他记得绫裳在玄荒之时曾经说过，对方就在天池神朝。
<br/>　　“不错，神域共有九大神朝，天池神朝乃是地域最广大的神朝，我大夏神王殿，在天池神朝八十一个界域之中，也算是比较强大的势力！”微胖青年点头说道。
<br/>　　“哪九大神朝？”苏莫好奇的问道。
<br/>　　“混沌神朝、亘古魔国、幽冥神朝、天星神朝、霸天神朝、不死神朝、天龙神朝、轮回神国、天池神朝。”
<br/>　　微胖青年徐徐说道，他并未隐瞒，这些众所周知的事情，苏莫日后都能轻易知晓的。
