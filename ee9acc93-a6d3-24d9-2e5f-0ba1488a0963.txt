        那估计是傅夜沉派在她连星儿身边的几个贴身保镖吧！
<br/>
<br/>        明司寒忽然发现，那几个保镖并不是二十四小时都跟在这丫头身边。
<br/>
<br/>        “喂，你在想什么？”连星儿见明司寒凝眉沉思，于是抬起手来在他眼前晃了晃。
<br/>
<br/>        明司寒回过神来，看着连星儿会心一笑：“你还挺能跑。”
<br/>
<br/>        “谁说我挺能跑呢？再跑下去，我都快要缺氧窒息了！”连星儿一边喘着气一边说道，摇摆着自己另一只手的时候，才发现自己还牵着明司寒的手。
<br/>
<br/>        倏地，她的双颊微微泛红，连忙松了手往后背去。
<br/>
<br/>        明司寒看了一眼自己刚刚被她牵过的手，还有她手心里的余温缱绻，嘴角情不自禁地微微上扬。
<br/>
<br/>        “要不，我们走后门回学校？”明司寒淡淡地问。
<br/>
<br/>        连星儿点了下头。
<br/>
<br/>        两人并肩走在了一起，气氛有点儿怪异。
<br/>
<br/>        连星儿受不了这种不明所以然的气氛，开始有一搭没一搭地说道：“我还以为学校里的男生都怕你哦。”
<br/>
<br/>        “嗯，也有不怕死的。”明司寒云淡风轻地回答。
<br/>
<br/>        连星儿随后又没下文了。
<br/>
<br/>        回到学校里后，连星儿要赶着去集合，明司寒将那一袋子的饮品递给了她，在她赶时间要走之前，他说了一句：“晚上，三教一楼105自习室见。”
<br/>
<br/>        “好！”连星儿眯眼一笑，爽快地便答应了。
<br/>
<br/>        体育课下课前点完名后，女体育老师宣布正式下课。
<br/>
<br/>        路娅玟看着自己的这杯已经浑浊得不见奶盖的“抹茶芝士奶盖”，瘪了瘪嘴，看着连星儿打趣地问道：“你和明司寒回来的时候，是打野战去了吗？”
<br/>
<br/>        “打野战倒不是，只不过遇上了几个高年级的坏学长，跟他们干了一架。所以……”连星儿尴尬地笑道。
<br/>
<br/>        路娅玟决定不计较了，虽然这茶长得不好看，但至少味道还行。
<br/>
<br/>        连星儿接着将袋子里剩下的饮品，拿了两杯出来分给了顾千凝和赵安彤。
<br/>
<br/>        四人一边喝着饮品，一边有说有笑地往回寝室的校道上走去。
<br/>
<br/>        等到上晚自习的时候，连星儿去晚了，还以为三教一楼105自习室没座位了，直到她走进自习室后才发现，明司寒早已坐在第一桌替她占好了位置。
<br/>
<br/>        连星儿轻轻地坐了过去，见明司寒正在做英语听力题，便没跟他打招呼，自顾自地翻开书背记起了英语语法知识。
<br/>
<br/>        但是，一到自习课间休息时间，周遭又会有女生抱着测试卷走到明司寒身旁求讲解。
<br/>
<br/>        明司寒每次一句“没空”，便让女生们悻悻而归。
<br/>
<br/>        连星儿扬了扬眉梢，故意学着刚刚离开的那个女生的语气，从手机里随便翻了一道英语入门题，自己看都没仔细看，便直接递到明司寒的眼皮子底下，撒娇道：“明学长，我这道题不会，你可不可以教我？”
<br/>
<br/>        明司寒看了一眼连星儿递过来的题目，咧了咧嘴，反问：“默写二十六个英文字母，你不会？”
<br/>
<br/>        “……”一听这话，连星儿这才把手机拿回来自己看了看，对着明司寒赧然一笑。
<br/>
<br/>        而就在这时，洛羽的经纪人林苏突然发了一条短信过来。
<br/>
<br/>        星儿，你现在有空来校门口一趟吗？你们学校的正大门口外，左边有一辆黑色的保姆车，我有话想跟你说——林苏。
<br/>
<br/>        嗯，有空，我马上过来——连星儿。
<br/>
<br/>        回复完短信后，连星儿收拾了一下自己的课本，对明司寒说道：“我要去校门口见我一个朋友，待会就不会过来接着自习了。我先走了哈！”
<br/>
<br/>        “嗯。”明司寒淡淡地应了声。
<br/>
<br/>        连星儿微微一笑，转身便离开了。
<br/>
<br/>        而明司寒却抬起头来，若有所思地看着连星儿走出自习室的背影。
<br/>
<br/>        难道……
<br/>
<br/>        又是去见洛羽了吗？
<br/>
<br/>        他和她之间，果然还是隔着洛羽。
<br/>
<br/>        连星儿如约而至，在正校门外的左边，她上了那辆黑色的保姆车。
<br/>
<br/>        起初她以为洛羽也在，没想到保姆车里除了司机，就只有林苏一个人。
<br/>
<br/>        林苏命令司机下车出去走走，司机二话不说地便打开车门离开了。
<br/>
<br/>        此刻，车内只剩下她和林苏两个人。
<br/>
<br/>        “苏苏姐，你是有什么事情要跟我说吗？”连星儿关问道。
<br/>
<br/>        林苏黯然神伤地垂下眼眸，忧心忡忡地回答道：“洛羽最近的状态不大好。”
<br/>
<br/>        “洛羽他到底怎么呢？”连星儿微微皱起眉头，追问道，顿了顿后，她又补充了一句，“他也有好一段时间没跟我联系了。”
<br/>
<br/>        就连这次他在她的大学里拍完戏回去了，也未跟她打招呼。
