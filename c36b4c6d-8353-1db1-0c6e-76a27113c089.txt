　　苏莫告别了黄衣青年之后，便一路向北，向那所谓的天山赶去。
<br/>　　四日之后，他看到了一座巨大的万丈高峰，高峰的山巅之上，还有建筑屹立其上。
<br/>　　“看来这就是天山！”苏莫暗道，随即加快速度，向那座万丈巨峰飞去。
<br/>　　嗖！
<br/>　　片刻之后，苏莫落在了万丈高峰的山巅之上。
<br/>　　只见山巅上是一个宽敞的的平台，平台上只有一个星空传送大阵。
<br/>　　而在星空传送大阵的旁边，坐落着一座小型的石殿，殿门敞开，其内有五、六名武者。
<br/>　　苏莫打量了一番，这几名武者其中一名是武帝境九重的修为，其它几人修为则是从武帝境一重到八重不等。
<br/>　　“使用星空传送大阵，五千下品灵晶！”一名中年武者走了出来，向苏莫说道。
<br/>　　苏莫点了点头，随即取出五千下品灵晶，交给了对方。
<br/>　　“每日傍晚，启动一次星空传送大阵，等着吧！”中年武者说了一句，便再次返回了宫殿中。
<br/>　　苏莫无奈的摇了摇头，便走到一旁盘膝而坐，对于这种情况，他也没有什么意外。
<br/>　　毕竟，这掌控传送阵的势力，也是在赚取灵晶，不可能为他一个人开启传送大阵。
<br/>　　反正他也不急在这一时，安静等候便可。
<br/>　　时间流逝，偶尔有零零散散的武者来到山巅，均是要乘用星空传送大阵的人。
<br/>　　到了傍晚时分，山巅上聚集了二十余人，而后，石殿中的几名武者走了出来，开启了星空传送大阵。
<br/>　　苏莫乘用星空传送大阵，向另一颗星辰而去。
<br/>　　接下来的三个多月时间，苏莫辗转在一颗又一颗星辰之上，有了那名黄衣青年提供的星辰路线，他只要按照路线传送就可以了。
<br/>　　随着不断的传送，路过一颗又一颗星辰，苏莫对于明虚星河的大致背景，也有了一些了解。
<br/>　　在明虚星河，太阴族是绝对的霸主，无可匹敌的大势力，没有之一。
<br/>　　整个星河之中，有无数的势力，都臣服于太阴族，以太阴族马首是瞻。
<br/>　　而强大的太阴族之下，下辖七个超级势力，这七个超级势力，完全臣服于太阴族，受太阴族支配调遣。
<br/>　　这明虚星河的情况，与古灵星河完全不同，完全就是一家独大的场面。
<br/>　　一颗小型星辰之上，苏莫身形伫立在高空中，抬头仰望星空。
<br/>　　浩瀚无垠的星空，星光璀璨，在他的视线中，一颗巨大的星辰，在星空深处，仿佛亘古长存。
<br/>　　而在那颗巨大的星辰四周，遍布着数百个小型的星辰，所有小型星辰将那颗巨大的星辰拱卫在其中。
<br/>　　苏莫知道，那颗巨大的星辰，就是所谓的太阴星。
<br/>　　太阴星以及周围的数百颗小型星辰，是整个明虚星河的中心，所有大势力全部聚集在此。
<br/>　　而他这几个月来，路过这这些星辰，都只是明虚星河的外围之地。
<br/>　　“还需传送两次，就能到达太阴族了！”苏莫轻叹了口气，本就神魂受损，精神力减弱的情况下，连续不断的赶路，让他感觉非常的疲惫。
<br/>　　到了现在，他的脸色依旧苍白，显得较为虚弱。
<br/>　　少倾之后，苏莫身形一动，再次向打听到的传送之地飞去。
<br/>　　……
<br/>　　三日之后。
<br/>　　中午时分，苏莫从一座传送大殿中走了出来。
<br/>　　“好浓郁的灵气！”
<br/>　　深吸一口气，苏莫惊讶不已，这太阴星的天地灵气，比古周星还要浓郁的多，真乃是绝佳的修炼之地。
<br/>　　眼眸扫视四方，他现在身在一座城池之中，大殿之前就是宽广的街道。
<br/>　　随即，苏莫抬脚在城中走动了起来，刚到太阴星，太阴星的情况他还不清楚，准备先打探些情况。
<br/>　　还有就是，青璇虽然是被一个太阴族的宫装女子从巫族中带走，但具体在不在太阴族之中，他也不能确定。
<br/>　　若是青璇不在太阴族，那就麻烦了，天下之大，到何处寻找？
<br/>　　苏莫不知道那名太阴族的宫装女子叫什么名字，所以只能慢慢先来到太阴星查探。
<br/>　　不多时，苏莫在城中找到了一家名为‘清风’的大型酒楼，抬脚走了进去，然后在大堂中靠窗的位置坐了下来。
<br/>　　酒楼，三教九流皆有，永远是最好的探听消息的地方，这在哪里都是如此。
<br/>　　这家酒楼很大，大堂宽敞，酒客众多，嘈杂不已。
<br/>　　“客观，您需要点什么？”身穿青衣的小二来到苏莫身前，客气的问道。
<br/>　　“你们这里有什么好酒？”苏莫轻笑着问道。
<br/>　　“本店最好的酒，当然是招牌清风醉，一千下品灵晶一坛！”小二面带得色的道。
<br/>　　“哦，这么贵的酒？”
<br/>　　苏莫闻言惊讶，心中立刻好奇了起来，一坛酒居然要价一千下品灵晶，这未免太夸张了吧！
<br/>　　“客观，一看您就不是太阴星武者，我们清风酒楼的清风醉，可是闻名整个太阴星，您来到太阴星，若是不平常清风醉，那真是一大遗憾！”
<br/>　　小二摇头晃脑的吹嘘了起来，他倒是没有认为苏莫买不起，他虽然修为很低，但眼力还是有的。
<br/>　　“是吗？那给我来一坛！”苏莫笑道，他倒是要品尝一番，什么酒能价值一千下品灵晶。
<br/>　　“好嘞！客官稍等！”小二应声，旋即便前去取酒，很快便为苏莫奉上了一小坛美酒。
<br/>　　苏莫支付了小二一千下品灵晶，便自顾自的打开了酒坛，而后便闻到了酒坛中飘出一阵清香。
<br/>　　这清香很纯、闻之沁人心脾、心旷神怡。
<br/>　　“好酒！”苏莫还未品尝，便不由得暗赞一声，随即直接倒了一杯，一口见底。
<br/>　　美酒入腹，辛辣如烈火，而后化为滚滚热流，通之四肢百骸，让他全身一阵轻微的酥麻，修为都要一丝丝的增长。
<br/>　　苏莫不由的眼前一亮，这酒真是宝贝，喝一杯下去，简直比吸纳数十块下品灵晶的效果还要好。
<br/>　　他乃是武帝境八重的武者，都能有这么明显的效果，若是武王境以下武者，喝一杯估计要直接提升修为了！
<br/>　　难怪敢要价一千下品灵晶，当真是不凡！
<br/>　　随后，苏莫便一边细细品尝美酒，一边聆听周围酒客的交谈。
<br/>　　不过，让他郁闷的是，从中午一直坐到傍晚，大堂中的酒客换了一批又一批，他已经喝了三坛清风醉，也没有听到任何有用的消息。
<br/>　　这些酒客，多是在谈论风月，谈论奇闻轶事，根本探听不到任何消息。
<br/>　　“话说，那云悠悠真是厉害，修为可谓是直线攀升，如今已经达到了武帝境八重的境界了！”
<br/>　　“可不是嘛！拜虚神境大能为师，想不提升都难！”
<br/>　　“主要还是天赋吧！天生纯阴之体，这可是最适合太阴族道统的体质之一！”
<br/>　　三名年轻的武者，坐在大堂的中央，一边饮酒一边闲聊。
<br/>　　而苏莫，听到这三人的谈话，眸中顿时精光爆射。
<br/>(本章完)
