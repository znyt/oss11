　　分身回到了惊天府。
<br/>　　端坐在惊天府之中，分身有些踌躇了，不知该如何将手中的资源，给予本体。
<br/>　　他很小心，不能让本体陷入任何风险之中。
<br/>　　万一他出去见本体，或者本体来到人王宫，或许会有危险。
<br/>　　思索了良久，他还是决定，让本体来人王宫。
<br/>　　直觉告诉他，宫主和玄天长老，应该不会加害于他。
<br/>　　不然的话，根本不必如此麻烦。
<br/>　　目前来看，这个宫主人德神王，还是可以相信的。
<br/>　　……
<br/>　　山洞之中，苏莫从静修之中，苏醒了过来。
<br/>　　“轮回古城！”
<br/>　　苏莫低声自喃，两年的时间，差不多够他突破到一品高级了。
<br/>　　随即，苏莫长身而起，向人王城而去。
<br/>　　当然，苏莫也留了个心眼，将天梦元神，留在了山洞之中。
<br/>　　毕竟，若是真的有危险，本体在劫难逃，他还有一个元神活着，就不会彻底死亡。
<br/>　　元神，已经能彻底独立，不需要肉身的滋养，也能长久存活，遨游天地。
<br/>　　并且，就算只是单独的元神，战力也同样不可小觑。
<br/>　　没过多久，苏莫回到了人王宫，回到了惊天府之中。
<br/>　　从分身手中，结果存放资源的储物戒，苏莫静静的等待了起来。
<br/>　　等了一个时辰，也没有任何的变故，他就彻底的放下了心。
<br/>　　若是有危险，当他踏入人王宫的那一刻，就应该到来了，不会拖这么久。
<br/>　　随后，苏莫进入密室，准备开始修炼。
<br/>　　而他的分身，则是离开了人王宫。
<br/>　　当然，万化神剑元神和不败战气元神，被苏莫收了回来。
<br/>　　分身的肉身，只是在他一缕神魂的控制之下，离开了人王宫。
<br/>　　分身会掌控苏莫的天梦元神，自行修炼，蛰伏在不为人知之地。
<br/>　　密室之中。
<br/>　　苏莫在整理源石，五行源石不多，共有一百六十多万。
<br/>　　不过，其中有四分之一，是中品源石。
<br/>　　除了源石之外，还有四株神草，每一株神草都非同小可，其内蕴含的源力，都非常惊人。
<br/>　　只不过，这神草之中的源力，并不是五行源力，苏莫无法使用，这让他很是无奈。
<br/>　　当然，虽然不能使用，但是，他可以拿出去交易，换取五行源石。
<br/>　　少倾之后，苏莫开始吞噬源石修炼。
<br/>　　源源不断的五行源气，进入了他的体内，被炼化成混沌源气，壮大圣丹，增强修为。
<br/>　　从一品中级，提升到一品高级，没有什么大的壁障，无非就是凝聚源力。
<br/>　　苏莫进入了闭关之中，抛却了一切的杂念，争取在两年之内，达到一品高级。
<br/>　　虽然，在百岁之内，他有无敌的自信，但是，也不会浪费一丝一毫的时间。
<br/>　　他已经比帝一魂和拓跋青云，落后太多太多看，必须奋起直追。
<br/>　　时间缓缓的流逝，苏莫的修为，缓缓的增长。
<br/>　　一月！
<br/>　　两月！
<br/>　　三月！
<br/>　　一年！
<br/>　　两年！
<br/>　　两年时间，悄然流逝。
<br/>　　密室之中，苏莫缓缓的睁开了眼眸。
<br/>　　他的修为，在一个多月之前，便已经达到了一品高级。
<br/>　　一切，都水到渠成！
<br/>　　没有天赋的桎梏，这种小境界，对于他来说，只要资源足够，一切都不是问题。
<br/>　　呼！
<br/>　　苏莫长出了一口气，算算时间，估计也差不多了。
<br/>　　他起身离开了惊天府，准备去找玄天长老问问，具体什么时候去轮回古城。
<br/>　　不过，他还没走到玄天神殿，沁儿迎面走了过来。
<br/>　　“沁儿姑娘！”苏莫向沁儿抱拳示意。
<br/>　　“段惊天，你终于出关了！”沁儿面带笑意的说道。
<br/>　　“刚刚出关！”苏莫笑了笑，他的眸中，却是暗含惊讶之色，因为，此刻的沁儿，居然达到了二品低级的修为。
<br/>　　这个修炼速度，可是不慢啊！
<br/>　　“父亲让我通知你，半个月之后，在内宫广场集合，去轮回古城！”沁儿沉声说道。
<br/>　　其实，玄天长老很早就通知她，让她告知苏莫了。
<br/>　　只不过，她去找过苏莫两次，后者好像一直在闭关，她就没有打扰。
<br/>　　没想到，今日碰巧在这里遇到了。
<br/>　　“半个月之后？”
<br/>　　苏莫闻言，点了点头，道：“好，我会准时去的！”
<br/>　　“嗯！到时候我们同行！”沁儿笑道，其实，她心中很是好奇。
<br/>　　因为，她知道，自己的父亲根本没有收段惊天为徒。
<br/>　　但是，父亲却让她通知段惊天此事，明显是格外的重视。
<br/>　　但是，她也询问过父亲，为何如此重视段惊天，但是，她父亲却是闭口不答。
<br/>　　这让沁儿，心中充满了好奇。
<br/>　　“嗯！”
<br/>　　苏莫点头，笑着道：“沁儿姑娘，若是无事，我就先回去了！”
<br/>　　“那你回去吧！”沁儿摆了摆手。
<br/>　　随即，苏莫转身返回惊天府。
<br/>　　“真是个怪人！”
<br/>　　望着苏莫离去的背影，沁儿憋了憋嘴，低声自语。
<br/>　　这个段惊天，真是相当的奇怪，不和其他同门相处，仿佛与世隔绝了一般。
<br/>　　不过，这个段惊天的修为，居然增长的如此之快，达到了一品高级。
<br/>　　……
<br/>　　苏莫回到惊天府之后，便继续修炼了起来。
<br/>　　半个月的时间，几乎是转瞬而过。
<br/>　　这一日，苏莫换了一声淡蓝色的长衫，向内宫广场走去。
<br/>　　现如今，他精气神饱满，气血如龙，修为大增，实力完全处在巅峰。
<br/>　　即便天梦元神不在他的体内，对于他的实力，也没有什么影响。
<br/>　　当苏莫到达内宫广场之时，广场上已经人山人海。
<br/>　　广场中央，伫立着二十余人，而在广场周围，则是围拢了不少内宫的弟子。
<br/>　　这二十余人之中，苏莫只认识四人，那就是沁儿、褚彦、骆妤、姬月松。
<br/>　　前三者去轮回古城，苏莫并无意外，但是，姬月松居然也去。
<br/>　　那么此人也不足百岁？
<br/>　　这让苏莫稍稍惊讶，这个姬月松不简单啊
<br/>　　一眼扫过，不仅是夕儿的修为提升了，姬月松和褚彦、骆妤，修为都提升了。
<br/>　　姬月松，已经达到了二品高级，骆妤也达到了二品中级，褚彦这是二品低级。
<br/>　　苏莫大步而行，穿过广场外的人群，径直来到了广场中央，沁儿等人所在之地。
<br/>　　“这个就是传闻中的段惊天吧！”
<br/>　　“好像是！”
<br/>　　“我的天，他的修为居然达到了一品高级！”
<br/>　　“居然修炼的这么快！”
<br/>　　不少人认出了苏莫，看到他的修为，顿时惊讶了起来。
<br/>　　十几年前，还只是外宫弟子，这么短的时间之内，居然达到了一品高级。
<br/>　　这种修炼速度，未免太惊人了吧。
<br/>　　“一品高级！”
<br/>　　褚彦和骆妤两人，看到苏莫的修为，顿时眼眸一凝。
<br/>　　苏莫的修为，出乎了他们两人的预料。
<br/>　　姬月松大量了苏莫一眼，眸中亦是带着惊讶之色。
<br/>　　因为，这种修炼速度，他当初都没有。
<br/>　　当然，仅仅是修炼速度而已。
<br/>　　“段惊天，人还没齐，还要再等片刻！”沁儿对苏莫说道。
<br/>　　“嗯！”苏莫点了点头。
<br/>　　“段惊天，你得到了什么玄奇神药，居然修炼的这么快？”褚彦沉声问道。
<br/>　　“没有什么神药！”苏莫摇了摇头。
<br/>　　“没有？”
<br/>　　褚彦闻言，微微皱眉，道：“没有玄奇神药，你怎么可能修为提升的如此之快？”
<br/>　　“没有玄奇神药，我就不能提升的这么快了？”苏莫凝视着褚彦，反问了起来。
<br/>　　“你……！”褚彦顿时语塞，他可不信，若是没有那种超强的玄奇神药，这个段惊天根本不可能提升的这么快。
