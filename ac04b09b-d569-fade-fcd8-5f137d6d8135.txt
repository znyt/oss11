　　荒凉的小岛之上，一个僻静的山谷中，杂草丛生，枯藤密布。
<br/>　　山谷中盘坐着数百人，正是一众巫族的试炼者。
<br/>　　巫天御的脸色阴沉如水，他做梦都没有想到，一个鸟不拉屎的苍穹世界，居然会出现苏莫这等强大的土著！
<br/>　　一场战斗，打的他们措手不及，损失惨重，三百余名族人被杀。
<br/>　　这要是回到族中之后，族中高层定然会震怒吧！
<br/>　　“天御哥，难道我们就这样等着试炼结束吗？巫娴姐怎么办？”一名俊秀的锦衣青年向巫天御问道。
<br/>　　“是啊！天御族兄，我们要杀光那些土著！”
<br/>　　“这些土著必须全部诛杀，方可为那些族人报仇！”
<br/>　　“一定要救出巫娴姐，不然大祭司会杀了我们的！”
<br/>　　其他巫族天才，亦是纷纷开口，个个满脸不忿之色。
<br/>　　巫天御闻言，扫视了众人一番，心中的怒火也因为众人的不忿，一下子炸开了。
<br/>　　“你们这些废物，就凭你们还想报仇？连一些土著都打不过，简直是丢尽了我族的脸！”巫天御大声怒喝道，眸中蕴含无尽的冷意。
<br/>　　众巫族天才闻言，顿时不说话了，个个惊若寒颤，满脸的羞愧之色。
<br/>　　本来，没有人将孱弱的苍穹世界当回事，只当是一次简单的历练，杀些苍穹世界的土著强者，掠夺一些资源宝物就离开。
<br/>　　以往的试炼都是这样，只是简单的屠杀，根本不会有什么危险。
<br/>　　但谁能想到，孱弱的苍穹世界，如今居然有这么多厉害的武者！
<br/>　　而且，那名叫苏莫的青年，实力居然如此的逆天，能和巫天御抗衡！
<br/>　　那苏莫就是一个变数，惊天变数啊！
<br/>　　这一次他们天冥星支脉子弟损失惨重，日后定然会让其它两大支脉，以及主脉之人嘲笑。
<br/>　　“天御族兄，那……我们难道就这么走了？”有人满脸忐忑的开口问道。
<br/>　　“还能怎么样？”
<br/>　　巫天御叹了口气，道：“待出去以后，让尊者出手吧！轻易就能灭了这些不知死活的土著！”
<br/>　　巫天御心中很是无奈，他倒是没有为自己担心，以他的实力，无人能杀的了他。
<br/>　　但是，他却不能不为这数百名族人考虑，若是千名试炼的族人，死的就剩他一人，那他回到族人估计也难逃一死。
<br/>　　巫族不同于其他种族，族规森严狠辣，即便他天赋不凡，也难逃重罚。
<br/>　　所以，巫天御便召回了所有人，然后落脚在了这座岛屿上，待月余之后试炼之期结束，便直接离开。
<br/>　　到时候，他们说明情况，族中尊者定然会降临苍穹世界，灭杀那些土著，救出巫娴。
<br/>　　那个时候，不管巫娴有没有事，与他就没有多少干系了。
<br/>　　就算巫娴死了，族中也不会过多怪罪与他。
<br/>　　众巫族天才闻言，均是心中无奈，面露苦涩，这一次他们实在太狼狈了。
<br/>　　他们堂堂的巫族天冥星支脉的天才，居然栽在了一群卑微的土著手中，这简直是从未有过的事情。
<br/>　　就在一众巫族天才暗自叹息之时，异变陡生。
<br/>　　蓦地，一道长达数万丈的三色剑气，突然从天穹之上斩下，如同一柄擎天神剑，势不可挡。
<br/>　　磅礴的剑威浩荡天地，笼罩了整座岛屿，无边的锋锐之气，撕裂了整片空间。
<br/>　　澎湃的剑气波动，让得天地轰鸣，岛屿巨震，骇人无比。
<br/>　　“什么！”
<br/>　　“不好！”
<br/>　　“快跑啊！”
<br/>　　霎时之间，所有巫族试炼者，均是心中大骇，惊恐无比。
<br/>　　毫不犹豫，包括巫天御在内，所有巫族试炼者均是急速飞逃。
<br/>　　巫天御的实力虽强，但这袭来的一剑太过强大，仓促之间他也没有把握抵挡，所以他亦是毫不犹豫，身形如电，急速闪避。
<br/>　　下一息，惊天三色剑气，便狠狠的斩在了山谷中。
<br/>　　轰！
<br/>　　一声惊天动地的巨响，整动了整片海域，方圆数百里的小岛，被剑气直接劈开，化为了两半，两边的海水冲起万丈之高。
<br/>　　啊！啊！啊！！
<br/>　　凄厉的惨叫声此起彼伏，无数的血雾升腾而起，无尽的鲜血挥洒长空。
<br/>　　只是一剑，便有不下两百余名巫族试炼者惨死，连身躯都被斩成了碎片。
<br/>　　剩下的人虽然在紧急时刻避开了，但足有大半人被剑气波及，身受不轻的伤势。
<br/>　　岛屿上空，苏莫眸光如电，一击之后，便立刻锁定了巫天御，直接向对方杀了过去。
<br/>　　“你的死期到了！”
<br/>　　苏莫厉喝一声，身形如电，手中动作不停，魔剑连连挥挥，道道三色剑气伴随汹涌的魔气，向巫天御绞杀而去。
<br/>　　突破到武皇境九重巅峰的苏莫，实力之强，比之半个多月前，已不可同日而语。
<br/>　　他的攻击更加的犀利，剑威更加磅礴，惊天的玄力波动如同怒海狂涛。
<br/>　　即便他没有动用任何武魂的力量，攻击力也比之前强大了近三倍。
<br/>　　于此同时，苏莫心神一动，二十八柄本命灵剑，尽数****而出，开始收割一众巫族天才的性命。
<br/>　　“快走，所有人都离开此地！”巫天御的大喝声响起，他一眼便看出了苏莫修为更近一步，现在的实力已经超越了他，立刻命所有人逃离。
<br/>　　而他，眼见铺天盖地的剑气袭来，手中瞬间出现了一杆长枪。
<br/>　　长枪如龙，疯狂暴刺，无数枪影洞穿虚空，迎向袭来的漫天剑气。
<br/>　　巫天御并不恋战，一边出手抵挡苏莫的攻击，一边身形急速逃离。
<br/>　　轰轰轰！！
<br/>　　无数枪影和漫天剑气相击，顿时疯狂爆炸，化为滔天的冲击波，扫荡八方，犹如世界末日一般。
<br/>　　漫天枪影被轻易斩碎，剑气破开了抵挡，继续杀向巫天御。
<br/>　　不过，巫天御的速度极快，瞬息千里，剑气很难伤及对方了！
<br/>　　而那数百名巫族试炼者，遭到二十八柄本命灵剑的攻击，以及秦不死三人的出手，瞬间便又陨落了三十余人。
<br/>　　“逃啊！”
<br/>　　“快走！”
<br/>　　“跟上天御族兄！“
<br/>　　惊恐的呼声不断，数百名巫族天才，早已吓的胆寒，哪敢逗留分毫，全部都向着巫天御的方向逃离。
<br/>　　“逃得了吗？”苏莫见此顿时冷笑一声，一柄本命灵剑突兀的出现在了他的脚下，他的身形如同疾电，瞬间洞穿了虚空。
