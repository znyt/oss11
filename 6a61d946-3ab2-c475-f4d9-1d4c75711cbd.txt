　　很快场上便是开始了ban选画面，第一局白队这边在蓝色方，蓝队则是在紫色方。
<br/>　　“这边白队连犹豫都没有犹豫直接将霄神的锤石给ban掉了，现在在赛场上真的很难看到霄神玩锤石了，无论锤石这个英雄怎么削，除非这个英雄重做。”看着白队的ban人，米粒有些感叹的说道。
<br/>　　“是的，霄神当初手速不够的时候就能用锤石这个英雄为自己证明，现在他的手速已经是世界超一流选手的手速了，可想而知这个英雄他玩的会多么的恐怖。”林雅楠跟着点头。
<br/>　　场上双方来回ban人，叶帝他们主要是针对蓝队的下路，而蓝队这边主要则是针对叶帝他们这边的上野两路。
<br/>　　“蓝队的这个战术很明确，这就跟之前白队和皇族打的时候一样，你中路都有一个超一流的选手了，就算是大魔王真的比暗夜强，只要暗夜小心，那么单杀的几率其实不大的，这种情况下倒不如针对以下白队这边另外两个carry点。”看着蓝队的ban人，米粒继续说道。
<br/>　　“看来两支战队的战术都很明确，白队这边就是死针对白队的下路，而蓝队这边则是死针对白队的上野两路，这一波ban人双方55开，都取得了他们想要的成果。”看着双方已经ban掉6个英雄，林雅楠大声说道。
<br/>　　此刻场上叶帝他们这边已经ban掉了锤石、圣枪游侠还有女警这三个英雄。
<br/>　　而蓝队那边则是ban掉了盲僧、鳄鱼还有蜘蛛这三个英雄。
<br/>　　白队比赛室，
<br/>　　“咱们一手选什么？”香蕉对着叶帝问道。
<br/>　　“拿卡萨丁吧。”叶帝犹豫了一下后，开口说道。
<br/>　　说话间香蕉已经是确定好了卡萨丁这个英雄，看着这个头像亮起来的一刹那，现场顿时响起了一阵山呼海啸般的欢呼声。
<br/>　　“卡萨丁，这个大魔王一战封神的英雄，至今我还记得那一次大魔王单人偷基地的举动，而这个英雄也是因为大魔王的原因一削再削，不过就算是如此这也是所有战队都会ban掉的，今天又能够看到大魔王来表演了，这个真的期待。”看到叶帝的选人，米粒顿时激动的喊道。
<br/>　　“蓝队这边选择了武器和妖姬，暗夜这边也是丝毫不怂啊，不过妖姬这个英雄在前期面对着卡萨丁的时候确实是占据着很大的优势的。”林雅楠跟着喊道。
<br/>　　而在白队的比赛室，看到对方选择的是武器这个英雄打上单的时候，叶帝几人的脸上都露出了一抹笑意。
<br/>　　“果然啊，这个英雄放出来他们就会选，不过奶妈这个英雄可是非常克制武器的，就是不知道他们看到咱们选奶妈的时候会怎么想。”夜血狂歌笑着说道。
<br/>　　“拿奶妈吧，既然他们想看大招，那就给他们看看咱们的这个大招，茄子你玩扇子妈行么，这一局咱们来个盾奶组合。”叶帝开口问道。
<br/>　　“没问题。”听着叶帝的话，茄子当即开口应道。
<br/>　　下一刻在无数人一脸懵逼的状态中，白队这边直接确定了奶妈和扇子妈这两个英雄的人选。
<br/>　　“什么情况，不是已经选完中单了么，怎么又拿出奶妈和扇子妈这两个英雄了，难道是扇子妈去打上单？还是卡萨丁去打上单？”看着白队的选人，米粒直接懵了。
<br/>　　“不会是扇子妈辅助，奶妈去打上单吧。”林雅楠愣愣的说道。
<br/>　　“怎么可能，奶妈这个英雄怎么可能打上单。”林雅楠刚说完，米粒便是否认道。
<br/>　　大家都知道奶妈这个英雄多脆，这要是在上路和别人对拼，那部明摆着就是送呢么。
<br/>　　众多现场还有直播间的观众看到白队的这个选人，一个个也都是非常的不解。
<br/>　　“6666666666666，这是不是就是白队的大招？不过这个大招看的我真是一脸懵啊。”
<br/>　　“这是什么阵容，到底谁打中单谁打上单，我现在还没看明白。”
<br/>　　“其实白队这边下路奶妈辅助没毛病的，毕竟他们的下路弱么，现在用一个奶妈来续航，这是非常好的针对手段啊，不知道你们诧异什么。”
<br/>　　“就是，之前不是有一个肉扇子妈上单的那个体系么，我感觉白队现在就是要打这个套路，你们真是见识少啊。”
<br/>　　“这一局感觉有意思了，估计蓝队看到这个选人也是一脸懵。”
<br/>　　而在蓝队比赛室内，在众人看到叶帝他们的选人后，李景禹几人的眉头也都是皱了起来。
<br/>　　“他们这个选人是什么意思？”蒋念一脸的不解。
<br/>　　“扇子妈上单？我之前看过一个主播的视频，不过那种只适合路人局啊，如果他敢那么玩的话，就算是到了后期也没用。”楚南跟着说道，对于自己的武器他还是非常的自信的。
<br/>　　“香蕉说的大招会不会是奶妈上单，因为我实在是想不出扇子妈能够上单的理由，加上他们一直说大招，我只能这么猜了。”秦霄猜测道。
<br/>　　“不管怎么说，小南这一局你要顶住，这一局他们放大招了，咱们下路就别拿那个组合了，我怕出问题。”李景禹开口建议道。
<br/>　　双方飞快的选人，很快各自的五名英雄确定了下来。
<br/>　　叶帝他们这边是卡萨丁、奶妈、扇子妈、烬和螳螂。
<br/>　　蒋念他们那边则是选择了武器、妖姬、挖掘机、娜美和寒冰这对组合。
<br/>　　不过这一刻大家却是都将目光放到了白队这一边，他们都想看看最后白队会来一个怎样的变阵。
<br/>　　“不知道白队是哪个英雄打上单，哪个英雄打中单，就看这最后一手变阵了。”米粒开口说道。
<br/>　　下一刻当众人看到奶妈被调到上单那个位置的时候，所有人都惊到了。
<br/>　　“奶妈上单！！”林雅楠惊声喊道。
<br/>　　就是连带着蓝队那边也是有些小小的吃惊，虽然他们已经料到可能是这个英雄上单，但是真到了这一步，面对着他们从来没有在上单位置面对的英雄，几人心里都是有些紧张的。
<br/>　　“小南这一局你稳住，慢慢打，后期咱们优势。”李景禹拍着楚南的肩膀说道，随后便是摘了耳机向着后面的座位上走去。
