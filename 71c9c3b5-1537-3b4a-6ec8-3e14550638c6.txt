　　平安村的夜晚是风起云涌的，激烈的爆炸不绝于耳，别说是人，就连月亮也早就躲到了云雾之中，山林之中的生物很多都殃及池鱼，一片一片的无声死去。
<br/>　　二阶跟三阶的战斗，是没有可比性的，但就是那么的神奇，蛇与蛟战成了白热化，不相伯仲。
<br/>　　地面层层掀开，仿佛宛如火山爆发，有岩浆迸射。
<br/>　　也不知道过了多久。
<br/>　　“这不可能...”
<br/>　　上空那条黑色气焰的虺蛟发出一道深沉而沙哑的声音，跟着与一道磅礴的力量相击撞，随后就是地动山摇，飞沙走石。
<br/>　　虺蛟浑身遍体鳞伤，鲜血淋淋，而远处只见一颗巨大蛇头傲立在空中，眼瞳散发着妖异的寒芒。
<br/>　　看来三阶生灵不过如此。
<br/>　　涂小安久战之下，发现三阶虺蛟的实力貌似并没有他预想的强大。
<br/>　　难道是他牛逼到超出自己的想象了？
<br/>　　虺蛟压倒了一排的房屋，连连的咳出了数口血，每一滴血，都浑然天成，妖艳如珠。
<br/>　　张天师心中感觉到了前所未有的羞耻，他身为虺蛟，既然没将区区一条二阶的蛇精拿下。
<br/>　　“本天师要不是前段时间受了重伤，实力发挥不过五成，早将你碎尸万段了”
<br/>　　看着空中傲然而立的紫金巨蟒，张天师满是不甘的开口。
<br/>　　受重伤？
<br/>　　涂小安哑然一笑：“胜者为王败者为寇，天师，别找其他的借口，最后一博吧”
<br/>　　只见紫金巨蟒雷电跟火焰两种异象显化，其势滔滔，猛虎啸天，鹰击长空，朝着虺蛟撕咬而去。
<br/>　　“瘦死的骆驼也比马强，区区蛇精，也敢在老祖宗面前放肆，受死”
<br/>　　黑色气焰的虺蛟通体熠熠生光，如苍龙翻腾，一股无比骇人的气息毁天灭地，这便是三阶生灵的气息。
<br/>　　足够镇压一切。
<br/>　　蛇与蛟再次来势汹汹，携惊天之磅礴的力量冲撞到了一起去。
<br/>　　远远遥望，就是两条龙在争斗，迸发世界末日般的恐怖，两道光芒宛若两轮骄阳当空而照，璀璨的光芒耀人双目，巨大的能量流如山洪爆发一般喷放而出。
<br/>　　“轰...轰...轰...”
<br/>　　一股惊天巨响，顷刻间天地变色，飞沙走石。
<br/>　　同时显化三十米，相互缠绕，难分蛇与蛟，不管是蛇鳞片，还是蛟鳞片，全都遍地鳞伤，一片狼藉。
<br/>　　“我涂小安这条蛇，屠龙就在今天”
<br/>　　紫金巨蟒发出激动亢奋的声音，谁说蛇便不如龙，谁说蛇非要化作龙。
<br/>　　“天师，我会告诉你，已蛇证道，不比化龙弱”
<br/>　　涂小安的肉身强度，不会逊色虺蛟，他无所畏惧。
<br/>　　“休要妄语，本天师修炼五百年，化蛟成龙，势在必行，我是不会败的”
<br/>　　蛇与蛟疯狂的相互的撕咬对方的身躯，血与肉横飞，画面惨绝人寰，都疯了一样。
<br/>　　可就在他们即将要分出胜负的关键，在这深沉而恐怖的黑夜中，无任何生物接近，可此时，遥遥空中，一抹白引掠来，似浮光掠影。
<br/>　　就是那么的不可思议，蛇与蛟相互搏战的上方，一位白衣女子皎若太阳升朝霞，立于两庞然大物中间。
<br/>　　她风华绝代，冷若冰霜，周身缭绕一股深蓝色的气焰。
<br/>　　“是她...”
<br/>　　涂小安余光之下，诧异万分。
<br/>　　几乎是下一个念头，他就大感不好。
<br/>　　“冰封千里”
<br/>　　白衣女子一出现，发出清脆悦耳的声音，但杀机也随之弥漫出来，一股比寒冬腊月更要冰冷的力量铺天盖地下去。
<br/>　　霎时，诡异的事情发生了，只见缠绕在一起难解难分的蛇蛟骤然间化为了巨大的冰雕，之后已他们为中心的地面居然已肉眼可见的速度跟着冻结起来，一股寒冰朝着四面八方蔓延，寒冰所到之处的地面全都冻成了冰。
<br/>　　短短不过半响，整个平安村被冰冻了起来，一切仿佛被静止。
<br/>　　没有任何活物存在。
<br/>　　白衣女人绝代身姿傲然与空，举手投足间，尽显优雅脱俗之态，清丽美颜露出一丝寒冬腊月般的笑容，冷的阴寒。
<br/>　　她美眸高高俯视而下，欣赏自己的杰作。
<br/>　　她一个动作，将一片地带全给冰冻住了，最得意的是，将一条三阶虺蛟跟蛇精全都冰封其中。
<br/>　　这是何等伟大的壮举。
<br/>　　所谓螳螂捕蝉黄雀在后。
<br/>　　她，妖精管理局，二组组长，冷霜是也....
<br/>　　“咯咯...咯咯...”
<br/>　　冷霜在空中，花枝招展的发出银铃般的笑声，清脆悦耳，高兴极了。
<br/>　　“蛇就是蛇，就算开了灵智，也高明不到哪里去，还不是被本小姐利用”
<br/>　　狗咬狗一嘴毛，最后便宜的都是她。
<br/>　　“让本小姐来结束这一切....“
<br/>　　”冰之粉碎”
<br/>　　她的美眸光芒闪烁，却透着冷漠杀气，一只纤纤玉手顺势往下方一压。
<br/>　　砰...砰...砰.....
<br/>　　霎时间，粉碎的声音冠绝全场，就宛如一块巨大的玻璃从天砸到了地面，顷刻间粉身碎骨。
<br/>　　但凡被冰冻之物，全部粉碎，一件件房屋支离破碎，就跟纸糊的一样脆弱。
<br/>　　紧接着，两道闷哼的凄惨叫声冲天而来。
<br/>　　相互缠绕的蛇与蛟终于分开了，如断线的风筝狠狠的摔在了地上，遍体鳞伤，鲜血四溢。
<br/>　　要不是他们的肉身强度足够强悍，也会随之粉碎。
<br/>　　半响后，冷霜霸道的宣誓：“一条蛇，一条蛟，你们都是本小姐的战利品，猎物”
<br/>　　三十米的虺蛟半死不活的躺在地面，身受重伤，难以起来。
<br/>　　而那条紫金巨蟒，更好好不到那里去，一动不动，鲜血直流，无一丝气息散发出来。
<br/>　　“蛇精，你比本小姐预想的还要强，强太多，我应该要好好感谢你”
<br/>　　冷霜淡然的目光投放到那条紫金巨蟒身上，她真的没料到这条蛇精能跟三阶虺蛟战到不相伯仲，甚至在打下去，他都有赢的趋势。
<br/>　　不可想象，二阶对上三阶，都有战胜的胜算，简直就是妖孽了。
<br/>　　可惜，最后，她才是胜利者，人类为什么能站在这个世界的巅峰，掌控一切，主宰一切。
<br/>　　不但拥有强大的实力，更有非凡的智慧。
<br/>　　智慧这种东西，往往有时候，比实力还要重要。
<br/>　　“你终于出来了，本蛇王等你许久了”
<br/>　　突然，那条一动不动的紫金巨蟒发出了一道戏谑的声音，顿时杀意如狂风骤射，疯狂弥漫出来，带着致命的阴冷气息。
