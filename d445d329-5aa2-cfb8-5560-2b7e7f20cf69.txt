　　这其实是法师最喜欢的穿越方式，因为不用把肉身留下，投影过去之后，任何法术都不耽误。对叶少阳这样的天师以上牌位的法师来说，虽然依靠自己也可以做到，但一般弟子是不行的，龙虎山有这样一个空间裂缝，如果必要的话，可以让大批弟子穿越过去。
<br/>　　空间裂缝开启之后，道渊真人先跳进去，然后叶少阳和毛小方一起也跳了进去。
<br/>　　在虚空中穿梭了不一会，三人来到了酆都城附近，往北门走去。
<br/>　　“我带你去见崔府君，看看你的生死簿，就一切都知道了。”道渊真人头也不回地说道。
<br/>　　“崔府君会让你看？”叶少阳有些好奇。
<br/>　　“我陈述利害，然后试一试。”
<br/>　　说话间来到城门外，道渊真人和毛小方都是天师牌位，跟叶少阳一样出示了天师牌，顺利进入城内。
<br/>　　毛小方和道渊真人都不是第一次走阴，跟酆都城守门的阴兵也都认识，彼此还聊了几句，这些阴兵只是对叶少阳有些好奇。
<br/>　　因为人间的道门天师数量不多，都来阴司办事过，这些守兵对他们都有点面熟，唯独叶少***本没有见过，不过既然是跟着这两位一起，自然也不怀疑他的身份，只当是新晋受封的天师。
<br/>　　进城之后，三人直奔天子殿去。
<br/>　　叶少阳内心有些激动，想着如果再次见到萧逸云，自己一定要再次尝试说服他，不过一想上次崔府君驱逐自己时说的不让自己再来的话，又有些担心，天子金口玉言，绝对不会是吓唬人。自己这么长时间没有过来，就是顾忌这个，知道来了也是白来。
<br/>　　一路来到天子殿，又看到了守门的两个熟人，叶少阳故意躲在后面不给他们面见。
<br/>　　道渊真人恭恭敬敬地递了名帖，请求拜见崔府君。
<br/>　　一般来说，人间天师是没有资格见崔府君的，倒不是级别不够，而是天子殿负责的是人间所有人的生死休伤，人间法师为了避嫌，与天子殿要尽量减少接触，除非真的遇到了什么重要的事情，需要找崔府君解惑，才不得不求见。
<br/>　　守门的鬼武士也懂规矩，派了一个人进去送信，等了一会之后，那人出来，让他们几个进去。
<br/>　　叶少阳夹在中间，走过了殿门，进去之后，刚松了一口气，结果从对面突然吹来一阵黑风，将三人裹在中间，一时间什么也看不见，只感到风不断地吹，站立不住，被吹得飞了起来，身边一团漆黑，什么也看不见，也不知道是在往哪个方向飞。
<br/>　　三人慌乱之中运气抵挡，叶少阳就不用说了，现在法力不行，道渊真人和毛小方都是天师，然而面对这股霸道的阴风，也是一点法力都用不上。
<br/>　　在飘摇之中，叶少阳突然感觉自己的衣领被人抓住了，然后急速前行，脚下猛然一沉，落在了实地上。风已散去，叶少阳定睛看去，是一个白衣人抓着自己，还没等看清，白衣人又遁飞到身后，一只手拽着一个，将道渊真人和毛小方也从风暴中提了出来。
<br/>　　叶少阳一眼看到白衣人的相貌，眼前一亮，情不自禁地喊了一声：“小白脸！”
<br/>　　萧逸云白皙的脸立刻就黑了。
<br/>　　道渊真人和毛小方也认识萧逸云，打了招呼，转头看去，几个人已经是在酆都城外面的一片旷野之中了。
<br/>　　“刚才怎么回事，怎么一阵风把我们吹到这里来了？”道渊真人和毛小方都震惊无比。
<br/>　　“那是府君大人放的阴风，他不想见你们……所以把你们送出来。”萧逸云瞥了叶少阳一眼，“主要还是因为他。”
<br/>　　道渊真人和毛小方更加惊奇。
<br/>　　“你怎么又来了，府君大人说过，不许你再踏入阴司一步，这次对你算是手下留情了！”萧逸云望着叶少阳，冷冷说道。
<br/>　　叶少阳耸了耸肩，道：“我们总是兄弟一场，你不用这样子对我吧？”
<br/>　　“谁跟你是兄弟，我压根就不认识你！”
<br/>　　叶少阳道：“你是怀疑的，不然的话，你根本就不会现身，也不会跟我说这么多了。”
<br/>　　萧逸云被他这么抢白，说不出话来，只好冷哼了一声。
<br/>　　道渊真人对萧逸云拱了拱手，说道：“萧郎君，我这次来找崔天子，就是想看看这位叶天师的生死簿……”
<br/>　　萧逸云一摆手，说道：“不用看了！”
<br/>　　道渊真人愣住。
<br/>　　萧逸云叹了口气，左右看了看，说道：“这里不是说话的地方，我先随你们去人间吧。”
<br/>　　于是三人破开虚空，回到了龙虎山上道渊真人面壁的地方，站在亭子里。
<br/>　　“还请萧郎君明言。”道渊真人急不可耐地说道。
<br/>　　萧逸云眉头紧锁，喃喃说道：“我之前偷偷查验过了，生死簿上……没有他的记录。”
<br/>　　道渊真人和毛小方一下子就傻眼了，面面相觑。
<br/>　　“生死簿上没有记录……”道渊真人喃喃说道，“那除非是证道飞升，脱离了三界六道，像那些青冥界的阐教金仙一样……”
<br/>　　萧逸云瞥了叶少阳一眼，道：“你看他像吗？”
<br/>　　两人哑口无言。
<br/>　　“那没有别的可能了……”毛小方迟疑着说道。
<br/>　　叶少阳在一边默默地看着他们的表情，说道：“我早就说过了，你们不信。如果不是我真的有问题，为什么个崔府君不愿意见我？”
<br/>　　道渊真人抬头看着萧逸云，道：“萧郎君，他果真是……”
<br/>　　“我不知道，我也没见过这种事。”萧逸云嘴上这么说，停在道渊真人和毛小方耳中，却是一种承认。
<br/>　　叶少阳刚要说什么，萧逸云转过头来看他，默默说道：“就算你是从百年后来的，我只希望你不要骗我，不要利用我。”
<br/>　　叶少阳拍着他的肩膀，郑重地说道：“我们的关系，是能够为对方去死的那种，这个，我绝不骗你，不过我也不需要你为我死，我在这边没什么朋友，我希望你能帮我，哪怕是出出主意也行。”
<br/>　　萧逸云盯着他看了半天，摸了一下鼻子，说道：“我就暂且相信你吧。”
