　　玉石俱焚，碎成十块碎片……
<br/>　　楚涵细细咀嚼鱼柔女所说的话，心中的震惊让他表情都变得有些不自然。
<br/>　　如果推测没错，那么回炉一门的守护者，就是回炉系统，或者说就是旺财！
<br/>　　而失去记忆的旺财，自出现开始就不断要求他去寻找十块回炉碎片，楚涵只知道十块碎片的聚集可以让回炉系统的功能恢复到100%，哪怕有三块碎片已经被用在了其他地方，但还是能够找到代替品。
<br/>　　并且他还已经找到了一块，那就是水族圣物海蓝之心！
<br/>　　可是……
<br/>　　回炉这么重要的东西，关系到这么多神秘家族的至高存在，怎么会落在他身上？
<br/>　　莫非他和回炉一门有什么联系？
<br/>　　望着楚涵苍白的神色，鱼柔女只是以为他初次听闻这些被震惊，没有想太多就继续道：“所以说想要破解禁制，就需要将阴谷的平衡重现，回炉一门的守护者，以及其余七门的家族成员入住。”
<br/>　　楚涵一愣：“也就说八门聚齐在此，禁制就解了？”
<br/>　　鱼柔女点头：“不错，不过回炉的守护者早已破碎消失，想找到何其难？更何况还碎裂成了十块碎片，哪怕聚齐恐怕也难以恢复当年盛况，能不能镇压阴谷最高禁制地回炉一门，还有待考究。”
<br/>　　楚涵有些震惊也有些后怕，道：“必须所有七门聚齐才行，那罗家胆子也太大了，他们设下了陷阱直接让肖家灭了门！这禁制岂不是永远都不可能接触？这些神秘家族难道完全放弃了阴阳谷？”
<br/>　　鱼柔女再次一笑：“恐怕对于白家和罗家来说，已经习惯了在这个星球上称霸吧？至于肖家的灭门，不是还有一个肖七么？”
<br/>　　“她只有一半的神秘家族血脉！”楚涵脱口而出。
<br/>　　“可她返祖了。”鱼柔女直视楚涵。
<br/>　　“所以……”楚涵愣住了，呐呐开口：“她一个人也算？”
<br/>　　“制裁者也是一个人，制裁家族也很特殊。”鱼柔女回道。
<br/>　　楚涵点头：“这么说来如果七门全部聚齐在这阴谷之内，再找到回炉的守护者，阴谷的禁制就会解除，到时候水火两族也就会碰面？”
<br/>　　鱼柔女笑道：“你似乎很期待？”
<br/>　　楚涵连忙摇头：“不，我害怕这一天的到来，顺便一问，你们水族可以让我进入水族旧址，那如果火族的人有同样的想法，是否也可以让普通人进入他们火族旧址？”
<br/>　　鱼柔女眉头一挑：“的确可以。”
<br/>　　楚涵盯着鱼柔女的眼睛：“我有个直觉，你还没告诉我全部……”
<br/>　　正当楚涵想要继续发问的时候，忽的一个明显的声音在两人前方响起，打断了他的思维。
<br/>　　只听见‘轰’的一声，远在前方沙滩上的地方，一个黑色的不明飞行物忽然从海面冲出，歪歪扭扭又跌跌撞撞的冲在了沙滩上！
<br/>　　见到这一幕的楚涵和鱼柔女同时失了声，惊讶的望着那片沙滩。
<br/>　　鱼柔女有些回不了神，这是她第一次遇到这么难以预测的状况，那片沙滩是阴谷入口进来的海岸，一般来说是会有人被海浪冲上来，但这飞行物……
<br/>　　楚涵则是在最开始的惊讶之后，就转而变成了呆愣。
<br/>　　今天有太多的事让他情绪起伏，也让他思维都混乱，好巧不巧的是这飞行物他认得，不仅认得还乘坐过。
<br/>　　这是天照3000！
<br/>　　那个被他和高少辉从白忧秘密基地内取出，随后还被狼牙科技部的和上拆开再重组的天照3000。
<br/>　　“你认识？”鱼柔女一看楚涵的神色就立即发现了不对。
<br/>　　楚涵点头，嘴角抽·搐：“来了个熟人。”
<br/>　　两人快速朝着海岸线走去，不多时就来到了天照3000的旁边，这飞行物以一个非常不好看的样子倒在沙滩上一动不动，显然在从南沙港的入口进来时，里面的驾驶员被那恐怖的漩涡和震荡弄晕了。
<br/>　　楚涵轻车熟路的找到天照3000的机舱门，而后用了最大力气狂敲好几下。
<br/>　　碰碰碰！
<br/>　　高少辉只觉得好像在地震，将他强行从混昏迷中震醒，他迷迷糊糊的睁开眼，首先引入眼帘的就是前方玻璃外的沙滩场景。
<br/>　　进来了？
<br/>　　索性高少辉并没有忘记昏迷前自己在做什么，他的目的就是带着天照3000进入阴阳谷，从而躲避白忧的追杀。
<br/>　　只不过当他清醒后的第一眼，就看到了站在沙滩上的鱼柔女。
<br/>　　女人？
<br/>　　高少辉愣了愣，有些不知所措，因为眼前这个女人的衣着太古怪，简单至极的一块白布条包裹全身，裸·露在外的手臂白·皙细嫩。
<br/>　　这装扮，让高少辉下意识的异常警惕！
<br/>　　但紧接着——
<br/>　　碰碰碰！
<br/>　　又是一阵震动，高少辉这才反应过来有人在敲机舱门，他连忙将内部的仪器调整，很快就看到了大屏幕上楚涵的那张脸。
<br/>　　“我靠！”高少辉一声大叫，当即打开天照3000跳了出去，还没来得及站稳就脱口而出：“楚涵？你怎么在这里！”
<br/>　　楚涵望着有些凌·乱的高少辉，上下打量了一番道：“我还想问你，你怎么把天照3000带了进来？”
<br/>　　高少辉一下子想了起来他来此处的重点，连忙就想往天照3000内部钻：“对对对，来不及了！回头再聊，我得赶快把天照3000放进高家旧址，这样白忧就找不到了！”
<br/>　　哪知还不等高少辉钻进去，楚涵就一把将其衣领揪住，道：“不用去了，这玩意儿借我用下。”
<br/>　　大尸潮的时间即将来临，这一点楚涵当然没忘，他本想接着水族圣女的能力从水路走，这样能快速到达距离南署基地附近的河流，三天时间应当足够，但眼下既然天照3000就在眼前，楚涵当然没必要白白折腾。
<br/>　　“借你用？”听到楚涵的要求后高少辉惊讶反问：“你不知道白家在追杀你啊？你还想开着天照3000招摇过市，你真是找死啊！”
<br/>　　楚涵无所谓的一笑：“总比你开着招摇过市的好，白家要杀我已经是个定局不可能改变，我对白家再多做几样恶事也无所谓，但高家可就不一样了，要是白忧发现天照3000在你手上，高家和白家的关系就会更紧张吧？”
<br/>　　高少辉一听，深觉极有道理，他刚想继续说些什么的时候，却骇然发现楚涵已经先一步进入了天照3000机舱，稳当当的坐在了副驾驶位置上。
