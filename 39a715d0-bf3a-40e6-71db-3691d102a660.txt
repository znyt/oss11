<br/>　　迎宾女人立刻战战兢兢，讪讪的说：“不是说蛇经常十天半月才吃一顿的吗，我以为是正常的”

<br/>　　野生的动物，有自己的习性，忽然变成“囚犯”，坐吃等死，会不适应。
<br/>　　尤其还是这种珍贵的龙蛇，有一定的灵性跟傲骨，如今被当成了观赏蛇，可谓生不如死。
<br/>　　生命可以归结为一种简单的选择，要么忙于生，要么赶着死。
<br/>　　人会已绝食进行自杀，动物也会。
<br/>　　“很显然，这条龙蛇是想用绝食来反抗”涂小安最后下了一个定论。
<br/>　　“有没有如此玄乎呀，冷血动物的蛇，还会已绝食自杀呀”
<br/>　　不少人被涂小安的说法说的有点懵逼，摇头表示不太相信。
<br/>　　涂小安也不想过多的解释，他自己就是蛇，也被人囚禁过，但是有些蛇注定不会被关在牢笼里的。
<br/>　　它们的每一片鳞片都闪耀着自由的光辉。
<br/>　　这条龙蛇及其珍贵，更不想如家雀，供人欣赏。
<br/>　　“小兄弟，按你的说法，我这条龙蛇就是无救了？”苏羽担忧的看着涂小安询问。

<br/>　　之所以重金买来，不是因为喜欢，纯粹是为了气运跟店里的风水。
<br/>　　但这条龙蛇要是死了，可不是钱的事情，而且你也无法在拿出五百万在重新觅得一条龙蛇。
<br/>　　况且这条龙蛇现在关于这苏羽自身的气运。
<br/>　　“也不是无药可救，只要帮这条龙蛇撑过这一关，到时候它会慢慢的适应新环境”
<br/>　　但凡是求死的人，或许都是一时想不开，时间会消磨一切。
<br/>　　野生的蛇，不适应家养的环境，很多中途会死掉。
<br/>　　但是这一关要是过去了，也就万事大吉了。
<br/>　　此话一出，苏羽大喜，：“小兄弟，你是不是有办法救我的龙蛇，要是你救了龙蛇，我苏某一定有重谢”
<br/>　　涂小安出来BB了那么多，自然不是光出来说废话，这条龙蛇及其珍贵，要是就这样的死了，到是有点可惜。
<br/>　　“把蛇给我”涂小安伸出手。
<br/>　　龙蛇现在在贴身保镖的手中，他看了苏羽一眼，只见苏羽点了点头，没有反对，就将龙蛇交到了涂小安的手里。
<br/>　　涂小安也是第一次接触龙蛇，细细的观察了一遍，顺着它的蛇躯一阵抚摸，它的蛇鳞片都是凸起的状态，非常的与众不同。
<br/>　　“让我也摸摸”
<br/>　　李清照一只白皙的手也伸了过来，涂小安拍打了一下，现在显然不是玩蛇的时候。
<br/>　　“想帮这条龙蛇撑过这一关，那么必须让它吃东西，但是它这个样子，就算给它吃再好的肉食，它也不会感兴趣”
<br/>　　有句话叫，你无法叫醒一个装睡的人，同样，求死的人或动物，也非常的棘手。
<br/>　　这种事情，在别的人手中或许有点手足无措，但对于涂小安并不是什么难事。
<br/>　　他甚至可以用蛇王的身份命令这条龙蛇进食，不过，他没有那么做，而是不知何时，他的手中出现了一片晶莹的蛇皮。
<br/>　　“来，宝贝，张嘴”
<br/>　　这片蛇皮放到龙蛇的面前，只见萎靡的龙蛇猛然的睁开了蛇眸，原本黯淡无光的眼神，变是满是贪婪起来。
<br/>　　“吃吧”
<br/>　　吃吧两个字还未落下，这片蛇皮已经被龙蛇给大快朵颐的吃了进去。
<br/>　　吃进去之后，倏然地，这条蛇的萎靡状态一扫而空，蛇眸灿灿起来。
<br/>　　“大家看，这条蛇有精神了”
<br/>　　不少人惊奇的叫了出来，涂小安手中又出现了一片蛇皮，再次的喂给了龙蛇吃。
<br/>　　这蛇皮是他第一次蜕皮下来的，而不是第二的脱皮，第二次脱下来的蛇皮甚至能让一条珍稀蛇种突破觉醒的可能。

<br/>　　救活这条蛇，已经是它的造化。

<br/>　　龙蛇的反应，大家都历历在目，苏羽好奇的问。

<br/>　　大厅中的客人不禁的冲着涂小安拍了拍手掌，一条看似奄奄一息的蛇，既然被他一下子给整的生龙活虎起来了。

<br/>　　李清照再次见识到了涂小安的手段，激动的跟一个小迷妹似的。
