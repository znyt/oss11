        打开小盒子后，里面有一张卡片。
<br/>
<br/>        明司寒拿出卡片，展开后，只见上面写着：我喜欢你。
<br/>
<br/>        落款的签名是“星儿”。
<br/>
<br/>        他就知道，一定是这丫头！
<br/>
<br/>        明司寒会心一笑，再去看盒子里的时候，才发现里面竟然还有七个“安全套”。
<br/>
<br/>        顿时，他脸色有些尴尬了，连忙将盒子给盖上。
<br/>
<br/>        这丫头也太直接了吧！
<br/>
<br/>        明司寒左想右想，犹豫了许久后，才掏出手机，给连星儿发了条信息。
<br/>
<br/>        这周星期六上午九点，临海主题公园门口，不见不散。——明司寒。
<br/>
<br/>        另一边，连星儿收到明司寒的信息后，以为他是要约朱小溪，于是把这条简讯转发给了朱小溪，连带着把明司寒的手机号码也给了朱小溪。
<br/>
<br/>        朱小溪在收到连星儿的这条短信时，纳闷不已。
<br/>
<br/>        明司寒怎么会约她呢？要约也是约连星儿吧！明明她在卡片上写的是连星儿的名字。
<br/>
<br/>        什么意思？——朱小溪。
<br/>
<br/>        明司寒约你见面啊！你有戏了！到时候要给我发喜糖！——连星儿。
<br/>
<br/>        真的？——朱小溪。
<br/>
<br/>        当然是真的！——连星儿。
<br/>
<br/>        那谢谢你了，星儿。——朱小溪。
<br/>
<br/>        随后，朱小溪把这件事情说给了同寝室的张蓉蓉听。
<br/>
<br/>        张蓉蓉冥思苦想了一番后，振振有词说道：“我觉得，连星儿可能是诓你的。你想想看，明司寒拒绝了那么多女生，连校花都不放眼里，又岂会看上连星儿？”
<br/>
<br/>        “但是……”朱小溪欲言又止，本来她是想说，连星儿比上任校花漂亮。
<br/>
<br/>        这话说出来，怎么就长连星儿的气焰了？
<br/>
<br/>        所以，朱小溪干脆不说了。
<br/>
<br/>        “要不，我们那天去看看，躲在暗处。如果明司寒真的来了，就证明连星儿没骗我们。她要是骗我们，正好我们还可以拿这事怼她。”张蓉蓉又接着说道。
<br/>
<br/>        朱小溪觉得张蓉蓉这话有理，认同地点了点头。
<br/>
<br/>        话说回来，连星儿觉得最近这几天，明司寒的行为太过异常。
<br/>
<br/>        中午去食堂吃饭，明司寒带着他的三个室友非得跟她们四个女生坐一桌。
<br/>
<br/>        下午去打个水，明司寒又带着他的三个室友过来帮她们提水。
<br/>
<br/>        然而，寝室里的话题却围绕着明司寒的另外三个室友展开。
<br/>
<br/>        “听说，自从上次KTV的事情过后，姚莹莹和仲正浩分手了。仲正浩现在单身。”路娅玟说道。
<br/>
<br/>        赵安彤则一脸不屑：“仲正浩是出了名的花心大萝卜，还是邢乐游好，酷酷的！”
<br/>
<br/>        “我觉得张子晋不错，安安静静，温文儒雅。”顾千凝满心欢喜地说。
<br/>
<br/>        连星儿就纳闷了，那个明司寒是他们四个男生里最帅的，她家这三个室友，怎么就没一个议论明司寒呢？
<br/>
<br/>        “喂，你们都是一些什么眼光啊？人家明司寒长得帅，球也打得好，还请你们吃糖醋排骨，怎么就不见你们夸他呢？”连星儿反问。
<br/>
<br/>        顾千凝、路娅玟和赵安彤立即齐刷刷地盯住连星儿。
<br/>
<br/>        十秒后……
<br/>
<br/>        “明司寒是你男朋友，咱们几个姐妹从不抢一个男人。”赵安彤率先表态。
<br/>
<br/>        路娅玟啧啧嘴，点头应道：“就是。”
<br/>
<br/>        顾千凝接着说道：“我比较喜欢文静点的男生。”
<br/>
<br/>        “？！”连星儿一脸懵，顿了顿后，不明所以然地反问，“他什么时候成我男朋友了？”
<br/>
<br/>        “你那天拽着千凝当中递礼物这事，早就在学校里传开了。都说你倒追明司寒，而且还追到手了。”赵安彤一本正经地说。
<br/>
<br/>        连星儿“切”了一声，解释道：“我是帮我一高中同学递的好不好，再说了，人家还主动提出要跟我那朋友约会了！”
<br/>
<br/>        顿时，寝室里这三个女生不约而同地瞪着眼睛，看着连星儿。
<br/>
<br/>        “真的假的？”三人几乎是异口同声。
<br/>
<br/>        “当然是真的！”连星儿斩钉截铁道。
<br/>
<br/>        三人面面相觑后，仍旧一脸狐疑地看着连星儿。
<br/>
<br/>        “你们不信啊？”连星儿随即摆了摆手，“不信也没关系，反正这是事实，说不定他们这次约会后，我就有糖吃了！”
<br/>
<br/>        “那这段时间，明司寒带着他的室友跟我们套近乎，敢情是为了你那个同学？”顾千凝难以置信地问。
<br/>
<br/>        连星儿微笑着点了点头。
<br/>
<br/>        肯定是这样！
<br/>
<br/>        明司寒“讨好”她，一定是为了要她替他追朱小溪。
<br/>
<br/>        诶？！等等！
<br/>
<br/>        好像哪里不大对劲！
<br/>
<br/>        连星儿想了想，没想出个所以然来，索性不想了。
<br/>
<br/>        就在此时，她的“小纽扣”终于给她发了一条简讯。
<br/>
<br/>        星儿，你们的军训，这个星期五就结束了吧？周六是否有空？我会来临海城。——小纽扣。
