　　大禹治水之后，在人族的威望更进一步，直逼三皇。数百年时间匆匆而过。
<br/>　　大禹的修为也一直在进步，两百年前便突破到了准圣境界，有人族的气运在辅助，就仿佛是天地主角，运来天地皆助之，又有无数资源供给，修炼起来自然快速无比。
<br/>　　是日，大禹突然将自己的证道法宝镇世青铜九州鼎取出，他手中不断打出各种玄奥印诀，随着时间的推移，那九州鼎开始晃动起来。
<br/>　　“分！”
<br/>　　突然，夏禹大喝一声，眼前九州鼎一分为九，分成九个青铜大鼎，每个青铜鼎上都刻满了神州每个州的山河社稷图，仿佛一个鼎，便囊括了一个州。并且每个鼎上都刻着一个字，分别是兖、杨、荆、青、梁、雍、幽、冀、徐、豫。
<br/>　　九州鼎，一分为九，每州一鼎，一鼎镇一州。
<br/>　　“去！”
<br/>　　突然，大禹大喝一声，将手中九鼎抛起，九鼎化作九道流光，朝着九个不同方向飞去。最后，每鼎对着一州，落了下去，没入大地之中。
<br/>　　“起！”
<br/>　　大禹手捏印法，突然一者清风吹起，一道玄妙的光线传出，之后将九个神鼎相连，最后按照某种特定的轨迹，运转起来，同时，一条无形的苍龙横贯于神州天地之上，最后缓缓隐没，光线也消失不见。
<br/>　　“嗡……”
<br/>　　同时，人族每个人心中都升起一股莫名的轻松，仿佛有一种来自心底的喜悦之情。
<br/>　　“怎么回事？我怎么有点兴奋呢？”
<br/>　　“我也是。”
<br/>　　“……”
<br/>　　人族祖地，众多人族祖老、武祖玄天也在此时睁开眼睛，看了一眼人族九州，九州相连，气运永存，青铜九鼎，能够镇压人族气运无数年，使人族气运绵长，天长地久，更能抵御外族的入侵。
<br/>　　在明白了原因后，众人都兴奋了。
<br/>　　“大禹这人，果然厉害，我人族将气运将永存于世。”
<br/>　　天庭、昆仑山、五庄观、北冥宫、幽冥血海、须弥山……每个人都心中一叹，人族算着彻底成长起来了，成为洪荒第一大族，为天地主角。
<br/>　　人族都城之中，大禹在做完这一切之后，有种心累的感觉，做这样的事情，就算以他如今的修为，也不是很容易，对他损耗很大。
<br/>　　但紧接着，他便感受到一股轻松之意，有种大解脱，大自在，大喜悦的心情。
<br/>　　同时，天上开始聚集无量功德，无边金色功德汇聚成云，染黄了整片天空。功德降落，直接落入大禹身体之中，无数感悟随之而来，助大禹悟道，他对大道的领悟在飞速提升着，同时修为也在快速进步。
<br/>　　转轮境初期……
<br/>　　凝聚第二道**，转轮境中期……
<br/>　　凝聚第三道**，转轮境后期……
<br/>　　体内天地阴阳风水山泽雷九大神鼎开始汇聚，隐隐有融合的趋势。
<br/>　　“轰——”
<br/>　　一股莫名的气息引动，大禹的身体突然一震，他体内的九鼎完全凝聚成一个大鼎，大鼎呈紫金色，一口，两耳，三足，像是在演绎这道生一，一生二，二生三，三生万物的道与理。
<br/>　　紫金大鼎鼎身上的龙之九子早已消失不见，取而代之的是一条巨大的九爪紫色神龙，神龙横贯其上，隐隐有种睥睨天下的威势。
<br/>　　大禹的修为彻底突破到了半步混元，亦是半步帝境。可惜，功德已经消耗完了，他的三道**还没有合一，代表着他的大道还没有彻底完善，不过，他却是证道了，证得人族五帝之一。
<br/>　　至此，人族三皇五帝彻底圆满，人族气运大涨，坐实了天地主角之位，此势已成定局，不容更改。
<br/>　　当然，天地间从来没有什么永恒的主角，真正主角永远是实力最强的那个人。这个世上，拳头便是法则，实力便是气运，强者便是主角，自古至今，不外如是！
<br/>　　在大禹证道大帝之位的同时，元始徒弟玉玲珑也吸收了教导人族大帝的功德，斩去善尸，彻底突破到了准圣境界，而另外三个也或多或少的得到点功德，修为虽然没有太大的进步，但好歹也有精进。
<br/>　　“恭喜师姐（师妹）突破准圣！”
<br/>　　旁边，玄青、白衣、风晓月向着恭喜道，白衣与风晓月眼中更是流露出强烈的羡慕之情，他们虽然也到了大罗金仙后期，但想要突破准圣，还需要很长一段时间，毕竟领悟不够。
<br/>　　“也恭喜你们修为进步。”玉玲珑脸上带着喜意，对着三人说道。
<br/>　　“我们哪有师姐你进步大啊，你都准圣了，我们还在大罗境界呢。”白衣与风晓月羡慕道。
<br/>　　“你们早晚也会有如此机会的，好了，我们的任务也完成了，去跟大禹说一句，也该回去禀告老师了。”玉玲珑开口道。
<br/>　　“嗯。”四人自然是没有意见。
<br/>　　就在他们想要走的时候，一道神秘的声音传入他们的脑海之中，让他们的脚步一滞：
<br/>　　“禀告就不必了，本座也没有在昆仑山，你们可以自行回去，或者在洪荒历练一番，再回去。”
<br/>　　“是老师的声音，不会错，就是老师的声音！”
<br/>　　四人顿时激动了，不过，他们也知道自己的老师恐怕并没有在附近，刚才那只是传音罢了。
<br/>　　四人慢慢回过神来，便向着都城走去，他们想要向大禹告知一声，便去游历大陆。
<br/>　　……
<br/>　　大禹退位之后，便传位于他的儿子夏启，改变了原始王朝的禅让制，开创了沿用千万年的世袭制的先河，自此，国天下变成家天下，国家中央王室血脉上有宗法关系，政治上有分封关系，经济上有贡赋关系。
<br/>　　夏启成为共主之后，组建军队，修建城池，制定刑法，建立了人族历史上真正意义地第一个奴隶制王朝——夏朝，之前的虽然也叫王朝，但实际上大王只是相当于领袖，而不是帝王。
<br/>　　自此，神州大地亦称华夏！
<br/>　　此时，人族帝位的争夺已经引不起洪荒大能们的关注了，诸圣更是没有在意，就算是人族祖地之中，对此也保持平常态度，公天下变家天下，又何尝不是一种文明的进步呢？
<br/>　　“世袭制取代禅让制，这是时代发展的必然，人族乃是智慧种族，拥有无限创造力，但随着时代的进步，人族的私心便也会越来越重，这是无法阻拦的。再说，没有私心，又怎么会有动力，没有动力，社会又如何进步？”
<br/>　　一个神秘的秘藏里，又换了一个身份游玩的元始看着人族部落喃喃低语道。
<br/>　　“洪荒经此以后，倒是可以平静一段时间了，不过，平静的背后，却又是酝酿着无尽凶险，我仿佛看到了下一量劫的腥风血雨了。”
<br/>　　元始站在山头，默默望着远处染红天际的云霞，久久不语。
<br/>　　旭日西落，又何尝不是大日东升的开始？
<br/>　　……
