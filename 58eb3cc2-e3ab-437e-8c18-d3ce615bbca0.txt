　　第217章
<br/>　　李乐天是一个很敬业的王子，做人坦坦荡荡，做事兢兢业业，关心民生，经济，军事，既能和文官诗词歌赋，也能和武将谈笑风生。而一旦有人提起，他必然会说：唉，我是替父皇分忧，可对皇位没兴趣喔。
<br/>　　是个人都能看出李乐天的野心，是个人也能看出他……没多大机会。
<br/>　　明帝陛下有很多王子，若真说有人对皇位没兴趣，可能就是六皇子李行哉，而偏偏就是他机会最大。
<br/>　　六皇子李行哉，嬉笑怒骂，视宫廷威仪无无物，他能和戏子做朋友，也可以在青楼大醉数月。身为一个皇子，未来皇位的继承者，如此作践自己的可以说有史以来独一份。
<br/>　　而就连他那些贩夫走卒的朋友都说，六皇子做个朋友是不错的，但若是做皇帝，怕是不行。
<br/>　　如此行径，经常将明帝气得暴跳如雷，见到他就恨不得揍其一顿。可明帝这么多儿子，挨过他打的也只有六皇子一个而已。
<br/>　　所有人都说，只要六皇子稍微上进一些，其他皇子就不用争了，反正争也争不过。在这些皇子中，有兢兢业业的，有假装淡泊不争为争的，有处心积虑的阴谋家……可这些加在一起都斗不过一个纨绔，于是大家什么心情，也就可想而知了。
<br/>　　可六皇子失踪了。
<br/>　　下落不明，音信全无，其他王子都觉得自己机会来了，于是又都蹦了出来。
<br/>　　百里胜在赶往王府的路上，他不是很喜欢参加今日的聚会。帝国诗风盛行，一旦聚会，看罢歌舞，那帮文官就开始舞文弄墨。
<br/>　　“喔，我今日兴之所至，得诗一首，与诸位念念。”
<br/>　　却是不知道你在家里头发揪掉了多少，才做出一首诗，就想趁这个机会人前显贵，在歌姬口中一传，搏个名动京师。
<br/>　　还特么兴之所至。
<br/>　　可无奈何，武将这边都是些大老粗，每次想插话也插不上，只能喝着闷酒，受着文官团体的嘲笑。
<br/>　　百里胜走到王府门口时，正好和程大雷碰在一起，百里胜不想和程大雷走在一起，若是二人并肩而行，很容易让人误会自己和程大雷是一伙的。
<br/>　　现在的情况是，百里胜想让程大雷以为和自己是一伙的，但不想让别人以为他们是一伙的。
<br/>　　这样，程大雷就能为自己卖命，而当他闯祸的时候，也不会牵扯到自己。
<br/>　　程大雷和徐神机、徐灵儿先百里胜进去，见这里果然很热闹，大殿内载歌载舞，士子席地而坐。程大雷只是一个不入流的小官，位置紧挨着大殿门口。坐在这里，就连大王子的模样都看不清楚。
<br/>　　程大雷看了半晌，也摸明白李乐天摆这次宴会的用意。
<br/>　　武举在即，把一些有名望，有潜力的举子叫到一起，显示你们是天子门生，帝国皇子都请你们喝酒了，你们以后还不多想想精忠报国。
<br/>　　一来士子可以混个脸熟，二来为朝廷招揽一下人心，互相都有好处。
<br/>　　第一个节目是剑阵舞，二十名皇城禁卫身着黑衣黑甲，长剑舞时，一片杀气泼出，大殿内顿时寒了几分。
<br/>　　程大雷也看的全神贯注，神情竟一时呆住。
<br/>　　“好。”宋天函赞了一声，道：“如此儿郎怕是我帝国才有，九江，你素有才子之名，可否就此赋诗一首。”
<br/>　　来啦！
<br/>　　所有人心中呼了一声，程大雷奇怪的看到，自己身边一人掏出一张小纸条，默默背诵着。
<br/>　　程大雷却是不知道，一个出头的机会对这些士子有多重要。今天这么多达官贵人都在，若自己能拿出一首了不起的诗，在京城扬名，岂不是就能引起主考官的主意。
<br/>　　大概类似于选秀的时候，选手先来段表演，给评委留下印象。
<br/>　　于是许多人都等着今天呢，有些文采的提前好几天就开始揪头发，没有文采的，只能花钱请人代笔了。
<br/>　　而李乐天，宋天函，百里胜等知道大家的心情，自然要给他们一个机会表现一下。
<br/>　　曲九江起身，向冲李乐天拜了一拜，道：“才子之名，愧不敢当，刚才倒是的确偶得两句，那屈某就献丑了。”
<br/>　　不知多少武将咧嘴，还特么偶得几句，还不知道你准备了几日。
<br/>　　只见曲九江走到案前，挥毫泼墨，片刻一首七言律师已经落在纸上。李乐天看了一遍，微微点头：“果不愧京州才子。”
<br/>　　如此算是给曲九江这个才子的名头盖戳了。
<br/>　　一众士子赞不绝口，此时柳轻名站了起来，道：“柳某是个武夫，不懂诗词，刚才也得了两句，屈兄既然有才子之名，不如请兄台指点一二。”
<br/>　　柳轻名也走到案前，笔走龙蛇，李乐天看罢后赞道：“当真称得上文武双全呐。”
<br/>　　曲九江得了个京州才子，柳轻名得了个文武双全，大家都知道，这次武状元，二人是最有力的争夺者。
<br/>　　在二人之后，纷纷有人站起来说：“某也想到几句，还请诸位赐教”
<br/>　　李乐天看着这一幕，哈哈笑道：“帝国实在是人才济济，那今日小王、常忠伯、宋大人就为大家做个评判，选出其中好的，由一支桃姑娘传唱。”
<br/>　　如此就拉开了今夜诗词大会的序幕，习得文武艺，货与帝王家，谁不想人前扬名。如今的帝国，一战扬名有点扯淡，一诗扬名的机会还是有的，往年也不是没有突然窜出黑马的事情，每个人都以为自己是黑马。
<br/>　　一番评点，自然是文官占优，武将太弱。一群糙汉子聚在一起，脸黑得像锅底一样。
<br/>　　“喂，你听的懂么？”
<br/>　　“听不懂啊，谁知道他们瞎扯什么淡。”
<br/>　　“打仗，靠写诗也能打仗么？”
<br/>　　“这帮酸文人太猖狂了，若是此刻有人能教训教训他们就好了。”
<br/>　　程大雷此刻想溜，因为他发现一支桃竟然也在场，这若是被认出来，自己的计划岂不是竹篮打水一场空。
<br/>　　“牛三斤牛校尉，听说你胸藏锦绣，腹隐山河，可否就此情此景赋诗一首？”李乐天。
<br/>　　猫着腰想溜的程大雷被喊住了，他一回头，发现许多人的目光都聚集在他身上。
<br/>　　『胸藏锦绣，腹隐山河』
<br/>　　谁特么造得谣！
