　　回归萧家后。
<br/>　　萧逸并没有在家族多留。
<br/>　　而是带着小星前往雷鸣荒泽，日夜牵引滚滚天雷修炼。
<br/>　　时间飞跃。
<br/>　　一月时间，眨眼便过。
<br/>　　雷鸣荒泽某座高山之上，萧逸****着上身。
<br/>　　滋滋…
<br/>　　一阵阵雷电，在他那精壮而极其匀称的身躯上，劈啪作响。
<br/>　　咔咔…
<br/>　　爆炸性的肉体力量，在他的拳头上充斥着。
<br/>　　一月时间的天雷炼体。
<br/>　　他的肉体力量，已达地元八重。
<br/>　　今日，他停下了修炼。
<br/>　　因为，滚滚天雷，已对他无用。
<br/>　　这种炼体办法，终究有着限度。
<br/>　　不过，他的收获却极其巨大。
<br/>　　因为，他的肉体力量是地元八重，但修罗战体的修炼程度，只在地元五重。
<br/>　　简单来说，天雷炼体，是让他原本地元五重的肉体力量，飙升了三重强度。
<br/>　　而他体内的修罗之力等等，丝毫没有增加。
<br/>　　日后，待得他将修罗战体第四层真正修炼到巅峰。
<br/>　　也就是地元九重巅峰的程度时。
<br/>　　他的肉体力量，会远远超过本身修罗战体赋予的地元九重力量。
<br/>　　地元九重程度，绝对有着不亚于天元境的实力。
<br/>　　而且还是不借助修罗之力的情况下。
<br/>　　当然，他现在的武道修炼和实力，其实已经远远超过了肉体力量。
<br/>　　但这毫无办法。
<br/>　　修罗战体，本就是逆天功法，修炼的难度和限制极大。
<br/>　　高级的妖兽精血，太难得到。
<br/>　　修罗战体的修炼，本就停滞了许久。
<br/>　　轰…轰…轰…
<br/>　　这时，天雷仍旧在高空中肆虐。
<br/>　　萧逸虽停下了修炼，但小星没有。
<br/>　　滚滚天雷，乃是小星的大补之物。
<br/>　　一月时间的吸收，加上萧逸赋予的大量高品丹药支撑下。
<br/>　　小星的实力，从地元一重，飙升至地元六重。
<br/>　　小星继续修炼着。
<br/>　　萧逸，则自顾地走到一边，取出了游云杀步和游云杀剑，参悟了前来。
<br/>　　之前在剑阁中得到的，只有游云杀步，而且还是不完整的。
<br/>　　而剑宗大长老交予自己的，是完整的游云杀步和游云杀剑。
<br/>　　事实上，萧逸对此很是疑惑。
<br/>　　自己与段云长老间，并无太大交情，更不是他的弟子。
<br/>　　他的一生传承和绝技，怎么可能这么轻易交给大长老。
<br/>　　再由大长老交到自己手上。
<br/>　　这其中肯定有着什么自己不知道的原因。
<br/>　　萧逸未能想明白，但并未多想。
<br/>　　之前并不完整的游云杀步，尚且给予了自己一定的实力增幅。
<br/>　　如今完整的游云杀步，加之有游云杀剑配合。
<br/>　　其威力，必定强得多。
<br/>　　半月时间，转瞬便过。
<br/>　　萧逸停下了参悟。
<br/>　　他的武道天赋，极其卓绝。
<br/>　　半月时间，已然将这两门绝技参悟透彻。
<br/>　　同一时间，滚滚天雷之下的小星，也结束了修炼。
<br/>　　小星的修为，达到了地元八重。
<br/>　　此时，四根引雷木，对他也无大用了。
<br/>　　引雷木的真正功效，其实远不止如此。
<br/>　　否则不可能被药王谷用于镇压星辰之火。
<br/>　　不过，药王谷是以四根引雷木为基，配合阵法，镇压的星辰之火。
<br/>　　如无意外，应该是这阵法，能引动更恐怖的天雷。
<br/>　　萧逸并不知这阵法，自然也没办法了。
<br/>　　一人一兽的修炼，暂时在这里划下了帷幕。
<br/>　　随后，双双御空飞回萧家。
<br/>　　一月半来，萧家并未再受到袭击。
<br/>　　萧逸已经知道背后对付萧家之人，就是血雾谷。
<br/>　　血雾谷，估计也是已经知晓萧逸回到了北山郡，知道奈何不了萧家。
<br/>　　便再没有动作。
<br/>　　……
<br/>　　接下来的数天，萧逸没有修炼，也没有外出。
<br/>　　一直在家族中。
<br/>　　不时陪一下长老们，指导一下族人的修炼。
<br/>　　这样‘空闲’的生活，对于惯了四处闯荡的萧逸来说，显得很是温馨。
<br/>　　但这种温馨的生活，只维持了数天。
<br/>　　萧逸很清楚，自己还有许多事要处理。
<br/>　　议事厅内，萧逸告别长老们。
<br/>　　萧重最是不舍，但还是说道，“逸儿，我们就知道，你要不了多久又要离开的。”
<br/>　　萧逸笑笑，手中光芒一闪，出现了一把锋芒利剑。
<br/>　　利剑上，散发着一股妖异光芒。
<br/>　　正是血戮剑。
<br/>　　“极品灵器？”长老们脸色一惊。
<br/>　　萧逸点点头，道，“剑上有我烙印下的一道禁制。”
<br/>　　“禁制可随时触发。”
<br/>　　“除非是天元九重武者来袭，否则，萧家可保无虞。”
<br/>　　“什么？可挡天元九重强者？”众长老脸色惊骇。
<br/>　　萧逸肯定地点了点头。
<br/>　　血戮剑上，烙印了一道‘寒冰裂天斩’。
<br/>　　而且还是萧逸耗尽了体内所有元力，压缩而成。
<br/>　　其威力，甚至比之前对付‘风云二老’，以及‘血成河’时，强得多。
<br/>　　哪怕巅峰强者来了，都绝对讨不了好。
<br/>　　若是巅峰强者以下，则根本抵挡不了。
<br/>　　当然了，如此强大的攻击，等闲之物，根本承载不了。
<br/>　　萧逸只能将其烙印在血戮剑上。
<br/>　　不过，萧逸猜测。
<br/>　　若这道寒冰裂天斩真的被释放出来，恐怕血戮剑也会直接被毁坏。
<br/>　　血戮剑，只是极品灵器。
<br/>　　以前，无论是剑派考核，还是在剑宗的考核和比武，都给予了自己极大的帮助。
<br/>　　但以自己现在的实力，已用途不大。
<br/>　　当然了，若是血戮剑损坏了，萧逸会想办法修补回来。
<br/>　　另外，事实上萧逸现在的最强手段，是四种强悍火焰的融合。
<br/>　　不过，这四种强悍火焰，外人根本无法操控得了。
<br/>　　加之在不稳定的情况下，会直接爆炸，太过危险。
<br/>　　故萧逸并没有留下这样的手段。
<br/>　　言归正传。
<br/>　　萧逸再次交待了一声，“这把剑上的禁制虽强，但只能使用一次。”
<br/>　　“若萧家真的有危，立刻通过猎妖令子令通知我。”
<br/>　　“我会立刻赶回来。”
<br/>　　“好。”萧离火点了点头。
<br/>　　交待好一切，萧逸再次对着长老们行了一礼。
<br/>　　而后彻底离去。
<br/>　　飙升到极致的速度，在天际划过一道绚烂的流光。
<br/>　　他不得不回王都。
<br/>　　无论是为了萧家，还是裂天剑派。
<br/>　　宗门，已经开始着手要脱离北山郡裂天剑派了。
<br/>　　对于裂天剑派而言，剥除这个身份，绝对不是没了一个‘头衔’那么简单。
<br/>　　少了北山郡武道圣地这个身份，对裂天剑派来说，必是致命的打击。
<br/>　　另外，对萧家而言。
<br/>　　既然血雾谷要对付萧家。
<br/>　　萧逸就更加要赶回王都。
<br/>　　自己在王都，可以随时牵制血雾谷的真正强者，例如血谷主和血成河。
<br/>　　而只要这两人不出手。
<br/>　　有‘寒冰裂天斩’以及两个六级傀儡护身的萧家，必可保安全。
<br/>　　……
<br/>　　萧逸，正往王都而回。
<br/>　　而在大半月之前，王都血雾谷内。
<br/>　　议事厅中，再次强者齐聚，似在讨论着什么，个个脸色难看。
<br/>　　“该死，萧逸怎会赶回北山郡去了。”在座的某个血雾谷长老，愤怒地说着。
<br/>　　“金大师，你不是说萧逸绝不可能赶回北山郡吗？”
<br/>　　“我们的计划，本就没将萧逸算进去，这下子，全失败了。”
<br/>　　血成河闻言，脸色也是有些难看。
<br/>　　“我也不知道，那小子，被我废了一臂，又中了我的金轮之火。”
<br/>　　“理应在很长时间内都无法恢复伤势才是。”
<br/>　　“怎么可能一下子就伤势痊愈，还赶回北山郡去了。”
<br/>　　“再者，对付萧家之事，我们做得如此隐秘，他怎么可能知道家族有危？”
<br/>　　“事情不可能这么巧合，其中必有什么我们不知道的事。”
<br/>　　一直没有说过话的血谷主沉声道，“无论什么原因，我们很难再对付萧家了。”
<br/>　　“剑宗四长老，既已在北山郡裂天剑派看到萧逸。”
<br/>　　“而以萧逸此子一直追查血意丹之事，很可能已经查到我们与圣子以及剑宗长老的事了。”
<br/>　　“此子心智过人，恐怕已经猜到是我血雾谷对付他们萧家了。”
<br/>　　“难道我们就这样放过萧家？”一个长老冷声说道。
<br/>　　在他旁边，是‘楚楚可怜’的慕容娇儿。
<br/>　　显然，慕容娇儿没少在这位长老面前诉苦。
<br/>　　“当然不是。”血谷主冷笑一声。
<br/>　　“我们不对付他，但有别人会对付他的。”
<br/>　　“这一点，我已经收到确切的消息。”
<br/>　　“呵呵，我敢肯定，萧逸此子要不了多久一定会回来王都。”
<br/>　　“届时，他一定会惊掉下巴的，不，应该是害怕。”
<br/>　　“哈哈哈哈。”
<br/>　　血谷主狰狞的笑声，响彻整个议事厅。
