　　骆妤和沁儿两人，从执事那里了解一番，便轻易的就找到苏莫的阁楼。
<br/>　　不过，面对苏莫闭关不出，两人有些无奈。
<br/>　　最终，两人只能暂时作罢，准备待苏莫出关以后再来。
<br/>　　玉镯空间之中，苏莫双眸紧闭，不停的吸纳源石。
<br/>　　他的体内，混沌之力浩荡，在不断的凝练。
<br/>　　五行源气进入体内，化为混沌之力，再加以凝练，只留下精纯的混沌源力。
<br/>　　混沌源力，乃是混沌本源之力，比普通的混沌之力，要更加的强大。
<br/>　　虚神四变，突破到真神之境，是一个脱变的过程。
<br/>　　需要大量的源气支撑，好在苏莫得到了大量的五行源石，可以不断的冲击。
<br/>　　时间如同白驹过膝，转瞬间便过去了两年。
<br/>　　苏莫的五行源石，消耗了足有七八十万之多。
<br/>　　两年时间，他闯过熔岩窟第十八层之时的浪潮，也渐渐的消退了。
<br/>　　人王宫之中，虽然很多人都知道，宫中出现了一个妖孽，但是，并没有多少人见过。
<br/>　　而一些高层长老，则并没有人主动来找苏莫。
<br/>　　毕竟，苏莫虽然厉害，那些神王境的长老，还不至于上门收徒。
<br/>　　玉镯空间之中，苏莫全身源力鼓荡，气息深邃浩大，他距离真神境已经不远了。
<br/>　　其实，他的混沌源力，已经凝练到了极致，但是，法则限制了他的突破。
<br/>　　本源力量，需要法则之晶来平衡，才能达到真神之境。
<br/>　　没有法则之晶，本源之力无法彻底凝练成功。
<br/>　　法则，是奥义的晋阶，而他的奥义之力远远不够。
<br/>　　这两年，苏莫一边修炼，一边参悟吞噬奥义。
<br/>　　靠着拥有吞噬元神，他直接参悟元神中的吞噬之力，吞噬奥义进展很快，已经达到了九阶初期。
<br/>　　修炼无岁月，苏莫抛却了一切杂念，安心的修炼。
<br/>　　一晃眼，三年的时间又过去了。
<br/>　　苏莫的吞噬奥义，达到了九阶大圆满。
<br/>　　九阶圆满的吞噬奥义，开始向吞噬法则转化。
<br/>　　苏莫所要做的，便是凝练吞噬法则，晋升真神之境。
<br/>　　一切，都有条不絮，和当初凝练奥义之晶非常的相似。
<br/>　　又过去了半年，苏莫身上陡然升起了一股庞大的气势，如妖似魔，如渊似海，汹涌浩大。
<br/>　　这股气势一出，玉镯空间开始了大崩溃，舔她地陷，一片末日般的场景。
<br/>　　唰！
<br/>　　苏莫立刻睁开了眼眸，全身气息收敛，闪身离开了这片空间。
<br/>　　这空间玉镯，实在是太低级了，已经承受不住他的气息了。
<br/>　　不过，他身上还有空间宝器，实在神路得到的圣器。
<br/>　　阁楼之上，苏莫闪身而出，深深的吸了口气。
<br/>　　他终于晋升到了真神之境，百岁之内成神了。
<br/>　　他如今的年纪，也才七八十岁而已。
<br/>　　虽然他目前，只是一品低级真神，但是，他的战力，不是任何一品真神可以比拟的。
<br/>　　少倾，苏莫在阁楼中坐了下来，仔细思考拜师之事。
<br/>　　不过，考虑了片刻，他暂时放弃了这个想法。
<br/>　　现在贸然去拜师，实在太过冒险了。
<br/>　　因为，他对于人王宫内的那些神王长老，完全不了解，包括火陵神王。
<br/>　　所以，不能确定后果的事情，他不能去做。
<br/>　　诚然，他若是拜了火陵神王，或者是其他神王为师，很可能得到重点培养，地位飙升。
<br/>　　但是，若是这个神王，和天王宫或者是地王宫有联系呢？
<br/>　　要知道，六大王宫，本来都是轮回天尊的下属，六大王宫之间，关系千丝万缕。
<br/>　　虽然轮回天尊不再了，六大王宫也各自为政，但是，六大王宫的联系，绝对不可能断了。
<br/>　　所以，他若是贸然去拜师，太过危险了。
<br/>　　比如，那个火陵神王，若是和天王宫或者地王宫的某位强者是至交，那么他去拜师，不是自投罗网吗？
<br/>　　苏莫放弃了拜师的想法，此事等日后再说吧！
<br/>　　随即，苏莫手掌一番，掌中出现了一个金色的光团。
<br/>　　这金色的光团，真是火陵神王的一部分道统。
<br/>　　苏莫虽然不准备修炼火陵神王的道统，但是，也想要查看一番，了解一番。
<br/>　　随即，他意念一动，裹挟住了金色的光团。
<br/>　　仔细的感知，没过多久，苏莫便发现光团之上，有一层无形的防御，让他无法窥探其内的信息。
<br/>　　他开始用意念炼化这层防御，此防御不清楚是如何布下，但是，只有精神意念才能感知的到。
<br/>　　耗费了整整三个时辰的时间，苏莫才将这层防御炼化。
<br/>　　而这个金色的光团，也化为无形的气流，进入了苏莫的脑海之中，其中的道统，立刻被他全部吸收。
<br/>　　“火源之道！”
<br/>　　苏莫静静的查看，此道统内容不多，但是，的确是不凡。
<br/>　　这是一种特殊的火属性修炼之法，不仅玄妙，而且还能让火属性之力极度强大。
<br/>　　“可惜，却是不适合我！”
<br/>　　查看完了这道统，苏莫无奈的叹了口气，完全不适合他。
<br/>　　当然，也并不是说完全不适合，他若是修炼的话也可以。
<br/>　　能让他的火属性之力威力倍增，但是，火属性之力太强，其它属性弱的话，就不够平衡了。
<br/>　　所以，他完全不能修炼，除非，其它四种属性之力，也等得到类似的道统，五种属性道统同修。
<br/>　　不过，这显然不太现实。
<br/>　　“此道统，日后传于司空炎！”苏莫心中暗道。
<br/>　　其实，修为到了真神之境，已经没有功法的限制了。
<br/>　　因为，修炼之路是固定的，而如何去走，则看每个人的想法了。
<br/>　　比如，到了真神之境，根本没有神级的功法，有的只是道统。
<br/>　　好比修炼火系之道，到了真神之境，不存在改修神级的火属性功法了，只能说找到适合自己的道统。
<br/>　　这个道统，可以修炼前人的道统，也可以走出自己的路。
<br/>　　因为，每个人的元神、体质奥义等等都不一样。
<br/>　　道统，则包含了元神、体质、法则、属性之力等等。
<br/>　　而苏莫，他只能修炼自己的道统，他的混沌道统，也只是混沌功法而已。
<br/>　　至于元神、血脉、法则等等，没有人和他相同，他只能走自己的路。
<br/>　　“还是先成为内宫弟子吧！”
<br/>　　深深的吸了口气，苏莫打开了大门，走出了阁楼。
