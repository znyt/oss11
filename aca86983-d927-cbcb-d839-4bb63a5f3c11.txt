　　下午时分，涂小安来到了副院长的办公室，见这个老头直接朝他伸出手。
<br/>　　涂小安还能不懂吗，将青枭的九彩色羽毛抛了过去，直接飘到了老头的眼前。
<br/>　　“这羽毛真是漂亮，不愧是仙灵孔雀的羽毛呀”
<br/>　　副院长笑眯眯了起来，看着他道：“你要进超等天才班的事情我已经安排好了，等下就带你过去挑战一人，打败了，你就可以顺利加入天才班了”
<br/>　　涂小安其实对进不进天才班兴趣不是很大，只是当初被白君然给挑衅的。
<br/>　　问道：“天才班，跟普通班有什么区别呀”
<br/>　　“待遇不同”
<br/>　　待遇？涂小安蹙眉，这又不是去上班，还有什么待遇呀。
<br/>　　副院长道：“天才班每个学员每个月会被分配到十颗的灵石，要是排名在前五的学员，将会有二十颗，而上等班的学员就只有五颗，下等班一颗”
<br/>　　灵石？
<br/>　　还有这种东西啊，涂小安没见过。
<br/>　　“老头，你拿一颗给我看看”
<br/>　　副院长见他的样子，诧异：“你堂堂白镇蛇王，没见过灵石什么样？”
<br/>　　“我在小说中见过，现实中没见过”涂小安也不脸红。
<br/>　　“真不知道你这条蛇精是怎么修炼的，不过也不能怪你没见过灵石这种宝物，一般也只有我们人类修武者才有”
<br/>　　说着副院长朝着涂小安扔过来了一颗晶莹剔透的石头，这块石头很小，大概只有刚出生婴儿的拳头那么大。
<br/>　　“好强的能量”
<br/>　　涂小安徒手接住灵石，有股冰凉，最关键是里面蕴含灵气十足，能量充沛，很明显就感受到。
<br/>　　“叮...下品灵石，宿主吸收可以获得100点的进化能源”
<br/>　　我去，一百点进化能源，这等于是涂小安吞噬了一条觉醒生物了。
<br/>　　“吸收”
<br/>　　涂小安的手紧紧的握住灵石，顿时间，灵石闪烁起来，里面纯粹的能量被快速的吸走，旋即晶莹剔透的灵石变成了普通的石头，然后粉碎了开来。
<br/>　　进化能源：3300/5000。
<br/>　　“小子，谁让你吸收灵石的”
<br/>　　副院长站了起来，又诧异道：“你既然可以在几个呼吸内，将一块灵石里面的能量彻底吸收，怪物呀”
<br/>　　一颗灵石，让一阶的修武者修炼，需要十天的时间，才能彻底吸收里面的能量，
<br/>　　被二阶的修武者修炼，也要三天的时间。
<br/>　　大惊小怪，涂小安吞噬生物，直接就化为进化能源了，这算什么。
<br/>　　不过这话，他自然也不会说。
<br/>　　“这灵石不错”涂小安笑了笑。
<br/>　　副院长冷哼一声：“当然不错，只有成为古学院的正式学员，才能每个月得到灵石，不然你以为各方势力都将家族子弟送过来干嘛，有绝大部分的原因，是为了灵石”
<br/>　　一颗灵石100点，十颗1000点，一百颗一万点。
<br/>　　涂小安顿时呼吸都重了，这东西好呀，五十颗，就可以让他提升一级。
<br/>　　要是这样，他靠着灵石，就能轻松提升到三十级。
<br/>　　“老头，在多给我几颗”涂小安贪婪了起来。
<br/>　　副院长冷笑：“你当这是什么呀，糖果吗，每一颗灵石都得来不易，经国家编号掌控起来”
<br/>　　“古学院是国家人才的摇篮，才会每个月下发一部分给学员修炼”
<br/>　　涂小安明白了：“你刚才说进入天才班每个月可以获得十颗，要是实力排名班级前五，可以每个月二十颗对吧”
<br/>　　“没错”副院长点了点头。
<br/>　　“那我过去将里面排名第一的学员打败，是不是就替代了对方的排名”
<br/>　　“哼，想的挺美”副院长自然明白涂小安打的什么鬼主意：“新学期，学员的实力排名会重置，不过不是通过个人的挑战，而是通过学院特殊的试炼任务”
<br/>　　涂小安道：“那么麻烦呀”
<br/>　　“你也不用急，你现在要做的是进入天才班，因为新学期开始，会发放一次灵石”
<br/>　　“什么时候”
<br/>　　“三天后”
<br/>　　涂小安懂了，笑了起来，按这样说，三天后，他就可以获得十颗灵石了，也就是轻松得到一千的进化能源。
<br/>　　而且不光他有，别的学员也有，嘻嘻。
<br/>　　“老头，现在就带我去天才班考核吧，别浪费时间了，大家都挺忙”
<br/>　　副院长吹胡子瞪眼：“蛇精，你要是在敢乱叫，小心老夫将你赶出去，老夫已经对你很优待了，纵观古学院的历史，还没妖精可以进入古学院学习的”
<br/>　　“你不是有求于我吗”涂小安不吃他这一套。
<br/>　　他可是先答应了这个老头一个要求，以后指不定要他做什么呢。
<br/>　　不过古学院果然不简单，连灵石这种宝贝都有，以后肯定会有很多的惊喜等着涂小安。
<br/>　　“跟我走吧”
<br/>　　副院长亲自带路，涂小安这个最强新生非常有牌面，主要也是对方当心他乱来。
<br/>　　“等会你跟一个学员比试，记得点到为止”
<br/>　　副院长特地的交代了一句。
<br/>　　“如何才叫点到为止”
<br/>　　“对方认输，你就不能在出手了”
<br/>　　一会儿后，涂小安被老头带到了一个比武场上，里面已经汇聚了不少超等天才班的学员，一个个脸上布满了傲气，冷笑的看着涂小安这位新生。
<br/>　　副院长冲着人群中一位青衣男子招了招手，对方身法极快，化为青色影子掠了过来，站到了比武场地上。
<br/>　　“老头，你放心，我保证不打死他”
<br/>　　涂小安笑嘻嘻的丢下一句话，上去了，对他现在的实力来说，这不是比武，而是玩耍。
<br/>　　这个比武场，莫约有一百多个平方，让两个人比武，足够用了，不过这个比武台也没想象的那么简单。
<br/>　　涂小安眼神环视一圈，发现有好几处地方都藏着不少气息强大的修武者，而比武台的位置，竟正好能让古学院各个方向的人都处于可视的地方。
<br/>　　也就是说，你从各个方向，都能看到比武场的比试。
<br/>　　只要你的视力足够强，站在几公里处都可以。
<br/>　　恐怕这场比试，不但吸引了古学院的学员，还有很多古学院的高层都在密切的关注。
<br/>　　涂小安不禁暗暗的想，除了这个副院长老头，古学院中，到底有多少人知道他是蛇精的身份。
<br/>　　正常来说，涂小安现在的修为，只要不释放蛇类气息，三阶之下的人，是看不出他的伪装。
<br/>　　因为昨天他跟白君然对拼了一招，白君然压根他看出他蛇精的身份。
<br/>　　这个白君然，可是一只脚迈入三阶了，他都没看出来，古学院学员应该就没人能看破了。
