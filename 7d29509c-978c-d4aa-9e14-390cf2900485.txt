着很好的断玉能力！”南宫雪说道。<br/>　　<br/>　　“啊！要是这样，能把这位陈先生招到咱们公司，岂不是非常的好？”杨顾问说道。<br/>　　<br/>　　但是南宫雪却摇摇头，并没有接话。<br/>　　<br/>　　“怎么？难道这位陈先生不喜欢，还是其他？”杨顾问问道。<br/>　　<br/>　　“不是不喜欢，也不是其他，而是这个家伙他又其他的工作，断玉什么的，不过是打酱油的角色罢了！”想到这，南宫雪就有些咬牙。<br/>　　<br/>　　甚至想起南宫亮想掏钱买陈默炼制的丹药，那种钱无所谓，你随便说，只要丹药能卖给我就成的架势，就能知道丹药有多赚钱，那么也就说明陈默他自己不缺钱。<br/>　　<br/>　　大概给杨顾问说了一下关于陈默的信息后，杨顾问也是摇头，没有想到这么个年轻人，还真的有些妖孽了！<br/>　　<br/>　　“不过有些可惜了！”南宫雪将另外一块玉石拿在手里，翻看着，也有些感叹的说道。<br/>　　<br/>　　“嗯？可惜了？雪总，你说的可惜了是怎么了？难道这两块芙蓉种不好么？”杨顾问说道。<br/>　　<br/>　　“不是不好！而是很好，那么也就可惜了！我是在想，这两块毛料都有着这么好的表现，那么刚才那一块，被陈默看的很紧的毛料，会是什么表现呢？”南宫雪有些不明所以的说道。<br/>　　<br/>　　杨顾问一听到这个话，就想到陈默对他脚下那块毛料很是在意，那么岂不是表明陈默对那块毛料也是非常看好，也就是说，那块毛料绝对要比之手头的芙蓉种表现的好！<br/>　　<br/>　　“高冰种？！”杨顾问转头对南宫雪说道。<br/>　　<br/>　　“也不是不可能！”南宫雪也是有些郁闷的说道。<br/>　　<br/>　　“哎！可惜了！错过了啊！雪总，要是有可能，回去后多多询问一下这位陈先生，看看那块石头他买不买！”杨顾问也是一脸的便秘，没有想到可能是一块冰种的玉石，从自己眼前闪过溜走！富品中文<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
