　　张无生表示没事，让钟馗等自己一会，跟他聊聊天去——他们两个也是挺熟的。
<br/>　　“这老爷子咋了，失魂落魄的。”叶少阳看着张无生的背影，嘀咕了一声。
<br/>　　林三生不愧是来救火的，利利索索地将大军都安置妥当，连防卫工作也做好了，然后问叶少阳还有什么事，叶少阳也是刚回来，把瓜瓜叫过来问他。
<br/>　　瓜瓜表示没什么事，在他们离开这段时间，腾永清跟那个曲波也回来了，看他们不在，又主动去守城去了。“还有就是，我娘说她想吃柚子，或者酸梨，我不知道哪里有，我打算去给她找找去。”
<br/>　　“额，为什么想吃酸东西，不会是怀了吧？”
<br/>　　“怀什么啊，我连爹都没有！”瓜瓜愤愤不平地争辩，立刻跑走了。
<br/>　　“他娘是谁？”林三生好奇。
<br/>　　“说来话长，慢慢再告诉你。”叶少阳看了看左右，把他拉到角落里，道：“说正事，道风现在到底怎么样了？”
<br/>　　林三生看钟馗走了，四下没人，于是把事情跟叶少阳讲了一遍，叶少阳虽然也不能理解道风现在是什么样一个境界，但至少他又活了，而且芮冷玉跟他在一起，也就彻底放心了。
<br/>　　一直到第二天早上，关于“小四哥”的线索还是丝毫没有，大伙急得不行，这每拖一分钟下去，星月奴那边都有可能完成大阵啊。
<br/>　　这本来就是在跟声明赛跑，关键是他们什么也没干，到现在还在原地踏步。
<br/>　　林三生把自己关在房间里，苦苦思索，天快亮的时候，这种全无线索的事，居然真的让他想出了眉目，赶紧着急叶少阳一干人，聚集在一起讨论。
<br/>　　“这个小四哥，就在我们中间！”林三生第一句话就让大伙瞠目结舌。
<br/>　　“我们？”
<br/>　　“确切说，是我们这个阵营。你们先别着急，听我分析下，首先，这人一定不是普通人，必然有一定的实力，否则自身安危都没法保障，所以，这九星阁一定是在一个有强大实力的人体内，这样才能保证安全……”
<br/>　　大伙纷纷点头。
<br/>　　“还有就是，如果是普通人，没什么存在感，鲲鹏也不会想到他身上去，还有——”
<br/>　　“行行，你确定了就行，跳过这个。”叶少阳知道他喜欢长篇大论，赶紧叫停，急着听结论。
<br/>　　“所以这人有一定实力，八成跟鲲鹏是一个量级的，但不太可能是他们那边的人，一来，像鲲鹏那个等级的，那边也最多不超过十个，彼此也都熟悉，谁叫小四哥，鲲鹏按理不会不知道……
<br/>　　还有一点，万一真发生意外，鲲鹏要向星月奴汇报，必然要找到这个人，九星阁在这人体内，必须把他杀了——至少要把肉身毁了，星月奴不会这么坑自己手下吧？如果这人不愿意牺牲，鲲鹏跟他打，就一定打得过？有这么多的不确定因素，我想，星月奴这么谨慎的人，应该不会这么做。”
<br/>　　大伙听了，又觉得很有道理。
<br/>　　“那你快说啊。这个人在哪？”
<br/>　　“我说了，在我们这个阵营！先别忙着惊讶，大家最初进来，都是被抓了的，那时候被洗脑了，星月奴绝对有办法把九星阁放进一个人身体，但他自己又不知道……放在我们这个阵营的人身体里，是最安全不过的，就算我们拿到谜题，也绝对不会在自己人身上动手，同时，这个人跟鲲鹏一定是认识的，至少鲲鹏能查到他的底细……嗯，以上就是我的分析。”
<br/>　　大伙听得心中发凉，一个个都不作声。
<br/>　　小马嘀咕道：“你说星月奴这安排，我们绝对猜不到真相，现在不还是猜到了。”
<br/>　　“那是因为有我，星月奴想不到我会来，就算知道，她也想不到我竟然这么聪明。”林三生骄傲地挺着胸膛。
<br/>　　小马一头黑线。“你这个逼装的，我竟然无言以对。”
<br/>　　叶少阳沉吟了一下说道：“那不对啊，星月奴闭关有一阵子了吧，我们都是在她闭关之后进来的，时间也对不上。”
<br/>　　林三生道：“这个倒也不确定……因为所谓闭关，应该不是从一开始就得关在里头不能出来，我猜测是分阶段的，可能现在到了最关键的阶段，的确就得一心一意地修炼，但在这之前，多半是可以行动的，而且，在她闭关之初，我们都还没到承德，她犯不着这么提防我们。
<br/>　　所以，我猜应该是在大批法师来到承德之后，她担心出意外，正好那时候愿力也吸收差不多了，这才将计就计地施展了计划……”
<br/>　　叶少阳道：“这么说，那个九星阁有可能在我们任何一个人肚子里？”
<br/>　　他这么一说，大家都本能地摸着自己的肚子。
<br/>　　雪琪道：“我们是昨晚才来的，肯定跟我们没关系。”
<br/>　　吴嘉伟低声道：“这话可不要传出去，不然人人自危，不利于团结……”
<br/>　　林三生沉吟着点点头。
<br/>　　“啊！”小马突然大叫一声，“我知道了！”
<br/>　　一时间大家都转头去看他。
<br/>　　小马喃喃道：“会不会是……和尚？他名字里不是有个‘四’字么，而且人也被星月奴绑架了，这……”
<br/>　　大伙心中都颤抖了一下，求助地看向林三生。
<br/>　　林三生捻须不语。
<br/>　　叶少阳沉吟道：“应该不是四宝……”
<br/>　　“应该不是他。”林三生道，“星月奴没道理留这么浅显的谜语，如果能让人一下子猜到，那就没意义了……”
<br/>　　“如果星月奴想到咱们会这么想，所以故意反着来，故意弄得这么浅显，迷惑我们呢？”小青很没底气地插了一句，其实是将自己的担心说出来，希望林三生反驳。
<br/>　　林三生无奈摇头，“其实，是存在这种可能，毕竟人心不同，没法算到极致，不过我还是认为，星月奴这么狡诈的人，应该不会这么浅白……反过来想，如果九星阁在四宝体内，那她就没道理再控制四宝，把他不声不响地放走，反而不引人注意，你们想是不是这个道理？”
