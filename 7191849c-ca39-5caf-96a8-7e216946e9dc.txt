　　“十六万，好的，还有比十六万更高的价格吗？十六万第一次！十六万第二次！十六万第三次！恭喜这位先生！”
<br/>　　建中拍卖行，看着拍卖师一锤定音，纪敏长出一口气，然后狠狠地挥舞了一下拳头。
<br/>　　“小纪啊，恭喜恭喜，山泉药酒一炮而红。之后就不愁销路啦。”姜长飞看着纪敏激动模样笑道。他能理解这种事业成功的兴奋和满足感。
<br/>　　“还要谢谢姜大哥提携，不然我也没机会主导胖猫食品公司。”纪敏满脸感激。
<br/>　　“我只是推荐，能有如今成就是你自己够努力。”姜长飞感慨。
<br/>　　药酒项目还真是纪敏死皮赖脸搞来的。
<br/>　　“要不我也去撒泼打滚试试？”姜长飞想到那个小黑无人机，心里就觉得没底。
<br/>　　卖了那么长时间无人机，他对苏重的无人机小黑还是无解。
<br/>　　根据实验室人员估计，小黑无人机硬件没什么，但控制软件起码超过公司现有公司两代！甚至存在一定人工智能痕迹。
<br/>　　“老板真是个天才啊。”姜长飞下意识感慨。
<br/>　　“老板确实是个天才！”纪敏心有同感。自己当初拿着图纸都组装不起来净化水设备，苏重只用半天就轻松搞定。
<br/>　　还有活性水，他从未在其他地方见过。正是有活性水，山泉系列药酒才会那么成功。
<br/>　　“小纪小姜，恭喜恭喜啊。”孙建中大步走到两人身前道贺。
<br/>　　一品山泉药酒，没支都卖出十五万以上的高价。而且他知道山泉药酒的品质，口碑会越来越好。药酒项目已经大获成功。
<br/>　　“多谢孙老帮忙。”纪敏赶紧道谢。
<br/>　　“那你想怎么谢我？”孙建中笑呵呵。
<br/>　　纪敏一懵，这套路有点儿不对啊，这么直接的吗？
<br/>　　“一品药酒每月会给您单独供应！”纪敏认真道。
<br/>　　处于销售策略，也有材料限制原因。纪敏早就决定，限制一品山泉药酒数量。
<br/>　　但每月给孙建中提供一批还是没什么压力的。
<br/>　　“这个好说。”孙建中仍旧笑眯眯：“我帮你打响山泉酒的名气，借用了好多朋友的关系，你不得请我吃顿饭？”
<br/>　　“那必须得请！”纪敏立刻点头。
<br/>　　“那你打算请我去哪吃？”
<br/>　　纪敏：“……”这么直接的吗？
<br/>　　“我看苗家村养殖场就不错。”孙老头眼睛放光。
<br/>　　纪敏：“……”原来目的在这里。
<br/>　　“山泉药酒项目成了，身为大老板，他怎么也得庆祝一下吧！你去和小苏说说。”孙建中暗搓搓撺掇。
<br/>　　“对，我觉得也得让老板搞场庆祝宴会。”姜长飞眼睛眯着。
<br/>　　他得想法子从老板那里再掏出些东西，给他的无人机产业开辟新产业。这次聚会如果办成了，那就是一个机会。
<br/>　　纪敏都快哭了。这个老吃货，为了吃脸都不要了。这是让我去趟雷啊！大老板的饭是那么好吃的吗？
<br/>　　“行，我去试试！”纪敏咬牙道。每次在门口被小黑无人机盯着，他就有种毛骨悚然的感觉。
<br/>　　他可是知道，姜长飞曾被那小小无人机放倒过，为了感谢孙老，大不了拼了！
<br/>　　纪敏打算豁出去了。
<br/>　　“不过，老板做的饭却是挺好吃的哈！”纪敏想着想着就歪楼了，他有点儿想流口水。
<br/>　　第二天，纪敏早早来到山下铁栅栏门口。他刚想伸手推门就浑身一僵。头顶上，一台无人机就那么无声无息出现。
<br/>　　他敢肯定，如果不是他停下推门的手，这小黑无人机一定会偷摸给他一下！
<br/>　　太阴险啦！怎么还偷偷就摸过来了呢！
<br/>　　纪敏收回推门的手，抬头和天上的小黑瞪眼。
<br/>　　山上小楼内，苏重看着和小黑大眼瞪小眼的纪敏有些无奈：“他怎么又来啦！”
<br/>　　“昨晚山泉系列药酒上拍，他应该是来报喜的！”红后解释。
<br/>　　“上拍？结果怎么样？”他还真没注意这事情。
<br/>　　要是让纪敏知道，自家大老板连这么重要的事情都不知道，不知道要去哪里哭死。
<br/>　　“单支一品山泉药酒，最高价十六万！”
<br/>　　“不错啊。”这么高的进项，他以后买设备什么的，可以从纪敏上上薅羊毛啦！
<br/>　　“让他进来吧！”
<br/>　　纪敏兴冲冲走进小楼，尽量把眼中忐忑掩盖起来。
<br/>　　“老板，我们的山泉药酒大伙成功啦！”纪敏兴奋道。
<br/>　　“我知道，我一直在关注公司发展，恭喜！”苏重淡定道。
<br/>　　“同喜，同喜！”纪敏心里一阵感动。老板果然还是关心公司的！
<br/>　　他要是知道，苏重刚刚才知道药酒上拍的事情，估计得想死的心都有。
<br/>　　“年终我会给你发个大红包的。”苏重笑道。然后就盯着纪敏。他这是要赶人！
<br/>　　纪敏这会儿就想哭。哪有这么当老板的。也不说给口水喝，直接就赶人！不带这样的。
<br/>　　“老板，您看，咱们是不是得庆祝一下！”纪敏硬着头皮不看苏重眼神，期期艾艾道。
<br/>　　“庆祝？你想怎么庆祝？”苏重玩味。
<br/>　　“您看哈，咱们山泉药酒能一炮而红，孙董可是出了大力的。又是联络朋友举办酒会帮咱们打响名气，又亲自安排上拍，怎么也得感谢一番对方不是。”纪敏循循善诱。
<br/>　　苏重是看出来了，纪敏这小子不老实。今天过来，肯定不只是道喜！
<br/>　　“我就说嘛，这个厚脸皮的家伙，怎么可能会单纯的过来道喜。”
<br/>　　“孙董事大集团董事长，咱们也没什么那得出手的谢礼。我这边答应每月给他一批一品山泉药酒。到他那地位，也就对养生啦美食啦有兴趣。您说是吧？”纪敏小心翼翼打量苏重表情。
<br/>　　“是孙老头让你来的吧？”苏重算是看出来了。这就是想让他请孙老头吃饭。
<br/>　　这个老头也是够可以的，为了顿饭就这么拐弯抹角？
<br/>　　他想吃直接来不就行啦？他苏重还能真把他赶出去？他不是那样的人！
<br/>　　嗯，赶出去是不能赶出去的，不过可以让小黑无人机提前堵门啊！
<br/>　　“行啦，让他来吧！把姜长飞一块喊过来，时间就定在明天晚上！”苏重答应下来。
<br/>　　送走纪敏，苏重不忙着研究红后核心。
<br/>　　既然请孙老头来吃饭，答应给他的好东西也就一并送给他。
<br/>　　这段时间孕养，阵法核心的药材已经能用。
<br/>　　换了一身衣服进山，把药材全都取出来。苏重就一头扎厨房不出来。
<br/>　　为了炼药，他之前已经专门买了些工具。在不影响药材效果前提下，慢慢抽出水分，然后按照比例将各种药材糅合在一起，最后搓成弹丸。
<br/>　　苏重之前给红玉和小灰制了不少零食弹药，现在只做培元丹，手艺纯属很快就做好。
<br/>　　每颗弹药黄豆大小，棕色弹丸圆润光滑，捏着一些软弹，没有刺鼻气味，只有一股淡淡药香。
<br/>　　“这弹药算是成了。”
<br/>　　找到提前准备好的水晶瓶子，一瓶子里面装了三十粒，一共装了五瓶。
<br/>　　“还是有些太少，以后有了药材再多准备一些吧。”
<br/>　　第二天下午，孙老头早早的就上了山，问苏重今晚吃什么。
<br/>　　苏重都懒得搭理这老小孩。拿出准备好的培元丹扔给他一瓶。
<br/>　　“这什么东西，模样倒是挺唬人。”孙老头翻来覆去打量。
<br/>　　“之前答应给你的好东西。培元固本延年益寿。”
<br/>　　“真有那么好的功效？”孙老头满脸狐疑：“你不会用糖豆糊弄我吧？”
<br/>　　“不要就还给我？”苏重翻了个白眼，真是不知道什么是好东西。
<br/>　　“那不能够，送出去的东西，怎么还往回要！一点儿都不大气！”孙建中立刻把小瓶踹兜里。
<br/>　　他已经见识过药酒功效，这段时间每天喝一点儿，睡觉踏实，身体舒泰，精神非常好。
<br/>　　苏重都说这丹丸是好东西，那肯定就是好东西！
