　　第601章
<br/>　　有人生来就是为了毁灭而存在的。
<br/>　　白无双恰恰就是这种人，他身后自然也有一段经历，只是不愿再对任何人说起。传言说的不错，他的确是来自帝国，许多年前只身来到戎族，靠脑袋吃饭，一步一步往上爬。直到如今，成了呼延力身边的谋士，每句话至少能抵三分数。
<br/>　　身为一个谋士，最可怕的事情倒不是一场战役的失败，或者是某些地方没算计到。糟糕的事情是，你所辅佐的人根本不认同你想做的事。
<br/>　　这大概就是道不同不相为谋的意思。
<br/>　　白无双想要的，自然不是做戎族的大长老，过衣食无忧的生活。这，自无法令他满足。但很显然，呼延力很满足现在的生活。无聊的时候成个亲，生几个孩子，便是他生活乐趣的全部。
<br/>　　所托非人呐。
<br/>　　这个念头不是第一次冒出来，但这次明显比任何一次都要强烈。戎族有天下无双的骑兵，若有一日挥兵南下，帝国花花江山垂手可得。只不过，呼延力明显不是可以纵马中原的人。
<br/>　　或者，自己要重新选择要辅佐的人了，应该是一个有野心的人。白无双扭过头，目光落在那飘忽的烛火上。
<br/>　　……
<br/>　　程大雷使出看家本领，一心往外逃。可恼的是燕不归铁了心要杀死程大雷，死咬在他后面不松口。
<br/>　　二人一追一逃，程大雷离了戎族王城，奔向女儿湖。燕不归仍带着几个人缀在后面，丝毫没有放过程大雷的意思。
<br/>　　程大雷心里也发了恨，只要脱离包围，程大雷不介意回身一剑，取走燕不归的性命。在这人鬼不见的草原上，最后谁杀死谁，也是说不好的事情。
<br/>　　小白狼一直在湖边等候程大雷，心里也是十分担忧，不知程大雷能否从王城平安脱身。正这时候，远处突然传来一阵啸声。小白狼还未弄明白是怎么会事，只见一旁的黑牛突然来了精神，撒开蹄子向远方奔去。
<br/>　　程大雷的身形从远方出现，他身子突然跃起，落在黑牛背上。带着它兜了个圈，立稳了身形。
<br/>　　燕不归带着人从远处快马奔来，程大雷冷哼一声，亮出了匹夫剑。
<br/>　　“狗贼，与我纳命来。”燕不归提着重剑大喝。
<br/>　　对方来势汹汹，也是杀心旺盛，这一点恰恰和程大雷相似。他将剑凌空一挥，口中喝道：“来。”
<br/>　　双方都想杀人，互相捉对冲锋，宛若两柄满弓射出的箭，在空中狠狠相撞。只是相遇的一刹那，双方就交手了数个回合，之后互相都没讨到便宜，短暂分开。
<br/>　　程大雷心中暗惊，相比上一次交手，燕不归的实力有增无减。自己想要拿下他，确实不是一件容易的事，况且戎族大部队随时会赶过来，时间若是拖长，怕是会对自己不利。
<br/>　　但燕不归很显然没有程大雷这心思，现在他是杀人为乐，一心一意要去取程大雷性命。
<br/>　　双方交锋数个回合，两柄剑在空中碰撞，打得十分凶险。小白狼在一旁为程大雷压阵，看得出来，程大雷一时半刻间拿不下燕不归。而她向远处看，已经有一队人正向这边涌来。
<br/>　　“风紧，扯呼。”小白狼冲着程大雷高喊。
<br/>　　“扯呼，扯呼。”
<br/>　　程大雷口里答应两声，连环几招快剑逼退燕不归。自己骑着黑牛，与小白狼一起向远处逃去。
<br/>　　程大雷的黑牛，小白狼座下的瘦马都是系统奖励的座骑，燕不归绝不能轻易追上。如果他真的铁了心要杀程大雷，一旦与大部队走散，孤身来追，程大雷绝不介意用雷霆手段取了他的性命。
<br/>　　不过，结果令其很是失望，燕不归虽然走火入魔，但明显没到无药可救的地步。追了一程，见连程大雷的影子都追不到，便被手下人拦了回去。
<br/>　　程大雷和小白狼并没有走远，他们就在女儿湖附近，程大雷杀燕不归的心不死，定要等到他落单，杀了他复仇。
<br/>　　反正现在已经知道燕不归的下落，程大雷就守在城外，他还不相信燕不归会不出来。
<br/>　　……
<br/>　　大帐内，野原火看着面前的地图，终于满意的点点头。
<br/>　　这份地形图由钟牧山所绘，包括北蛮部方圆的山脉河流走势。戎族以前也绘图，不过是用炭笔在羊皮上勾勒几笔，但书院出来的弟子，人人都是琴棋书画俱全的人才。
<br/>　　钟牧山忐忑的看着野原火，这是自己答应辅佐对方后做的第一件事。狠心斩断之前的事，愿意跟在他身后成一番事业，也总是希望自己能够得到重用。
<br/>　　“很好。”野原火点点头，目光看着钟牧山：“钟先生放心，跟在我身边，定不会埋没了你一身才华。”
<br/>　　钟牧山沉默的点点头，选择投靠戎族，终究不是一件荣耀的事情。目前，他还在适应中。
<br/>　　地图摊开在大帐中，上面已标明北蛮部的位置。有了齐如海的兵器助力，又得到钟牧山这样的谋士，野原火接下来要做的，便是拿下北蛮部。
<br/>　　“我们现在能出动的兵力，不过六千余人，兵力上并不占任何优势。想以少胜多，以小吞大，就只能夜袭，趁着敌人不备，直捣黄龙，未必没有机会拿下北蛮部。”
<br/>　　钟牧山说得是未必，但机会其实并不大，毕竟北蛮部统治许多部落，在这片草原上还是很有声望的。于是，也只能夜袭，方才有搏一搏的机会。
<br/>　　野原火目光看着地图，轻轻摇摇头。
<br/>　　钟牧山心中有些忐忑，难道自己出的第一条计策，就被野原火看不上。
<br/>　　“钟先生说的很好，也并没有什么不对的，不过，我们不能这样打。”
<br/>　　钟牧山奇怪的看着野原火，不太理解对方的意思。
<br/>　　“我们需要一场胜利，一场光明正大，干干净净的胜利。”见钟牧山越来越困惑，野原火缓缓道：“用这场胜利告诉草原上所有部落，我们来了，接下来，我们将统治整片草原。”
<br/>　　钟牧山一怔，目光盯着野原火，隐隐觉得，自己选择放弃心中的忠孝，辅佐一个戎族，未必是一个错误选择。
<br/>(本章完)
