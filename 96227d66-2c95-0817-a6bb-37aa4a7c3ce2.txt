　　正当景天一行人焦头烂额，忙着破开结界的时候，慕清霄与天鬼皇已经乘坐青冥骏回到了阴寒域。
<br/>　　此时，阴寒域与半个月前相比截然不同。
<br/>　　忘川城中的鬼族修士，明显比原先多了数倍不止，城中鬼族后裔依旧过着各自的生活，对于周围变化浑然不觉。
<br/>　　走在城中，慕清霄微微点头。
<br/>　　看来天鬼皇的子女确实有些手段，在不知不觉中就已经将阴寒域完全掌控住。
<br/>　　甚至，没有走漏一点风声。
<br/>　　恐怕，其他鬼王还不知道，双煞鬼王已经死亡，天鬼皇则取而代之。
<br/>　　收敛气息，悠闲的行走在忘川城繁荣的街道上，慕清霄目光不断在街道的摊位上扫过，目标自然是各种灵药。
<br/>　　鬼界中的灵药，一般都是被鬼王或者更高层收购。
<br/>　　只有一些没有收购的，他们本身不能用，自然会拿到街道上便宜售卖，许多高级灵药，价格出乎意料的便宜。
<br/>　　来到鬼界一趟，慕清霄收获不是一般的大，至于天鬼皇早已赶回去，准备开启鬼界通往人界的空间门。
<br/>　　似乎，慕清霄在他看来相当于瘟疫，避之不及。
<br/>　　在忘川城中逛了整整一天，天鬼皇派来的部下告知，传送阵已经准备好，随时都能启动。
<br/>　　慕清霄也不想在阴森森的鬼界中多做停留，赶往忘川城中的庙宇，与天鬼皇一同前往传送阵。
<br/>　　此时，漆黑如墨的平原上，数十名鬼族修士分别站在不同的方位，双手结印，一股庞大的能量融入虚空……
<br/>　　不多时，中心位置的空间荡起一阵涟漪，伴随着一道黑芒冲起，上空被撕出一道裂缝。
<br/>　　仿佛有两只无形的手掌，慢慢将裂缝撑开，最终形成一个偌大的空间门，其中透着令人余悸的沧桑气息。
<br/>　　空间门开启，天鬼皇朝着慕清霄抱了抱拳，双眼中满是诚恳的感激，道：“阁下，本座在阴寒域有要事处理，保重。”
<br/>　　“保重。”
<br/>　　慕清霄轻轻抱拳，也不多说，直接化作剑光，眨眼间就掠入空间门中。
<br/>　　待慕清霄消失，空间门也以肉眼可见的速度慢慢缝合，最终消失不见，露出一片昏沉的天空。
<br/>　　于此同时，极乐世界。
<br/>　　红色的岩浆世界中，景天握着魔剑，不断朝面前的空间结界劈开而出，脸上露出一副苦不堪言的模样。
<br/>　　“白豆腐，都砍了一整天了，本大爷手臂都肿了，还是用你们蜀山的符印吧！”景天握着魔剑，双臂肉眼可见的颤抖着。
<br/>　　徐长卿露出一副无奈的模样，道：“景兄弟，你在加把劲，结界的能量已经被消耗了大半，很快就能劈开了。”
<br/>　　“哥哥，你要不要休息一会啊？”龙葵站在一旁，脸上满是心疼的说道。
<br/>　　……
<br/>　　……
<br/>　　人界，酆都百里外的一座山谷中，山谷中有一片巨大的湖泊，湖水清澈透着诡异。
<br/>　　就在这时，湖泊中央泛起一丝涟漪，涟漪扩散，一波又一波，湖水不断动荡起来，波浪也越来越大。
<br/>　　陡然，一束黑色的光芒冲天而起，最终与四周山峦相连接，湖泊上方就裂开一道庞大的裂缝。
<br/>　　裂缝被慢慢撑开，最终形成一扇漆黑的大门。
<br/>　　“咻咻……”
<br/>　　不多时，一道璀璨的剑光自漆黑的大门中掠出，仿佛要将天地照亮。
<br/>　　长剑冲出空间门后并未停留，而是化作剑光，犹如追星赶月，风驰电掣一样向酆都方向掠去。
<br/>　　几十秒后，长剑散发出一阵晶莹的光芒，最终显现出一道修长的身影。
<br/>　　望着脚下的酆都，城市中不见一个鬼影，神识中也没有任何的生命痕迹。
<br/>　　看来，天鬼皇在夺得阴寒域后，便将酆都的鬼族后裔，全都转移到了鬼界中。
<br/>　　酆都虽是鬼城，但却在人界。
<br/>　　酆都城中空荡荡的，慕清霄身影一闪，转眼就出现在鬼皇宫中。
<br/>　　鬼皇宫中的侍女也全部转移，唯有正殿与偏殿中的各种杀阵遗留了下来。
<br/>　　慕清霄找了一间空旷的休息室，布下结界后，直接盘坐在床榻上，微微闭目，五心向天，神识则出现在阴阳玉中。
<br/>　　此时，火灵珠已经被转移到了阴阳玉中，心法运转，慕清霄控制魂剑，准备炼化火灵珠。
<br/>　　火灵珠中拥有极其浩瀚的火属性能量，将其全部炼化，慕清霄相信，自己的实力绝对会突飞猛进，神火的威力估计也会增强。
<br/>　　“轰轰……”
<br/>　　随着火灵珠中的精纯能量牵引而出，一股恐怖的高温突兀的汹涌起来。
<br/>　　恐怖的温度犹如滔滔不绝的江水，以慕清霄为中心，向着四面八方席卷而出，整个休息室中都变得通红一片。
<br/>　　最后，除了慕清霄与结界本身，周围的事物居然全部开始液化瓦解……
<br/>　　不多时，整个偏殿就在恐怖的高温下溶解。
<br/>　　慕清霄对此仿若未觉，如同老僧入定，不断的牵引着火灵珠中的能量汇入四肢百骸中，以至于他整个身体都变得通红无比。
<br/>　　甚至，头顶白烟缭绕，热气升腾。
<br/>　　好在慕清霄本身不惧怕高温，比起吸收土灵珠和雷灵珠，火灵珠无疑要简单许多，所以能够肆无忌惮的牵引出火灵珠中的能量。
<br/>　　这样做的后果一目了然，整个宫殿几乎在眨眼间就消失不见，连地面都在高温下下降了几丈。
<br/>　　被牵引出的精纯火属性能量流淌在他的四肢百骸中，不断的淬炼着慕清霄的细胞，经脉与肉身。
<br/>　　慕清霄已经是大乘四重的修为，突破是所需要的能量或灵气，简直到了一个令人发直的地步。
<br/>　　在他吸收精纯能量的同时，他的上空则出现一个灵气旋窝，以酆都为中心，方圆几十里甚至百里，空间中的灵气都不断汇聚过来。
<br/>　　“轰轰……”
<br/>　　随着旋窝越来越庞大，苍穹之上突兀荡起一阵巨大的波动。
<br/>　　天地间浩瀚的灵气，甚至带着各方生灵与自然界的能量，在空中彼此交汇，仿佛产生共鸣，最后向着同一个方向涌去。
<br/>　　方圆百里外，灵气被抽取一空。
<br/>　　甚至，生活在范围内的生灵，忽然觉得空气变得压抑，连呼吸都变得困难了起来。
<br/>　　……
<br/>　　……
