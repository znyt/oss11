        明司寒再次甩开贺思思的手，见她衣服都湿了，冷着脸说道：“你还是赶紧去换衣服！”
<br/>
<br/>        “好！我立马就去！那你要等我噢！”贺思思顿时兴奋地向明司寒送上一个飞吻。
<br/>
<br/>        明司寒懒得理会她，自顾自地提着连星儿刚刚塞给他的礼品袋，回到了寝室。
<br/>
<br/>        他坐在书桌前，小心翼翼地打开礼品袋后，看到里面的两个杯子变成了六大块，以及杯身上的卡通人像，忽然间意识到自己刚刚真的是抽疯了。
<br/>
<br/>        未等邢乐游他们问及他原因，他便收拾了一下手里的东西，什么也没交代便提起礼品袋出了门。
<br/>
<br/>        连星儿一个人在寝室里待了很久，才收到顾千凝发过来的信息，说她和赵安彤还有路娅玟三个人在逛步行街。
<br/>
<br/>        “真没义气，三个人出去逛街，把我一个人丢寝室。”连星儿不禁自言自语地发起来牢骚。
<br/>
<br/>        当她决定去祖爷爷家过双休日的时候，顾千凝正巧给她发了条短信过来。
<br/>
<br/>        星儿，晚上我们一起吃饭，为了补偿你，晚饭我们三请了，你就乖乖在寝室里等我们回来呦！——顾千凝。
<br/>
<br/>        连星儿顿时喜上眉梢，高兴地回了条短信过去。
<br/>
<br/>        那我就乖乖地在寝室里等你们回来好了！——连星儿。
<br/>
<br/>        嗯，么么哒——顾千凝。
<br/>
<br/>        可是当跟顾千凝发完信息后，她又回想起明司寒的事情来时，心里又是极其的不悦。
<br/>
<br/>        明司寒真的是瞎了眼！居然把那种女孩子当朋友看待！
<br/>
<br/>        连星儿愤愤不满地在拿起笔和纸，一边在心底抱怨，一边在纸上写着明司寒的名字然后又打了把叉。
<br/>
<br/>        另一边，贺思思换了身干净的衣服，给自己补了妆后，直接去了明司寒的寝室。
<br/>
<br/>        当她一袭青绿色蛋糕长裙，上搭一件樱花花瓣点缀的白色露肩纱衣，剪着斜刘海，烫着齐肩的栗色梨花烫，出现在门口时，门内的三个男人瞬间看呆了。
<br/>
<br/>        “你就是贺思思？追求咱们寒哥的那位美女？”仲正浩惊艳地问道。
<br/>
<br/>        贺思思腼腆地笑了笑，抬起手将两鬓的发丝撩到了耳根后，声音娇柔地说道：“对呀！三位大帅哥们好！初次见面，还请多多关照。”
<br/>
<br/>        “多多关照！”邢乐游第一个冲到贺思思跟前，向贺思思友好地伸出手来。
<br/>
<br/>        贺思思微微一笑，轻轻地握了握邢乐游的指尖，以示问候。
<br/>
<br/>        但是，她那双水灵灵的大眼睛，却在扫视着这间还算整洁的男寝室。
<br/>
<br/>        “快进来坐吧！”仲正浩随即搬了条凳子过来，然后回头看着发愣的张子晋，使唤道，“子晋，快去泡茶！”
<br/>
<br/>        “不用了！”贺思思连忙摇手，甜甜地一笑，“我是来找司寒哥哥，他不在吗？”
<br/>
<br/>        “他出去了。”张子晋应了声。
<br/>
<br/>        邢乐游点头道：“嗯，他出去了。”
<br/>
<br/>        “那他去哪里呢？”贺思思又问道。
<br/>
<br/>        这回三个男生不约而同地摇头。
<br/>
<br/>        “哪张床是他的呀？”贺思思接着问道。
<br/>
<br/>        仲正浩指向靠窗的那张床后，贺思思便直接走了过去，坐在了床下的书桌前，并从书架上随手挑了一本书出来。
<br/>
<br/>        “我就坐在这里一边看书，一边等他回来好了。”贺思思微笑道。
<br/>
<br/>        仲正浩忍不住上前搭讪：“你的小提琴拉得真好听！”
<br/>
<br/>        “谢谢。”贺思思微笑着点了下头。
<br/>
<br/>        仲正浩开始有一搭没一搭地找贺思思聊天。
<br/>
<br/>        两个小时过去了，眼瞧着快到午饭时间了，明司寒仍旧没有回来。
<br/>
<br/>        贺思思便让仲正浩给明司寒打电话，询问一下他大概什么时候回来。
<br/>
<br/>        仲正浩也不知道是着了贺思思的什么魔，她说什么，他便做什么。
<br/>
<br/>        明司寒接听了电话后，仲正浩直接开门见山地问：“阿寒，你去哪儿呢？”
<br/>
<br/>        “我在忙了，你有事吗？”明司寒不冷不热地反问。
<br/>
<br/>        仲正浩看了贺思思一眼，接着说道：“贺思思小姐正在咱们寝室等你了。”
<br/>
<br/>        “叫她回去。”明司寒冷冷地命令道。
<br/>
<br/>        仲正浩顿时十分诧异地看向贺思思，回答道：“她都已经等了你两个小时了，而且，她是特意来给你过生日的吧！”
<br/>
<br/>        “你告诉她，我跟星儿在一起，今天不会回来了。”明司寒淡然道，说完后直接把电话给挂断了。
<br/>
<br/>        仲正浩看了一眼手机屏幕上的“通话结束”，一脸无奈地再次看向贺思思，瘪了瘪嘴，说道：“阿寒说，他跟连星儿在一起过生日，而且，今天都不会回来了。”
<br/>
<br/>        “他……”贺思思欲言又止，鼻子一酸，泪水瞬间在眼眶里打转。
<br/>
<br/>        她这一哭，同屋子里的三个男生顿时不知所措。
