　　轮回神朝，天王宫。
<br/>　　一座古色古香的偏殿之中，天君河和天尘宇两人，相对而坐。
<br/>　　“宫主，根据天龙神朝探子来报，龙族大长老龙昌，不久之前离开了龙都城。”天尘宇面色凝重的道。
<br/>　　“去往何地？”天君河沉声问道。
<br/>　　“不清楚，是混沌神朝的方向！”
<br/>　　天尘宇摇了摇头，继续道：“根据我们收买的龙族奴仆的消息，在龙昌离开龙都城之前，苏莫曾进入了其府邸之中，一直没有出来。”
<br/>　　“哦，你怀疑龙昌带着苏莫离开了龙都城？”天君河面色凝重，眸中闪过璀璨的精光。
<br/>　　“有这个可能！”
<br/>　　天尘宇点了点头，道：“不过，我们的探子，修为都不足神王，无法追踪龙昌。”
<br/>　　天君河闻言，沉默不语，心中在思索利弊，即便知晓苏莫有可能离开了龙都城，但是目前也没有什么办法。
<br/>　　首先，龙昌的实力很强，他亲自去追踪，现在也来不及了，其次，就算找到了，他也不可能在龙昌的手中擒拿苏莫。
<br/>　　再者，估计此刻，苏莫已经脱离了龙昌，不知去往何地了，而且，帝一魂目前不在天王宫，他也不好轻易离开。
<br/>　　“苏莫不一定是去混沌神朝，多派些人手在各大神朝查探。”良久之后，天君河沉声吩咐道。
<br/>　　“嗯！”天尘宇点了点头，随即又道：“最近，终极神王又去了龙都城，而且，终极神王殿对于藏身在龙族的苏莫，不知道有什么计划？”
<br/>　　“终极神王去龙都城，不外乎是去谈判，想要拿回终极神剑和终极剑道，此事不必理会！”天君河摇了摇头，继续道：“当然，我们要对付苏莫，能拉拢终极神王殿便劲力拉拢。”
<br/>　　“嗯！”天尘宇点头。
<br/>　　“这样吧，把苏莫和龙昌的情况，告知紫霄，让他去追查苏莫！”稍一沉吟，天君河继续道，目前，紫霄就在他们天王宫。
<br/>　　对于紫霄的情况，他们也都知晓了，紫霄做了一些事情，无颜再回终极神王殿了。
<br/>　　对此，他们愿意给予紫霄帮助，当然，也可以说是合作，毕竟，他们都有苏莫这个重要的敌人，
<br/>　　“如此最好！”天尘宇叹道，苏莫有可能离开了龙都城，若是不去追查，难免会错过机会，让紫霄去再好不过了。
<br/>　　踏！踏！踏！！
<br/>　　就在此刻，一阵轻微的脚步声在殿外响起，一道高大的蟒袍身影，来到了殿门之前。
<br/>　　“一魂，你回来了！”天君河和天尘宇两人，见到来人，顿时微微一怔，因为来人正是帝一魂。
<br/>　　“怎么样？”天尘宇急忙出声问道。
<br/>　　“妥当了！”帝一魂面色淡淡，大步的走到殿中端坐了下来。
<br/>　　“哦！”天君河和天尘宇闻言，顿时眼眸一亮，不由自主的站了起来。
<br/>　　这一次，帝一魂秘密外出，去往各大神朝，在商谈一件大事。
<br/>　　这件大事的终极目的，便是对付苏莫，一旦成功，那么大事可期。
<br/>　　“我去了混沌神朝、不死神朝、霸天神朝、幽冥神朝、天星神朝、亘古魔国，除了混沌神朝之外，其它五大神朝，都同意了我的要求，只要他们得到更换神域神阵阵基的资格，便允许我下界一次！”帝一魂声音低沉的说道。
<br/>　　“那真是太好了！”天君河闻言大喜，这次帝一魂这次外出的目的，并不是拉拢各大神朝，而是获得下界的机会。
<br/>　　想要拉拢各大神朝，太过困难了，他们两人已经去了多次，都无功而反，所以，这一次帝一魂单独行动，就是要获得下界的机会，给予苏莫致命一击。
<br/>　　“五大神朝同意了，但是，还是不太保险，更换阵基的资格，也有可能被天龙神朝，人王宫、以及天池神朝八十一个界域的大势力得去。”天尘宇面色凝重的说道。
<br/>　　虽然这五大神朝的实力，极度的强大，天才辈出，获得更换神阵阵基的资格希望很大，但是也不绝对。
<br/>　　毕竟，混沌神朝、天龙神朝，天池八十一个界域，乃至于人王宫的天才，都很强大。
<br/>　　“你们认为，苏莫会去参加千选战吗？”帝一魂面色郑重的问道。
<br/>　　“应该不会吧，他才三品的修为，没有资格参加！”天君河摇了摇头道。
<br/>　　“不，他一定回去参加！”帝一魂笃定的说道，他比所有人都了解苏莫，苏莫一定会去参加。
<br/>　　“他能在五年之内，达到五品的修为，这不太可能吧？”天尘宇诧异的说道，他也知道苏莫肯定想要参加千选战，但是，想要在五年之内达到五品的修为，无异于天方夜谭。
<br/>　　“你们不了解，只要他愿意，就能做到！”帝一魂面带感慨之色的说道。
<br/>　　“你的意思是？”天君河眉头微皱，心中有所猜测。
<br/>　　“这一次千选战，五大神朝的万古天骄，如古魔太子、霸云公子等人，会助我镇压苏莫！”帝一魂沉声道。
<br/>　　“什么？”天君河和天尘宇闻言，顿时面带惊色，前者诧异的问道：“你真的拉拢了这五大神朝？”
<br/>　　天君河和天尘宇很震惊，五大神朝的万古天骄出手，那么，只要苏莫去参加千选战，那么便是粘板上的鱼肉。
<br/>　　无人神朝的天骄，虽然在天赋方面，都不如苏莫，但是，个个修为强大，而且人数众多。
<br/>　　再者，如同亘古魔国的古魔太子，九品高级的修为，霸天神朝的霸云公子，八品低级的修为，天星神朝的无极公子，七品高级的修为，这些人的天赋或许略逊于苏莫，但是实力根本不是苏莫能比拟的。
<br/>　　“不能算是拉拢了！”
<br/>　　帝一魂摇头，叹了口气道：“每个神朝都只是私下里答应，愿意在千选战之时，让弟子对苏莫出手，但是，不会做的太过明目张胆；而且，只此一次。”
<br/>　　这五大神朝，并不愿意表明态度，只是在此次千选战中，愿意帮助一次。
<br/>　　而且，这五大神朝之间，并不互相知晓，换言之，亘古魔国愿意出手，但是，亘古魔国并不知道，还有其它四大神朝也愿意出手。
<br/>　　没办法，神域各大神朝，各大势力之中，局势非常的复杂，哪个神朝都不想率先表态。
<br/>　　“足够了，五大神朝的万古天骄，足以彻底镇压苏莫了！”天君河微微颌首，心中大大的松了一口气。
<br/>　　苏莫若是参加千选战，将被镇压，而若是不参加千选战，那么帝一魂就要下界，擒拿苏莫的软肋。
<br/>　　“能镇压自然最好，若是事不可为，那就只能镇杀了！”帝一魂微眯着眼眸，声音低沉的说道。
<br/>　　他自然想要融合苏莫的神魂，但是，若是机会不大，他宁愿镇杀苏莫，不要神魂。
<br/>　　因为，他感受到了来自苏莫的威胁，若是这一次无法解决苏莫，以后更加的困难了。
<br/>　　六魂融合吗？
<br/>　　就算舍弃一魂，他也同样能成为至强者，这只是时间的问题。
<br/>　　至于天尊复生？
<br/>　　帝一魂的心中，无比的复杂，纠结不定，这是他宿命中的挣扎。
<br/>　　“虽然这个计划，成功率极高，但也不是有十成的把握，对于苏莫依旧不能放松！”
<br/>　　天尘宇道，此事虽然一片想好，但是，他们却是要做好最坏的打算，比如，苏莫没参加千选战，更换神域大阵的资格，被混沌神朝或者天龙神朝，亦或者人王宫得去了。
<br/>　　“的确！”天君河点了点头，随即将苏莫可能跟随龙昌离开龙都城的消息，详细告知了帝一魂。
<br/>　　“查，立刻追查苏莫的下落，但凡有一丝可能，也不能放过！”对于苏莫可能离开了龙都城，帝一魂极为果断，当即决定追查苏莫的行踪。
<br/>　　若是能顺利的擒拿苏莫，带回天王宫，这是最完美的事情。
<br/>　　三人一番商讨之后，最终，决定由天尘宇和紫霄一起，前去追踪龙昌，追查苏莫的下落。
