　　众人在山洞之中，前进了约莫一刻钟之后，前方出现了一个宽敞的石室。
<br/>　　石室巨大，方圆数千丈，石室四周有着上百个石洞，漆黑幽深，不知通向何处。
<br/>　　石室的中央，半空之中，悬浮着一柄通明的宝剑。
<br/>　　此剑剑柄在上，剑尖在下，长达五丈有余。
<br/>　　即便没有走进石室，都能感觉到宝剑之上散发的剑威，犀利而浩大，磅礴而冷冽。
<br/>　　众人对于万道剑冢的情况，基本上都有所了解，所以，均是准好了准备，鱼贯而入，进入了宽敞的石室之中。
<br/>　　霎时之间，石室中的透明大剑，爆发出璀璨无比的剑光，几乎刺的人睁不开眼睛。
<br/>　　而后，璀璨的剑光，化为数千利剑，向众人暴刺而来。
<br/>　　这些剑光化作的虚幻之剑，仿似每一柄都有各自的目标，威力也不尽相同。
<br/>　　“又是这种攻击？”苏莫眼眸微眯，他看到一柄利剑，再次向他袭杀而来。
<br/>　　这柄剑仿佛锁定了他的气息，只攻击与他，对于其他人没有任何的影响。
<br/>　　当然，其他人也各自受到一柄利剑的攻击。
<br/>　　霎时之间，光芒四起，几乎所有人都出手了，苏莫也没有懈怠，一记火焰剑气斩击而出。
<br/>　　轰轰轰！！
<br/>　　爆响不断，大量的虚幻之剑被斩碎，化为无形的气流，消散无踪。
<br/>　　苏莫非常的轻松，便将攻向他的虚幻之间斩碎。
<br/>　　啊！啊！啊！
<br/>　　惨叫声再次响起，有数十人挡不住虚幻之间的攻击，纷纷受伤。
<br/>　　对于这些人，透明大剑之中，不断射出森寒的剑光，化为同样的虚幻之剑，再次攻击。
<br/>　　这些受伤的人，有的拼命抵挡，有的急忙退出了石室。
<br/>　　不过，实力不济之下，那些抵挡的人，根本抵挡不住，很快就退出了石室。
<br/>　　苏莫伫立在石室之中，目光直直的望着中央的透明大剑，从刚才抵挡住了攻击之后，他发现，自己从此剑之上，能够感受到其磅礴的意志。
<br/>　　这一点，不只是他自己，其他人同样如此。
<br/>　　这股意志没有属性，有的只是单纯的绝世锋芒，森冷萧杀。
<br/>　　有少数一些人，走到透明大剑之前，盘膝而坐，静静的感受此剑的意志。
<br/>　　“我们走！”
<br/>　　此时，终极神王殿的韩生，招呼一声，便带着二十余名终极神王殿之人，走进了石室中的一个石洞之中。
<br/>　　慕容雷利，作为终极神王殿之人，亦是和韩生等人同行。
<br/>　　唰！唰！唰！
<br/>　　拓跋流明、千幻邪君、云沐、公孙嘉、古剑山等人，均是没有迟疑，纷纷选择一个石洞，飞掠了进去。
<br/>　　他们都是万古天骄，自然不可能在这入门之停留。
<br/>　　留在此地的人，只是极少之数，均是实力较低，不敢深入之人。
<br/>　　但是，大部分人，均是纷纷散开，各自选择一个石洞，进入了其中。
<br/>　　苏莫环视四周，石室四周有上百个石洞，可以随意的选择。
<br/>　　上次，他从绫裳口中，也了解了一些万道剑冢的情况，每个石洞之内，都不相同。
<br/>　　所以，具体的情况，他也不太了解。
<br/>　　总之，这里就是一层层的考验，每通过一次考验，都能得到一些机缘。
<br/>　　而通不过，要么生死道消，要么立刻退出。
<br/>　　当然，这里得到的机缘，也全部是与剑道有关，并无什么天材地宝。
<br/>　　呼吸之间，数千名妖孽，便纷纷涌进了上百个石洞之内，只留下数十人在此感悟，不敢深入之人。
<br/>　　唰！
<br/>　　苏莫认准了一个石洞，闪身便飞了进去。
<br/>　　“这人才二品高级的修为，居然还敢进去？”
<br/>　　“找死而已！”
<br/>　　“他能挡住虚幻之剑的攻击，真是不可思议！”
<br/>　　“他去的是千幻邪君进入的石洞！”
<br/>　　石室之外，先前抵挡不住虚幻之剑攻击的人，看到最后离开的苏莫，纷纷暗暗摇头。
<br/>　　很多三品低级修为之人，都不敢继续深入，此人居然无所畏惧。
<br/>　　不过，他们也没有多想，纷纷打坐调息，准备再次进入石室。
<br/>　　只有挡住虚幻之剑的攻击，他们才能留在石室之中，感悟这剑道真意。
<br/>　　这透明大剑，据闻乃是终极剑尊的剑道真意所化，若是参悟一二，对于他们的剑道，均有莫大的裨益。
<br/>　　……
<br/>　　唰!
<br/>　　苏莫进入石洞之后，缓慢的前进，前方一片漆黑，即便他混沌源力注入双眸，视距都只有百余丈。
<br/>　　他之所以选择这个石洞，是因为此石洞中进入的人较少，除了千幻邪君之外，只有寥寥八九人。
<br/>　　而其它的石洞，进入的人，最少都有二十人以上。
<br/>　　他本想进入慕容雷利所去的石洞，找机会斩杀此人，但是，想到有二十余名终极神王殿的弟子与慕容雷利同行，还有四品修为的韩生，他便暂时放弃了。
<br/>　　石洞幽深，苏莫没有看到先一步进入的千幻邪君等人，他缓慢的前进。
<br/>　　前进了约莫半刻钟，前方豁然开朗，他来到了一片宽敞的空间。
<br/>　　这片空间，同样是一座石室，但是却宽敞无比。
<br/>　　石室方圆千丈，中央耸立着一座祭坛，高约数十丈。
<br/>　　在祭坛的中央，倒插着一柄大剑，此剑如同是水晶锻造，通体泛光，散发出惊人的威势。
<br/>　　最主要的是，感受到此剑的气息，苏莫体内的万化神剑元神，竟然有些悸动。
<br/>　　这种感觉，仿佛是猫见了老鼠，有一种深深的渴望。
<br/>　　“这是什么剑？”苏莫心中讶然，随即大步向祭坛走了过去。
<br/>　　这祭坛占地极大，方圆数十丈，四周均是有九层台阶，通向祭坛之上。
<br/>　　而此时此刻，千幻邪君和另外八名妖孽，正在台阶之上，向上缓慢的走动。
<br/>　　九人的动作极慢，仿佛半天都迈不出一步，其中千幻邪君在第三个台阶，其他人都在第一个台阶和第二个台阶之上。
<br/>　　每一个人，望着祭坛上的水晶宝剑，眸中都露出了渴望之色。
<br/>　　“必须要一步步的登上祭坛吗？”
<br/>　　苏莫心中惊疑，快步的来到了祭坛之下，凝望着祭坛上的宝剑。
<br/>　　略一沉吟，他大手抬起，轻轻一探，试着抓取祭坛上的宝剑。
<br/>　　霎时之间，一只火焰大手，便冲天而起，直上祭坛。
<br/>　　虽然苏莫知道可能无法抓去，但是他还是想要试探一番。
<br/>　　“他居然想直接抓取剑晶！”
<br/>　　“找死！”
<br/>　　“呵呵，不知所谓！”
<br/>　　众人见此，顿时纷纷不屑，这也太无知了吧，居然要直接抓去剑晶。
<br/>　　毫不意外，大手还没有冲上祭坛，便触动了某种玄妙的力量，立刻湮灭。
<br/>　　随即，一股磅礴的伟力，从祭坛上冲下，直接撞击在了苏莫的身上。
<br/>　　嗖！
<br/>　　苏莫立刻被撞退十数丈，体内气血翻涌，辛苦他早有防备，不然说不定会因此受伤。
<br/>　　“看来只能一步步蹬上去了！”苏莫叹了口气，便向台阶走去。
<br/>　　“咦？”
<br/>　　“厉害啊！”
<br/>　　众人一见苏莫无事，不仅个个心中惊疑，因为苏莫的修为，不过是二品高级。
<br/>　　这种修为，能来到此地，已经是很惊人了，没想到违规抓取剑晶，居然都没有受伤！
<br/>　　这就厉害了，不少人仔细的大量了苏莫一番，但是并不能看出丝毫的不凡之处。
<br/>　　千幻邪君微微瞥了苏莫一眼，便不再关注，对于他来说，此地无人能威胁到他。
<br/>　　都是一些蝼蚁，或许天赋都不错，但是根本没有资格和他争夺剑晶。
<br/>　　随即，他脚步缓缓抬起，踏上了第四阶台阶。
