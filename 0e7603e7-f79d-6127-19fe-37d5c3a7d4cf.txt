　　“剑道唯我！天下之间，无人可以跟我抗衡！”
<br/>　　楚风眠长啸一声，他身上的剑意猛然凝聚在了极点，在他的剑锋之上，爆发出了一股光辉，组成无上剑气，猛然向着剑祖荒的身躯，斩杀而去。
<br/>　　这一剑，猛然打中剑祖荒的这一缕化身。
<br/>　　剑祖荒，也感觉出来，楚风眠这一剑的恐怖，急忙想要后退，但是他的速度，哪里比得上楚风眠的剑光。
<br/>　　“小子，你的确很强，以剑道将我逼到这一步的人，你还是第一个，不过你永远不可能是我的对手！”
<br/>　　剑祖荒的声音，突然响起。
<br/>　　从天空之上，突然一道隧道，猛然打开，似乎是剑祖荒的本体，将源源不断的力量，注入到这一缕化身之中，想要拯救这一缕化身。
<br/>　　楚风眠所说的没错，这一缕化身，乃是剑祖荒的剑意凝聚。
<br/>　　若是今日真的是被楚风眠这样斩杀，对剑祖荒的实力也有着一定的损伤，剑祖荒绝不会坐视不理。
<br/>　　“想要救？这一缕化身，还是留下吧！”
<br/>　　楚风眠冷笑一声，他的剑锋再次闪耀，化为一道冲天剑芒去，直接要随着隧道，直接斩杀剑祖荒的本体。
<br/>　　“轰！”
<br/>　　在这一剑之下，那隧道都是突然崩裂，被一股剑气直接剿灭，不过却并没有伤及到剑祖荒的本体。
<br/>　　在最后时刻，剑祖荒还是关闭这一道隧道，让楚风眠的剑气，无法伤及到他的本体。
<br/>　　只是这样一来，这一道化身，他也再也无法接走。
<br/>　　“死！”
<br/>　　楚风眠剑光闪动，那剑祖荒的化身，也在楚风眠的剑光之下，全部破碎，他身上的剑意，如今被楚风眠，全部吞噬。
<br/>　　“很好，小子，想不到在这归墟天中，居然又出现了一个足以跟我抗衡的人物，今日我不动用本体，算是给你一个机会，留你一条性命，等我下次见到你，一定要斩杀了你！”
<br/>　　剑祖荒的声音，从天空之上传来，还是显得无比愤怒。
<br/>　　他的化身，如今都被楚风眠斩杀，对剑祖荒来说简直是奇耻大辱。
<br/>　　下一次见面，剑祖荒跟楚风眠之间，必定有着一场血战。
<br/>　　“下次见面，我要斩杀的，就不止是你的一道化身了，而是你的本体。”
<br/>　　听着剑祖荒的话，楚风眠也是冰冷的开口道。
<br/>　　不过今日，楚风眠也没有想要在战下去的必要。
<br/>　　楚风眠刚刚降伏镇压了那天魔宝车，这天魔宝车，还在源源不断的冲击这镇压，令楚风眠不得不分心去对付天魔宝车才行。
<br/>　　所以这一次，楚风眠也没有打算真正跟剑祖荒生死之战的打算，等他将这天魔宝车彻底镇压炼化，在来找剑祖荒一战为好、
<br/>　　这一战中，楚风眠感觉到，周围至少有着数十道灵识，都在观察这一战，人多眼杂，楚风眠也必须要找个地方，将这天魔宝车，彻底炼化。
<br/>　　这一次斩杀小魔主，楚风眠所得到的好处最大的，便是这天魔宝车。
<br/>　　九魔城的人只怕也想不到，这小魔主，居然会在归墟天中陨落。
<br/>　　小魔主在归墟天中，有着九魔城弟子的保护，更是有着天魔宝车，可以说其中的任何人都休想要伤及到小魔主。
<br/>　　因为这样，九魔城的那些老古董才会放心让小魔主带着天魔宝车来到归墟天中吧。
<br/>　　只是九魔城的人也想不到，居然会出现一个异数，便是楚风眠，让这天魔宝车，落入到了楚风眠的手心之中里来。
<br/>　　“走。”
<br/>　　楚风眠身形一动，直接也是离开了原地，回到了九魔城驻地所在的宫殿之中。
<br/>　　另一边，另外一座高大的山峰之上，其中最为庞大的宫殿之中，一声愤怒的吼声，突然暴怒而起。
<br/>　　“该死！该死！一个区区北域小子，居然敢斩杀我的化身！”
<br/>　　这咆哮的声音，正是那白衣男子，剑祖荒，如今他被楚风眠斩杀化身，简直是愤怒到了极点。
<br/>　　剑祖荒，在这天羽神域之中纵横数十年，都从来没有吃过这种亏。
<br/>　　“剑师兄，您不是在闭关吗？发生了什么？”
<br/>　　在周围宫殿中，一名名天剑宗弟子，听到这剑祖荒的咆哮声，纷纷敢来，有些吃惊的询问道。

<br/>　　剑祖荒冰冷的开口道。
<br/>　　“我们天剑宗，这一次进入归墟天的武者，都是被此人，全部斩杀。”
<br/>　　“什么？”

<br/>　　“还有那小魔主，不是号称带着九魔城的至宝，天魔宝车进来的吗？居然被人斩杀了？那天魔宝车，也被人夺走了？”
<br/>　　“剑师兄的化身，居然都被此人斩杀了！”
<br/>　　“这人，是我们天剑宗的大敌，绝不能让此人活着离开！”
<br/>　　在场天剑宗弟子，听到剑祖荒的话，都是无比愤怒，震惊，群情激昂。
<br/>　　“剑师兄，我们该怎么办？”
<br/>　　一名天剑宗弟子，恭敬的向着剑祖荒试探性的说道。
<br/>　　“要不要我们派人，去围杀此人？”
<br/>　　“不用，此人的实力极强，贸然出手，不好对付，更何况现在云羽天宫，已经出现种种异变，距离开启已经不远，现在没有功夫对付这小子。”

<br/>　　“禀告剑师兄，那女子的一身实力，极为诡异，已经是逃出了无数次我们的围捕，现在逃到了冰裂谷中去，应该受了不小的伤。”
<br/>　　那天剑宗弟子，开口回答道。

<br/>　　“走，去冰裂谷！”
<br/>　　剑祖荒大手一挥，催动身法，直接从宫殿中离开。
