<br/>　　不得不说，它是有先见之明。

<br/>　　“啧啧...是吗”
<br/>　　霎时间，庭院飞沙走石，一股大范围的狂风肆意席卷，只见十几米巨大的金蚕蛊蛇全身迸发金黄色光芒，亮的刺吓人的眼。
<br/>　　它的体形开始出现了变化，在暴涨，虎虎生风的暴涨，一息之见就是二十米。
<br/>　　在几个眨巴眼的工夫，噌的一声，就跟金箍棒变大了一样，直接三十米，于紫金巨蟒同一个高度。
<br/>　　与此同时，这条蛊蛇的气息愈演愈烈，凝结了庞大的气势，金黄的光芒遮天蔽日，宛若一条黄金浇铸的真龙形成，浩浩荡荡的扶摇直上。
<br/>　　天空忽明忽暗，似乎周围空间全部被这股气势给波及到了。
<br/>　　轰然一声，四十米...
<br/>　　“祖奶奶我怕吓死你”金蚕蛊蛇此时蛇形龙象，威武不凡，仰天睥睨。
<br/>　　这蛇妖？？？
<br/>　　要不要这样啊，我三十米，你就四十米，怎么地，比身高？
<br/>　　“哼...”
<br/>　　涂小安岂能弱余它蛇，恐怖的气息迸发，三十米的体形又出现了变化，已肉眼可见的速度，也暴涨到了四十米。
<br/>　　开玩笑，他当初吞了吸血蛇后，就是三十米了。
<br/>　　现在等级二十九级，还能是三十米吗。
<br/>　　我不过想低调点而已。
<br/>　　等涂小安变成四十米的紫金巨蟒，顿时一脸懵逼，对方已经是五十米了。
<br/>　　这特么的真的跟自己比身高？
<br/>　　也好在这片区域不可见，否则能将温城的人民吓死一片一片的。
<br/>　　“大...大...大...”
<br/>　　涂小安表示不服，体形又是开始暴涨，显化五十米。
<br/>　　“小小蛇精，不错，还没证道能有这番作为”浑身金黄灿灿的蛊蛇，说话间，就六十米了。
<br/>　　涂小安突然好想骂街，三阶蛇妖就了不了？欺负他还没证道吗。
<br/>　　罢了，这老妖精心眼太小，非要压自己一头。
<br/>　　五十米是一个什么概念了，那就是一栋移动的高楼大厦呀。
<br/>　　张开血盆大口，能把大卡车当成玩具来吞。
<br/>　　此时，涂小安不在变化体形，他已经达到顶点了，除非使用力量药剂，否则不能在变大了。
<br/>　　不过，对方毕竟是妖王级别，修炼了不知道多少年，跟它比这个，貌似也没什么意义。
<br/>　　好一条金蚕蛊蛇。
<br/>　　涂小安精光一闪，貌似自己第三阶段的进化蛇种，就是它了。
<br/>　　他第一次开启进化，是拿变异鸡冠王蛇，雷电赋予，第二次进化，是拿变异吸血蛇，火焰赋予，如今，他二十九级，很快要进行第三阶段的进化。
<br/>　　那么目标就是吞噬这条金蚕蛊蛇了。
<br/>　　妖王层次的蛇种，摆在他的面前，没有比这个更合适了。
<br/>　　只是涂小安的进化能源还没到，现在吞噬好像有点浪费了，无法直接开启第三阶段的进化模式。
<br/>　　他必须要等级在三十级以上，才能开启第三阶段的进化模式。
<br/>　　另外，事情的走向远远的超出了涂小安的预计，鬼知道尼玛的堂堂隐藏家族的祖奶奶，会是一条蛇妖化形的存在。
<br/>　　还特么的是妖王级别。
<br/>　　这弄的现在涂小安很尴尬，本以为对方是强弩之末，看这样子也不像。
<br/>　　打不一定打的过，打的过吞噬了，又有点浪费资源，要收服就更是不容易了。
<br/>　　就算收服了，他以后也舍不得用来开启第三阶段的进化了。
<br/>　　只见一条五十米的紫金巨蟒，傲立半空，另外一条金黄灿灿蛇形龙象的金蚕蛊蛇更是睥睨可怕。
<br/>　　一时之间，双方都这样杵在了那里，谁都没有立刻动手。
<br/>　　隐隐间，好像它们的变大，只是为了呼吸高空上的新鲜空气。
<br/>　　涂小安有顾虑，而蛇妖更是在之前的战斗中受伤，底气远远没有表面的那么足。
<br/>　　它很想知道，为什么一条蛇精，能强大到这种地步，谁能告诉它。
<br/>　　在两条巨蟒对峙而僵住的时候。
<br/>　　突然，这片封闭的区域波动涟漪了起来，一道美丽的身影浮光掠影，诡异的出现在了高空中，出现在了两条惊天巨蟒的中间地带。
<br/>　　她宛如就九天仙女下凡，白云飘飘，美的不可方物，一张绝美的容颜含妖含俏。

<br/>　　卧槽？？？
<br/>　　姐姐？
<br/>　　蛇妖还有姐姐，什么情况呀...

<br/>　　“你们两个就此罢手吧”她的声音透着空灵，白衣飘飘，容光绝艳。
<br/>　　金蚕蛊蛇道：“姐姐，这条蛇精敢在太岁头上动土，等妹妹活吞了他，在来跟姐姐叙旧”

<br/>　　白衣女子飞到了金蚕蛊蛇的眼前，六十米的巨蟒，一颗眼瞳都有一个人那么大了，但她一点都不怕，还含着笑。

<br/>　　金蚕蛊蛇有点震惊，看了看对面的紫金巨蟒，既然诡异的听话了，身上的金黄光芒闪烁，就地缩小。
