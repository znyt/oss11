        &lt;h3&gt;第一百五十九章  别有意味的家宴&lt;/h3&gt;
<br/>
<br/>        凌辰及刘诚之和厂家的工作人员成紫将工作进行了对接，心中算是松了口气，准备打道回府。
<br/>
<br/>        在临行之前，董总请凌辰又赴了一次家宴，这一次带上了刘诚之。
<br/>
<br/>        凌辰和刘诚之到达董总的家里，董总、董亚子及董巧奕在家，这确确实实是一场家宴。
<br/>
<br/>        凌辰心里在想，这一次的家宴和上一次的家宴有何区别？
<br/>
<br/>        “快来，快来，就差你俩了。”董总笑眯眯的看着凌辰和刘诚之，并且多看了两眼刘诚之。
<br/>
<br/>        凌辰是山西的，刘诚之是苏州的，而桌上的饭菜既有山西菜也有苏州菜，看来董总确实是费了一番心思。
<br/>
<br/>        凌辰的心里有些感动，这么多年，董总确实对他照顾有加，尤其是这一次，背着压力让他继续坐上了总代的位置，恐怕他们后期都不会太顺利。
<br/>
<br/>        他凌辰再怎么有能力，再怎么努力，如果厂家不认可，他，怕是也不会轻易能拿到总代的，这一点凌辰是明白的，所以他对董家在座的这三个人都充满了感激。
<br/>
<br/>        “你们什么时候回去？”入座后董总问凌辰，但是眼睛却是看着刘诚之。
<br/>
<br/>        “定了明天上午的机票，原本还说来向董总辞别的，却没有想到让董总如此费心。”凌辰微笑着回答董总。
<br/>
<br/>        “诚之也是回山西？”董总又笑眯眯的问刘诚之。
<br/>
<br/>        “是的，先回公司！”刘诚之看了一眼董亚子，董亚子正给董巧奕夹菜。
<br/>
<br/>        董总：“诚之到公司几年了？”
<br/>
<br/>        “快两年了。工作能力不错，有他我很轻松的。”凌辰是真心的夸奖刘诚之，小小年纪处理事情周到而且很能考虑别人的感受。
<br/>
<br/>        “凌辰现在还没有结婚吧？”董总随意的问着。
<br/>
<br/>        凌辰和刘诚之相互看了一眼，莫非这就是今天吃饭的目的？
<br/>
<br/>        “目前事业不成，有何颜面谈婚论嫁呢？”凌辰不显山不露水的回答。
<br/>
<br/>        “做到全国总代的位置了，你还想要什么事业？莫非还想要我这个厂子？”董总哈哈哈大笑。
<br/>
<br/>        董亚子和董巧奕都看着凌辰。
<br/>
<br/>        “哈哈哈，如果真能拿到董总的厂子我就可以去迎娶我心爱的姑娘了！少奋斗很多很多年啊！”凌辰也哈哈大笑。
<br/>
<br/>        “这是有了心爱的人？身份地位不太匹配？”董总一针见血：“真是很奇怪一些老古董，只要孩子们情投意合就好了，还非要什么门当户对。”
<br/>
<br/>        “像董总这么开明的父亲确实很少，亚子妹妹比我和我女朋友幸福很多了。”凌辰一脸羡慕，对董总充满了崇拜。
<br/>
<br/>        “哈哈哈，现在都什么时代了，还整以前的老一套。”董总笑眯眯的说：“吃菜，吃菜。”
<br/>
<br/>        “诚之家是？”
<br/>
<br/>        “董总，我家是苏州的，一个农村家庭。”刘诚之淡淡的回答，反正你们也查不到我的家庭信息。
<br/>
<br/>        “农村家庭能培养如此优秀的人才很不错啊，家里应该是书香世家吧？苏州的安林曦认识吗？”董总点点头看着刘诚之。
<br/>
<br/>        “安教授是泰山北斗，无人不知无人不晓，但是我却不认识的。”刘诚之并不想大家知道他的身份，所以也就不承认他是认识安教授的，况且刘诚之也并不知道董亚子的经历。
<br/>
<br/>        作为刘诚之来说，他就是凌辰单位的一个工作人员，能干多长时间他也不清楚。
<br/>
<br/>        “诚之有女朋友了吗？”董总一个问题接一个问题的问刘诚之
<br/>
<br/>        “诚之还没有谈朋友呢，董总有合适的给他介绍一个？”凌辰直接就替刘诚之回答了。
<br/>
<br/>        刘诚之心里说：“又想让我给你背锅？”但是嘴里却说：
<br/>
<br/>        “我们家是农村的，家长作风比较古板，总是喜欢让我相亲，喜欢媒妁之言。唉！我呢还算是比较孝顺的孩子，有时候还是很听父母亲的话的。”
<br/>
<br/>        董亚子立马就感觉自己被嫌弃了：“切，还媒妁之言呢，你还活在古代呢？土！像凌辰哥哥就知道追求自己所爱，就算是头破血流也不回头，宁家可是很厉害的家族呢。”
<br/>
<br/>        “是啊，我是很土啊，我怎么能和凌总比呢？”刘诚之微笑着说。
<br/>
<br/>        “刘诚之，你！”董亚子不知道为什么见了刘诚之就是气。
<br/>
<br/>        “亚子，你看看你说的什么话？诚之哪里土了？连话都不会说，难怪嫁不出去了，看来只能将你嫁给何少华了。”董总生气的说。
<br/>
<br/>        “什么？何少华？你要将亚子嫁给何少华？”刘诚之吃惊的问。
<br/>
<br/>        想起何少华他都会起鸡皮疙瘩，他给了董亚子一个可怜的眼神，摇了摇头。
<br/>
<br/>        “何少华不好吗？”董总又笑眯眯的问。
<br/>
<br/>        “也不是不好吧，我总觉得何少华配不上亚子。”
<br/>
<br/>        “凌辰，你觉得何少华怎么样？说实话，何少华让人来提亲了，甸城，太远了，不想将女儿嫁过去啊。”
<br/>
<br/>        “董总刚还在说你不是老古董呢，人家来提亲你这是准备答应了？”凌辰心里在说，老狐狸就是老狐狸啊。
<br/>
<br/>        “我当然不是老古董，那现在亚子也不是没嫁吗？人家还说要俩个人先处着，说实话不知道该如何回绝。”董总叹了口气，他如花似玉的女儿怎么能便宜了何少华？
<br/>
<br/>        宁愿便宜眼前这俩个帅小伙子，虽然身世一般，但是也不能说是一般，凌辰的家世算是中等，但是小伙子肯努力愿上进，比起那个一无是处的何少华要强很多。
<br/>
<br/>        眼前的这个刘诚之，人小鬼大，身世更是不一般，不过小伙子不愿意暴露身份，他也不去揭露他，两下一对比，越发的觉得眼前的俩个小伙子人品更好，人家才是真正的有教养。
<br/>
<br/>        可惜凌辰有了喜欢的人，刘诚之看样子对董亚子没有兴趣，可是亚子之前是明明喜欢凌辰的，为什么就突然转了性子了，非要拉上刘诚之？
<br/>
<br/>        “董总是畏惧何少华身后的人？俗话说得道多助失道寡助，你好好做你的生意，怕他做什么？还想搭上女儿的幸福？”刘诚之一语双关，听得董总心里一惊。
<br/>
<br/>        凌辰也看了一眼刘诚之。
<br/>
<br/>        “正因为我们是做生意的人，所以还是要权衡利弊的啊。”董总的声音多少有些无奈。
<br/>
<br/>        “亚子，你喜欢谁就嫁谁，幸福是掌握在你自己的手中的，别成了牺牲品。”刘诚之毕竟年轻，血气方刚。
<br/>
<br/>        “我知道了。”董亚子脸一红，看了一眼刘诚之低下了头。
<br/>
<br/>        “哈哈哈，还是年轻好啊，有气魄，来，凌辰，我们几个喝一杯。”董总哈哈大笑。
<br/>
<br/>        “年后我们怕是有一场大战要打了，我希望在三一五的时候能看到成效。”董总看着凌辰。
<br/>
<br/>        “如果想三一五要看到成效的话，只能是别的地方，甸城是不行，我们对甸城还不熟悉。”凌辰不打没有把握的帐。
<br/>
<br/>        “我呢，可以提前走一趟甸城。”刘诚之微笑着说，何少华是不会拿他这个不起眼的助理当回事。
