        到了楼下，傅昼景走了一大段路程，才上了季思妍的车。
<br/>
<br/>        季思妍说要直接开车来他家楼下等他，但是被傅昼景拒绝了。
<br/>
<br/>        或许是做贼心虚，又或许是别的原因……
<br/>
<br/>        季思妍在傅昼景的心里，是永远都见不得光的。
<br/>
<br/>        就在车上，傅昼景当着季思妍的面，打了一通电话给傅夜沉。
<br/>
<br/>        “大哥，我要出差半个月。筱颖想出去工作，我把她安排到你那儿总部的设计部去了，明天她便会去你那儿的总部报到。对了，这半个月的时间里，还麻烦大哥替我关照一下筱颖。”傅昼景直接开门见山地说。
<br/>
<br/>        他开口让大哥关照筱颖，就像当初大哥跟他开口，要让他关照一下来滨江城玩的苏漫雪一样。
<br/>
<br/>        “好。”傅夜沉声音冷淡地答应了。
<br/>
<br/>        傅昼景接着闲聊了几句，就跟大哥傅夜沉挂了电话。
<br/>
<br/>        季思妍瞥了傅昼景一眼，话中有话地问：“你觉得，你大哥信得过你吗？”
<br/>
<br/>        傅昼景不假思索地回答：“当然信得过！”
<br/>
<br/>        “那就好！”季思妍只笑不语，见傅昼景系好安全带后，便发动了车子。
<br/>
<br/>        这次他们两一起出差，纯粹只是为了自己的新公司去谈业务。
<br/>
<br/>        在工作上，他俩是山鸣谷应的搭档，在私生活上，他俩是耳鬓厮磨的床伴。
<br/>
<br/>        那她童筱颖对于他傅昼景来说，又是什么呢？
<br/>
<br/>        只不过是个有名无实的妻子罢了！
<br/>
<br/>        季思妍在心底这般嘲讽童筱颖的时候，却忘记了，童筱颖终归到底都是傅昼景的“合法妻子”，而她自己却成了名副其实的“第三者”。
<br/>
<br/>        “阿景，如果……我早一点回来……你还会娶筱颖吗？”季思妍一边开车，一边若有所思地问。
<br/>
<br/>        傅昼景身心一怔，微微凝眉，良久后才缓缓地回答：“或许不会。”
<br/>
<br/>        他只是说，或许不会，而没有肯定的用“不会”两个字。
<br/>
<br/>        由此可见，筱颖是真的走入了阿景的心。
<br/>
<br/>        季思妍心痛了一下，转瞬间，她又释然地笑了笑。
<br/>
<br/>        这次他们谈的合作项目是在外省，出差的时间比较长，所以，她有很多的时间可以和他单独相处。
<br/>
<br/>        她又何必去计较呢？
<br/>
<br/>        季思妍开了两个小时的长途后，和傅昼景一起进了服务区里休息。傅昼景去上洗手间的时候，季思妍买了一些小吃坐在了就餐区的餐桌前等傅昼景。
<br/>
<br/>        她买的，全是以前阿景爱吃的。
<br/>
<br/>        傅昼景回来后，看到桌上的那些小吃，不禁欣慰地咧嘴一笑。
<br/>
<br/>        五年过去了……
<br/>
<br/>        思妍还记得他喜欢吃什么。
<br/>
<br/>        傅昼景的口味一直没变，或许，他是一个真的很念旧的男人。
<br/>
<br/>        “酒店我已经订好了……不过……只剩一间豪华套间了……行吗？”季思妍叉起一个小笼包，体贴地送到傅昼景的嘴边，温柔地问道。
<br/>
<br/>        傅昼景毫无顾虑地咬住季思妍亲手喂过来的包子，点了下头。
<br/>
<br/>        季思妍抿嘴一笑，心底乐开了花。
<br/>
<br/>        剩下的路程，是傅昼景开的车。
<br/>
<br/>        他们下榻酒店的时候，已经晚上七点多了。
<br/>
<br/>        傅昼景就在酒店里点了餐，让服务员送上门来，然后拿起睡衣打算先去洗个澡。
<br/>
<br/>        季思妍则在房间里整理自己和傅昼景的衣物。
<br/>
<br/>        “滋滋——滋滋——”
<br/>
<br/>        此时，傅昼景换下的西裤里的手机震动了起来。
<br/>
<br/>        季思妍放下手里的活，走过去捡起浴室门口矮凳上的西裤，将手机掏了出来。
<br/>
<br/>        来电显示上是——
<br/>
<br/>        筱颖……
<br/>
<br/>        季思妍的手指抖了抖，刚准备接听的时候，浴室门开了。
<br/>
<br/>        “是筱颖的电话……”季思妍惊怔了一下，心虚地将手机递给了从浴室门缝里伸出一条胳膊的傅昼景。
<br/>
<br/>        傅昼景从季思妍的手中拿走电话后，便滑动到了接听键。
<br/>
<br/>        “嗯，我到了。”
<br/>
<br/>        “……”
<br/>
<br/>        “好。”
<br/>
<br/>        “……”
<br/>
<br/>        “你一个人在家里也要注意安全。”
<br/>
<br/>        “……”
<br/>
<br/>        “遇到什么麻烦了，你可以找我大哥。”
<br/>
<br/>        “……”
<br/>
<br/>        “不想找他的话，你找赵斌也行。”
<br/>
<br/>        “……”
<br/>
<br/>        “好，那就这样。拜拜，晚安。”
<br/>
<br/>        季思妍站在门边，只听得到傅昼景的回话，并不知道电话那头的筱颖和傅昼景说了什么。
<br/>
<br/>        不过，从傅昼景的回话声中倒是可以听出，他们只不过是寻常的寒暄而已。
<br/>
<br/>        “思妍，以后不要乱接我的电话。”傅昼景将手机从浴室门缝里递了出来。
<br/>
<br/>        季思妍立即接过手机，微微凝眉，弱弱地应了声：“好。”
<br/>
<br/>        就在傅昼景将要关上浴室门的时候，季思妍突然将手伸了过去挡了一下。
<br/>
<br/>        傅昼景便顺着季思妍挡住的这只手，直接抓住了她的手腕，将她整个人拉进了浴室里，利索地搂住了她的腰肢，抵到墙上热吻了起来。
<br/>
<br/>        花洒释放的热水，冲掉了长途行车的疲惫，悠长而放纵地化成水雾，萦绕在他们的周遭。
<br/>
<br/>        傅昼景褪去了季思妍身上的衣物，在这片温暖的水雾里，与她相拥。
<br/>
<br/>        季思妍缓缓蹲下身去，任水流划过炙热的肌肤，在唇舌间流连……
<br/>
<br/>        筱颖……
<br/>
<br/>        此时此刻，傅昼景的脑海里，却浮现出了童筱颖的倩影。
<br/>
<br/>        一切的火热和黏腻，禁锢得让人无法自拔。
<br/>
<br/>        童筱颖刚准备揭开锅盖，却不小心被锅盖的边缘给烫到了小拇指，痛得她松了手，锅盖“哐当”一声落在了地上。
<br/>
<br/>        某种不详的预感漫上心头……
<br/>
<br/>        “叮咚——叮咚——”
<br/>
<br/>        此时，门铃响了。
<br/>
<br/>        童筱颖不得不拿起一块抹布隔着将锅盖从地上捡起来，然后关了炉火，走出厨房去玄关处，按开了可视对讲机，查看来访者是谁。
<br/>
<br/>        可视对讲机的屏幕上，出现了一个身穿物业管理制服的青年男人。
<br/>
<br/>        童筱颖温和地问道：“你好，请问你找谁？”
<br/>
<br/>        “你好，我是物业公司的，楼下的住户跟我反应，你们家厨房地面漏水，让我上来跟你们说一声。然后，我顺便看看是你们厨房里具体哪个地方漏水，我好请水电工过来修理。”对方彬彬有礼地说道。
<br/>
<br/>        “你稍等！”童筱颖转身去了厨房，上上下下找了一圈后，才发现洗菜盆下的水管确实在滴水。
<br/>
<br/>        随后，童筱颖又回到了门口，将门给打开来，一脸抱歉地说道：“对不起，让你久等了。我家厨房的水管坏了。”
<br/>
<br/>        “我进去帮你看看是什么情况，然后再叫水电工过来给你换。”青年男人客客气气地说道。
<br/>
<br/>        童筱颖点了点头，放了这青年男人进屋。
<br/>
<br/>        青年男人径直走到厨房，询问：“是哪里的水管坏了？”
<br/>
<br/>        “洗菜盆下面。”童筱颖回答道。
<br/>
<br/>        青年男人这才蹲下身去，打开了洗菜盆下面的柜门。
<br/>
<br/>        童筱颖心里刚放松警惕，抬起手来抓在防盗门的手柄上，准备关门的时候，一只戴着黑色手套的大手突然从门外伸了进来，将她吓了一跳，并防不胜防地捂住了她的口鼻。
<br/>
<br/>        “唔——唔——”童筱颖一边挣扎，一边用力吭声，把所有的希望，都放在了那边蹲身查看水管的青年男人身上。
<br/>
<br/>        然而，那青年男人听到这边传来动静后，只是淡定地站起身来，对着童筱颖咧嘴一笑。
<br/>
<br/>        紧接着，一丝冰凉的感觉，挨在了童筱颖的脖子上。
<br/>
<br/>        身后的人，轻轻地带关了大门，并扼制住童筱颖，让她不敢轻举妄动。
<br/>
<br/>        而那个假装在查看水管的穿着物管制服的青年男人，则从衣兜里掏出橡胶手套、帽子和鞋套，穿戴好后便开始在家里翻箱倒柜。
<br/>
<br/>        十多分钟过去后，青年男人将这个家搜了个底朝天，也只是从卧室里搜出了上次傅夜沉买给她童筱颖的那几瓶未用完的高档香水，和她的一个钱包里傅昼景给她的生活费，还有她的手机。
