　　青铜古棺内，一片漆黑。
<br/>　　漆黑意味着未知，漆黑意味着神秘。能够带给人无限恐惧。
<br/>　　黑暗再加上此处是棺材，顿时，一股阴嗖嗖的寒意弥漫着，像是有恐怖的大魔王真张开狰狞的血盆大口，等待着猎物的上门。
<br/>　　“黑暗虽然能够带给人神秘之感，但它也会带给人无限恐惧。这样的黑暗对不起主神的威名，所以——”混沌道君立于铜棺之中，幽幽道。
<br/>　　“我要改变它。本道君号称混沌道君，自然是混沌色比较好了。棺内变成混沌光彩，这样就有情趣多了。”混沌道君想了想，决定改变棺内色调。
<br/>　　只见他单手遥遥一指，顿时，发出混沌色的灰蒙蒙光芒。
<br/>　　一丝光，
<br/>　　两丝光……
<br/>　　无量光！
<br/>　　顿时，不到三秒，青铜棺内便充满幽深的混沌色，驱逐了无尽黑暗，增添了几分神秘的色彩。
<br/>　　道君看着眼前青铜棺内，刻着无数的图案。
<br/>　　有凶残的九头神鸟振翅高飞，有浑身上有尺长硬刺毛的巨大凶兽仰天咆哮，有……
<br/>　　布满绿铜锈迹的青铜刻图，有些是《山海经》中记载的荒古凶兽，诸如饕鬄、穷奇、梼杌、混沌等，体型庞大，面目狰狞，栩栩如生，让人望而生畏。
<br/>　　当然，也有一些事上古先民以及远古神话的神邸图像。
<br/>　　最后，是一大片秘密麻麻，星星点点，像极了一片星空的星空图。
<br/>　　道君眼中精光一闪，一个瞬间，便将这些星空图记下，只等以后主神计划开启，就可以在这些星域之中活动了。
<br/>　　有了星空图，以后想怎么玩，就怎么玩？天下何处不可去？
<br/>　　“此棺内的图案不变，但其它的需要改变一番。嗯？”道君看着棺内，略作沉吟。
<br/>　　片刻后，只见他大手一挥，无尽混沌神光，向着棺内刷去，神光所过之处，画面变得更加苍茫古老，充满了神秘的气息。这是道君还在其中加入了一点东西，使得这些刻画变得徐徐如生，宛若下一刻它们便会从图中跳了出来。
<br/>　　比如，那九头神鸟，你好似能够听到它在鸣叫，再如那巨大凶兽，只要你仔细听去，就能听到那巨大的咆哮声，再如那饕鬄等凶兽，它们身上都带着一种魔性的威力，能够摄人心魂，若是胆小之人，一不小心，便会被吓晕。
<br/>　　同时，在它们的身上，有着道光流转，大道的气息，充斥着大殿。
<br/>　　在棺内，“混沌神殿”四个巨大的潦草字横亘虚空，闪烁着道韵的光辉，无尽道则流转，包容一切，它们就仿佛是大道的化身。让人一看就知道非比寻常。
<br/>　　“主神光球呢？该怎么办？”道君沉吟道。
<br/>　　看着眼前的小青铜棺，混沌道君顿时就有了想法。这青铜棺是荒天帝演化的小仙界，其内蕴含着荒天帝的道与法，还有乱古时期的仙界法则，很值得混沌道君参悟。
<br/>　　“就用小青铜棺代替主神光球，至于功法、灵宝、仙法、灵药等需要兑换的东西，我自己就有。不过，这外观需要重新改变一番。”
<br/>　　道君眼前一亮，祭出混沌神光，扫向小青铜棺，将其改变成一只闪烁着九彩光芒的小棺，空间法则加持，将它隔绝出这一片虚空，使得人们只能看得见，却摸不着它，更增添几分神秘色彩。
<br/>　　若是有人想要兑换什么，只要心中默念一声“兑换”即可。在他的脑海之中，就会出现兑换的列表。
<br/>　　“总感觉还差点什么？”道君心中默默思索，紧接着，眼前一亮，道：
<br/>　　“还差一个资深的引导者。”
<br/>　　说完，他屈指弹出一道自己的血液，也不算精血，毕竟以他现在的修为，就算是一滴精血，都有堪比洪荒大罗金仙的战力了。
<br/>　　那滴鲜血落地，迅速就变成一个满头黑发飘逸，一身黑袍披身的青年男子模样，他的样子与混沌道君前世没有修仙之前一模一样。
<br/>　　“这个世界的特殊体质太多了，你如今已经算是纯正的混沌体了，不过，吾却是要你改变。混沌体虽好，但吾已经算是天地间最强的混沌体了，你就不要同我走一样的路了。”
<br/>　　道君祭出法则之火，将他体内的道痕法则全部毁灭，之后再在他体内重塑了一团魔道本源。
<br/>　　“你体内现在只有魔道法则，你如今就是真魔体了。”说完，道君将他脑海中的记忆封住，改动了一些记忆，并告知如今的情况。
<br/>　　他，是一名穿越者，穿越之前，他是地球上的一名杀手，名为夜邪，在执行任务之时，与敌人同归于尽，最后穿越到遮天世界，同时，他也知道遮天世界的发展剧情。
<br/>　　道君一个传送，便将他传送道了地球。
<br/>　　“如此，应该可以了。本道君还是到小青铜棺之中创法去吧。如若不然，到时若是没有兑换的东西，自己这个主神才算憋屈呢。就让一道神念融入这青铜棺内，为吾监视这棺内的动静。”
<br/>　　混沌道君说完，一道神念从他体内飞出，迅速融入到青铜棺中，谁也不知道他在哪里。接着，道君掀开小青铜棺，化作一道流光，飞入小仙界之中，去参悟此界的道则与创主神需要的兑换法与兵器。
<br/>　　小仙界中，仙光纵横，道则清晰，触手可及，道音袅袅，霞光纷飞。
<br/>　　道君一进入此界，全身仙光涌出，又有无数仙岛法则翻飞，有如仙光，有如飘带，道音阵阵，悦耳动听。
<br/>　　他本是仙人，体内蕴含着仙道法则，这一刻催动体内的仙岛法则，立刻就与青铜棺内的仙岛发生共鸣：
<br/>　　“天之道，损有余而补不足……”
<br/>　　神秘的大道道音，浩大而深奥，像是从远古洪荒划破时空，传荡而来。最终如同黄钟大吕一般在混沌道君耳旁震动，传入他心中。
<br/>　　大道天音，如渊如海，深奥浩瀚，每一个字响起，都如山海崩裂，响彻天地。
<br/>　　聆听大帝帝文，每时每刻，都有新的大道法则生成！
<br/>　　混沌道君陷入悟道之中，无数法则一举领悟，脑后菩提古树悬浮，阵阵智慧之光飘散而来，助他悟道。
<br/>　　混沌道君整个人立在虚空，神音如钟，悠远而浩大，大欢喜，大解脱，大彻大悟的感受浮上心间，他越来越强，越来越壮大，就仿佛一尊混沌之中诞生的魔神，他无比古老。
<br/>　　轰！
<br/>　　不知过了多久，道君睁开眼睛，身边所有异象消失，混沌道君再次变为原样，普普通通，谁也不知道，他在刚才那几个呼吸的时间中，便领悟了数百道大道法则。
<br/>　　“这个世界，吾越来越喜欢了。不过，还是先创法吧。本体那边的各种道法，可以全部搬来，吾还需要创些简单的道法与灵器。毕竟，没有人能一开始就兑换顶级的仙法道法。”
<br/>　　就这样，道君再次闭目，陷入了创法状态之中，在他的身后，菩提古树摇摇曳曳，无数造化玄光纷飞，智慧之光飘散而来，助他创法。
<br/>　　而此时，地球上，一个谁也没有发现的角落里，时空扭曲，出现一个黑洞。紧接着，黑洞中出来一个人，他昏迷在地，正是夜邪。
<br/>　　一会儿，一个名叫叶梵的人，路过此地，见此，将他带走。
<br/>　　……
<br/>(本章完)
