夏夕绾迅速抬眸，她在门边看到了一道高大英挺的身躯，陆寒霆来了。
<br/>
<br/>        陆寒霆刚回来，一身的正装，白色衬衫打着领带，精良合体的黑色西装，外面罩了一件黑色大衣，举手投足弥漫着一股优雅倨贵，还有高高在上的强大气场。
<br/>
<br/>        苏希也看到了陆寒霆，他并没有见过陆寒霆，自然是不认识的。
<br/>
<br/>        但是陆寒霆那双狭眸辗转落在了他还扣在夏夕绾腰肢的那只大手上，苏希就感觉头皮一麻，下意识里松开了手。
<br/>
<br/>        这时霍西泽拍了拍手跑了过来，“二哥，那些人都被我解决了。”
<br/>
<br/>        这一下熟悉的人来了，苏希看到霍西泽一惊，霍西泽可是海城的小霸王，谁人不识？
<br/>
<br/>        这时顾夜瑾迈着斯文的步伐走了过来，很明显刚才他没出手，等霍西泽把人解决了才进来了，他清寒的黑眸看向苏希，带了一些玩味，“苏少，这大晚上的我们都过来抢人了，你还是把人放了吧。”
<br/>
<br/>        苏希看到霍西泽时是心惊，当看到这位簪缨世家顾家少主顾夜瑾时，他的心里已经掀起了巨浪。
<br/>
<br/>        海城顾霍两家，他今天全部见到了，那…这个男人又是谁，苏希看向了陆寒霆。
<br/>
<br/>        陆寒霆没有再看苏希，他拔开长腿来到了夏夕绾的面前。
<br/>
<br/>        夏夕绾不知道他怎么这么早就回来了，刚才他踹门而入的时候她还被苏希抱着，如今衣衫凌乱，这让她觉得很难堪。
<br/>
<br/>        夏夕绾伸手捂了一下自己的领口。
<br/>
<br/>        陆寒霆看着女孩低垂的羽捷，嗓音低沉沁凉，“你没告诉他你是谁的女人？”
<br/>
<br/>        夏夕绾手指一蜷，有点像被老师训斥了的孩子。
<br/>
<br/>        陆寒霆伸手将她打横抱起，往门外走去。
<br/>
<br/>        苏希从来没有在海城见过这样的大人物，这男人有顾家少主和霍家太子伴随左右，身份非富即贵，况且他身上沉淀着一个上位者该有的冷戾杀伐，相当的危险。
<br/>
<br/>        苏希忍不住对着陆寒霆英挺的背影问，“你是谁？”
<br/>
<br/>        陆寒霆脚步没停，只是唇角勾出了一点薄凉讥诮，“她，是我的女人，今天的事情如果再发生第二次，到时你自然就会知道我是谁。”
<br/>
<br/>        话音落下，陆寒霆和夏夕绾的身影也消失在了视线里。
<br/>
<br/>        苏希一下子拽紧了拳，好猖狂的语气，不过…
<br/>
<br/>        苏希迅速想到了一种可能性，这个男人是不是就是…传说中十几岁开始玩转商界的一代大佬…陆寒霆？
<br/>
<br/>        怎么可能？
<br/>
<br/>        夏夕绾怎么跟陆寒霆这样的大人物扯上了关系？
<br/>
<br/>        但是，顾夜瑾和霍西泽怎么解释，帝豪酒店派出的那辆劳斯莱斯怎么解释，今天晚上这海城最尊贵的几个男人空降这里将她给抢走，又怎么解释？
<br/>
<br/>        这天下怎么可能有这样的巧合？
<br/>
<br/>        …...
<br/>
<br/>        夏夕绾被陆寒霆抱上了加长版的商务豪车，陆寒霆坐在后座上，她还坐在他的怀里。
<br/>
<br/>        夏夕绾突然想到了林婶，迅速道，“林婶她…”
<br/>
<br/>        “送去医院了。”
<br/>
<br/>        夏夕绾抬眸看着男人，陆寒霆也在看着她。
