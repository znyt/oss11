　　青云门主修太极玄清道，在修成第四层御物前，只专心修炼玄清道积攒法力。
<br/>　　等修成御物找到合适法宝后，再专心修炼攻伐之法。
<br/>　　青云每六十年举行一次的七脉会武，最低标准就是御物。没自己的法宝上去就等着挨打吧。
<br/>　　在御物之前，弟子们会修炼些凡俗剑法强身健体。这也是藏书楼会有那么多世俗武功的原因。
<br/>　　对于修炼玄清道增强神魂的一众青云弟子来说，世俗武功并不难。这些还没有超脱凡俗的功法，比的就是眼疾手快。
<br/>　　普通人只能靠常年苦练才能形成肌肉记忆，在对抗的一瞬间，身体超过意识做出神而明之的反应将对手击倒。
<br/>　　但对修仙之辈来说，一个个神智清明，几乎很快就能把招式上手。且修为越高神魂越强，对需要技巧的世俗武功来说，就越发掌控由心。
<br/>　　杜必书修到玄清道第四层，虽不能御物但也不远。所以他才会自信能瞬间击败苏重。凭他第四层玄清道，神魂强大反应迅速。苏重不过修炼半年，理论上根本就不是一合之敌。
<br/>　　“师弟，你先来吧。”杜必书手里握着木剑随意的站着。
<br/>　　“不了师兄，还是你先来吧，我对世俗武功比较熟悉。”苏重笑眯眯道。
<br/>　　“师弟，玄清道有助于修炼世俗武功，还是你先来吧。”杜必书突然想到苏重不能修炼太极玄清道，脸上露出歉然笑容。
<br/>　　“真让我先出手？”苏重挑了挑眉，以为修炼了太极玄清道就能胜过我？
<br/>　　苏重转头看了眼师娘苏茹。
<br/>　　“尔旦，不用留手，给我教训教训这个不知天高地厚的混蛋小子！”苏茹气的柳眉倒竖。
<br/>　　太极玄清道？就你那第四层修为都挡不住一枚剑气！苏重那天一招就把青铜鼎切成两半，她可是亲眼所见！
<br/>　　“那我来了？”苏重了然一笑。
<br/>　　“快出招吧！”杜必书满脸不在意。
<br/>　　苏重嘴角一翘，身形一闪消失。下一刻，木质剑尖突兀出现在杜必书咽喉前。
<br/>　　杜必书双眼圆瞪，这会儿他连剑都还没抬起来！竟然已经中剑？
<br/>　　“小师弟你这不对，你这是偷袭，我还没准备好呢。”杜必书满脸懵逼，他还稀里糊涂呢。
<br/>　　旁边众人也一脸懵然，怎么还没开始就结束了？
<br/>　　听到杜必书下意识争辩，苏茹气的差点儿一剑劈了他。
<br/>　　“那这次不算，再来一次？”苏重给师娘使了个颜色，打定主意要来一次狠的。
<br/>　　“好，再来！”杜必书满脸欣喜的答应。
<br/>　　苏茹气的双目含煞，真是个蠢蛋，被自家师弟一招打败，竟然还有脸再来一次？！她也不开口提醒，任由自家小徒弟施展。
<br/>　　苏重虽然没修炼太极玄清道，但却修成奇诡庚金剑气。虽不会御剑飞行，但如果突然出手，就是修炼时间最久的宋大仁都不一定是对手。
<br/>　　庚金剑气可是攻杀利器，虽然不如玄清道厚积薄发，却最擅长杀伐。
<br/>　　苏重后退几步和杜必书拉开距离。
<br/>　　“师兄，准备好了吗？”苏重一手提着剑，浑身上下充满随意气息。
<br/>　　“好了，来吧！”杜必书这回认真起来，一手捏着剑诀，把剑横在胸前，双眼死死盯着苏重。
<br/>　　青云弟子修炼玄清道，身体灵魂被不断洗练，反应速度极快。一个修炼了一二层玄清道的青云弟子，对上世俗武林高手，就好似大人戏耍小孩。不管对方速度多块，技巧多好，在对方眼里都是小孩跳舞一般，轻易就能掌控。
<br/>　　杜必书相信，只要他认真起来，就能挡住苏重的剑法。
<br/>　　“那我来了。”苏重也不含糊，随意抬脚而行，身影却骤然消失原地。
<br/>　　杜必书眼睛睁的老大，只觉一柄煞气长剑悍然飞来！无形波纹如影随形！
<br/>　　他下意识的挥出一剑，然后就感觉喉咙一凉，浑身汗毛都倒竖起来。
<br/>　　苏重木剑再次抵在他咽喉上。杜必书下意识咽了一口唾沫。
<br/>　　“师兄，咱们再来一次怎么样。”苏重不等杜必书开口，再次邀请道。
<br/>　　杜必书还有些恍惚，下意识点头。
<br/>　　苏重再次回到原地：“师兄，小心了！”
<br/>　　话音刚落，苏重木剑再次放在杜必书咽喉下。
<br/>　　“师兄，还好吗？”苏重温和笑道。
<br/>　　杜必书看到苏重温和笑容，不由自主的打了个寒颤。他怎么也无法把眼前这个温和师弟，和刚才那柄充满煞气的木剑联系在一起。
<br/>　　“丢人现眼！刚才是谁说一招就能打败人家来着。”苏茹瞪着仍然在恍惚的杜必书呵斥道。
<br/>　　其他几个师兄也满脸不可思议看着长剑。杜必书虽然修炼时间不长，但修到玄清道第四层。竟一招就败在苏重剑侠。
<br/>　　他们看不明白，苏重到底是怎么赢的。
<br/>　　苏重剑法速度快，但他们能看清楚。可杜必书却呆愣愣站在原地，好像等着苏重把剑放在他咽喉前。
<br/>　　难道是故意放水？
<br/>　　开始他们还能这么想，可连续三次被毫无还手击败，几人就看不明白了。
<br/>　　“你们一个个都上，不允许用法宝，和你们师弟比一比！”苏茹今天是打算好好给这几个徒弟上一课。
<br/>　　苏重剑法奥妙别人看不明白，她却能看出一二。
<br/>　　剑法普通，却带着庚金剑气威势。庚金剑气是攻杀利器，充满凶狠霸道气势。
<br/>　　田不易就曾吃过暗亏。大竹峰弟子修为不错，实战经验却差了些。凶狠剑气威慑，瞬间被夺取神志，一剑失败也就可以理解。
<br/>　　其他几个师兄弟没办法只能和苏重放对。
<br/>　　然后从五师兄到大师兄宋大仁，一个都没能撑过苏重一剑，全都被苏重放到。
<br/>　　“到底是怎么回事？”一众师兄弟更加摸不着头脑。还没反应过来就被击败，几人满心郁闷。
<br/>　　“尔旦，给你这几个心高气傲的师兄们涨涨见识！”苏茹见一众徒弟被打击的不轻，仍然不想放手。
<br/>　　这些徒弟一贯懒散，大竹峰本来就人少，手上功夫还差，七峰里面常年垫底，田不易心里不畅，她心里也憋着一口气。
<br/>　　这次逮着机会，她就要狠狠打击一下这几个徒弟，让他们有上进心。
<br/>　　苏重无奈看了眼苏茹，知道这是让他展示庚金剑气。转头打量四周，祠堂门口那个全新的大鼎怎么看怎么耀眼。
<br/>　　挥手就一道庚金剑气飞了过去。
<br/>　　“住手！”一声爆喝从祖师祠堂传来。
<br/>　　然后当啷一声，大鼎应声碎成两半。
<br/>　　一众师兄目瞪口呆，这还是那个不能修炼太极玄清道的小师弟？
<br/>　　田不易从祠堂里冲出来，一脸痛心疾首的瞪着苏重：“我说二蛋啊！大竹峰祖师爷们没招你惹你吧？啊！你怎么就逮着我这大鼎不放啊！”
