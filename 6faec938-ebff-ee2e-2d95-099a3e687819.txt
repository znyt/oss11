　　“Move！”
<br/>　　麦克米兰握着战术步枪冲出楼房，在经过那辆倒扣在地面上的悍马车时，手中的战术步枪对着悍马车漏到地面上的汽油打出一个点射。
<br/>　　“呼！”
<br/>　　被子弹击中的瞬间，火焰便蹿了起来，眨眼间，就把悍马车包围起来。火焰引燃了悍马车的油箱，引发了爆炸，火光冲天而起，炽热的高温逼迫着周围人远离。
<br/>　　重机枪子弹箱中的子弹在高温下发生爆炸，发出“噼啪噼啪”的爆裂声，飞出的子弹头做着无法用物理定律解释的不规则运动，朝着四周飞去，几名“中彩”的影子部队士兵被子弹击中。
<br/>　　不规则运动下的弹头对那几名影子部队士兵造成的伤口也是不规则的，飞旋的弹头打着转击中了一名影子部队士兵的右大腿，然后又打着转从他右大腿内侧飞出，这个短短的过程中，弹头打断了他的大腿腿骨，也拉断了他的股动脉，鲜血呈箭状从他腿部伤口中喷射而出。
<br/>　　这名影子部队士兵哀嚎着抱着他受伤的大腿躺在地上，但是此时却没有人上前去救他，因为周围人都在躲避着从悍马车中零星飞出来的子弹，毕竟谁也不想成为下一个被不规则运动的子弹头击中的对象，而且即便是上前救援，也不可能将那名手上的影子部队士兵从死神的手里抢夺回来。
<br/>　　这样的伤势，即便是进行战地手术也无济于事，因为在股动脉被拉断的瞬间，断裂的动脉血管就蜷缩到他身体中，无法再被连接起来。
<br/>　　罗昊、麦克米兰这些人趁着这个机会，带着Parker往军事监狱外突围，可是即便有了Bravo小队的支援，但是在失去了“黑鹰”直升机的接应，以及直升机上M134机枪压制性的火力支援，罗昊和麦克米兰的小队人员和火力上的弱点很快就呈现了出来。
<br/>　　距离军事监狱大门还有不到两百米的时候，Alpha和Bravo两只战术小队已经被前方的影子部队士兵彻底堵死，根本无法继续突围。
<br/>　　麦克米兰看了眼手表上的倒计时，说道：“还有一分钟，海军航空部队就会发起空袭。”
<br/>　　“来不及了！”
<br/>　　罗昊轻轻摇头，如果是在平时，一分钟的时间冲过不到200米的距离，可以说是绰绰有余，但是现在前方被影子部队完全封锁，要在一分钟的时间内突破前方的封锁突围出去，是根本不可能的事情。
<br/>　　“就近寻找掩体，进行隐蔽！”
<br/>　　罗昊对着周围人大声命令，“等到第一轮空袭过去后，我们再突围！”
<br/>　　“现在也只有这个办法了。”
<br/>　　麦克米兰苦笑了一下，说道：“但愿开飞机那两个家伙的眼神能好一点，别把JDAM扔到我们头上。上帝保佑美利坚！”说着麦克米兰在胸口比划着，向他们伟大的上帝进行祈祷。
<br/>　　美国的报纸杂志电视电影有时会这么说，连美国的战斧导弹和JDAM都爆炸出这样的声音，“上帝保佑的是美利坚。”
<br/>　　所以，美国人一直都坚信，上帝是保佑美国人的！
<br/>　　在这个军事监狱中要寻找掩体很容易，沙袋堆积出来的机枪堡垒，还有用钢筋混凝土浇筑出来的射击掩体，要寻找一个暂时用来躲避空袭的掩体并不算一件困难的事情。
<br/>　　“还有三十秒！”
<br/>　　麦克米兰给自己手中的战术步枪换上一个备用弹匣，对着身边的Bravo队员喊道：“烟雾弹！”
<br/>　　两名Bravo队员取出烟雾弹，拉开保险，扔了出去，短短几秒钟的时间，从烟雾弹中释放出来的浓烟，就形成了一道掩护带。
<br/>　　“放低身体，不要起身，对方可能有热成像仪！”罗昊向身边的行动队员提醒道。
<br/>　　影子部队是西伦科特按照特种部队的标准打造出来的精锐部队，他们用精锐的单兵装备，掌握精准的枪法和近身格斗技巧。
<br/>　　可即便罗昊做出了提醒，还是晚了一步。一名Bravo队员在起身撤离的时候，被从后方飞来的一颗子弹打中了脖子，栽倒在地上。
<br/>　　“Damn！”麦克米兰咒骂一声附身去查看倒地队员的伤势。这名Bravo队员几乎被子弹打穿，伤口皮肉外翻，能看到颈椎骨，右侧颈动脉完全断裂，就算是上帝在场也不可能救回他的性命。
<br/>　　“他死了。”
<br/>　　麦克米兰伸手将牺牲队员睁大的眼睛合拢，轻轻摇了摇头，然后猫着身体，快速向一处机枪堡垒靠近。
<br/>　　罗昊两个点射将原本待在机枪堡垒中的影子部队士兵击毙，然后把他们的尸体拽出机枪堡垒，自己跳了进去，对着追在其他人身后的影子部队士兵不断进行点射，掩护着其他人撤离。
<br/>　　这个用沙袋堆积而成，足有一点三米高的机枪堡垒并不算小，但是一下子挤入了近十几个人，还是立刻就显得拥挤起来。
<br/>　　“十秒钟！”
<br/>　　麦克米兰看了眼手表上的时间，大声提醒。
<br/>　　沙袋背后的所有人立刻收回武器，矮身藏了起来。突然中断的枪声，令得试图包围他们的影子部队士兵有些意外。
<br/>　　联合行动司令部中，所有盯着正前方屏幕的人员全都屏住了呼吸，在这一刻，仿佛就连他们的心跳也全都停止了跳动，屏幕右上角有个位置，鲜红色的倒计时数字正在飞快跳动，还有最后五秒钟。
<br/>　　两架满载弹药的F/A-18E“超级大黄蜂”战斗机以俯冲的姿态冲向地面上的军事监狱，并且投下了第一轮空袭的四颗JDAM，以及两枚“铺路3”型雷射制导炸弹被扔进军事监狱。
<br/>　　听着从天空中传来的，炸弹落下所发出的尖锐呼啸声，军事监狱中的影子部队士兵被吓得魂飞魄散。这一刻他们明白了，为什么原本突围的特种部队会突然后撤，为什么他们会突然停火，这是为了躲避即将到来的空袭！
<br/>　　轰！轰！轰！轰！轰！
<br/>　　原本因为被破坏了电源而漆黑一片的军事监狱，这个时候在炸弹爆炸的火光下，宛如白昼。那栋Parker原本躲藏的楼房，在两枚JDAM和一枚“铺路3”型炸弹的攻击下，遭到致命摧毁。
