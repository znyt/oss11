　　怪物老者似乎看穿了叶尘的想法，微微一笑，道：
<br/>　　“我猜你心里一定在想，我怎么会知道渡劫之后的事情，对不对？”
<br/>　　“那是因为，我的主人，便是修真界亿万年以来，极少数渡劫成功之人！”
<br/>　　说到这里，怪物老者微微一顿，接着又道：
<br/>　　“至于我家主人的尊号，说出来你也未必知道，他便是百万年前，曾经一统修真界的昊天大帝！”
<br/>　　“昊天大帝？”
<br/>　　叶尘顿时两眼一瞪，
<br/>　　他还真知道！
<br/>　　在修真界的历史上，能够渡劫成功者固然不多，可是无数纪元下来，却也积累了不少。
<br/>　　可是在渡劫之前，能够一统整个修真界的，也不过寥寥数人罢了。
<br/>　　而这其中，最后一次统一整个修真界的，便是百万年前的这位昊天大帝！
<br/>　　因而在前世，叶尘倒是没少听说这位一代大帝的事迹，所以自然也就记得。
<br/>　　“前辈，那昊天大帝和你说的昊天界，又是什么关系？”
<br/>　　叶尘忍不住开口问道，连称呼也变成了前辈，态度比之前恭敬了许多。
<br/>　　这怪物老者，既是昊天大帝的仆从，无论是论实力，还是资历，都当得起叶尘称呼一声前辈。
<br/>　　怪物老者微微一笑，
<br/>　　“你总算问到点子上！其实，这昊天界，是由我的主人昊天大帝，在渡劫成功之后，开创的一方宇宙！”
<br/>　　轰！
<br/>　　怪物老者这话一出口，叶尘顿时有些懵了，
<br/>　　他所在的地球，居然是昊天大帝创造的一方宇宙！
<br/>　　不过，
<br/>　　叶尘惊愕过后，旋即又发现了问题所在，
<br/>　　“这不对啊！前辈，我所在的那片宇宙，可是已经有着一百多亿年的历史，而昊天大帝不过是百万年之前的人物，怎么会……”
<br/>　　没等叶尘说完，怪物老者摆了摆手，道：
<br/>　　“时间不过是一个计量单位，对于创造主来说，时间是不具备任何意义的，而且昊天界的时间，和修真界里的时间，自然也不能等同……”
<br/>　　经过老者一番解释，叶尘似懂非懂，他对时间法则了解不多，在这方面的悟性，也比不上曦月。
<br/>　　怪物老者接着又道：
<br/>　　“当初，我主人成就混沌仙体之后，自知在修真界待不了多久，于是便混沌世界中，开创了昊天界，并且在修真界和昊天界之间，强行打开了一个通道！”
<br/>　　“如此一来，主人的家人、朋友，还有我们这些仆从，就可以通过昊天界，继续和主人待在在一起……”
<br/>　　听到这里，叶尘早已经惊骇的说不出话来。
<br/>　　自己所在的那一方宇宙，居然是那位昊天大帝所创！
<br/>　　不愧是能够一统整个修真界的一代大帝，此等手段，就连他都觉得匪夷所思，继而又欣喜不已，
<br/>　　“如此说来，前辈应该知道如何返回地……昊天界？”
<br/>　　不料，那怪物老者却摇了摇头，道：
<br/>　　“本来是可以的！可惜在十万年前，我家主人突然在混沌世界中失踪，昊天界成为了无主之界，没有了主人的维持，昊天界无法从混沌世界中吸收养分，和修真界之间的通道，也开始逐渐封闭……”
<br/>　　“再后来，昊天界中的灵气越来越稀薄，我和其他强者，只好全都退回到了修真界，只留下了一些修为较弱的人留守。”
<br/>　　“之后，又过去了数千年，昊天界和修真界之间的通道彻底断开，如今连我也找不到进去的入口了！”
<br/>　　“而当初和我一同返回修真界的强者，陨落的陨落，失踪的失踪，我走遍了整个修真界，也未能找到进入昊天界的入口……”
<br/>　　听完老者的话之后，叶尘顿时一阵失望，
<br/>　　“难道就真的没有办法回去了吗？”
<br/>　　老者道：
<br/>　　“当然有！而且这个办法，可能还要靠你来完成！”
<br/>　　叶尘顿时更懵了，
<br/>　　“前辈这话是什么意思？”
<br/>　　老者一脸郑重道：
<br/>　　“你的身上有我家主人的烙印，便是他老人家认定的传人！”
<br/>　　“若是你能完全继承我家主人的传承，就有可能重新掌控昊天界！”
<br/>　　“到时候，你不但可以再次打开昊天界和修真界的通道，还将会继承整个昊天界，成为昊天界的新主宰，甚至将来进入混沌世界，为我的主人报仇！”
<br/>　　叶尘听到这话，顿时满脸惊骇，
<br/>　　“前辈的意思，昊天大帝死了？他不是已经成就混沌仙体，不死不灭了吗？”
<br/>　　老者惨然一笑，
<br/>　　“不死不灭？这世上哪有真正的不死不灭？便是一方宇宙，也终有尽头！”
<br/>　　“更何况，我曾听主人提起过，在那混沌世界之中，有很多比他老人家还要强大许多的存在，还有一些混沌异兽，专门猎杀渡劫成功的修真者，我家主人已经失踪了十万年，极有可能已经……”
<br/>　　老者说到这里，已经老泪纵横，陷入十分难过。
<br/>　　叶尘心中唏嘘不已，也不便再继续多问，于是道：
<br/>　　“那要如何才能继承昊天大帝的传承？如何成为昊天界的主宰？还请前辈教我！”
<br/>　　叶尘曾和家人定下十年之约，是无论如何也要返回地球的，这个机会他自然不能放过。
<br/>　　老者抹了抹眼角的泪痕，眼眸中露出一抹赞许之色，道：
<br/>　　“好！本帝就喜欢你这种爽快之人！”
<br/>　　“不过，想要继承我家主人的传承，可不是一件容易的事情！”
<br/>　　老者说到这里，微微一顿，
<br/>　　“小子，你听说过迷罗星海吗？”
<br/>　　叶尘眉头一挑，当即点了点头，道：
<br/>　　“知道一些！迷罗星海，是修真界十大绝地之一，据说里面杀阵重重，凶险无比，就算是渡劫仙帝进入其中，也有陨落的危险！”
<br/>　　老者听到叶尘这话，顿时老眼微微一眯，
<br/>　　“你知道还挺多嘛！一点也不像是刚刚进入修真界的菜鸟！”
<br/>　　叶尘心里一突，也顿时意识到，自己表现的有些太过了。
<br/>　　不过好在，老者并没有追究这个问题，继续开口道：
<br/>　　“我来告诉你吧！其实这迷罗星海，是我家主人留在修真界的一个后手，而他的传承，便存放在第九重星海海域之中！”
<br/>　　“可惜的是，这百万年以来，却从来没有人，能够进入第九重星海海域。”
<br/>　　“所以现在，你还敢吗？”
<br/>(本章完)
