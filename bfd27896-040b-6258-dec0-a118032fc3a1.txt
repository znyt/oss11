　　待得幽光彻底在无形漩涡中被绞灭。
<br/>　　萧逸才松了口气。
<br/>　　虽说他经历过许多生死一线。
<br/>　　但像刚才那般身不由己的迎接死亡的感觉，还是让他后怕不已。
<br/>　　那种无力，才是最让人内心恐惧的存在。
<br/>　　“呼。”萧逸深呼吸一口气。
<br/>　　危机已除，暂时不必多想。
<br/>　　据萧逸猜测，那股幽光，绝不单单是灵识那么简单。
<br/>　　否则，不可能使用‘魂技’。
<br/>　　很显然，刚才使他心神迷失，意识模糊的，便是诡异莫测的‘魂技’。
<br/>　　但，使用魂技的前提，必须是武魂。
<br/>　　只有武魂，才能释放魂技。
<br/>　　萧逸也不太清楚那抹幽光，也就是冰尊者的灵识，具体是怎么回事。
<br/>　　但，这抹灵识，必然与‘武魂’脱不了干系。
<br/>　　否则，冰尊者不可能释放魂技。
<br/>　　而太阴太阳之眼，恰恰是‘武魂’的克星。
<br/>　　萧逸估计着，这也是太阴太阳之眼，能绞杀幽光的原因。
<br/>　　同时，也是因此，幽光被绞杀后，悉数被无形漩涡所吸收。
<br/>　　萧逸从‘幽光’之中，得到了不少信息和记忆。
<br/>　　只不过，这些信息和记忆，并不完整，不，或者说，是残缺得相当厉害。
<br/>　　冰尊者的灵识，存在了千万载以上。
<br/>　　在这漫长岁月中，必是被消磨了不少，记忆之类的，也缺失了许多。
<br/>　　除了武神，任何东西，任何存在，都无法抵挡时间力量的磨蚀。
<br/>　　哪怕灵识也不行。
<br/>　　言归正传。
<br/>　　关于冰尊者为何身死，或者来历等等，这些时间太久远之事，并未存在于信息中。
<br/>　　倒是近数百年之事，很是清楚。
<br/>　　萧逸大概明白了这极寒裂缝，以及这洞内的一些事。
<br/>　　另外还得到了一些修炼功法之类的。
<br/>　　萧逸沉思了一会儿，随后摇摇头。
<br/>　　这些事，他暂时不打算管。
<br/>　　如今，最重要的是提升修为，尽快脱离此地。
<br/>　　冰心圣果，是现今救易老的唯一办法。
<br/>　　萧逸绝不会让冰承天带回冰武王国。
<br/>　　想罢，萧逸身影一闪，往那汪水泉而去。
<br/>　　水泉之内，是无数的寒冰之髓。
<br/>　　寒冰之髓，只有雪脉之地能产生。
<br/>　　雪脉之地，亦分品阶。
<br/>　　小型雪脉之地，一年可产一斤寒冰之髓。
<br/>　　中型雪脉之地，一年可产十斤。
<br/>　　大型雪脉之地，一年可产百斤。
<br/>　　不夜城之下，便有一条小型雪脉之地，一年可积蓄产出一斤寒冰之髓。
<br/>　　而观这汪水泉内寒冰之髓的凝聚速度，以及气息浓度。
<br/>　　水泉之下，必有一座大型雪脉之地。
<br/>　　每年可产百斤寒冰之髓。
<br/>　　萧逸认真感知了一番水泉的深度，顿时脸色一喜。
<br/>　　水泉的深度，远远超过了自己的想象。
<br/>　　原本还以为，这水泉内的寒冰之髓，只有万斤左右。
<br/>　　但现在看来，起码在数万斤以上。
<br/>　　这汪水泉内的寒冰之髓，积蓄了绝对千万年以上。
<br/>　　本应数量更多。
<br/>　　但很显然，里头的力量，有许多都通过冰层，牵引到最上方了。
<br/>　　也是因为这些力量，上方那片风雪之地，才会生长出几乎漫山遍野的珍稀天材地宝。
<br/>　　冰尊者的灵识，只是一无形之物，不可能吸收这些寒冰之髓。
<br/>　　倒是被他牵引到上方，弄出了那片风雪之地，无数年来，吸引了不少‘倒霉蛋’前来。
<br/>　　成了他的猎物。
<br/>　　此时，萧逸盘膝坐于水泉旁，彻底开始修炼。
<br/>　　他困在地元九重修为已经许久了。
<br/>　　身上已积蓄了大量修炼之物。
<br/>　　他一直等。
<br/>　　现今，便是机会。
<br/>　　体内，小世界中，有一股寒冰灵气。
<br/>　　这是在不夜城客栈中，数月修炼所得。
<br/>　　并未转化为元力提升修为，仅仅是储存着。
<br/>　　此时，萧逸一个念头间，这股寒冰灵气，悉数化作澎湃元力。
<br/>　　元力一出现，立马涌向气泉。
<br/>　　气泉内，本就储存满了元力，新的元力的加入，立刻让得原本的元力出现压缩。
<br/>　　不过数秒，元力出现液态。
<br/>　　普通元力，是地元境的标志。
<br/>　　而液态元力，乃是天元境的标志。
<br/>　　十成气泉，瞬间一空。
<br/>　　上方‘冰山火海’，瞬间冰消瓦解，渐渐不复存在。
<br/>　　“天元境，成了。”萧逸眼中泛过一道精光。
<br/>　　他现在，已是天元一重修为。
<br/>　　不过，战力，不增反降。
<br/>　　冰山火海，已然消失。
<br/>　　他的力量来源，只剩下气泉中高度为一成的液态元力。
<br/>　　萧逸没有浪费时间。
<br/>　　各种修炼之物，快速吸收着。
<br/>　　乾坤戒内的三头吞灵冰鲸王，20头吞灵冰鲸，无数吞灵冰鱼。
<br/>　　待得这些修炼之物，被吸收殆尽。
<br/>　　萧逸的气泉高度，却几乎没有变化。
<br/>　　上升了不足一成；不，根本还不足半成，甚至更少。
<br/>　　三千三百三十三丈气泉，果真恐怖。
<br/>　　这时，一股澎湃力量，忽然涌入气泉。
<br/>　　这是属于冰山火海的力量。
<br/>　　冰山火海，虽然消失了，但蕴含的力量可不会凭空没掉。
<br/>　　虽九成九都被冰鸾剑吸收了，只有一些保留了下来。
<br/>　　但这一些，已足以让萧逸的气泉提升一成有余。
<br/>　　也就是说，现今气泉高度，仅在二成多些。
<br/>　　萧逸微微皱眉，将目光看向小世界，气泉一旁那头已经缩水了七八成的吞灵冰鲸皇之上。
<br/>　　吞灵冰鲸皇，蕴含力量之精纯度，远超吞灵冰鲸王。
<br/>　　若是完整的一头，蕴含之力量，怕是比千头吞灵冰鲸王加起来还要多。
<br/>　　当然了，现在只剩二三成左右。
<br/>　　但，这仍旧是一股澎湃得惊人的力量。
<br/>　　小世界内，冰鸾剑、控火兽，两大武魂齐开，不过半晌，便已经将这股力量吸收殆尽。
<br/>　　如今，冰山火海已消失。
<br/>　　冰鸾剑，不会再吸收这些力量。
<br/>　　待得吞灵冰鲸皇被吸收殆尽。
<br/>　　萧逸的气泉高度，升了足足三成，达到五成。
<br/>　　萧逸眉头一皱。
<br/>　　将乾坤戒中的五百斤寒冰之髓，悉数吸收。
<br/>　　气泉高度，升了一成，达到六成。
<br/>　　也就是说，他现在的修为，乃是天元六重。
<br/>　　再提升一重，便真正踏入天元境后期了。
<br/>　　可是，这剩下的一重…
<br/>　　萧逸的眉头，头一次紧紧地皱了起来。
<br/>　　天元七重，与天元六重，虽只差一重，但实力，却天差地别。
<br/>　　而这，来自于天元境后期的一项极其特殊的要求。
<br/>　　萧逸的目光，终于看向那汪水泉。
<br/>　　体内两大武魂，瞬间触发。
<br/>　　水泉内的寒冰之髓，以肉眼可见的速度，快速被吸收着。
<br/>　　萧逸的气泉高度，亦在不断上升。
<br/>　　在足足吸收了近五千斤寒冰之髓后。
<br/>　　气泉高度，终于达至10成，已满。
<br/>　　而现在，萧逸的修为，才算是天元七重。
<br/>　　不错，天元境后期的要求，便是气泉填满。
<br/>　　而并非单单七成高度。
<br/>　　“天元七重，成了。”萧逸自语一笑。
