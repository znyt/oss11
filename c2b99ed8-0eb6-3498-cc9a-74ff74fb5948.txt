　　但情况已经容不得她多想，木棺此时的震动愈加激烈，光靠不完全的灵气墨斗，已经压制不住木棺里面的那头僵尸了。
<br/>　　虽然是后天强行催生的伪百年僵尸，但也比一般的走僵强横了许多。
<br/>　　具体来算的话，也非常接近于百年毛僵了，以刘哔现在的实力，应对起来也并不是那么容易。
<br/>　　“咔嚓！”
<br/>　　就在刘哔手持桃木剑警戒时，木棺突然发出一声脆响，表面上浮现密密麻麻的诸多裂缝，一眨眼的时间，整个木质棺材炸裂，一团黑雾涌了出来。
<br/>　　眼中红光一闪，刘哔抬起手中木剑，闪电般冲进了前方的那团黑雾，瞬间，一阵金戈交鸣的声音从黑雾中传来。
<br/>　　过去了大概几分钟的时间，场上的黑雾方才消散，露出了他和那头僵尸的身影，紫女目光落在僵尸身上，不由皱起了眉头。
<br/>　　和先前义庄那些四目道长赶过来的僵尸不同，这头僵尸通体都长着一些黑色的毛发，模样倒是没多大变化，只是腐烂程度要严重了许多。
<br/>　　可是，之前刘哔他们查看的时候，这具尸体还没有任何腐烂掉的迹象。
<br/>　　“是黑僵，小心一些。”身处封印，紫女无法出手，只能开口，给刘哔带来一些提醒：“而且你看他身上的毛发，都有了一些发绿的征兆了，估计距离升级也不远了，你千万不要大意！”
<br/>　　正在和僵尸交手的刘哔，听见紫女的话语之后目光一凝，仔细分辨了一些，确实发现这僵尸身上的毛发确实有些不太正常。
<br/>　　之前身处黑雾之中，他看的也不是非常清楚，开始的时候，他以为这只是普通的走僵而已。
<br/>　　最近几天，他一直在查询有关僵尸的一些知识，房间里有九叔留下的一些书籍，他倒是也了解了一些东西。
<br/>　　而这个世界中，僵尸也是有分级的，最常见的，就是刚刚觉醒的白僵和修炼时间尚浅的黑僵。
<br/>　　这类僵尸，尸体的颜色呈白色或黑色，毛色也是同样颜色。
<br/>　　因为修炼时间尚欠的缘故，它们的行动非常迟缓，比较容易对付，害怕的东西有很多，人们通常将这两类的僵尸统称为走僵。
<br/>　　当然，细分下来还是有区别的，有的地方也会称呼白僵为跳僵，只不过它们的危险性，都不是很大，还是比较容易对付的。
<br/>　　然而修炼了百年之后，它们的毛发就会逐渐变成深绿色，实力也会得到一个比较大的提升，这种级别的僵尸，就很难应对了。
<br/>　　因为它们的速度极快，且身体素质强横，一般武器都不会对他们造成什么损伤，只有精通法术的道长，方能有机会制服它们，这类僵尸，也被人成为毛僵。
<br/>　　再修炼千年，毛僵的毛发就会全部脱落，表面看上去也和人类没很大的区别，此类僵尸擅长法术，身体不坏，且有飞天之能，被人称为飞僵。
<br/>　　据说在飞僵之后，还会有诸多变化和等级，比如飞僵，不化骨，伏尸，游尸，旱魃，犼等等，不过这类僵尸，谁都没有见过，只存在于传说当中罢了。
<br/>　　现在刘哔所要面对的，就是一个即将变成毛僵的僵尸，此时应对起来，也异常吃力。
<br/>　　所幸是他熟知剧情，提前再此截住了它，否则等它去到任家，吸食了至亲之人的鲜血之后，就会真正的发生蜕变，那时想要再将其击杀就十分困难了。
<br/>　　但即使是这样，他依旧感受到一股庞大的压力，这头走僵的肉体实在强横，要不是靠手上的这炳桃木剑，估计他根本对它造成不了什么伤害。
<br/>　　“简直和铜墙铁壁一样。”嘀咕了一句，刘哔将体内的天地灵气灌注在剑柄上，随后向僵尸头部挥砍而去。
<br/>　　二十多点的力量增幅，让木剑在空中舞起了虎虎风声，他并不像将战斗拖延很长时间，故此没有任何留守。
<br/>　　木剑上带着猩红色的流光，劈砍在僵尸腐烂的头颅上，溅起了一连串的火化，就像是劈在了一块生铁疙瘩上面一般，但却没起到太大的效果。
<br/>　　“砰！”
<br/>　　一声巨响，僵尸像是被刘哔的攻击惹烦了一般，一脚将旁边的棺材踢向刘哔，电光火石之间，来不及闪躲，刘哔只能用手臂护在了胸前。
<br/>　　一股巨大的冲击力从手臂上涌来，他从半空中倒飞了出去，将一只木椅砸的粉碎，胳膊软绵绵的耷拉在地上，骨头像是错位了。
<br/>　　没敢犹豫，刘哔立即从地上爬了起来，将胳膊上错位的骨头强行拧了回来，随后，疾退几步，躲开了刚刚扑过来的僵尸。
<br/>　　“攻他眼睛！”身后，九叔的声音传来，刘哔苦笑一声，终归没能在九叔到来之前击杀僵尸，不知道最后还能不能抢到积分。
<br/>　　但这也是没办法的事情，一开始他也没有想到，木棺中的僵尸，居然有这样强横的实力。
<br/>　　“愣着干什么，我们快去帮忙啊！”随后，文才也从门口迈了进来，看了一眼站在原地的刘哔，伸手拉了一把，手中还握着一柄明晃晃的菜刀。
<br/>　　看着文才手中的武器，刘哔嘴角抽动了一下，也没多说什么，提起木剑加入了九叔和僵尸的战团。
<br/>　　本来他和僵尸的实力就在仲伯之间，此刻有了九叔的帮忙，自然是大感轻松，而僵尸也是节节败退，很快就被逼退到了墙角之处。
<br/>　　“吼啊！”
<br/>　　怒吼一声，僵尸愤怒的看着九叔和刘哔，面对两人的夹攻，它根本没有还手的余地，此刻更是想逃走都成为了一件困难无比的事情。
<br/>　　原著中，它成功进化成了毛僵，才有了和九叔一战之力，更何况旁边还有一个实力不俗的刘哔，此刻的它，根本就不是对手。
<br/>　　即使是这样，僵尸依靠着自身强大的防御力，强撑了一个多小时的时间，才被刘哔一剑捅穿脑袋，倒在了地上。
<br/>　　“叮~恭喜宿主击杀高级黑僵一名，获得积分三百，积分余额四百。”
<br/>　　听到系统的声音，刘哔才放下心来，轻呼了一口气，松开了手上的木剑，任由它插在僵尸的眼眶中。
