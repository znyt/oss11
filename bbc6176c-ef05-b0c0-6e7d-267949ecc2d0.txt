　　随后，大祭司身边的一名核心长老，一挥手臂，一座看台便出现在了广场上。
<br/>　　这座看台高约五丈，长约百丈左右，其上整齐的摆放着数十个金光闪闪的黄金座椅。
<br/>　　而后，大祭司、四祭司以及一众核心长老，身形缓缓下落，落在了看台上，纷纷就座了下来。
<br/>　　哗啦啦！！
<br/>　　众人纷纷后退，立刻给看台区域让出了很大的一片区域。
<br/>　　大祭司端坐在中央主座之上，沧桑而又深邃的目光，环视广场上数以亿计的巫族族人。
<br/>　　少倾之后，大祭司朗声道：“这一次族中的少司选拔，今日正式开始，共分为三关，能闯过三关的人，便最终成为少司；若是有多人同时闯过了三关，那就以战力来决定最终的结果。”
<br/>　　整个广场上都寂静无声，众人听着大祭司的话语，心中的冲劲都升了起来。
<br/>　　三关！
<br/>　　该有多难呢？
<br/>　　因为每一次的少司选拔，考核项目都不同，众人也不知道这一次会是哪三关！
<br/>　　“好了，老夫也不多言，现在出第一关！”
<br/>　　大祭司淡淡的说了一句，随即袖袍向前一挥，霎时之间，一座小塔飞射而出，飞到了广场中央的上空。
<br/>　　这小塔呈灰白之色，只有三层，但比较怪异的是，塔顶并无塔尖，而是呈现纯平之态，这让整个小塔都仿佛像是一根圆柱。
<br/>　　下一刻，在无数人的注视之下，小塔迎风见长，只是呼吸之间，便暴涨了千倍大小，变成了一座直径足有数百丈，高达近千丈的巨塔。
<br/>　　轰！
<br/>　　巨塔落在了圣巫广场的中央，让得整个广场都猛烈一震，发出一声剧烈的爆响声。
<br/>　　“这塔共有三层，每一层内都有巫凌的投影，所有进入塔中之人都会遇到同阶的巫凌，第一层的投影拥有巫凌三成的实力，第二层五成的实力，第三层七成的实力。”
<br/>　　“打败巫凌的投影，成功闯过三层，达到塔顶，便算通过了第一关！
<br/>　　大祭司声音淡淡，却响彻在所有人的心底。
<br/>　　众人闻言，俱是心中一惊，居然是要打败巫凌的投影？
<br/>　　巫凌，是五年前上一届少司选拔的胜出者，现在族中尊贵的少司，其天赋战力极度逆天。
<br/>　　想要通过三层，到达塔顶谈何容易啊！
<br/>　　即便第三层最强的投影，只有巫凌七成的实力，也依旧恐怖异常了！
<br/>　　“好了，闯关者进塔吧！”大祭司最后说了一句，然后三层宝塔的大门边轰然打开。
<br/>　　不过，众人都没有急着进入，不少人目光转动，都好似在寻找某人。
<br/>　　此刻，众人心中不免有些疑惑，这次少司候选榜第一名的巫人王，为何还没来？难道不准备参加少司选拔了？
<br/>　　咻！
<br/>　　就在此刻，一声刺耳的破空声响起，众人循声望去，便见到天际的尽头，一道黑色的闪电划破长空，风驰电擎而来。
<br/>　　这黑色的闪电不仅气息恐怖，更是裹挟着无比狂暴的气息，让人心中发寒。
<br/>　　众人见此先是心中一震，随即立刻反应了过来，巫人王终于回来了！
<br/>　　唰！
<br/>　　瞬息之间，黑色的闪电来到了广场之上，瞬间便停了下来，化为了一名身穿黑色劲装的青年。
<br/>　　这名青年长得眉清目秀，唇红齿白，乍一看仿佛只有十六、七岁，年轻的有些过分。
<br/>　　此人的头发极长，垂落到了大.腿之下，整个人给人一种阴柔之感，与其身上狂暴的气息截然不同，形成了强烈的反差。
<br/>　　“呵呵！总算没有错过！”
<br/>　　青年止住身形之后，立刻微微一笑，随即向看台上抱拳行礼，道：“见过大祭司、四祭司和各位核心长老！”
<br/>　　“嗯！”
<br/>　　大祭司微微颔首，原本淡漠的脸庞上露出了一丝笑容，问道：“巫人王，听说你去了三化星历练？有什么战绩？”
<br/>　　众人闻听大祭司的问话，也均是来了兴趣，因为巫人王看似一个阴柔的少年，实则是一个战斗狂人。
<br/>　　其每一次出去历练，说白了就是到处挑战，挑战各地的超级天才，铸就同阶无敌的威名。
<br/>　　巫人王闻言自信一笑，傲然道：“回大祭司，三化星虽大，但无论是巨灵族、飞灵族、黑鳞族亦或是普通的人族，同阶之中无人能击败我，均是徒有虚名之辈，不值一提！”
<br/>　　三化星在古灵星河是一个特殊的星辰，很多种族都在此星有极强的势力，但没有谁能单独主导此星，是以此言是一个多种族混乱之地。
<br/>　　“是吗？没有遇到厉害的天才？”大祭司显然对于巫人王很是看重，淡笑着询问道。
<br/>　　“的确是遇到了一个厉害的人物，此人不是三化星之人，乃是神武学府的弟子，名为尤天韩，实力不错，和我对战了千招，也没有分出胜负！”巫人王叹了口气说道。
<br/>　　他这次去三化星历练，几乎横扫了整个三化星同境界的天才，但却是遇到了神武学府的妖孽，最终依旧没有击败对方。
<br/>　　苏莫闻听巫人王之言，心中很是惊讶，对方说得居然是神武学府中，太史世家一脉的尤天韩。
<br/>　　尤天韩好像是武帝境九重境界，那看来这个巫人王，也是武帝境九重境界了。
<br/>　　只是，尤天韩有这么强大吗？居然能和天冥星巫族支脉的妖孽天才巫人王同阶一战而不败！
<br/>　　苏莫心中很是诧异，他以前只知晓尤天韩很厉害，是南苑一等一的高手，但没想到对方居然厉害到了如此的程度。
<br/>　　看来他回到神武学府之后，想杀对方的话，还有很长一段路要走。
<br/>　　“神武学府，收容古灵星河大量的天才，能有如此人物，不足以意外！”大祭司道。
<br/>　　“大祭司，我巫人王一向同阶无敌，这个尤天韩，待我成为少司之后，会直接前往神武学府挑战他！”巫人王沉声说道，可见他对于这次的少司选拔，是志在必得了！
<br/>　　“嗯！好了，先参加少司选拔吧！”大祭司点头说道，没有再多谈，随即向对方简单说了下规则。
<br/>　　“呵呵！”
<br/>　　听闻大祭司之言后，巫人王轻笑一声，随即身形一掠，稳稳的落在了三层宝塔的塔顶上。
<br/>　　苏莫见此眼眸顿时一凝，心中惊疑不已，这个家伙居然不闯塔！而是直接飞到了塔顶上！
<br/>　　此人这是要干嘛？
