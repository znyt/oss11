　　边上一个人说道：“废话，不想活命怎么会从战场上逃出来！我特么当兵是为了混口饭吃，哪想到真的打仗了，再不走真的要死在战场上了。”
<br/>　　身边几个士兵跟着点头，他们也都是一样的想法。
<br/>　　小三说道：“那就是了，我们现在是逃兵，回去也是个死，既然我们手上有枪，为什么不去搏一把吧？当土匪有什么不好，现在时局这么乱，官兵不会来缴费，吃香的喝辣的，比我们当大头兵熬日子要强的多吧？兄弟们你们说呢？”
<br/>　　吴连长说道：“去当土匪，也是给人卖命，跟给督军卖命有什么差别？”
<br/>　　小三道：“我们当然不是去当卒子，我们现在有十几个人，一人手里一条枪，只要我们一条心，上山的话，黑云寨至少给吴连长你一个二当家三当家的坐坐……”
<br/>　　吴连长眼神一动，说道：“那你们还不是要卖命？”
<br/>　　“打仗随时会死，当土匪，也就是拦路劫道，抢些路人……”小三把眼睛瞟向叶少阳这边，“就像这几个过路的，你抢他们，他们还能反抗不成？”
<br/>　　老万他们一听，吓破了胆。老万急忙拱手，哀求道：“老总们饶命，我们都是些苦命人，做点小买卖……”
<br/>　　当兵的没人理他们。小三望着吴连长，说道：“吴连长，你觉得兄弟这路子怎么样？”
<br/>　　吴连长把玩着手里的佩刀，望着面前火堆的余烬，说道：“我不想再寄人篱下，我若当土匪，必然做老大，兄弟们一起吃肉喝酒，不受别人掌控。”
<br/>　　小三一巴掌拍打在自己腿上：“这好办啊，黑云寨是这边几个寨子里最小的，也就几十个人，手里最多就是盒子枪，哪里比的过我们的长杆子，我们假意投奔，先稳住，找机会办了他们大当家的，再杀几个人，不怕下面人不服……”
<br/>　　老万等人面面相觑，一个个吓的腿软，说不出话来。这些个当兵的，居然想去当土匪，还要跟土匪火并……
<br/>　　吴连长犹豫再三之后，接受了小三的提议，于是大家讨论具体的计划，这时候一个人跳起来说道：“我不当土匪！我绝对不当土匪！”
<br/>　　吴连长阴沉着脸看着他，说道：“石头，人各有志，我也不勉强你。你走吧。”
<br/>　　小三道：“把枪留下！”
<br/>　　石头没有任何犹豫，把自己的枪扔在了地上，转身要走，吴连长说道：“等一下。”
<br/>　　对其余部下说道：“你们还有谁不想当土匪的，要走现在就走，别当时候等上了山再反悔，再当一次逃兵。”
<br/>　　大伙互相看看，又有两个人站起来。
<br/>　　吴连长问了几声，没人再站起来，于是让他们把枪和子弹都交出来，让他们走。
<br/>　　这三人对着吴连长等人抱拳行礼，也没什么好说的，转身就走，走到山神庙门口的时候，小三突然站起来，说道：“石头，虎子，陈柱！”
<br/>　　三人本能地回过头去。
<br/>　　小三猛地抬起步枪，拉动枪栓，啪啪啪一连几枪打在他们身上。
<br/>　　“啊——”
<br/>　　翠云还是第一次看杀人，吓的当场尖叫起来，叶少阳蹲在地上，把她抱在怀里，抓着她的手安慰着。
<br/>　　老万等人也是吓的瘫软在地，浑身颤抖。
<br/>　　二蛋当场就吓尿了。
<br/>　　吴连长和手下那些士兵也都吓傻了，看着三具尸体倒下去，半晌才回过神来，一个个转过头，震惊地望着小三。
<br/>　　小三放下枪，对吴连长和大伙说道：“兄弟们别怪我，不是我狠心，我也不想对他们动手，你们想想，当土匪最怕被人认出来，他们三个回去，说不定会去告密，就算不说，万一他们被督军抓到，肯定也会供出我们的事情，到时候我们就成了通缉犯。”
<br/>　　吴连长说道：“那有什么关系，反正你说的，官兵不会来剿匪。”
<br/>　　小三笑道：“当土匪那是不怕，不过我们难道还真当一辈子土匪了，总要进城的吧，我可不想一进城就被人唔出是通缉犯，把我绑了见官。再说，将来万一我们势力坐大，被官兵注意到，被人认出脸，总比不认识脸的好。”
<br/>　　吴连长等人都不作声，显然也都是默了他的说法。而且还有一层，他们也都想到了，他们中大部分家都在江西，当逃兵是没什么，督军也没空去他们老家抓人，但是当土匪那就不一样了，很容易就连累到家人，再者，他们也不想被家人知道自己当了土匪。
<br/>　　吴连长对小三说道：“我记得之前劝我们逃出部队的，也是你，你果然是比我考虑的周到。”
<br/>　　小三笑道：“连长，现在我们可以去做正事了。”
<br/>　　吴连长望着他，说道：“等一下，我觉得……你考虑事情比我周到，不如由你来做大哥，我跟着你混，如何？”
<br/>　　小三立刻摇头，抱拳说道：“连长千万别这么说，我这个人，就适合做个师爷，出出主意，大局上我可不行，连长你性格比我沉稳，弟兄们也都听你的，这个老大绝对你来当，我安心给你当个师爷、副官就行，兄弟们一起吃香的喝辣的，大哥如果信不过我，我们今天就喝血酒拜把子！”
<br/>　　吴连长一听，也是立刻同意，但是找了半天没酒也没碗，于是找叶少阳等人索要。
<br/>　　老万身上带着酒壶，立刻颤抖着递了过去。吴连长和小三用刀子割破手指，滴了血进去，然后小三提议大家一起拜把子，于是大伙挨个割手滴血。
<br/>　　叶少阳在一旁看着这奇诡的一幕，目光不时扫向那个小三，此人印堂开阔，五官端正，耳垂肥厚，从面相上看是个能做大事的人，不过两条眉毛中间有杂乱的毛发，将两条眉毛连在一起，相术上说，这种人内心邪恶，阴险狡诈，果不其然。
<br/>　　其实就算不看面相，叶少阳也看出了这个小三内心的险恶：之前那个吴连长提议让他做大哥的时候，他没有做，是对的。一来吴连长当惯了老大，支配小三，突然要给他当手下，肯定不习惯，虽然是他自己提出的，将来也肯定会积怨在心，指不定会在背后捅他刀子。
