　　马球其实是一种极为现代的说法，在大唐，多数人都会将其称为‘击鞠’。
<br/>　　这是一项十分古老的竞技运动，在我国魏曹时期便有记载，而到了唐代更是盛行一时，是贵族间经常用到的一种交流方式。
<br/>　　李承乾甚至记得一些历史文献中还记载了很多关于大唐皇族如何喜爱马球的实例。
<br/>　　只不过作为正式比赛，这还是李承乾第一次提出来的，为此他还安排长乐制定了详细的比赛规则，计分制度以及犯规后的判罚标准。
<br/>　　要知道，在这次比赛之前，马球比赛并没有什么太过细致的规则，左右不过就是把球打进对方的球门就好，球场之上一群马球流氓，追着一个拳头大的木质小球跑来跑去，时不时还会把球杖丢出去砸人。
<br/>　　就是因为这样，大唐的贵族们往往分成两个圈子，文官圈子绝不与武将圈子的人打马球，反之亦然。
<br/>　　不过这一次制定好了标准之后，一切都好了许多，再有人丢球杖砸人，那就直接罚下场，而且还要禁赛数场。
<br/>　　而且这一次比赛标准的制定不单单有犯规的判罚，比赛的时间、场地范围、比赛的人数，球杖的材质长度，大小等等等等，都有了详细的规定。
<br/>　　……
<br/>　　书归正传，却说这一日早朝，还没等到殿内的内侍扯着嗓子喊出那句‘有事早奏，无事退朝’，大殿上就炸了庙了。
<br/>　　只见得几个世家的官员齐齐站出来，指着程咬金的鼻子直接开喷：“程知节，你，你什么意思，你怎么可以让你的女儿下场，要知道她虽是你的女儿，但却也是我大唐德妃，你，你这是有辱国体！”
<br/>　　“扯特么蛋，琳丫头是老子的闺女，为啥不能下场比寨，你们这个老不死的就是嫉妒，自己打不赢就开始耍无赖，你们以为自己是谁，还有辱国体，亏你们想得出来。”程老魔头自然不会对几个文官有什么好脸色，这老人渣向来都是见到吵架没够，有人找上门来他又怎么可能退缩。
<br/>　　“胡说，老夫等人一心为国，岂会像你一般无耻，德妃若真是上场，徒惹人笑是一回事儿，若真是受了些伤又该如何！”
<br/>　　“真是笑话，就凭你们这些老货还能让俺闺女受伤？也不撒泡尿照照自己什么得德性，老子的闺女让你们一手一脚，能靠到她身边的，老子给你们磕头。”程咬金这家伙看来是打算将不要脸进行到底了，不管别人怎么说，就是不松口，执着的认为程琳应该代表程家出战。
<br/>　　最后还是孔颖达这老头儿精明，哆哆嗦嗦站出来说道：“卢国公，德妃是你的女儿不假，但她现在已经是皇家之人，就算是要出战，也是代表皇家，而不是程家吧！”
<br/>　　“呃……”这下，程老匹夫被将住了，张口结舌半晌无语，抬头看看李承乾，发现某人正用手支着脑袋在打盹，一副事不关已高高挂起的样子。
<br/>　　“是啊，卢国公此举的确是有欠考虑！”
<br/>　　“就是，闺女都已经嫁人了，已经是夫家的人了，怎么还可以替娘家出战！”
<br/>　　一群老货见程妖精不说话了，纷纷揭竿而起，落井下石。
<br/>　　李承乾等了半天不见程老货反击，知道热闹是看不下去了，直起身子敲了敲桌面说道：“好了，都消停点吧，下面说正事！”
<br/>　　“是！”几个占了便宜的老家伙带着得意的笑容退到了一边，而老程则是不怀好意的瞪着那几个老家伙，也不知道是不是打算半夜去砸玻璃。
<br/>　　“这样啊，马球的事情呢，咱们先不说，这个是小事儿。”李承乾等到老家伙们都归位了，这才继续说道：“今天咱们要议一议的是自行车，有谁能够告诉朕，现在大唐每年的自行车产量是多少？”
<br/>　　“大概有万余辆，工部那边每年产量是五千辆，其他外面民间产量低一些，但一年各地加起来大概也有六到七千。”户部侍郎大致上算了一下，然后回禀道。
<br/>　　李承乾略一沉默，算是在算什么东西，随后说道：“那这段时间工部那边的自行车就不要卖了，都给朕留着！”
<br/>　　“留着？”户部侍郎愣了一下，那东西留着能干啥？
<br/>　　“对，留着，那些异族就要来了，朕总得给点礼物给他们。”李承乾点头说道。
<br/>　　拿自行车送人？不是应该送丝绸和瓷器么？大殿上静的可怕，谁都不知道这位皇帝陛下到底葫芦里卖的什么药。
<br/>　　最后还是刑部侍郎李敬玄站出来：“陛下，这自行车是不是格局有点低了？”
<br/>　　“低就低吧，你们都清高，都不想沾上一身铜臭，那就只有朕亲自出马喽！”李承乾不无讥讽的说道。
<br/>　　下面众人不知道李承乾又在发什么疯，不过还是习惯性的一齐躬身：“臣等有罪！”
<br/>　　“有罪？你们也知道有罪？”李承乾拍着桌子，站起身来踱了几步，而后沉声说道：“你看看你们一个个，天天讨论什么球赛、球赛，还想点正经事情不？朕把那些异族找来参加马球比赛，你们以为朕真的在乎输赢！”
<br/>　　“啊？！”下面一群老头儿，半大老头儿齐齐一晕，不在乎输赢？那还找那些异族来干什么？如果打输了，那得多丢人啊！这可是大大的丢脸啊！
<br/>　　李承乾看着懵逼的一群手下，无奈的叹了口气：“你们什么时候能让朕省点心，你们就不能动动你们的脑子，想想我们大唐都有什么是那些异族没有的，还拿丝绸送人，送个屁，那东西朕还留着卖钱呢。”
<br/>　　“朕给你们说，今天回去之后，马上给朕制定一份价目表出来，丝绸、瓷器、自行车、推土机、拖拉机……，所有东西，就算是一颗钉子，也要把外销的价格给老子翻上三倍，明白没有？”
<br/>　　“陛下，您，您这是……”房玄龄隐约间大概想通了李承乾想要干什么。
<br/>　　“钱！朕只要钱！名声什么的都不重要，球赛的输赢朕不在乎，不过如果这次那些来的异族不留下五万万贯的订单，全体官员，从上到下，集体罚俸一年！如果超过五万万贯，俸禄翻倍！”
<br/>(本章完)
