　　轮回神朝境内，沁儿和姬月松、褚彦、骆妤一起，向天池神朝飞去。
<br/>　　他们的目的地，是去天池神朝一个叫无双界域的地方，这是很早之前便商议好的。
<br/>　　高空之中，几人一边飞行，一边闲聊。
<br/>　　“沁儿，你怎么了？”
<br/>　　褚彦比较关心沁儿，见对方面色仿似略带忧愁，疑惑的问道。
<br/>　　骆妤和姬月松闻言，也转头望着沁儿，对方今日的确是少言寡语。
<br/>　　“我们别去无双界域了，还是去时空界域吧！”沁儿环视三人，沉声说道。
<br/>　　“去时空界域！”三人闻言一怔，他们早就商量好了去无双界域，沁儿怎么又想去时空界域了呢！
<br/>　　“沁儿，不是说好了去无双界域吗？”姬月松诧异的问道。
<br/>　　“无双界域没什么特色，还是时空界域比较好，时空之城能更快的提升修为！”沁儿沉声道。
<br/>　　“沁儿，你不是从不去以前去过的地方吗？“褚彦疑惑的问道。
<br/>　　他对沁儿太熟悉了，对方出来历练，其实就是出来游玩，增长见识的，从不去以往去过的地方。
<br/>　　“哪有，我觉得时空之城很好，再去一次也无妨！”沁儿撇了撇嘴说道。
<br/>　　“沁儿，你不会是去找段惊天的吧？”骆妤面带沉吟之色，试探性的问道。
<br/>　　“怎么可能……你想多了……我去找段惊天干什么？”沁儿面色微变，吞吞吐吐的否认了。
<br/>　　但是，其神态实在不适合撒谎，根本隐瞒不足骆妤三人。
<br/>　　一瞬间，骆妤和褚彦、姬月松三人，脸色都沉了下来。
<br/>　　“沁儿，那段惊天不会回来了，玄天长老不是说了吗？”褚彦面色难看的说道，这都几年了，对方居然还在想着段惊天！
<br/>　　“父亲也不能肯定他不会回来！”沁儿反驳道。
<br/>　　“你……！”褚彦脸色阴沉，沁儿居然如此执着。
<br/>　　“沁儿，段惊天天赋固然妖孽，但是，明显和我们不是一路人，你何必还去找他呢？”骆妤叹了口气道。
<br/>　　她承认自己以前看走了眼，那段惊天是万古天骄，甚至比万古天骄都要妖孽，她也曾想过，日后多和段惊天接触接触。
<br/>　　但是如今，段惊天已经走了，如同一股刮过的狂风，没有留下任何的痕迹，注定只是人王宫的一个过客。
<br/>　　“不是一路人！”沁儿闻言，面带一丝落寞之色。
<br/>　　随即，她心中暗暗的叹了口气，的确，骆妤说的或许不错。
<br/>　　但是，她就是想去找段惊天，对方连一个告别都没有给她留下，就这么走了。
<br/>　　“别想了，就是先去无双界域历练一段时间再说吧！”
<br/>　　姬月松朗声道，想到段惊天，他的心绪有些复杂。
<br/>　　“好吧！”沁儿无奈的叹了口气，既然三人都不同意，那就先去无双界域，至于时空界域，过段时间再做决定。
<br/>　　玉瓶空间之中，苏莫能感知到外界的一切，他不由的叹了口气。
<br/>　　沁儿居然如此的想要见他，但是却不知他就在对方的身上。
<br/>　　“不能向你告别，还请见谅！”苏莫心中暗道，不过，他肯定会回来的。
<br/>　　只不过，不知道多久才能回来，可能数十年，也可能数百年甚至更久。
<br/>　　随即，苏莫暗暗计算了一下时间，离开人王城已经五个时辰了，差不多可以准备出去了。
<br/>　　不过，就在此时，外界突生变故。
<br/>　　嗖！
<br/>　　只见长空之上，沁儿四人正在飞行，后边响起刺耳的破空声。
<br/>　　“嗯？”
<br/>　　沁儿四人转头一看，顿时微微一怔，只见后方追来了一位男子，这人也是人王宫的弟子。
<br/>　　此人三十岁许，身穿一袭黑衣，体型矮小，脸颊欣长，下巴上还留有一小撮胡须。
<br/>　　“裘蒙！”
<br/>　　沁儿几人一怔，他们都认识这个黑衣男子，对方是数百年前，通过考验加入人王宫的弟子。
<br/>　　因为外界之人加入人王宫的弟子不多，所以，他们基本都认识。
<br/>　　“姬月松、骆妤、沁儿、褚彦，你们怎么在这里？”黑衣男子的速度很快，快速的追了上来。
<br/>　　“裘蒙大哥，我们是要去无双界域历练，你呢？”沁儿问道，四人纷纷的停了下来。
<br/>　　“无双界域？”
<br/>　　裘蒙闻言，眸光微闪，笑着道：“真是巧了，我也要去无双界域，那我们同行如何？”
<br/>　　“哈哈，有裘蒙兄加入，求之不得！”姬月松大笑着说道，这位裘蒙可不是弱者，修为高达三品低级，虽然不是妖孽，但也是厉害的天才人物。
<br/>　　“既然你们不介意，那我就跟着几位了！”裘蒙面上带着和煦的笑意。
<br/>　　“裘蒙兄，有你这个高手加入，是我们跟着你才对！”褚彦沉声道。
<br/>　　“哈哈，褚彦你抬举了！”
<br/>　　裘蒙朗笑一声，道：“我们边走边聊！”
<br/>　　“嗯！”众人点头，随即再次前进，一边飞行一边闲聊。
<br/>　　玉瓶空间之中，苏莫眉头一皱，居然又有人加入沁儿等人的队伍。
<br/>　　“怎么会如此巧合？”苏莫很谨慎，感觉事有蹊跷。
<br/>　　而且，此人的修为，高达三品低级，不容大意。
<br/>　　他没有急着出去，准备先观察一番再说，反正距离天池神朝，还有很远的距离。
<br/>　　长空之中，沁儿、骆妤、姬月松、褚彦，以及黑衣男子裘蒙，一起疾驰于空，向天池神朝赶路。
<br/>　　几人一路闲聊，倒也惬意。
<br/>　　“话说我们人王宫的段惊天，怎么没了消息？这种万古天骄，我还从未见过！”
<br/>　　良久之后，裘蒙好似随意的说道。
<br/>　　众人闻听此言，均是脸色微沉，尤其是褚彦，面色不禁稍稍难看了下来。
<br/>　　“裘蒙兄，不要再提那个段惊天了！”褚彦沉声道。
<br/>　　“哦？这是为何？”裘蒙诧异的问道，他的眼眸深处，闪过一道微不可查的异茫。
<br/>　　“别提就对了！”褚彦声音低沉的说道。
<br/>　　裘蒙有些疑惑，他看了看姬月松和骆妤、沁儿，见三人都是沉默不语，心中很是疑惑。
<br/>　　不过，既然褚彦都如此说了，他也不好再问。
<br/>　　“奇怪！”裘蒙心中惊疑不定，他来此自然是奉了天尘宇的命令，来找寻苏莫的线索。
<br/>　　事实上，他已经跟踪了四人一段时间，逼不得已才现身而出的。
<br/>　　主要是，他的修为，并没有领先太多，又没有高深的隐藏之法，便索性现身而出了。
<br/>　　毕竟，姬月松是顶级妖孽，很容易发现他在跟踪，他也是人王宫弟子，还不如正大光明的同行。
<br/>　　他也没有抱太大的希望，能找到苏莫，毕竟这个概率很低。
<br/>　　但是此刻，他谈论起苏莫的化名段惊天，这几人为何是这种表情？
