　　大战将近，近段时间，各路神仙全来相助，有洪荒各地散修，也有各教派之人，这些人都是来噌人皇功德的。毕竟，功德是个好东西，谁都想要。
<br/>　　甚至就连镇守四海的龙族都出现了，派出应龙，前来相助。
<br/>　　龙族也曾是天地主角，曾辉煌一时，奈何神通不及天数，一场龙凤大劫，三族败退，退出了洪荒大舞台，龙族主动镇守四海，才得以保存。
<br/>　　天庭的昊天上帝与瑶池金母，派遣了九天玄女，率领一些天兵前来相助，真可谓是众仙来朝啊。
<br/>　　对于这些，阐教众仙自然是没有好脸色，毕竟功德本来就不多，还要被别人分去，谁都不会高兴。不过，人家是来帮忙的，他们也没有多说什么。阐教众仙，基本上都是大罗金仙的实力，再加上众多法宝，也不用担心会出现意外。
<br/>　　……
<br/>　　一个月后。
<br/>　　公孙轩辕、蚩尤于逐鹿之地，摆开阵势，欲一决高下，胜王败寇，争夺人皇之位。
<br/>　　只见战场上，声势浩大，场面宏大，滔天气势，直冲九霄。
<br/>　　九黎一方的军阵上空，黑云煞气聚集成一大片，遮天蔽日，滚滚黑烟直冲九霄，魔威滔天，杀机逼人。
<br/>　　黑云之中，可见鬼影飘飞，血气起伏，幽绿色的火焰飘飞，黑烟幻化万物，时而变成黑龙，咆哮不已，时而变成黑虎，仰天怒吼，时而变成魔神，杀声震天……
<br/>　　黑烟上下，翻滚不休，阴风阵阵，发出凄厉惨叫声。
<br/>　　蚩尤乃是上古巫妖人三族大战后留下的大巫，一身实力早已达到大巫巅峰，更是堪比祖巫，可战准圣。他精通巫族法术，进而大多数九黎族族人都修炼巫法，拜祭魔神，实力滔天，魔威逼人。与轩辕一方完全不同，这是杀出来的凶威，魔神蚩尤，可不是说说而已。
<br/>　　又看向公孙轩辕一方，人族大军按七星八卦九宫十方玄妙阵理摆布，形成一方绝域，整整齐齐，排列出一方大阵，大阵上方军煞之气依然沸腾，其间黄云滚滚，青光玄气漫天，演化神龙，麒麟，朱雀，玄武等各种神兽，比之九黎一族的阵势，亦毫不逊色。
<br/>　　公孙轩辕去过昆仑山，身为阐教弟子，自然也进过仙神楼与三界楼，读过各种阵法，军阵自然也在其中，而且还是最为顶级的。兵书律法，排兵布阵等，他身为人皇，自然需要向这方面努力。而且，再加上临走时元始让他再经历了一遍人皇的生涯，他自然也知道该怎么做。
<br/>　　双方大军都各自有数百万之多，从上往下看去，密密麻麻，简直吓死人。
<br/>　　大军虽然人数众多，声势震天，但说到底，却也只是陪衬而已。
<br/>　　真正的实力，最终结果的如何，还是需要看双方的高端战力。
<br/>　　也就是双方金仙以上的存在，如阐教的众多大罗金仙，又如九黎族的大巫们，他们之间的对决，决定着双方的胜败。普通修士大军虽然人数众多，但实力不够，人数再多也是枉然，法力高强的修士，挥手间便能够灭杀一大片，人数再多，又有何用？
<br/>　　众人都知道这个理，所以一开始上来便是顶尖修士之间的战斗。
<br/>　　九黎一方，于众将之首走出一九尺高的大汉，手中拿着一柄通体乌黑，血光掺杂魔气缭绕的魔刀，大声喊道：
<br/>　　“轩辕小儿，就凭尔等便想战胜我九黎部众，真是找死，还不速速投降，免得落个尸骨无存的下场！”
<br/>　　此人正是魔神蚩尤，有时也称他为“兵神”，他手中握着的是他的魔兵虎魄刀，相传这是他斩杀一头白虎所炼，封白虎魂魄在其中，能够噬血吞魂，不断进化，杀敌越多，魔刀便会越强。如今蚩尤不知道杀了多少人，此刀的威力，已经堪比大多数先天灵宝了。
<br/>　　此刻面对轩辕一方众仙，他也不认为自己会败，他就从来没想过自己会败，因为他有那个资本。
<br/>　　他自己本身就是顶尖大巫，肉身强悍，可战准圣，最为接近祖巫的存在。
<br/>　　放在大罗金仙与大巫之中，也是最强的，而且当年还参加过三族大战。公孙轩辕再强，也不过是大罗金仙，比他修炼的时间短多了，他也只是稍稍重视而已，要不是近来突然崛起，他还不放在眼里呢。
<br/>　　而且，最为重要的是，他这边还有顶尖大巫，如雨师，风伯。这可不是人族人巫混血的雨皇雨伯与风皇风后，而是水之祖巫与风之祖巫的嫡系后代，为第二代巫族之人，实力也极为强悍，只比他稍弱罢了。
<br/>　　而且，他也相信他的九黎魔兵不会败。九黎魔兵可是有九千强悍巫族强者组成的传奇战兵，其中更有他的八十一个兄弟，八十一个堪比大罗金仙的大巫。虽然，那只是普通的大巫。九黎魔兵一向是攻无不克，战无不胜的存在，一直令人闻风丧胆。
<br/>　　而轩辕一方，虽然看似众仙来朝，也有准圣与大罗金仙，但数量并没有他的大巫多，所以他并不担心。
<br/>　　面对蚩尤的叫喊，公孙轩辕毫不畏惧，上前一步，朗声道：
<br/>　　“大胆蚩尤，你妄动刀兵，导致人族混乱，生灵涂炭，天地不容。你此时若是退去，可免遭劫难，若是不从，必遭天谴！”
<br/>　　“哈哈，天谴？笑话，就凭你们吗？哈哈……”
<br/>　　蚩尤听了此话后，哈哈大笑，语气中充满不屑与狂傲，显然没有将他们放在眼里。
<br/>　　同时，蚩尤的心中也有些凄凉，想想上古时期，巫族何等强大，妖掌天，巫霸地可不是说说而已。那时的人族又是何等弱小，需要来巫族寻求庇佑，那时他们还有些嘲笑人族的弱小。可如今……
<br/>　　巫族与妖族的辉煌早已不见了，巫族，哪还有巫族，只有一些巫人躲在蛮野之地罢了，而当初弱小得随时可欺的人族却是成了巫妖两族的敌人，成了天地主角，这是何等的讽刺！
<br/>　　巫族的威名，当真是落寞了！
<br/>　　公孙轩辕不为所动，不会因为对方实力强悍便害怕，他轩辕之名也不是浪得虚名的，那是杀出来的，天地人三皇，人皇杀伐最重，谁不服，打到他服为止，若还是不服，那便杀了。他想来杀伐果断，面对上古大巫蚩尤，他也是大罗金仙，亦敢一战。
<br/>　　两者交锋多次，虽然他一直落入下风，但也不是毫无抵抗之力。蚩尤的实力增加不多，但他有着人族气运加持，这些年实力却在不断进步，所以最终孰胜孰败，还尚未可知。
<br/>　　巫族的肉体强大，他也一样修炼了阐教的《九转玄功》第七转，肉身堪比大巫，而且还有元神修炼之法，这些事巫族没有的，真要厮杀，他也不怕。
<br/>　　公孙轩辕的表现让众仙非常满意，不愧是人皇，这种气度便令人钦佩，同时也让他们的信心大增。一支军队首领的态度，决定了军队的面容。既然轩辕不怕，他们又有什么好怕的。
<br/>　　只见公孙轩辕气沉丹田，大声喝道：
<br/>　　“蚩尤，可敢一战！”
<br/>　　……
