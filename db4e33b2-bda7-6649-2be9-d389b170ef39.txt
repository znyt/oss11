<br/>				陈默缓缓收功，不练了。现在这种情况，还是先静一下，等休息一晚上再说其他的。<br/>　　<br/>　　甚至对于乾坤珠内的时间加速，增加到了20倍，他都没有去体验和启动，而是以恢复自己的神识为主。<br/>　　<br/>　　修真者，最最重要的其实就是神识，还有灵魂。这两者是一体，也是各不相同的。如果灵魂受伤之后，那么修炼停滞不前都是小事。<br/>　　<br/>　　灵魂受伤后，本源也就损伤了。一个修士的本源损伤，那么不仅身体体质渐弱，就是寿命也会减小，而且还修炼不了，功力直接停滞或者会渐渐散功，对于修士来说，散功是很可怕的。就像是晚期癌症患者一样，除了等死没有别的出路了。<br/>　　<br/>　　当然，在修真界有很多治疗灵魂损伤的药材或者天材地宝，但是想要获得，可能有些万千之难。<br/>　　<br/>　　陈默在一个灵气荒漠的星球，如果灵魂受伤，那么可能自己除了等死以外没有别的出路了。所以对于刚才他的行为，感到了深深的后怕，自己怎么能够不管不顾的就去实验呢？万一要是出个事啥的，那么不就悲催了么？<br/>　　<br/>　　身体此刻已经恢复正常，而且神识什么的都恢复了，头也不痛了。将练功房中的一切阵法收好，走了出去。<br/>　　<br/>　　这个时候天还没有亮，他一推开房门后，大黄和大灰，还有母狐狸和两只小狐狸，都迅速的从窝里钻了出来。<br/>　　<br/>　　大黄和大灰摇着尾巴，倒是没有叫出声，就是围着陈默转圈，并不时的半蹲在他的面前，表现自己，求抚摸。<br/>　　<br/>　　母狐狸算是比较矜持的动物了，只是半蹲在陈默不远的地方，看着他。两只小狐狸，就调皮了，直接爬到了陈默的鞋上，并不是的咬着他的裤腿，看着像是撒娇求玩耍。<br/>　　<br/>　　陈默轻轻的抚摸了一下每个动物，甚至是母狐狸都没有放过。全部安慰了一下，然后让两只小狐狸回到了他们的窝里，轻声对母狐狸说了声看着他们，转身带着大黄和大灰走了出去。<br/>　　<br/>　　他走到平时练功的地方，由于葫芦谷现在在建设期间，有些道路已经被挖断，所以需要绕路，就走的时间比原来要稍稍长些。<br/>　　<br/>　　来到这里，主要是他想练习长拳。这套拳法已经被他学习的滚瓜烂熟。先是沉默了一会，让自己心情平静下来，沉浸在山风徐徐吹来，周围发出：“沙沙！”声的环境中。<br/>　　<br/>　　很长一段时间后，他才直接一个起手式，就闭着眼睛开始打拳，一招快似一招，短短的一刻钟，就将拳法所有招式练习了一遍。<br/>　　<br/>　　不过他并没有停下来，开始了第二遍的拳法。<br/>　　<br/>　　在他开始第一遍拳法的时候，身体莫名的非常舒展。另外似乎是经历过威压以后吧，等神识回到身体内后，感觉身体有些滞涩，就好像是操控有些滞后，不灵敏。<br/>　　<br/>　　这也是他走出来，到这里练习拳法的原因。神识变大了，但却是突然之间增长的，所以身体一时之间就有些不协调。这个事情，一定要尽快解决，不然等以后，就会留下一些问题。<br/>　　<br/>　　身体的灵活度和自己识海是相辅相成的，一旦某个方面快速增长，那么另外一个方面就会滞后，产生拖累。<br/>　　<br/>　　当十遍长拳打完，浑身气血有些微涨，微微出汗。<br/>　　<br/>　　陈默缓缓收拳，胸腹之间有些短促的起伏着。刚才打拳的时候，他是百分百投入，等到最后几遍的时候，他已经沉浸其中。<br/>　　<br/>　　王氏长拳在他的施为下，已经渐渐有了拳意。这种玄之又玄的意境，在他的身上出现，王氏家族要是见到了之后，绝对会将陈默抓过去，解剖一百遍来研究个透彻。<br/>　　<br/>　　对于王氏家族而言，几百年的传承中，在练习长拳的过程中，进入拳意的修炼者，可以说不足一手之数。<br/>　　<br/>　　而且王氏长拳使用的时候，必须配合王氏家族的心法，才能使得长拳百分百能够发挥出其威力来。<br/>　　<br/>　　但是就在今天，一个不是王氏家族的人。一个拿到王氏长拳的拳谱后，就按照拳谱上的说明和拳谱图像，在没有任何心法的前提条件下，竟然一遍遍练习，然后就打出了拳意，这不能不说陈默的天分，真的是非常的高。<br/>　　<br/>　　当拳意出现后，第十遍长拳也施展完毕。陈默收拳后，接着开始了第十一遍。<br/>　　<br/>　　但是这一遍拳法的施展，却似乎有些别扭，不是前面顺着长拳拳谱上的招式，一一对应的施展出来。而是将拳谱中的招式打乱了，前后都不连贯后，施展开来。<br/>　　<br/>　　所以看上去就显得无比怪异，但是陈默却闭着眼睛，缓缓的施展着，似乎他在总结着什么，也似乎在打拳过程中他悟到了什么。<br/>　　<br/>　　一遍的不连贯和别扭，那就两遍，三遍！……<br/>　　<br/>　　渐渐，又一次十遍拳法施展后，虽然招式之间变得不连贯，但是在陈默的施展过程中，渐渐变得有些浑然天成的感觉。<br/>　　<br/>　　就好像，一招施展完以后，第二招本来是连贯的施展下去，却被陈默从另外一个角度，施展出一招。犹如天马行空，又犹如羚羊挂角般随意。<br/>　　<br/>　　如果此时有外人在旁边看着他施展长拳，绝对会被他施展的招式所吸引，有一种韵律在其中，有一种创意在其中。<br/>　　<br/>　　长拳，在陈默的施展中见其真意，也许按照这种练习方法，终究可以将长拳改变，成为陈默他所专有的拳法。<br/>　　<br/>　　终于，再一次十遍过后，陈默已经是满身大汗，在微微显亮的晨曦中，身体升腾出氤氲缭绕的白雾，这是汗水蒸发后所形成的景象。<br/>　　<br/>　　最后一招使出后，陈默随之收功，站立之后，将胸腹中一口气随着吐出，顿时他的面前由于云雾翻腾般出现一团白色云雾。<br/>　　<br/>　　这口气吐出后，他的身体也随之一阵：“噼啪！”声，身体骨骼和肌肉一阵犹如爆豆般的响声，陈默感觉练气三层的身体素质增加了不少。<br/>　　<br/>　　另外就是他感觉到练气三层的功力也增加了一些。他本来进入练气三层初阶并没有多长时间，但是现在却依然功力接近练气三层中阶了。<br/>　　<br/>　　心中一阵的欣喜，没有想到乾坤珠内那个神秘生物，威压虽然有些让他有些不爽，可是却是练功的好材料，如果每次这么增加的话，那么不就能够加快修炼速度么？<br/>　　<br/>　　嘿嘿！看来不是灾祸，而是福气啊！就是因为那个生物的威压，才让他有所感悟的练习长拳，一遍遍的练习，也让他不仅感悟到了拳意，还能渐渐创造自己的拳法，如果不是体力不容许，再这么练习个上百遍，陈默绝对会依靠王氏长拳的基础上，创造出自己的拳法。<br/>　　<br/>　　另外就是神识，也在威压下有了增长，还有就是自己的功力，也有了增长。这威压不是坏处，好处多多。<br/>　　<br/>　　为了巩固功力，就站在那里，缓缓行功并吐纳着。好在他知道这里是荒郊野外，所以并没有沉浸其中，只是缓缓运行的功法。<br/>　　<br/>　　此时，天色终于天色大亮，新的一天已经开始了。<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
