        &lt;h3&gt;第三卷第八十六章  悲伤也是动力&lt;/h3&gt;
<br/>
<br/>        其实陈子昂还没有闹明白是怎么回事，她和五姐聊超市的事情聊的正欢，几个人吃饭吃得好好的，怎么就又扯上她了，二嫂是有多么的不待见她啊。
<br/>
<br/>        似乎今天她就不应该来，如果她不来，大家是不是就开心很多呢。
<br/>
<br/>        “子昂。”陈子乔看着大家沉默不语，而且子昂的心情看起来很差。
<br/>
<br/>        “四姐。”陈子昂的声音是有点哽咽了。
<br/>
<br/>        “子昂，二嫂那人嫁给二哥从来都没有当我们是她的家人，像这种人呢你也不必要理会。”陈子乔也不知道怎么安慰陈子昂。
<br/>
<br/>        他们今天就不应该来二哥家吃饭，二嫂说话太伤人了，都过去了多少年了，还当着大家的面那样说子昂，老是纠结在过去有意思吗？
<br/>
<br/>        “嗯。”陈子昂浓浓的鼻音。
<br/>
<br/>        陈子昂这会心里真的是很悲伤，被自己的亲人那样说，似乎这一生她都只能活在过去，她已经很努力了好不好？
<br/>
<br/>        江俞轩长臂一伸，搂着陈子昂的肩膀：“陈子昂，无论过去怎么样，都已经过去了，别人愿意怎么说就随别人说好了。”
<br/>
<br/>        “嗯。”陈子昂什么也说不出来，只会“嗯。”她极力的忍着心里泛滥的哀伤，不让它们发作，这么多年自己一直努力的生活，怎么就摆脱不了过去的阴影？
<br/>
<br/>        “大家下午也没有吃成饭，我们去外面坐坐？”陈子乔提议。
<br/>
<br/>        “子昂。”大家看着陈子昂。
<br/>
<br/>        “我明天一早要回北京，俩个孩子还在家里。”陈子昂忍住泪，既然大家都没有忘记她的过去，现在的她也没有必要隐瞒了，在这里的基本都是她的亲人。
<br/>
<br/>        “俩个孩子？”陈子乔和陈子悦齐声问，陈子霖也是张着嘴巴，原来江俞轩说的是真的，那江俞轩还说他要和张家姑娘结婚？
<br/>
<br/>        “是的，团团和嘟嘟，一对龙凤胎。”陈子昂站在夜色下，她轻轻的拨开江俞轩的手，离得江俞轩远一点，似乎是要和江俞轩拉开距离。
<br/>
<br/>        “孩子已经上小学了，等有时间我就带回来给你们看，是张函的。”陈子昂看着琉璃阑珊的街道，声音有些缥缈。
<br/>
<br/>        大家听陈子昂说孩子都上小学了，什么都不说了，陈子昂这几年是怎么过来的？他们谁都不知道，谁都没有过问过，大家只知道她消失了几年，大家只看到她光鲜的一面。
<br/>
<br/>        “子昂，如果孩子没有人带，带回来，我来带。”陈子悦轻轻的搂着陈子昂。
<br/>
<br/>        “有人带呢，春姨和财叔一直带着。”陈子昂微微一笑，泪水却随着那笑容悄悄的滑落。
<br/>
<br/>        潘青松想起来之前见到张函那嘚瑟的样子，张函知道他有孩子了吗？张函那话里有话。莫非陈子昂一直和张函来往着？
<br/>
<br/>        “子昂，你和张函还有来往？”潘青松问。
<br/>
<br/>        “子昂这几年一直是一个人带着孩子。她和张函没有任何瓜葛。”江俞轩替陈子昂回答。
<br/>
<br/>        “子昂，过去的就过去了，人要朝前看。”潘青松对陈子昂说。
<br/>
<br/>        “嗯。”
<br/>
<br/>        “俞轩，七哥，我明天要回北京，你们陪我走走，公司的事情刚好要和你俩商量。姐，姐夫，大家都回去吧。”陈子昂还是笑着。
<br/>
<br/>        “好。”
<br/>
<br/>        “那，子昂你保重。”
<br/>
<br/>        夜灯下，三个纤长的身影并排而行。
<br/>
<br/>        “我生了俩个孩子，龙凤胎，是张函的。”陈子昂对身边的两个男人说。
<br/>
<br/>        两个男人都没有接她的话。
<br/>
<br/>        江俞轩是知道的，陈子霖也不再奇怪了。
<br/>
<br/>        “张函你知道吗？”陈子昂问江俞轩。
<br/>
<br/>        “我知道，倩楠的堂哥。”江俞轩当然知道张函，陈子昂的行踪他都会一点一滴的告诉张函。
<br/>
<br/>        “我认识张函的那几年是我最快乐的时候，也是我奠基基础的时候，这一生怕是没有人能比得上张函在我心里的位置了。”陈子昂苦笑一下。
<br/>
<br/>        “他的母亲和妻子来找我了，可是我却怀孕了。”陈子昂有些艰难的说。
<br/>
<br/>        “我一直以为张函是有苦衷的，可是上次信息会的时候我才知道，原来我不过只是一个过客。”陈子昂有些语无伦次，但是俩个男人也没有打断她。
<br/>
<br/>        “我努力的不再去想起以前，可是我却被过去贴上了标签。我遮遮掩掩的怕孩子被张家发现，藏着我的过去，其实不过都是自欺欺人而已。我怕给家里的人丢人，其实家里人根本不在乎我这个人。”她的泪水顺着脸颊流下来。
<br/>
<br/>        夜，笼罩在陈子昂的悲伤里。
<br/>
<br/>        过了很久。
<br/>
<br/>        陈子昂对江俞轩和陈子霖说：“我还是不希望张家知道孩子。”
<br/>
<br/>        “子昂，张函可能不是你想的那样。”江俞轩替张函说话。
<br/>
<br/>        “无论他是什么样都已经过去了，虽然我被过去贴上了烙印，我不会重蹈覆辙，我也不会任人欺负了。”为母则刚，为了孩子她也要坚强。
<br/>
<br/>        “子昂，回去好好休息，什么都不要想，我们的梦想还没有实现，你什么都有，有可爱的宝贝，有事业，有我们！”江俞轩安慰着陈子昂。
<br/>
<br/>        “好！”陈子昂干脆利索的答应着。
<br/>
<br/>        “俞轩，再见，哥，再见！”
<br/>
<br/>        这个世界上，有人在乎你，有人不在乎你，你所有的生活只是你自己的，和别人无关，过得好或者坏，都是你自己的，何必伤春悲秋呢？
<br/>
<br/>        在乎你的人是永远都在乎你，不在乎你的人永远都不会在乎你。
<br/>
<br/>        再黑暗的夜总会亮起一盏灯光，照亮心里某个地方。
<br/>
<br/>        今天也应该感谢二嫂吧，让她有机会给大家说俩个萌宝的事情，要不然她还不知道怎么开口。陈子昂这样一想心里豁然开朗。
<br/>
<br/>        刚刚那卷起的汹涌的悲伤被她抛到了九霄云外，她可以带俩个小萌宝回来西城见姥爷姥姥和舅舅姨娘了。
<br/>
<br/>        现在在西城住的房子还是前几年租的，哥哥给她租了一个公寓，一室一厅，环境还不错，每次陈子昂回来算是不用住酒店，有了落脚之处。
<br/>
<br/>        回到房间，梳洗一番，看着摆在书桌上的还没有写完的方案，她静静的坐着，拿起笔，静下心来，边看着郝景文发的邮件，边以一个乘客的想法写着出行是的感受和需要。
<br/>
<br/>        当黎明的曙光突破天边的暗云，散发着绚丽的云彩时，陈子昂放下了手中的笔，站起身伸展了一下胳膊。
<br/>
<br/>        一个晚上，折腾了她好几个月没有理清的思路，在这个晚上，全部理清，静静的躺在电脑里。
<br/>
<br/>        她要赶回去，将这个方案交给他们进行试验，当然是前期要取得寒伯安和郝景文的认可。
<br/>
<br/>        她换上运动衣和运动鞋，去了附近的建大跑了一圈，回来之后感觉神清气爽，于是又坐下来，重新将方案过了两遍，分别发给了寒伯安、郝景文和江俞轩。
<br/>
<br/>        发给江俞轩的多了一句话：再黑暗的夜总会亮起一盏灯光，照亮心里某个地方。我在昨天的夜里找到了一条通往另一个顶端的路。
<br/>
<br/>        有人说，上帝关上了一扇门，同时也给你打开另一扇门，面对生活中的磕磕碰碰，我们没有必要纠结在一个点，而是应该放开眼光，打开自己的心扉，努力让另外的一扇门的光照耀进自己的心里和生活中，以平和的心态面对一切。
