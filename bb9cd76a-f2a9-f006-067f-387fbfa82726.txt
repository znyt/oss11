
<br/>　　<br/>　　    王冬至拢了下头发，眼神朝着那竖着长辫的老者瞥去，尽管她看得挺隐晦的，也只是惊鸿一瞥，但那老头似乎还是捕捉到了有人注视他的瞬间，顿时两眼中的精光如一把离弦之箭就扎向了王冬至，这一眼居然把王大小姐给望的禁不住往后退了两步，额头上冒出了几滴汗珠，心头狂跳了好几下。<br/>　　<br/>　　    那长辫老头淡淡的收回眼神，转身就回到了大宅门里，然后“咣当”一声，把朱红色的宅院给关上了。<br/>　　<br/>　　    王冬至木然呆愣，王玄真看她反常就问了一句怎么了，也有点察觉出她似乎紧张起来，王冬至说道：“他在警告我，告诉我别在这长白山上乱动，动一动那就有麻烦”<br/>　　<br/>　　    午桥看着紧闭的朱红色大门，脸顿时就沉了，一针见血的点了一句：“对，他就是个麻烦，叶赫那拉氏现在的定海神针，老逼灯一个，忒烦人了”<br/>　　<br/>　　    “他是谁啊？”王玄真和王冬至打听道。<br/>　　<br/>　　    “他爹叫宫宝田，传说中的大内高手……”午桥呲着牙花子解释道：“宫宝田光绪二十三年入宫加封四品带刀侍卫，是大清最后一任大内总管，庚子年间八国侵华，因为他的护驾有功还被赐封了黄马褂可紫禁城骑马，慈禧和光绪都相当得意他了，甚至都评价过说有宫宝田一人在，就算有千军万马来围堵都能来去自如，后来清廷完了宫宝田就被张大帅给请走了，民间传闻十米开外连子弹都打不中他，这老家伙叫宫昌鸿就是宫宝田的儿子，一身武艺自然也是源自他爹，放在以前的宫中肯定又是一个大内侍卫总管，不过宫昌鸿在江湖上可没啥名气，因为他从十八岁开始就随着叶赫那拉氏在长白山上守着龙脉了，从来都没有出过这座山，肯定也是打算老死在这了，我知道宫昌鸿是因为知道他的两次出手”<br/>　　<br/>　　    王玄真叹了口气，说道：“惊天地泣鬼神了呗？”<br/>　　<br/>　　    “他可能这辈子也就出过两次手，一次是四七年的时候宫昌鸿才二十出头，鬼子来了东北后听说长白山上有清廷皇室后代，也知道这帮皇室后人可能挺有钱的，于是遣了一队全副武装的士兵想要过来搜刮一番，但是人马还没上山就在山脚下，一天晚上就被宫昌鸿给摸了过去，扎营的这队鬼子兵一共二百多人全都被屠了个干干净净的，连一发子弹都没来得及开，几乎个个都是脖子上被抹了一刀，都死了”<br/>　　<br/>　　    午桥咽了口唾沫，看着沉默不语的王玄真和王冬至，说道：“第二次是十五年前，龙虎山的一位长老死了，这位长老生前来过长白山，相中了一处龙穴地，觉得自己死后埋在这里能福耀后代，于是他就留下遗言说自己死了就被葬在长白山上，当时是他的后人过来的……具体细节无人得知，反正是那次宫昌鸿出手以后，龙虎山整整五年没开过山门”<br/>　　<br/>　　    王玄真听完了以后，就烦躁的骂了一声说自己不应该来，说向缺太扯犊子了，自己的事非得舔着厚脸皮麻烦他，但说归说，不愿意归不愿意，王胖子的脚却没往后挪一步。<br/>　　<br/>　　    王玄真虽已年过四旬，儿孙全都有了，但一个人不管年岁多大，哪怕就是老态龙钟了，性子也不会变的多么离谱。<br/>　　<br/>　　    王胖子一直都是那个敢跟向缺横刀立马的王玄真，捅破了天都不怕的主，尽管他始终都不是战斗型选手。<br/>　　<br/>　　    午桥斜了着眼睛偷瞄王冬至，想看看这小娘们听了的自己的话后，会不会打了退堂鼓，可没想到的是，王冬至就只说了一句话。<br/>　　<br/>　　    “哎呀，天下间关于高手的传闻太多了，你不试试深浅你哪里知道谁高谁低啊，能讲理就讲，讲不了就打呗，难不成奔波千里的过来了，放个屁就打道回府了？哎呦呦，好丢人的……”<br/>　　<br/>　　    王冬至是觉得，既然来了就肯定没有回去的道理，因为这九百六十万公里土地上，龙脉就那二十四条，这次你退了，以后早晚都得碰上，何必搞那两遍麻烦事呢？<br/>　　<br/>　　    片刻后，三人顺着前路继续往上走。<br/>　　<br/>　　    这个季节的长白山已经开始有点冷意了，白山以外热的狗吐舌头，山上却吹的都是凉风了，山顶则是云雾缭绕的，那里就是长白山最有名的天池，天池的另外一半归属于棒子，不光满清把这里奉为龙脉地，棒子也把这山当成是自己的大兴之地，称为白头山。<br/>　　<br/>　　    这条路上此时还能看见来玩的游人，三人顺着山路走了半个多小时后，王玄真就摆了下手，示意他们别在接着往上走了，真正的龙脉地不可能是人来人往旅游的地方。<br/>　　<br/>　　    王玄真没有手拿风水罗盘，就一手背在身后，一手掐着三根手指，人站在一块巨石上，往附近眺望着。<br/>　　<br/>　　    手拿罗盘看风水的，顶天了能被称为一声大师，靠手指头掐出龙脉在哪的，那都是地师之姿，就比如此时的王玄真放在多少年前，已经初具一国之风水地师的姿态了。<br/>　　<br/>　　    四五分钟之后，王惊蛰手指北坡南边，说道：“龙脉再往上五里地，那就是龙抬头之处了……”<br/>　　<br/>　　    一时三刻后，三人顺着王玄真指的路，走上了一座平淡无奇的小山头，他们人还没到山头上，就看见上面负手而立着一道孤傲的身影。<br/>　　<br/>　　    此人穿着身青衣，脑后扎起来的大辫子随风轻轻而动，一张老脸上一双眼睛，不带任何感情色彩的扫了过来。<br/>　　<br/>　　    宫昌鸿先他们一步来了，就在这等着呢。<br/>　　<br/>　　    午桥如临大敌，王冬至轻轻的皱了皱眉，王玄真则是冷冷的盯着对方，声音不大不小的传了过去：“莫要装什么高手了，我就是要来动一动你们满清的龙脉，龙脉我能动，但是你们肯定动不了我，要说让你们叶赫那拉氏从此以后就被除名那可能有点吹牛逼了，但我要是死，或者残在长白山，你家那个大宅院肯定得要被夷为平地了……”
