　　裘蒙的心中惊疑不定，他之所以现身，就是不抱希望了。
<br/>　　因为之前跟踪之时，他隐约听到了，几人好像是要去寻找段惊天，貌似知晓段惊天在哪里。
<br/>　　所以，他便现身与沁儿等人同行，希望真的能找到段惊天，也就是找到苏莫。
<br/>　　但是现在，几人为何如此，均是对苏莫讳而不言。
<br/>　　“难道他们真的知晓苏莫在哪里？”裘蒙心中暗想，不由得怦然心动。
<br/>　　本来不抱希望的他，这一刻，心中突然火热了起来，如果能找到苏莫，它能彻底改变命运。
<br/>　　虽说他也是人王宫弟子，但在人王宫终究是一个外人，毕竟是通过考验加入的人王宫。
<br/>　　他的天赋很高，但是，加入人王宫三百年了，如今才三品低级的修为。
<br/>　　这是因为，在人王宫内，他能得到的资源和培养，实在是太少太少了。
<br/>　　远不能和沁儿、褚彦、骆妤、姬月松四人相比，四人均是人王宫高层的后代，每一个在人王宫都有很多族中长辈。
<br/>　　但是，若是他能找到苏莫，就能得到海量的资源，以及天尘宇许诺的其它宝物。
<br/>　　裘蒙心中沉吟了起来，跟着沁儿等人，说不定真的能找到苏莫。
<br/>　　这一次，是他的机会。
<br/>　　随后，裘蒙也不再说话，更不再提起段惊天。
<br/>　　众人都陷入了沉默之中，闷头赶路。
<br/>　　片刻之后，沁儿突然停了下来。
<br/>　　众人见此，纷纷停止飞行，疑惑的看着沁儿。
<br/>　　“各位，你们去无双界域吧，我自己去时空界域！”沁儿沉声说道。
<br/>　　“什么？”众人闻言，顿时脸色一沉。
<br/>　　“沁儿，你就能肯定，段惊天会在时空界域？”姬月松沉着脸问道，对于沁儿的执着，他有些无奈。
<br/>　　不过，他心中清楚，这个丫头肯定是看上段惊天了。
<br/>　　“当年我们就是在时空界域遇到段惊天的，说不定，他还会回时空界域！”沁儿不确定的说道。
<br/>　　“这怎么可能？”
<br/>　　褚彦摇了摇头，脸色难看的道：“沁儿，当年段惊天只是路过时空界域，或者只是在时空界域历练，怎么可能还在那里？”
<br/>　　褚彦的心中很是愤怒，但是，他又不能表现出来。
<br/>　　“真不知道段惊天给你灌了什么迷魂汤！”褚彦心中暗道，对于此事，他只能将责任都怪到段惊天的头上。
<br/>　　“沁儿，你别傻了，神域这么大，你不可能找到他！”骆妤摇了摇头，无奈的说道。
<br/>　　裘蒙沉默不语，眸光打量着沁儿，又打量了一番姬月松三人，见几人不似演戏，心中不由得叹了口气。
<br/>　　看来，这四人应该也不知道苏莫在哪里，想找到苏莫，希望太渺茫了。
<br/>　　“我一定会找到他！”沁儿深深的吸了口气，随即身形一转，向另一个方向飞去。
<br/>　　“这……！”
<br/>　　几人见此，均是脸色很不好看，沁儿居然要独自去找段惊天。
<br/>　　“诸位，不能让沁儿姑娘独行，太危险了！”裘蒙朗喝一声，便向沁儿追了过去。
<br/>　　姬月松三人，无奈的对视了一眼，纷纷叹了口气，只能跟随沁儿而去。
<br/>　　他们不可能让沁儿独自一人，前往时空界域，不然若是有个三长两短，那就麻烦大了。
<br/>　　几人很快便追上了沁儿，一番交谈之后，只能改变目的地，向时空界域而去。
<br/>　　裘蒙也没有离开，跟随着沁儿四人，一同前往时空界域。
<br/>　　……
<br/>　　玉瓶空间之中，苏莫清楚的知道外界发生的一切。
<br/>　　对于沁儿为何找他，他也能猜到，这个姑娘可能是对他产生了爱慕之心。
<br/>　　对此，他心中只有无奈，毕竟，他可再没有心思谈情说爱。
<br/>　　至于这个裘蒙，他无法判断，对方到底是什么目的。
<br/>　　是真的巧合，还是为了他而来。
<br/>　　不过，对此苏莫也不在意了，区区三品低级的武者，就算如同姬月松般妖孽，又能奈他何。
<br/>　　他暂时没有急着出去，准备等进入天池神朝境内，再行离开，这样更安全一些。
<br/>　　时间流逝，沁儿五人，商谈好去时空界域之后，便不再多谈，快速的赶路。
<br/>　　仅仅是小半个月的时间，五人便进入了天池神朝，一个叫御天界域的地域。
<br/>　　这个时候，苏莫不想再等待了，准备出去，同时他动用了天幻神丹。
<br/>　　只见高空之上，沁儿五人正在急速而行，异变陡然出现。
<br/>　　沁儿的身上，突然升腾起一团朦胧的光芒，而后，空间一阵波动，一道身影突然出现在了沁儿的身侧。
<br/>　　“这是……！”沁儿五人均是神情一怔，立刻停了下来，惊疑不定的打量着这道身影。
<br/>　　这是一位青年，体型高大壮硕，身穿黄衣，面容略显粗狂，给人一种凶狠之感。
<br/>　　“你是谁？”沁儿怔怔的问道，由于事出突然，她被吓了一跳。
<br/>　　这位青年突然出现，毫无征兆，好像是从他的储物戒内出来的。
<br/>　　这让她心中惊疑不定，居然有人藏在他的储物戒之中，这怎么可能？
<br/>　　姬月松四人的目光，均是紧紧的盯着高大青年，上下打量。
<br/>　　眼前的高大青年，身上气息深邃而不可测，根本看不出具体的修为。
<br/>　　而裘蒙眸中光芒微闪，虽然眼前之人，和苏莫完全不同，但是，他却深深的怀疑了起来。
<br/>　　毕竟，他此行的目的，就是看看苏莫有没有被沁儿等人带出人王宫。
<br/>　　此情此景，他第一个念头，就是此人是苏莫。
<br/>　　只是，若是苏莫的话，藏在沁儿的身上，对方居然好似不知晓。
<br/>　　这高大的青年，自然便是苏莫，他动用了天荒神丹，在别人的眼中，他已经变成了另外一个人。
<br/>　　看了一眼神情惊疑不定的沁儿，苏莫一眼不发，转身而去。
<br/>　　他没有必要，与几人交谈，离开此地才是正途。
<br/>　　“站住！”
<br/>　　然而，苏莫刚准备飞走，裘蒙却是身形一闪，拦在了苏莫的身前。
<br/>　　“何方鼠辈，报上名来！”
<br/>　　裘蒙大喝一声，言罢，立刻向姬月松几人喊道：“诸位，此人出现的诡异，不能让他离开！”
<br/>　　喊话的同时，裘蒙的掌心，悄悄的出现了一枚特制的传讯符。
<br/>　　只要证明眼前之人是苏莫，他就立刻传讯，将消息告知天尘宇。
<br/>　　此时此刻，他心中无比激动，若是眼前之人，真的是苏莫，他的机遇就来了。
<br/>　　唰唰唰！
<br/>　　姬月松等人，也是毫不含糊，瞬间就将苏莫合围了起来。
<br/>　　“你到底是什么人？”姬月松双眉之间充满了厉色，大声质问。
<br/>　　他自然不能让眼前来路不明之人就此离去，最起码要搞清楚缘由。
<br/>　　“沁儿，怎么回事？此人好似从你身上出现的？”骆妤疑惑的看向沁儿，不解的问道。
<br/>　　“不能让他离开！”褚彦亦是冷声说道。
<br/>　　沁儿望着苏莫，神情怔怔，没有回答骆妤的话。
<br/>　　她不傻，稍一回想，就能确定，眼前之人，就是从她的储物戒中出来的。

<br/>　　她立刻就想到了爷爷给予他的玉瓶，眼前之人，很可能就是从玉瓶之中出现。
<br/>　　她稍一回想，貌似此人出现之时，储物戒之中的玉瓶，的确波动了一丝。
<br/>　　沁儿眼眸亮了起来，普通的宵小之辈，不可能出现在爷爷的玉瓶之中。
<br/>　　那么，眼前之人，是爷爷故意所为。
<br/>　　答案呼之欲出了，此人，很可能就是消身匿迹的苏莫。
