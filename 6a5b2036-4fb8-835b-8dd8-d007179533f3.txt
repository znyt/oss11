　　梁师兄和一众爪牙郁闷不已，而一些其他窥视天九殿的弟子，也都在静观其变。
<br/>　　其他一些千渊殿的弟子，自然能看出梁师兄一众爪牙的动作和意图。
<br/>　　这些人均是没有任何动作，有梁师兄打头阵，他们也乐意坐享其成。
<br/>　　而天九殿之中，苏莫却是不理会外界的风雨，努力的修炼。
<br/>　　整个修炼之中，精纯的五行源气浓郁到了恐怖的地步，苏莫的每一次呼吸，都带动源气滚滚。
<br/>　　这是绝佳的修炼之地，对于任何武者来说，都是宝地。
<br/>　　苏莫体内，混沌圣丹不断的壮大，不断的凝练，发出璀璨的混沌之光。
<br/>　　圣丹之旁，万化神剑元神飞舞，元神之力和圣丹之力不断的交融。
<br/>　　时间流逝，无休无止，转瞬间过去了半年。
<br/>　　这半年的时间，再也没有人来打扰，苏莫也落得安静。
<br/>　　很快，便过去了一年半。
<br/>　　这一日，修炼室之中，苏莫的身上，混沌之光无比璀璨，强大的剑势暴冲而起。
<br/>　　很快，气势收敛，光芒褪去。
<br/>　　苏莫睁开了眼眸，眸中闪过淡淡的喜意。
<br/>　　他已经算是进入了虚神境四变，但是，并不是真正意义上的提升了境界。
<br/>　　因为，若是他只有万化神剑元神，他现在已经是虚神境四变的大能。
<br/>　　只不过，他还有吞噬元神和天梦元神，需要去与圣丹融炼，还不算是真正的虚神境四变修为。
<br/>　　当然，总体而言，他已经实力大增，提升了一个层次。
<br/>　　“一年半左右的时间，算是比较快了！”苏莫深深的吸了口气，心中却是离意大升。
<br/>　　一年半了，他不能继续逗留了，再继续逗留下去，风险会百倍提升。
<br/>　　随即，苏莫长身而起，走出了修炼室。
<br/>　　说实话，想到离开，他还真的有些舍不得这绝佳的修炼室。
<br/>　　但是，为了安全起见，他却是不得不离开。
<br/>　　离开千渊殿很容易，不需要向殿主或长老打招呼，毕竟外出历练、闯荡很正常。
<br/>　　不过，千渊殿有规矩，任何弟子不得背叛，要随时听从殿中的召唤。
<br/>　　咯呀！
<br/>　　天九殿的大门打开了，苏莫大步走出。
<br/>　　“离开之前，去取件兵器！”
<br/>　　苏莫心中暗道，闭关之前，他就从伏卓那里了解到，新晋弟子可以免费领取一件下品神器。
<br/>　　离开天九殿之后，他径直向千渊府的中央位置而去。
<br/>　　距离殿主所住塔楼不远处，有神兵宝库，其内神器众多。
<br/>　　对于神器，苏莫很是心动，这可不是下界那些圣级上品兵器可比。
<br/>　　他的紫虚神剑，虽然威力比圣级上品兵器大的多，但终究不是神器。
<br/>　　“上官昊出来了！”
<br/>　　“上官昊走出天九殿了！”
<br/>　　“一年半都不止了，这个家伙终于肯出门了！”
<br/>　　就在苏莫走出天九殿的这一刻，很多弟子纷纷知晓了。
<br/>　　嗖！
<br/>　　院落中，黑衣青年连师兄，火速的赶到了梁师兄的院落之中。
<br/>　　“梁师兄，那上官昊出来了，看样子是要去神兵宝库！”连师兄进入梁师兄打坐的房间，朗声说道。
<br/>　　房间之中，本来正闭目调息的梁师兄，闻听此言，顿时睁开了眼眸。
<br/>　　“连霆，你带人过去，这一次，务必让上官昊颜面尽失，向所有人证明，下界的低等人不配拥有天九殿。”梁师兄眸中寒光一闪，冷漠的说道。
<br/>　　“是，梁师兄！”黑衣青年连霆重重的点了点头，随即转身而去。
<br/>　　等了一年半，上官昊终于出来了，他们自然不能放过这次机会。
<br/>　　望着连霆离去的背影，梁师兄嘴角勾起了一丝笑意，机会终于来了。
<br/>　　随即，梁师兄心中沉吟，待整治了上官昊之后，他去求见殿主，让殿主将天九殿赐予他。
<br/>　　苏莫离开天九殿之后，没过多久，便来到了神兵宝库。
<br/>　　神兵宝库，是一座小型的塔楼，只有三层。
<br/>　　第一层空间不大，是一间大殿，其内有上百张石台，每一张石台之上，都摆放着一件神兵。
<br/>　　每一件神兵，都被一个小型的防护罩守护，防护罩内，各色的光芒闪耀，璀璨无比。
<br/>　　“都是神器！”苏莫眼眸大亮，这每一件兵器，都比他的紫虚神剑要强大，这均是真正的神器。
<br/>　　大殿之中，只有一名面容古板的灰衣中年人，正在一张没有兵器的石台之上打坐。
<br/>　　除此之外，没有任何的弟子。
<br/>　　千渊殿弟子不多，而且，都是修为高深之辈，不会经常有人来神兵宝库。
<br/>　　“拜见执事！”苏莫来到中年人身前，微微抱拳行礼。
<br/>　　千渊殿之中，除了殿主和三位长老之外，还有十几名强大的执事。
<br/>　　所以，苏莫心中清楚，此人便是千渊殿的执事。
<br/>　　“你就是上官昊吧？”中年人睁开眼眸，打量了苏莫一番，淡淡的问道。
<br/>　　“正是！”
<br/>　　“嗯，你刚入门不久，可以领取一件下品神器，这几十件神器之中，你挑选一件吧！”执事点了点头道。
<br/>　　“是！”苏莫应声，而后便开始挑选神兵。
<br/>　　这大殿之中，共有兵器约九十余件，其中宝剑最多，共有三十余件。
<br/>　　苏莫一件又一件查看了起来，每一件神器的下方，都摆放着一块木牌，其上刻着对应神器的名字。
<br/>　　“火空神剑！”
<br/>　　“暗夜神剑！”
<br/>　　“雷霄之剑！”
<br/>　　“……”
<br/>　　苏莫不断的查看，每一柄宝剑都光华夺目，极为不凡。
<br/>　　不过，大多数宝剑，都不太适合他，因为对应的属性不同。
<br/>　　比如火空神剑，明显的火属性神器，能让火属性武者实力大增。
<br/>　　但是，对于他来说，就有些不契合了。
<br/>　　挑选了片刻，很快，苏莫的目光，并锁定在了一柄银色的宝剑之上。
<br/>　　因为这柄宝剑，不是单属性的神器，而是蕴含了各种属性。
<br/>　　或者说，这是一柄属性力量比较综合的神器，适合各类武者使用。
<br/>　　“太云神剑！”
<br/>　　苏莫看向银色宝剑下方的木牌，其上雕刻着这四个小字。
<br/>　　“执事，我选这柄太云神剑！”苏莫转头向中年执事说道。
<br/>　　“嗯！”中年执事点了点头，随即，他一挥手，太云神剑之上的微型防护罩，便消失不见了。
<br/>　　当防护罩消失，一股夺目的锋芒，便从太云神剑之上冲天而起，锋锐而又凌厉！
<br/>　　“好剑！”看到此剑的锋芒，苏莫眼眸一亮，这太云神剑可比他的紫虚神剑强大多了。
<br/>　　虽然同为神剑，但是，紫虚神剑并不是真正的神器。
<br/>　　“去吧，当你修为达到四品真神之时，还可以来领取一柄中品神器！”执事朗声道。
<br/>　　“多谢执事大人！”苏莫抱拳道谢，随即郑重的拿起了太云神剑，收进了储物戒之中。
<br/>　　随即，苏莫也没有多留，转身向外走去，他要出去避避了，过些年再回来。
<br/>　　然而，苏莫刚刚走出神兵宝库的大门，迎面便走来一群人。
<br/>　　“上官昊，你终于出来了！”
<br/>　　轻笑之声响起，开口之人身穿黑衣，正是连师兄连霆。
<br/>　　“诸位师兄，你们有何事？”苏莫停下了脚步，眉头微微一皱，沉声问道。
<br/>　　他认出了这些人，正是一年多前，被他拒之门外的那群弟子。
<br/>　　“上官师弟，大家都是同门，何需这么见外，不如到我院中坐坐如何？”
<br/>　　连霆带着众弟子，来到了苏莫身前，笑着说道，他很是客气，一副热情的模样。
