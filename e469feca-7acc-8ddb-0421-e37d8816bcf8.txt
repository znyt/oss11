　　处理完李静仙的事情，这段时间以来李承乾的一块心病多多少少算是解决了一小部分，也终于让他的目光开始转向其它事物。
<br/>　　说起来，李承乾到底也还是个小市民，纵然有些小聪明，但胸襟这东西却怎么也锻炼不出来，总是喜欢在一些事情上刨根问底，如果不彻底解决的话就总也放不下心来。
<br/>　　这与李二同志显然有着很大的区别，老头子就像一个主频不高的CPu，配合多任务处理的操作系统，虽然速度不快，但却可以同步处理几件不同的任务，而李承乾却是一个有超高主频的CPu，可操作系统却是DoS，速度虽然快，但却一次却只能处理一个任务。
<br/>　　而且这个操作系统还是当初购买的时候就已经预装好了的，事后不可重装，甚至连售后都没有，就算是不满意也只能将就。
<br/>　　李承乾在意识到这一点之后，索性也就放弃了挣扎，比不过就是比不过，儿子比不过爹这事儿很正常，虽然人们都希望青出于蓝，但……还是算了吧。
<br/>　　“陛下，边疆战报，候将军已经深入拜占庭，如果有意外，半年之后可攻陷其国都！”早朝之上，马周见李承乾气色不错，出班奏道。
<br/>　　李承乾把思绪拉回来，不动声色的点了点头：“唔，那程大将军呢？怎么还没有消息？他那里战事如何了？”
<br/>　　“这……”马周犹豫了一下，将目光投向一旁的张行成。
<br/>　　老房这个时候因为身体的条件不允许已经退出朝堂许久，张行成作为老房的接班人，这个时候正是需要他站出来的时候。
<br/>　　“陛下，程大将军那边似乎遇到了一些困难，依旧在被挡在木鹿城外。”张行成见李承乾的目光被马周引过来，心中无奈一叹，出班言道。
<br/>　　大唐这些年来因为装备的关系，只要是对外战争往往都是以摧枯拉朽的姿态在极短的时间内结速战争，很少会将一场战争打上数年之久。
<br/>　　但大食这个国家似乎有着别样的顽强，程咬金带兵打了近半年，却有四个月停滞未前。
<br/>　　这让朝上众人不得不怀疑老程是不是有什么特殊的想法，否则就是敌人再顽强，四个月时间也应该突破防线了。
<br/>　　李承乾对此倒是颇有些意外，昏昏欲睡的神态一扫而空，兴致勃勃的问道：“木鹿城还在？看来这大食人似乎不如想像中那么怂嘛！”
<br/>　　“陛下，区区木鹿城竟能阻我大唐二十万大军四月之外，卢国公此举好像有畏战之嫌。”有李猫之称的李义府这几年职务提升很快，已经是五品监察御史，有李承乾罩着，今天弹弹这个，明天弹弹那个，在朝堂上倒也颇有些声势。
<br/>　　马周与张行成之所以开始没有提程咬金的事情，也就是怕这小子从中捣乱，毕竟这段时日看李承乾的架式似乎挺宠这小子的。
<br/>　　现在一看，果不其然，这小子还真是属疯狗的，见谁咬谁，就连老程都不被他放在眼里。
<br/>　　而就在与老程相熟的一般人略有些紧张，生怕李承乾会误听人言打算出来替老程说几句话的时候，李承乾已经开口了：“李御史说的似乎有些道理，既然这样不如调程咬金回来吧，顺便将那二十万军队也调回来。”
<br/>　　“陛下，陛下万万不可啊，程大将军于国有功，万万不可如此草率处置，如此势必影响军心……”
<br/>　　“好了好了，朕什么时候说要处置程咬金了！”李承乾看了眼急急站出来的李敬玄，摆了摆手说道：“朕说的是轮战，轮战懂不？”
<br/>　　“啊？”李敬玄愕然抬头，满眼不解之色，其他众人也是一脸的懵懂。
<br/>　　“你们这些人啊，一天到底总是想着打胜仗，今天想着灭这个，明天想着灭那就，怎么就不想想，如果把所有敌人都消灭了，没有外敌的情况下，我大唐的出路在什么地方？”李承乾无可奈何的叹了口气，显得有些失望。
<br/>　　“陛下，难道打胜仗不好么？大食屡次挑衅，难道不应该将之消灭么？”李义府同样是一脸的疑惑，心中隐隐有了一丝担心，总觉得事情有些脱离自己的控制。
<br/>　　“谁说打胜仗就一定是好了？战无不胜，攻无不克说来虽好，但却容易让人养成骄傲自满的坏毛病，你李义府不就是这样么。”李承乾看着李义府淡淡说道。
<br/>　　‘唰’的一声，李义府脸色瞬间一片惨白，双膝一软‘噗通’一声跪倒：“陛下，臣，臣有罪，但请陛下看在臣对大唐忠心耿耿，不敢有一日懈怠的份上，从轻发落！”
<br/>　　“行了，别在那里装模做样，如果朕真的发落了你，怕是立马就要哭鼻子！”李承乾哼了一声：“不过你记住，军方的事情以后你不要插手，文官就要管好政事，如何打仗，打的如何不需要御史衙门来操心，都听懂了没有？！”
<br/>　　“臣等遵旨！”不管文官一系如何，武将一系对李承乾的意见自然是喜闻乐见，一个两个眉开眼笑。
<br/>　　文官一系面面相觑，想要开口却发现似乎没什么可说的，监察御史似乎真的没有弹劾如何打仗的权力。
<br/>　　在这方面，李承乾做的很好，专业的事情就让专业的人来处理，文官不懂如何打仗那就不要参乱插言，以背后指手划脚。
<br/>　　毕竟打仗不是打架，万里之外的战争，不说当时天气如何，地势如何，士气如何，就算是消息传回长安那也是好几个月的事情，作为一个文官仅凭纸面上的东西，如何去判断前方将士有没有尽力？
<br/>　　后世的大宋与大明坏就坏在外行指导内行，明明是武将在前面打仗，可皇帝却非要弄些文官或者太监去指挥管理，到最后明明是胜仗，可打到最后却成了败仗。
<br/>　　说什么武将不可信，可难道那些文官就可信了？谁来保证他们的忠诚，谁又能保证他们的忠诚？
<br/>　　所以李承乾认为还是老头子那种疑人不用，用人不疑的态度才是最正确的，上位者不偏不倚，给下面的人指明一条前进的道路，把所有人都拧到一起，有一个共同的发展方向才是最正确的，总是怀疑下面的人会反对自己的话，那只能说明这个皇帝是个废物。
