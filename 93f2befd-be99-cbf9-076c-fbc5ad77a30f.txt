　　“你们这些人，虽然如今通过了考核，成为了我们北邙学院的外门弟子，但是为了验证你们的忠心，都必须要去完成一次学院交代的任务才行。”
<br/>　　苏凌长老，负手而立，站在空中，言语冰冷的开口道。
<br/>　　“完成了这次任务，你们才算是我们北邙学院真正的弟子。”
<br/>　　“这些内门弟子，便是会带领你们前去完成任务，现在，开始分配你们的队伍。”
<br/>　　“你，你，你，跟我走。”
<br/>　　那些内门弟子，走入到人群之中，各自挑选着，每个人便是挑选少则几人，多则数十人。
<br/>　　这外门考核人物，便是就由内门弟子带领，一是为了保证安全，二也是一种监视作用。
<br/>　　像是北邙学院这等势力，仇敌无数，经常会混入一些奸细，来到北邙学院中。
<br/>　　这外门考核，便是考验外门弟子的忠心。
<br/>　　“你，跟我走。”
<br/>　　一尊长相冷峻的内门弟子，便是将楚风眠挑走。
<br/>　　除了楚风眠，另外还有着三人，一共四人，被这位内门弟子调走了。
<br/>　　“此人的实力，倒是不弱，比起拓跋鼎还要强大几分。”
<br/>　　楚风眠看着眼前这内门弟子，心中暗自想到。
<br/>　　此人的实力，应该力量指数达到四，在内门中，算是一尊中规中矩的武者。
<br/>　　这一批内门弟子，其实都没有什么强者，至于地榜武者，更是一个没有，毕竟这等强大武者，都在修炼，哪有时间来帮忙完成任务。
<br/>　　一尊精状武者，也是那被选中的外门弟子之一，开口道。
<br/>　　“我叫莽通，不知道各位兄弟都怎么称呼？”
<br/>　　“大家这一次聚在一起，完成人物，也算是缘分，不如交个朋友。”
<br/>　　“我叫华新。”
<br/>　　一尊武者开口了，倒是语气极为平常。
<br/>　　“瞿木。”
<br/>　　最后轮到楚风眠的时候，楚风眠还未开口，那莽通便是笑了笑开口道。
<br/>　　“楚兄就不用说了，这一届的新人王，谁不认识。”
<br/>　　“恩。”
<br/>　　旁边的瞿木，华新，都是点了点头，对楚风眠，他们倒是全都认识。
<br/>　　毕竟楚风眠刚刚来到外门的第一天，便是废了两个新人，然后更是毁灭了风云会，就连人榜第十一的武风云，都败在了楚风眠的手中。
<br/>　　这样的风云人物，他们怎么会不知道。
<br/>　　莽通有些紧张的开口道。
<br/>　　“这一次任务，不知道好不好完成啊，据说每一届的新人中，都会有人陨落在考核任务之中。”
<br/>　　华新冷冷的开口道。
<br/>　　“北邙学院的考核，哪有那么容易，没有实力，就算是混进来，也会陨落。”
<br/>　　楚风眠开口道。
<br/>　　“不用担心，我们既然是一齐完成这个任务，自然是要齐心协力才行。”
<br/>　　听着楚风眠的话，莽通的脸上才终于没有那么紧张。
<br/>　　“楚兄说的对，跟楚兄一起，我们肯定能完成。”
<br/>　　“你们废话那么多干嘛，还不快点过来。”
<br/>　　挑选完人。
<br/>　　那内门弟子，冷声的开口道。
<br/>　　“你们便是被我看中，是我的人了，这一次人物，你们必须要听从我的话，知道了吗？”
<br/>　　“不知师兄是何名谓？”
<br/>　　被挑选离开，莽通突然询问道。
<br/>　　“还有我们这次的任务是什么。”
<br/>　　“废话真多，我叫雷绝，这次是任务，是负责捣毁地魔门的一处祭坛，具体的事，告诉你们你们也不懂，你们就乖乖听从我的指挥便是。”
<br/>　　雷绝不耐烦的开口道。
<br/>　　“要是你们不听话，笨手笨脚的，敢妨碍到我，小心死在这次任务之中。”
<br/>　　“具体的事，等到了在告诉你们，跟我走。”
<br/>　　说着，雷绝便是一道遁光，直接离开。
<br/>　　而楚风眠四人，则是互相对视了一眼，也是化为遁光，追了上去。
<br/>　　雷绝带领，足足是连续飞行了三个时辰，在停了下来。
<br/>　　除了楚风眠，那另外三人停止飞行的一刻，都是气喘吁吁，坐在了地上，直接休息了起来。
<br/>　　以他们的实力，连续飞行三个时辰，便是已经快要将灵力耗尽，一个个都不顾一切，在地上休息了起来。
<br/>　　只有楚风眠，倒是显得极为平静，虽然看起来不算轻松，但是也不至于如此气喘吁吁。
<br/>　　不过这也还只是楚风眠故意装出来的样子，
<br/>　　“哼，一群废物，飞行三个时辰就消耗了这么多的灵力，以后怎么对敌？”
<br/>　　雷绝瞥了一眼，冷哼一声，看了楚风眠一眼，眼神中的不屑才轻视了许多，并没有对楚风眠说些什么。
<br/>　　休息了半个时辰，众人便是再次启程，足足是这样休息了三四次，终于跟随这雷绝，来到了这北邙学院的边缘。
<br/>　　在这北邙学院的边缘，居然是另外一处极为古怪的空间。
<br/>　　雷绝看了一眼四周，便是脚下一动，直接是踏入到了这一处古怪空间之中。
<br/>　　众人看着眼前的空间，都似乎微微有些犹豫。
<br/>　　“这……”
<br/>　　雷绝出声大骂道。
<br/>　　“你们这群废物，还不快点跟过来！”
<br/>　　楚风眠看了一眼这古怪空间，眼神闪烁一阵，便是直接踏了进去。
<br/>　　“走吧。”
<br/>　　看到楚风眠走了进去，那剩下的三人，也就一起脚下一动，大步迈了进去。
<br/>　　进入到这一处空间之中，楚风眠便是感觉到周围的灵气，跟在北邙学院中，截然不同。
<br/>　　这里面的灵气，的确也极为浓郁，但是却不像是北邙学院中的灵气，极为精粹，可以直接吞噬，炼化。
<br/>　　这里的灵气，却是浑浊无比，在其中，灵气之中布满了无数妖邪魔气，所以才显得极为浑浊。
<br/>　　看向四周，周围也没有多少生机，看起来此地，倒像是一处魔域一般。
<br/>　　华新扫视了周围一眼，眼神中也露出了几分紧张道。
<br/>　　“这里是哪里？我们已经离开了北邙学院？”
<br/>　　“这里是地魔界，已经不是在北邙学院之中。”
<br/>　　雷绝冷冷的回答道。
<br/>　　“这地魔界，可是地魔门的老巢，你们可要小心一点，不然惹到了地魔门的大魔头，我可不会救你们。”
<br/>　　“地魔界，地魔门。”
<br/>　　听到这两个名字，也在其他几人的眼神中，露出了几分紧张。
<br/>　　此地，居然是地魔门的老巢。
