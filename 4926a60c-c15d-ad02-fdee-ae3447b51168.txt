        宋云汐立马挽住冷晏琛的臂弯，小鸟依人地撒娇道：“阿琛，你怎么才来呀？”
<br/>
<br/>        “我给你做排骨汤去了，所以来晚了！”冷晏琛淡定地笑了笑，随声附和道。
<br/>
<br/>        郑温韦怔愣地从地上站起来，转过身去看着冷晏琛，见这男人穿着军装，却只是个服役兵，不禁嗤之以鼻道：“云汐，你就看上了这么个义务兵吗？服役期满了后，安排的工作只不过每个月那么两三千块钱的工资，你也看得上？”
<br/>
<br/>        “阿琛，你别介意。就算你服役期满了后，分配给你的工作哪怕是没钱，我也不在乎。放心吧！我会养你的。”宋云汐柔情似水地说道。
<br/>
<br/>        冷晏琛倒是挺意外地看着宋云汐。
<br/>
<br/>        原来，这女人是天生的演技派。
<br/>
<br/>        郑温韦不禁握紧双拳，冷哼了一声后，眉开眼笑道：“云汐，为了气我，临时找个男人，也不知道找个比我有钱的男人。”
<br/>
<br/>        宋云汐挽着冷晏琛的臂弯，一脸无所谓地走到郑温韦的面前，会心一笑：“你确实是个有钱的男人，只不过，我觉得，我现在的这个男人长得比你帅就行了。”
<br/>
<br/>        “你……”郑温韦欲言又止，双眼中瞬间布满了血丝。
<br/>
<br/>        宋云汐嘴角微扬，将脑袋往冷晏琛的臂膀上靠去：“郑总，好走不送。”
<br/>
<br/>        郑温韦握紧手中的戒指盒，狠狠地瞪了冷晏琛一眼后，阔步走出了病房。
<br/>
<br/>        冷晏琛转头看了郑温韦一眼，回过头来时，宋云汐已经松了手，并与他保持着一段安全距离。
<br/>
<br/>        “你找我有什么事？”宋云汐单手抱臂，略显狼狈地问道。
<br/>
<br/>        冷晏琛将手中的保温饭盒放在了病床的床头柜上，十分淡定地说道：“这是我祖爷爷让我给你送过来的，宋小姐记得喝。”
<br/>
<br/>        “谢谢。”宋云汐微微点头谢道。
<br/>
<br/>        冷晏琛礼貌性地笑了笑，准备转身离开的时候，宋云汐叫住了他。
<br/>
<br/>        宋云汐微微欠身，温和地说道：“相亲那天的事情，我向你道歉。刚刚的事，谢谢你帮了我。”
<br/>
<br/>        冷晏琛却只是云淡风轻地应了声：“没关系，不客气。”
<br/>
<br/>        他说完后，便离开了。
<br/>
<br/>        宋云汐看了一眼床头柜上的那个保温饭盒，心里其实很不是滋味。
<br/>
<br/>        她果真被老妈给“臭骂了一顿”。
<br/>
<br/>        “你真是缺心眼啊！那个宋雨柔，要死就让她死掉好了！她跟她那个小三母亲当年害我们娘两害得还不够惨吗？你为什么要捐骨髓给她啊！这是老天爷对她们的惩罚！你是有多缺心眼？你这一做手术，就得停飞一年！一年啊！你知不知道，这一年里……”
<br/>
<br/>        “妈！我救的只是一条生命，仅此而已。”宋云汐打断了母亲的滔滔不绝。
<br/>
<br/>        听筒里，母亲的声音哽咽了：“你怎么这么傻！呜呜——你怎么这么傻！”
<br/>
<br/>        宋云汐握紧手机，贴在耳边，默默地听着母亲的哭泣声。
<br/>
<br/>        她傻吗？
<br/>
<br/>        她不傻啊！
<br/>
<br/>        她不捐，也会被郑温韦使用他的职权将她停飞。
<br/>
<br/>        救了，她就当是给自己积德。
<br/>
<br/>        宋云汐站在病房的窗户前发呆，听母亲哭诉完后，安慰了母亲后，才跟母亲挂的电话。
<br/>
<br/>        冷晏琛走后没过多久，宋云汐的父亲宋志远便走了进来。
<br/>
<br/>        他进来的第一句话，不是关心她的话，而是为了他的利益的话：“云汐，你和温韦结婚。对你，对我们宋家来说，都是一件好事。”
<br/>
<br/>        “怎么？宋雨柔得那么个病，被郑家的人嫌弃了吗？”宋云汐转过身去看着父亲宋志远，不禁嗤之以鼻。
<br/>
<br/>        宋志远顿时沉默了。
<br/>
<br/>        宋云汐冷冷一笑：“我虽然也姓‘宋’，但我不跟你姓。我跟我妈姓。所以，你们宋家的事情跟我有何关系？”
<br/>
<br/>        宋志远叹了口气，心里暗想，云汐肯捐骨髓救雨柔，那么久代表她不是个铁石心肠的女人。
<br/>
<br/>        “云汐，爸爸知道。这么多年来，爸爸在你面前，都没有做到尽一个父亲的责任。只要你答应嫁给温韦，爸爸马上请律师来立遗嘱，以后双宋集团，你拿百分之五十的股份。”宋志远信誓旦旦地说道。
<br/>
<br/>        宋云汐嗤笑道：“你还好意思提那百分之五十的股份？当年，是妈妈和你一起打拼下来的双宋集团。结果，就因为那个小三给你生了儿子。你和那个小三便联手，将我和妈妈逐出家门。还害得妈妈一分钱也没有！你现在来跟我提股份？还有，你这么急着跟郑家联姻。只怕是公司的资金周转不过来了吧？”
<br/>
<br/>        “云汐，爸给你跪下了！”宋志远突然“啪”地一声，双膝跪在了宋云汐的面前。
