　　黎明的晨光中，营地中的排水渠泛着一层诡异的光，缓缓流动着。
<br/>　　开始的时候秦吉胜还以为那是还没有凝固的鲜血，可是很快他就意识到那并不是血。
<br/>　　因为如果有如此多的鲜血的话，除非上面积尸盈尺才可能会发生，否则的话根本不会有如此多的鲜血。
<br/>　　再说人血的有腥味的，可是排水渠里流动的液体根本没有任何腥味，反而有一种十分奇怪的味道在里面，闻上去总是有些熟悉。
<br/>　　这一发现引起了他的好奇心，于是便从坐下‘哈士奇’上下来，亲自蹲到水渠边上去看。
<br/>　　结果不看还好，这一看顿时把秦吉胜吓的魂飞魄散。
<br/>　　因为那沟渠之中流动的哪里是什么鲜血，那明明就是洧水，属于见火就着的东西。
<br/>　　大唐的营地中出现洧水，而倭人又根本没有这种东西，这一切说明了什么？
<br/>　　秦吉胜根本来不及想太多，以最快的速度跳回到‘哈士奇’的背上，拼了命的喊道：“撤，马上撤，这是一个陷井，快撤！”喊话声未落，人已经窜了出去。
<br/>　　……
<br/>　　山背大兄此时正在不远处琢磨着大胜之后回去如何把自己的兵权夺回来，开始并没有意识到秦吉胜在喊什么，而等他反应过来的时候已经连秦吉胜的狗尾巴都看不到了。
<br/>　　接着，不等他再多想，便听到有阵阵雷鸣之声自远而近传了过来，那声音听来极是熟悉，山背大兄曾经听过无数次。
<br/>　　大唐的贞观炮，只有这东西才会发出这种声音，反应过来的山背大兄只觉得后背唰的一下就布满了冷汗，逃走的念头刚刚升起，无数团烟尘已经扬了起来，剧烈的爆炸声在营地里响成一边。
<br/>　　而随着爆炸声起，唐军大营之中已经迅速的升起了数道火墙，将整个大营全部围在了当中，包括还没有来得及离开的山背大兄。
<br/>　　出事了，有埋伏！山背大兄终于算是反应过来刚刚秦吉胜喊的是什么东西，可是面对四周升起的熊熊烈火，他已经别无选择，想活命那就只能向前冲。
<br/>　　唐军的贞观炮实在是太过凶悍了，每一次爆炸都会有无数人丧命，残肢断体飞的到处都是，整个大营几乎瞬间就变成了修罗地狱。
<br/>　　那些没有参与追击，正在打扫营地的倭人几乎瞬间就被打懵了，连唐军的面都没有见着就开始四下乱窜，什么命令，什么上官全都不顾了。
<br/>　　而且这还不算，重要的是那些洧水燃烧起来之后产生了大量的浓烟，直接将小半个唐军大营覆盖住了，浓烈的硝烟不断刺激着那些小鬼子的嗅觉神经，时间不大已经有人开始缓缓倒下。
<br/>　　洧水，其实就是现代的石油，燃烧之后会产生大量的有毒气体，所以就算是那些小鬼子远远的躲开那些沟渠中的烈焰，也依旧免不了被浓烟所侵蚀。
<br/>　　秦吉胜在的反应还算是快的，提前一点逃过了死劫，从埋伏圈中脱身而出。
<br/>　　但就算如此，依旧让他感到后怕，看着那冲天而起的滚滚浓烟，一种叫作绝望的情绪开始在心中蔓延。
<br/>　　二十三万大军全在烈火的另一头，那可是这么多年整个倭国留下的最后精锐。
<br/>　　可是看眼下的情况，那些人应该是回不来了，唐军既然布下如此之局，想来必然还有后手。
<br/>　　……
<br/>　　而事实证明，秦吉胜想的并没有错，就在炮声响起之后，前面那些还在追着唐军逃走的倭人也意识到了情况的不对，纷纷停下坐骑回身观望。
<br/>　　也就是这一望，立刻便被看己看到的一切吓了一跳，只见他们身后唐军的营地上空已经彻底被浓烟所覆盖，就连刚刚升起的太阳也被遮住。
<br/>　　这是什么情况？发生了什么？追击唐军的无数小鬼子有些纳闷，不过好在他们并不傻，意识到了事情必然有了其他变故。
<br/>　　在这样的情况下再去追击唐军显然已经不可能，到底是回去还要留在原地待命成了他们所纠结的问题。
<br/>　　不过，这份纠结并没有多少时间，十多万人聚集在一起的这种情况正是炮兵喜欢的。
<br/>　　所以只是那么短短的一瞬间，引燃了营地中洧水的炮兵便调转了炮口，将剩于的炮弹倾泻到了这一群不知所措的倭军头上。
<br/>　　炮弹落到地上击起无数泥土，飞溅的弹片肆意的收割着生命，一声声爆炸就好像是死神的咆哮，带走小鬼子那卑贱的生命。
<br/>　　回头，必须回头，被炮火覆盖之下，小鬼子心里只有一个念头：中计了。
<br/>　　而既然已经中计了，那就说明前面一定会有更加可怕事情在等着他们，毕竟十万唐军到现在为止他们还没有杀死几个，很有可能大队人马正在前面等着。
<br/>　　所以小鬼子们决定回头，回去与后续部队会合，然后再杀出重围。
<br/>　　于是，负责追击的小鬼子开始调头往回跑，而那些已经进入唐军大营，正在捡洋落的小鬼子则因为后路被断，开始向唐军的后营移动。
<br/>　　就这样，二十来万小鬼子稀里糊涂的就跑到了唐军营地的后营，也就是马厩的位置。
<br/>　　山背大兄在一群护卫的保护下跑的上气不接下气，不过总算是已经离开了那片满是浓烟的地域，新鲜的空气让人精神一振。
<br/>　　在这个时候，他已经知道自己这是中了埋伏，可是身边的二十万大军却在提醒着他，他没有输，还有一战之力，只是不知道那些大唐的伏兵现在在什么地方。
<br/>　　……
<br/>　　距离唐军大营五里左右的一处山坡上，钦陵和黑齿常之举着新配发的双筒望远镜，看着营地中发生的一切。
<br/>　　良久之后，黑齿常之才将望远镜收起来，叹了一声说道：“发动总攻吧，这些人已经完了，再拖下去也没有什么意思。”
<br/>　　钦陵默默点了点头，略顿了顿：“嗯，你去集合队伍吧，这边攻击一展开，你就按计划向倭人主营发起进攻吧，力争在天黑之前把通道打开。至于倭人，能消灭多少是多少，我们有很多时间慢慢收拾他们，千万不要上了他们的当。”
