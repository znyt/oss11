　　第233章
<br/>　　没人注意到的角落里，李婉儿探出个脑袋，自然仍是男装打扮。
<br/>　　今天这么大的事情，她怎么可能不来凑凑热闹。几个平日素有交往的郡主都被她忽悠过来，换上男装后，躲在角落里看热闹。
<br/>　　“哎呦哟，那个柳轻名好帅哦。”
<br/>　　“帅什么，和屈九江差远了。”
<br/>　　都是十七八岁的年纪，一帮少女聚在一起叽叽喳喳，平日大门不出二门不迈，很少有这样放松的机会。
<br/>　　李婉儿本来很高兴，毕竟程大雷某种程度上可以说是他的人，程大雷飞黄腾达，她这个幕后老板也感觉与有荣焉。
<br/>　　可是，程大雷突然刺出的一剑令情形急转直下。
<br/>　　为什么，一个人放着平步青云的阳关大道不走，却要踏入深渊。
<br/>　　杨龙停没有选择和程大雷战斗，虽然他的实力未必会弱于程大雷，可他的选择还是逃跑。
<br/>　　杨龙停随身的十几名侍卫前后从台上跃下来，瞬间便将程大雷团团围住。
<br/>　　“杀了他，杀了他。”杨龙停大喊。
<br/>　　众人都看向明帝，今天一连串发生这么多事情，帝国的脸面已经丢得一干二净。身为一国之君，高高在上的明帝自然不能像杨龙停一样大喊着杀呀，杀呀。现在就看他的意思，他只要挥挥手，周围的禁卫军就会一拥而上，纵然程大雷有三头六臂，也绝逃不走这里。
<br/>　　明帝没有反应，他微闭双目，此刻看上去，不知是醒着，还是已经睡着了。但他没有反应，手下人也不敢有反应。于是刘阿吉并没有令禁卫军捉拿程大雷，不过，杨龙停手下也多是好手，程大雷此刻已然是逃不掉。
<br/>　　的确，程大雷陷入重围中，这些人都是杨龙停手下的大将，平日指挥千军万马战斗。他们包围程大雷，是以标准的军阵方式，前后三层，不露半点缝隙。
<br/>　　程大雷手握匹夫剑，身体警惕的盯着周围，脸上却带出一丝笑意。
<br/>　　“诸位还认得着柄剑么！”
<br/>　　大家当然认得，看到这柄剑就想起一些不好的记忆。当初追杀程大雷时，不知有多少幽州兵将在这柄剑下饮血。
<br/>　　而看到这柄剑，程大雷的身份也已经呼之欲出。
<br/>　　齐德龙握着帝国的制式军刀，咬牙道：“程大雷，杀弟之仇今日可以报了。”
<br/>　　“我可以送你去黄泉下重逢。”
<br/>　　在突然的一瞬，程大雷出剑，他的剑法很简单，甚至称得上是粗糙，只有一个字：刺。斜刺，直刺，在各种不可能的角度刺出必杀一剑。刺就是杀，这才是第一剑直来直往的奥义。
<br/>　　因为直来直往才能快，这是阿飞快剑。
<br/>　　在战斗过程中，程大雷脸上的胡须剥落，露出一张干净的脸。
<br/>　　金问道可以说是突然站起，口中道出两个字：“是他。”
<br/>　　而通过刚才齐德龙的话，看台上的文官武将此刻已恍然明白，眼前这人并不是那九品校尉牛三斤，而正是杨龙停口口声声说的程大雷。
<br/>　　可如此事情就有些奇怪了，程大雷不是已经死了么，而真如果杨龙停和程大雷的感情如此好，为何程大雷提剑要杀他。
<br/>　　真的假的，这件事中究竟那些是真的，那些是假的。
<br/>　　齐德龙欺身逼近程大雷，程大雷现在的情况并不是很好，他的敌人不仅在前面，还在身后，左右，可以说四面八方都是要杀他的兵器。
<br/>　　也就是趁这个机会，齐德龙敢逼近程大雷，趁着程大雷一个防守失误，他的刀砍了下去。
<br/>　　“阿强，你的仇报了！”
<br/>　　程大雷的剑招忽然加快，一剑连刺，虽然后发，但是先至，洞穿齐德龙的喉咙。而齐德龙手中的刀甚至都未落下。
<br/>　　好快的剑！
<br/>　　在提升到顶级后，程大雷获得的不仅是体力，速度，还有丰富的对敌经验，似乎有另一个人的战斗经验灌输到他体内，每次战斗时，程大雷都有种不是自己本人在战斗的恍惚感。
<br/>　　如今他使用阿飞快剑已经没有体力的困扰，而经过这一年时间，程大雷已经获得阿飞快剑的第二式。
<br/>　　第二式仍然是刺，连刺。
<br/>　　一剑刺出，缤纷光辉，看上去是一剑，但光芒万道，就像天空中的烟花一样，由一个黑点，绽放出千树万树，璀璨黑夜。
<br/>　　所以这一剑有个名字，唤做：爆烟花。
<br/>　　以程大雷目前的能力，只能做到一剑御七星的地步。
<br/>　　简单来说，直来直往是单体攻击，爆烟花是群体攻击。
<br/>　　恰好适合现在。
<br/>　　乱战丛中，烟花绽放，只听铛铛的兵器碰撞声，诸人手中的兵器一一被程大雷击落。程大雷行走在万千杀机中，却胜似闲庭信步，步步踏来，向杨龙停逼近。
<br/>　　“陛下。”
<br/>　　刘阿吉忍不住开口出声，没人能想到杨龙停的随从没拦住程大雷，再这样下去，杨龙停就要被程大雷杀死了。
<br/>　　明帝终于睁开眼睛，抬起手，缓缓道出两个字。
<br/>　　“回宫。”
<br/>　　刘阿吉伸手扶他起来，在太监的陪侍下，明帝的身影缓缓消失。
<br/>　　明帝最后也没有下令，而他的离去也等于已经下令。
<br/>　　相府和将军府的几个高手此刻都跃下看台，禁卫军也将程大雷团团围住。
<br/>　　而离得最近的则是屈九江和柳轻名，他们几乎是最快接近程大雷，然后又同一时间向程大雷出手，只要拿住程大雷，他们就还有洗刷耻辱的机会。
<br/>　　大刀转动如飞去砍程大雷的脑袋，长枪直刺如龙去扎程大雷的心脏。
<br/>　　“滚开！”
<br/>　　程大雷刺伤屈九江的手握，另一只手接住柳轻名的长枪。
<br/>　　灵犀一指。
<br/>　　柳轻名脸上肌肉拧起，想要摆脱程大雷的束缚，可铁枪像陷入大山中一般，不动分毫。
<br/>　　“让你滚了。”
<br/>　　程大雷挥手夺枪，直接将柳轻名砸开。
<br/>　　暗处观战的李婉儿心惊胆战，像是第一次认识程大雷。帝国两位最有可能夺得武状元的人，竟不是他一招之敌。她突然明白一件事，程大雷已经变了，早已非初见时那个机关算尽方能勉强活下去的山贼，今日的他，凭实力已经可以平?许多难关。
<br/>　　跨过屈九江和柳轻名后，程大雷直面杨龙停。
<br/>
