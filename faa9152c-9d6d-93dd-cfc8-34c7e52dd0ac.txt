　　在楚风眠身边的灵识，纷纷的散开了，没有任何一人还敢观察楚风眠。
<br/>　　单单是如今楚风眠的表现，便是已经足以代表他，并不是一个好惹之人。
<br/>　　在这云羽天宫开启之前，也没有人想要在这个时候，跟楚风眠交手。
<br/>　　楚风眠大摇大摆，便是直接来到了天羽城之中。
<br/>　　他一眼望去，便是看到，就在这天羽城的心中，有着一座巨大无比的阵法。
<br/>　　这阵法，应该便是天羽城跟云羽天宫之间的传送阵。
<br/>　　只是如今，还没有开启。
<br/>　　楚风眠直接来到了这天羽城的中心，看了一眼，便是看到了其中的一座庭院，直接便是向着其中走了进去。
<br/>　　“一个人，居然敢占据这个位置。”
<br/>　　“好狂妄的小子，可惜这里是归墟天，越是狂妄的人，只怕是死的越快。”
<br/>　　“看着吧，七大宗门的人，会给他一个教训的！”
<br/>　　一道道目光，正是看着楚风眠走到了这庭院之中去，不由的冷笑连连起来。
<br/>　　在这归墟天中，在狂妄的天才，都要架起尾巴做人了，毕竟能够来到归墟天中的武者，都是绝对的天才。
<br/>　　要是被群起攻之，谁也无法扛得住。
<br/>　　就在楚风眠进入到庭院之中的一刻。
<br/>　　天羽城外，又是一群人，突然出现，一共五个人，每个人的身上，都是露出了古铜色的肌肤，散发出如同金属一般的光芒。
<br/>　　看起来，每个人简直都已经不是人了，而是如同金刚罗汉一般的存在。
<br/>　　“铁骨金身，这是圣王宗的人！”
<br/>　　“圣王宗的人，号称锻体无敌，不修灵器，纯粹以身体为战，他们的身躯，比起灵器都要坚韧！”
<br/>　　“一群将身体当做灵器来锻炼，这些圣王宗的人，都是一群疯子！”
<br/>　　“不好，不能招惹他们，招惹上他们，可都麻烦了！”
<br/>　　一道道灵识，看着这一群人的到来，便是纷纷议论起来，没有任何一道灵识敢去接近。
<br/>　　圣王宗，七大宗门之一，眼前这五人，绝对是在归墟天之中，都算得上是一方极大的势力了。
<br/>　　归墟天中的无数武者，都不想跟他们为敌，如今一个个不敢为难他们，任由他们，进入到了天羽城之中。
<br/>　　“这天羽城，也没什么了不起的，曾经的九域第一大宗门云羽天宫，也是不过如此。”
<br/>　　一名圣王宗弟子，打量着周围的种种，一脸不屑的开口道。
<br/>　　“在这里面，还隐藏有着一群宵小之人，要不要我出手，将他们都给赶出去，这云羽天宫，不是有资格窥探的！”
<br/>　　这圣王宗弟子开口，毫不掩饰的狂妄，直接便是要将在场所有人，都给赶出去，简直可以说是眼睛长在了头顶上，没有将任何人，放在眼中。
<br/>　　不过他们狂妄，也是有着狂妄的资本，七大宗门的弟子，在九域，便是足以横着走，来到归墟天，自然也是如此。
<br/>　　“不用，我们先找个地方休息。”
<br/>　　在这一群人中，其中为首的一名，乃是一名金衣男子，他身着一身金衣，身躯却是足足有着十尺之高，身上肌肉高高隆起，蕴含有着无尽力量。
<br/>　　听到他开口，那些圣王宗弟子，一个个便是附和答应一声，不敢说些什么。
<br/>　　一群人，直接便是向着天羽城中心走去。
<br/>　　“盘师兄你看，那边有个庭院，正好适合我们。”
<br/>　　一名圣王宗弟子，张望了周围一眼，便是直接看向了楚风眠所在的庭院。
<br/>　　“那里面，好像是被一个小子占据了，不过没事，我这就让他滚出来！”
<br/>　　这名圣王宗弟子，开口一声，看着那盘师兄点了点头，便是直接向着楚风眠所在的庭院走了过来。
<br/>　　看到这一幕，天羽城中的不少武者，都在纷纷冷笑。
<br/>　　刚刚楚风眠还在嚣张，现在这圣王宗的人，就找上门去了。
<br/>　　圣王宗在七大宗门之中，都算是最难以对付的宗门之一了，尤其是圣王宗的弟子，一个个都是好战之人，谁跟他们起冲突，都要脱下一层皮来。
<br/>　　一道道灵识，远远的就看着这一切，在看热闹。
<br/>　　这圣王宗弟子，看着在庭院中的楚风眠，一脚踢开了庭院的大门。
<br/>　　“小子，滚出去，这个庭院我们圣王宗看上了！”
<br/>　　这圣王宗弟子，看着楚风眠，命令一般的口吻道。
<br/>　　“圣王宗的人。”
<br/>　　楚风眠看到那圣王宗弟子，眼神不动，连桑都没有任何的表情。
<br/>　　这些圣王宗弟子，从刚刚进入到天羽城中，便是一直就被楚风眠观察的一清二楚。

<br/>　　虽然说是跟那些炎阳门弟子的境界一样，但是眼前这圣王宗弟子身上的力量，却是要比起那些炎阳门弟子，还要雄厚的多。
<br/>　　七大宗门的底蕴，的确不是这些新崛起的宗门，可以比拟的了的。
<br/>　　面对圣王宗弟子，这样的七大宗门弟子，大部分的武者，都要胆怯几分，毕竟七大宗门带来的压力，实在是太大了。

<br/>　　如今这圣王宗的人，没等楚风眠出手，居然就已经是送上门来了。

<br/>　　这圣王宗弟子，看着楚风眠毫不回应他，直接便是冷和一声，向着楚风眠猛然突然出手，一把对着楚风眠的肩膀抓了过来。

<br/>　　楚风眠目光凌厉，一双眼睛直接看了过去，在这圣王宗弟子抓过来的瞬间，他也是突然出手了。
